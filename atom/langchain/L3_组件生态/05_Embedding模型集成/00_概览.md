## 知识点简介

Embedding模型集成是LangChain生态系统中的核心组件，负责将文本转换为向量表示。本知识点深入探讨LangChain的Embeddings抽象接口设计、多提供商集成策略、批量处理优化技术以及缓存机制实现。通过源码分析、官方文档和2025-2026年最新社区实践，你将掌握从基础使用到生产级优化的完整技能链，包括OpenAI text-embedding-3系列模型、Sentence Transformers本地模型、自定义Embedding实现以及与RAG系统的深度集成。

**核心价值**：
- 理解LangChain Embeddings的设计哲学和架构决策
- 掌握多种Embedding模型的选择与配置策略
- 学会批量处理、缓存优化等生产级技术
- 实现高性能、可扩展的RAG应用

## 核心概念预览

### 1. Embeddings抽象接口

**核心内容**：
- LangChain Embeddings基类的极简设计（仅4个方法）
- `embed_documents()` vs `embed_query()` 的分离设计理由
- 异步方法的默认实现与优化策略
- 类型系统设计（`list[str]` → `list[list[float]]`）

**关键发现**：
- 使用ABC定义接口，遵循"接口隔离原则"
- 异步方法提供默认实现（渐进式优化）
- 不依赖numpy，保持依赖最小化
- 文档与查询分离允许不同的优化策略

**来源**：`reference/source_embeddings_base_01.md`

---

### 2. 模型选择与配置

**核心内容**：
- OpenAI模型选择（text-embedding-3-small/large, ada-002）
- Sentence Transformers模型选择（all-MiniLM-L6-v2等）
- 配置参数详解（dimensions, chunk_size, max_retries）
- 环境变量管理与API密钥安全
- 非OpenAI提供商支持（OpenRouter, Ollama等）

**关键发现**：
- text-embedding-3支持自定义维度（降维节省成本）
- Pydantic BaseModel提供自动验证和环境变量加载
- 支持同步/异步动态密钥获取（企业级场景）
- `check_embedding_ctx_length=False` 兼容非OpenAI提供商

**来源**：`reference/source_openai_embeddings_03.md`, `reference/context7_langchain_openai_02.md`

### 3. 批量处理与优化

**核心内容**：
- 批量嵌入策略（batch_size, chunk_size）
- Token管理与自动分块（embedding_ctx_length）
- 长文本处理（加权平均合并）
- 异步批量处理（aembed_documents）
- 并发控制（max_concurrency）
- 进度条显示（show_progress_bar）

**关键发现**：
- 自动处理超长文本（分块→嵌入→加权平均→归一化）
- 批量处理减少API调用次数
- 异步处理提升吞吐量
- 支持进度条提升用户体验

**来源**：`reference/source_openai_embeddings_03.md`, `reference/context7_sentence_transformers_01.md`

---

### 4. 缓存机制

**核心内容**：
- CacheBackedEmbeddings装饰器模式
- 缓存键生成策略（SHA-1, BLAKE2b, SHA-256）
- 文档缓存 vs 查询缓存
- 批量缓存更新（mget/mset）
- 序列化与反序列化（JSON）
- from_bytes_store工厂方法

**关键发现**：
- 装饰器模式不侵入原始实现
- 默认只缓存文档嵌入（查询可选）
- SHA-1作为默认哈希（性能优先，有安全警告）
- 支持增量索引更新（新文档计算，旧文档读缓存）

**来源**：`reference/source_cache_backed_02.md`, `reference/search_cache_optimization_03.md`

## 实战场景预览

### 场景1：基础使用流程

**目标**：掌握Embedding模型的基础使用模式

**内容**：
- 单文本嵌入（embed_query）
- 批量文本嵌入（embed_documents）
- 添加缓存优化（CacheBackedEmbeddings）
- 异步批量处理（aembed_documents）

**代码示例**：
```python
from langchain_openai import OpenAIEmbeddings
from langchain_classic.embeddings import CacheBackedEmbeddings
from langchain_classic.storage import LocalFileStore

# 1. 基础使用
embeddings = OpenAIEmbeddings(model="text-embedding-3-small")
query_vector = embeddings.embed_query("What is RAG?")
doc_vectors = embeddings.embed_documents(["doc1", "doc2"])

# 2. 添加缓存
store = LocalFileStore("./cache")
cached_embeddings = CacheBackedEmbeddings.from_bytes_store(
    embeddings, store, namespace="openai-3-small"
)
```

### 场景3：RAG集成实战

**目标**：构建完整的RAG管道

**内容**：
- 文档加载与分块
- 文档向量化
- 向量存储（Chroma）
- 语义检索
- 完整RAG问答

**代码示例**：
```python
from langchain_community.document_loaders import PyPDFLoader
from langchain.text_splitter import RecursiveCharacterTextSplitter
from langchain_community.vectorstores import Chroma
from langchain_openai import OpenAIEmbeddings

# 1. 加载文档
loader = PyPDFLoader("document.pdf")
documents = loader.load()

# 2. 分块
splitter = RecursiveCharacterTextSplitter(chunk_size=1000, chunk_overlap=200)
splits = splitter.split_documents(documents)

# 3. 嵌入 + 存储
embeddings = OpenAIEmbeddings()
vectorstore = Chroma.from_documents(splits, embeddings)

# 4. 检索
retriever = vectorstore.as_retriever()
docs = retriever.get_relevant_documents("query")
```
