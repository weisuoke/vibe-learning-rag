# 状态调试技巧 - 一句话总结

**状态调试技巧是通过 stream_mode 流式监控、get_state/get_state_history 快照检查、自定义工具和 LangSmith 可观测性，实现对 LangGraph 图执行过程中状态变化的全方位观察、追溯和重现，是构建可靠状态化工作流的必备能力。**

---

## 核心公式

```
状态调试 = 实时监控(stream_mode) + 快照检查(get_state) + 历史追溯(get_state_history) + 生产监控(LangSmith)
```

## 记忆锚点

| 调试需求 | 工具 | 一句话 |
|----------|------|--------|
| 看当前状态 | get_state() | 给执行拍张照 |
| 看状态变化 | stream_mode | 给执行录个像 |
| 找问题步骤 | get_state_history() | 回放监控录像 |
| 改历史重来 | update_state() | 时间旅行 |
| 生产监控 | LangSmith | 行车记录仪 |
