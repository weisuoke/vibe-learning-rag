# 实战代码 - 场景2：代码仓库分析

## 场景描述

构建一个代码仓库分析系统，使用语法感知的文本分块器（PythonCodeTextSplitter）智能分割代码文件，支持多种编程语言，并实现代码问答功能。该场景适用于代码搜索、技术文档生成、代码审查辅助等应用。

**应用场景**：
- 代码搜索引擎（如GitHub Copilot）
- 技术文档自动生成
- 代码审查辅助工具
- 开发者知识库构建

**核心价值**：
- 语法感知分块，保留代码结构完整性
- 支持20+种编程语言
- 智能识别函数、类、方法边界
- 适配代码问答和检索场景

---

## 技术选型

### 为什么选择 RecursiveCharacterTextSplitter.from_language()？

**1. 语法感知能力**
```python
# 普通分块器：可能在函数中间切断
def calculate_sum(a, b):
    result = a + b
    # ← 这里可能被切断
    return result

# 语法感知分块器：按函数边界分割
# 完整保留函数定义
```

**2. 语言特定分隔符**
- Python: `\nclass `, `\ndef `, `\n\tdef `
- JavaScript: `\nfunction `, `\nconst `, `\nclass `
- Java: `\npublic class `, `\nprivate void `
- Go: `\nfunc `, `\ntype `, `\nvar `

**3. 递归策略**
```
优先级：类定义 → 函数定义 → 控制流 → 段落 → 行 → 字符
```

**对比其他分块器**：
| 分块器 | 语法感知 | 语言支持 | 适用场景 |
|--------|---------|---------|---------|
| RecursiveCharacterTextSplitter.from_language() | ⭐⭐⭐⭐⭐ | 20+ | 代码分析（推荐） |
| CharacterTextSplitter | ⭐ | 通用 | 简单文本 |
| RecursiveCharacterTextSplitter | ⭐⭐⭐ | 通用 | 通用文本 |

## 完整代码实现

```python
"""
代码仓库分析系统 - 完整实现
支持：多语言代码分块 + 语法感知 + 代码问答

依赖安装：
pip install langchain langchain-text-splitters langchain-openai langchain-community chromadb python-dotenv gitpython
"""

import os
from pathlib import Path
from typing import List, Dict, Any, Optional
from dotenv import load_dotenv
import git

# LangChain核心组件
from langchain_text_splitters import RecursiveCharacterTextSplitter, Language
from langchain_community.document_loaders import TextLoader
from langchain_openai import OpenAIEmbeddings, ChatOpenAI
from langchain_chroma import Chroma
from langchain.schema import Document
from langchain.chains import RetrievalQA

# 加载环境变量
load_dotenv()


class CodeRepositoryAnalyzer:
    """
    代码仓库分析器

    功能：
    1. 多语言代码加载（Python、JavaScript、Java、Go等）
    2. 语法感知分块（保留代码结构）
    3. 代码向量化存储
    4. 代码问答系统

    设计原则：
    - 语法感知：按函数、类边界分割
    - 多语言支持：20+种编程语言
    - 元数据丰富：文件路径、语言类型、函数名
    - 可扩展性：易于添加新语言
    """

    # 支持的语言映射
    LANGUAGE_MAP = {
        ".py": Language.PYTHON,
        ".js": Language.JS,
        ".ts": Language.TS,
        ".java": Language.JAVA,
        ".go": Language.GO,
        ".cpp": Language.CPP,
        ".c": Language.C,
        ".rs": Language.RUST,
        ".rb": Language.RUBY,
        ".php": Language.PHP,
        ".swift": Language.SWIFT,
        ".kt": Language.KOTLIN,
        ".scala": Language.SCALA,
        ".cs": Language.CSHARP,
        ".lua": Language.LUA,
        ".pl": Language.PERL,
        ".hs": Language.HASKELL,
        ".html": Language.HTML,
        ".md": Language.MARKDOWN,
    }

    def __init__(
        self,
        chunk_size: int = 1000,
        chunk_overlap: int = 200,
        embedding_model: str = "text-embedding-3-small",
        persist_directory: str = "./code_chroma_db"
    ):
        """
        初始化代码仓库分析器

        参数说明：
        - chunk_size: 块大小（字符数）
          * 推荐值：1000（适合大多数代码文件）
          * 小函数：500-800
          * 大类：1500-2000

        - chunk_overlap: 块重叠（字符数）
          * 推荐值：200（保留上下文）
          * 代码场景：150-250

        - embedding_model: OpenAI Embedding模型
        - persist_directory: 向量数据库持久化目录
        """
        self.chunk_size = chunk_size
        self.chunk_overlap = chunk_overlap
        self.persist_directory = persist_directory

        # Embedding模型
        self.embeddings = OpenAIEmbeddings(model=embedding_model)

        # LLM模型（用于问答）
        self.llm = ChatOpenAI(model="gpt-4o-mini", temperature=0)

        # 向量存储（延迟初始化）
        self.vectorstore: Optional[Chroma] = None

        # 语言特定分块器缓存
        self.splitters: Dict[Language, RecursiveCharacterTextSplitter] = {}

        print(f"代码仓库分析器初始化完成")
        print(f"   - chunk_size: {chunk_size}")
        print(f"   - chunk_overlap: {chunk_overlap}")
        print(f"   - 支持语言: {len(self.LANGUAGE_MAP)} 种")

    def get_splitter(self, language: Language) -> RecursiveCharacterTextSplitter:
        """
        获取语言特定的分块器（带缓存）

        参数：
        - language: 编程语言类型

        返回：
        - RecursiveCharacterTextSplitter实例
        """
        if language not in self.splitters:
            self.splitters[language] = RecursiveCharacterTextSplitter.from_language(
                language=language,
                chunk_size=self.chunk_size,
                chunk_overlap=self.chunk_overlap
            )
        return self.splitters[language]

    def detect_language(self, file_path: Path) -> Optional[Language]:
        """
        检测文件的编程语言

        参数：
        - file_path: 文件路径

        返回：
        - Language枚举值或None
        """
        suffix = file_path.suffix.lower()
        return self.LANGUAGE_MAP.get(suffix)

    def load_code_file(self, file_path: str) -> List[Document]:
        """
        加载单个代码文件

        参数：
        - file_path: 代码文件路径

        返回：
        - Document对象列表
        """
        file_path = Path(file_path)

        if not file_path.exists():
            raise FileNotFoundError(f"文件不存在: {file_path}")

        # 检测语言
        language = self.detect_language(file_path)
        if language is None:
            print(f"   跳过不支持的文件: {file_path.name}")
            return []

        print(f"\n加载代码文件: {file_path.name} ({language.value})")

        # 加载文件内容
        try:
            loader = TextLoader(str(file_path), encoding="utf-8")
            documents = loader.load()
        except Exception as e:
            print(f"   加载失败: {e}")
            return []

        # 添加元数据
        for doc in documents:
            doc.metadata["language"] = language.value
            doc.metadata["file_path"] = str(file_path)
            doc.metadata["file_name"] = file_path.name

        print(f"   加载完成: {len(documents[0].page_content)} 字符")

        return documents

    def load_repository(
        self,
        repo_path: str,
        file_extensions: Optional[List[str]] = None,
        exclude_dirs: Optional[List[str]] = None
    ) -> List[Document]:
        """
        批量加载代码仓库

        参数：
        - repo_path: 仓库路径
        - file_extensions: 文件扩展名过滤（如[".py", ".js"]）
        - exclude_dirs: 排除的目录（如["node_modules", ".git"]）

        返回：
        - Document对象列表
        """
        repo_path = Path(repo_path)

        if not repo_path.exists():
            raise FileNotFoundError(f"仓库不存在: {repo_path}")

        # 默认排除目录
        if exclude_dirs is None:
            exclude_dirs = [
                ".git", "node_modules", "__pycache__", ".venv",
                "venv", "dist", "build", ".idea", ".vscode"
            ]

        print(f"\n批量加载代码仓库: {repo_path}")
        print(f"   排除目录: {exclude_dirs}")

        all_documents = []

        # 遍历仓库文件
        for file_path in repo_path.rglob("*"):
            # 跳过目录
            if not file_path.is_file():
                continue

            # 跳过排除的目录
            if any(excluded in file_path.parts for excluded in exclude_dirs):
                continue

            # 文件扩展名过滤
            if file_extensions and file_path.suffix not in file_extensions:
                continue

            # 加载文件
            try:
                documents = self.load_code_file(str(file_path))
                all_documents.extend(documents)
            except Exception as e:
                print(f"   加载失败: {file_path.name} - {e}")

        print(f"\n批量加载完成: {len(all_documents)} 个文件")

        return all_documents

    def split_code(self, documents: List[Document]) -> List[Document]:
        """
        分块代码文档（语法感知）

        参数：
        - documents: Document对象列表

        返回：
        - 分块后的Document对象列表
        """
        print(f"\n开始分块: {len(documents)} 个文档")

        all_splits = []

        # 按语言分组处理
        language_groups: Dict[str, List[Document]] = {}
        for doc in documents:
            lang = doc.metadata.get("language", "unknown")
            if lang not in language_groups:
                language_groups[lang] = []
            language_groups[lang].append(doc)

        # 分别处理每种语言
        for lang_str, docs in language_groups.items():
            try:
                # 获取Language枚举
                language = Language(lang_str)
                splitter = self.get_splitter(language)

                # 分块
                splits = splitter.split_documents(docs)
                all_splits.extend(splits)

                print(f"   {lang_str}: {len(docs)} 文件 → {len(splits)} 块")
            except Exception as e:
                print(f"   {lang_str} 分块失败: {e}")

        # 统计信息
        total_chars = sum(len(doc.page_content) for doc in documents)
        avg_chunk_size = sum(len(s.page_content) for s in all_splits) / len(all_splits) if all_splits else 0

        print(f"\n分块完成:")
        print(f"   - 总块数: {len(all_splits)}")
        print(f"   - 原始字符数: {total_chars:,}")
        print(f"   - 平均块大小: {avg_chunk_size:.0f} 字符")

        return all_splits

    def create_vectorstore(
        self,
        documents: List[Document],
        collection_name: str = "code_repository"
    ) -> Chroma:
        """
        创建代码向量存储

        参数：
        - documents: 分块后的Document对象列表
        - collection_name: 集合名称

        返回：
        - Chroma向量存储对象
        """
        print(f"\n创建向量存储: {collection_name}")

        self.vectorstore = Chroma.from_documents(
            documents=documents,
            embedding=self.embeddings,
            collection_name=collection_name,
            persist_directory=self.persist_directory
        )

        print(f"   向量存储创建完成")
        print(f"   - 文档数量: {len(documents)}")

        return self.vectorstore

    def load_vectorstore(self, collection_name: str = "code_repository") -> Chroma:
        """加载已存在的向量存储"""
        print(f"\n加载向量存储: {collection_name}")

        self.vectorstore = Chroma(
            collection_name=collection_name,
            embedding_function=self.embeddings,
            persist_directory=self.persist_directory
        )

        return self.vectorstore

    def search_code(
        self,
        query: str,
        k: int = 5,
        language_filter: Optional[str] = None
    ) -> List[Dict[str, Any]]:
        """
        搜索代码

        参数：
        - query: 查询文本
        - k: 返回结果数量
        - language_filter: 语言过滤（如"python"）

        返回：
        - 搜索结果列表
        """
        if self.vectorstore is None:
            raise ValueError("向量存储未初始化")

        print(f"\n搜索代码: {query}")

        # 构建过滤条件
        filter_dict = None
        if language_filter:
            filter_dict = {"language": language_filter}

        # 相似度搜索
        results = self.vectorstore.similarity_search_with_score(
            query,
            k=k,
            filter=filter_dict
        )

        # 格式化结果
        formatted_results = []
        for i, (doc, score) in enumerate(results, 1):
            formatted_results.append({
                "rank": i,
                "content": doc.page_content,
                "score": score,
                "language": doc.metadata.get("language", "unknown"),
                "file_path": doc.metadata.get("file_path", "unknown"),
                "file_name": doc.metadata.get("file_name", "unknown")
            })

            print(f"\n   [{i}] 相似度: {score:.4f}")
            print(f"       文件: {doc.metadata.get('file_name', 'unknown')}")
            print(f"       语言: {doc.metadata.get('language', 'unknown')}")
            print(f"       代码预览: {doc.page_content[:100]}...")

        return formatted_results

    def create_qa_chain(self) -> RetrievalQA:
        """
        创建代码问答链

        返回：
        - RetrievalQA对象
        """
        if self.vectorstore is None:
            raise ValueError("向量存储未初始化")

        print("\n创建代码问答链")

        qa_chain = RetrievalQA.from_chain_type(
            llm=self.llm,
            chain_type="stuff",
            retriever=self.vectorstore.as_retriever(search_kwargs={"k": 5}),
            return_source_documents=True
        )

        print("   问答链创建完成")

        return qa_chain

    def ask_code_question(self, question: str) -> Dict[str, Any]:
        """
        代码问答

        参数：
        - question: 问题

        返回：
        - 问答结果
        """
        qa_chain = self.create_qa_chain()

        print(f"\n问题: {question}")

        result = qa_chain.invoke({"query": question})

        print(f"\n回答: {result['result']}")
        print(f"\n引用文件:")
        for i, doc in enumerate(result['source_documents'], 1):
            print(f"   [{i}] {doc.metadata.get('file_name', 'unknown')}")

        return result

"""
常见问题处理：

问题1：代码块被切断
原因：chunk_size太小或分隔符不合适
解决方案：
- 增加chunk_size到1500-2000
- 使用from_language()确保语法感知
- 检查分隔符配置

问题2：检索结果不相关
原因：代码语义理解困难
解决方案：
- 使用语言过滤
- 增加chunk_overlap
- 使用更好的Embedding模型

问题3：加载速度慢
原因：文件数量多，逐个加载
解决方案：
- 使用多线程并行加载
- 排除不必要的目录
- 缓存已加载的文件

问题4：向量存储占用空间大
原因：代码文件多，块数量大
解决方案：
- 只索引核心代码文件
- 使用更小的Embedding模型
- 定期清理过期代码

问题5：多语言混合效果差
原因：不同语言的语法差异
解决方案：
- 按语言分别创建向量存储
- 使用语言特定的Embedding
- 在检索时使用语言过滤
"""
