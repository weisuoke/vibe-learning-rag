# 状态快照 - 一句话总结

**状态快照（StateSnapshot）是 LangGraph 在每个执行步骤自动捕获的完整状态记录，通过 get_state/get_state_history/update_state 三大 API 实现状态查看、历史回溯和时间旅行调试，是构建可观测、可调试、可恢复的 AI Agent 工作流的核心基础设施。**

## 核心公式

```
状态快照 = Checkpointer（持久化引擎）
         + StateSnapshot（用户层 8 字段 NamedTuple）
         + Checkpoint（存储层版本管理 TypedDict）
         + 三大 API（get_state / get_state_history / update_state）
         + 时间旅行（Replay 重放 + Fork 分叉）
```

## 记忆锚点

> 状态快照就像游戏的自动存档系统：每过一关（每个节点执行后）自动保存，
> 你可以随时读档（get_state）、查看所有存档记录（get_state_history）、
> 修改存档重新打（update_state + invoke = Fork）。
> 没有存档系统（Checkpointer），这些功能全部不可用。
