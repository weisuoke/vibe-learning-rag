# Inter-Agent Communication 2025-2026

## Key Sources
1. **Event-driven multi-agent systems** - Confluent patterns
2. **Message passing protocols** - Industry standards
3. **JSON-based communication** - Structured data exchange

## Pi-mono Approach
- **JSON mode output**: Structured, parseable results
- **Text-based passing**: Simple, reliable
- **No shared state**: Complete isolation
- **Unidirectional flow**: Parent → Child → Parent

## Key Insight
Pi-mono uses simple text-based communication (JSON output) rather than complex message queues or RPC protocols. This aligns with the simplicity principle.
