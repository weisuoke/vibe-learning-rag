## ✅ 学习检查清单

完成本知识点后，你应该能够：

### 基础能力
- [ ] 理解 Pi Coding Agent 的核心价值和设计理念
- [ ] 成功安装 Pi（npm 全局安装或从源码运行）
- [ ] 配置至少一个 LLM Provider（API 密钥或 OAuth）
- [ ] 成功启动 Pi 并进行基础交互
- [ ] 理解 Pi 的四大工具（read、write、edit、bash）

### 配置能力
- [ ] 使用环境变量配置 API 密钥
- [ ] 使用 /login 命令进行 OAuth 认证
- [ ] 创建和编辑 ~/.pi/agent/settings.json（全局配置）
- [ ] 创建和编辑 .pi/settings.json（项目配置）
- [ ] 创建 AGENTS.md 或 CLAUDE.md 上下文文件
- [ ] 理解配置的优先级和覆盖规则

### 高级能力
- [ ] 切换不同的 Provider 和 Model
- [ ] 配置自定义 Provider（models.json）
- [ ] 理解 auth.json 的结构和权限
- [ ] 排查常见的安装和配置问题
- [ ] 理解 Pi 与 Claude Code 的区别

### 实战能力
- [ ] 使用 Pi 完成一个简单的编码任务
- [ ] 在实际项目中配置 Pi
- [ ] 为团队成员配置 Pi 环境
