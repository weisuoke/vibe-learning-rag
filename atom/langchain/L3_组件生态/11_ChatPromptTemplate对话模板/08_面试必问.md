# ChatPromptTemplate 面试必问

> 本文档整理了 ChatPromptTemplate 的常见面试题，涵盖基础、中级和高级三个难度层次

---

## 基础题（3题）

### Q1: ChatPromptTemplate 和 PromptTemplate 有什么区别？

**难度**: ⭐⭐

**答案**:

核心区别在于输出格式和使用场景：

| 维度 | PromptTemplate | ChatPromptTemplate |
|------|----------------|-------------------|
| 输出格式 | 字符串 (StringPromptValue) | 消息列表 (ChatPromptValue) |
| 角色支持 | 无 | System/Human/AI/Chat |
| 对话历史 | 需手动拼接 | MessagesPlaceholder |
| 适用场景 | 文本生成 | 对话系统 |
| LLM 集成 | 需要转换 | 直接使用 |

**代码示例**:

```python
from langchain_core.prompts import PromptTemplate, ChatPromptTemplate

# PromptTemplate: 输出字符串
prompt1 = PromptTemplate.from_template("你是{role}。问题：{question}")
result1 = prompt1.invoke({"role": "助手", "question": "你好"})
print(type(result1))  # <class 'StringPromptValue'>

# ChatPromptTemplate: 输出消息列表
prompt2 = ChatPromptTemplate.from_messages([
    ("system", "你是{role}"),
    ("human", "{question}")
])
result2 = prompt2.invoke({"role": "助手", "question": "你好"})
print(type(result2))  # <class 'ChatPromptValue'>
print(result2.to_messages())  # [SystemMessage(...), HumanMessage(...)]
```

**关键点**:
- ChatPromptTemplate 专为对话式 AI 设计
- 输出是结构化的消息对象，而非纯文本
- ChatModel 需要消息列表作为输入

### Q2: 什么是 MessagesPlaceholder？它解决了什么问题？

**难度**: ⭐⭐

**答案**:

MessagesPlaceholder 是用于动态插入消息列表的占位符，专门用于管理对话历史。

**解决的问题**:
1. 对话历史管理：无需手动拼接历史消息
2. 灵活注入：运行时动态插入任意数量的消息
3. 可选支持：通过 `optional=True` 支持首次对话
4. 数量限制：通过 `n_messages` 控制历史长度

**代码示例**:

```python
from langchain_core.prompts import ChatPromptTemplate, MessagesPlaceholder

# 创建带历史的模板
prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant."),
    MessagesPlaceholder("history"),  # 对话历史占位符
    ("human", "{input}")
])

# 使用
result = prompt.invoke({
    "history": [
        ("human", "What's 5 + 2?"),
        ("ai", "5 + 2 is 7")
    ],
    "input": "Now multiply that by 4"
})
```

**关键特性**:
- `optional=True`: 允许不提供历史（首次对话）
- `n_messages=N`: 只保留最后 N 条消息
- 自动转换元组为消息对象

### Q3: 如何创建一个基础的对话模板？

**难度**: ⭐

**答案**:

使用 `from_messages()` 方法创建，传入消息列表。

**代码示例**:

```python
from langchain_core.prompts import ChatPromptTemplate
from langchain_openai import ChatOpenAI

# 1. 创建模板
template = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant."),
    ("human", "{input}")
])

# 2. 创建链
model = ChatOpenAI(model="gpt-4o-mini")
chain = template | model

# 3. 使用
response = chain.invoke({"input": "What is Python?"})
print(response.content)
```

**三种消息格式**:
1. 元组格式: `("human", "Hello")` - 最简洁
2. 消息对象: `HumanMessage(content="Hello")` - 功能完整
3. 模板对象: `HumanMessagePromptTemplate.from_template("{input}")` - 灵活强大

## 中级题（5题）

### Q4: MessagesPlaceholder 的 n_messages 参数是取前 N 条还是后 N 条？

**难度**: ⭐⭐⭐

**答案**:

**取最后 N 条**，这是一个反直觉的行为。

**源码证明** (chat.py:187-189):
```python
def format_messages(self, **kwargs: Any) -> list[BaseMessage]:
    value = convert_to_messages(value)
    if self.n_messages:
        value = value[-self.n_messages:]  # 负索引，取最后 N 条
    return value
```

**代码示例**:

```python
prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant."),
    MessagesPlaceholder("history", n_messages=2),
    ("human", "{input}")
])

# 传入 4 条历史消息
result = prompt.invoke({
    "history": [
        ("human", "消息1"),  # 会被丢弃
        ("ai", "回复1"),     # 会被丢弃
        ("human", "消息2"),  # 保留
        ("ai", "回复2"),     # 保留
    ],
    "input": "新问题"
})

# 输出：只有最后 2 条
# [SystemMessage(...), HumanMessage("消息2"), AIMessage("回复2"), HumanMessage("新问题")]
```

**为什么这样设计**:
- 对话系统通常关心最近的上下文
- 避免上下文窗口溢出
- 控制 token 消耗

### Q5: 如何使用 Partial Variables 预填充变量？

**难度**: ⭐⭐⭐

**答案**:

使用 `partial()` 方法预填充部分变量，减少重复传递。

**代码示例**:

```python
from langchain_core.prompts import ChatPromptTemplate
from datetime import datetime

# 1. 预填充固定值
base_template = ChatPromptTemplate.from_messages([
    ("system", "You are a {role} assistant."),
    ("human", "{input}")
])

python_assistant = base_template.partial(role="Python programming")
data_assistant = base_template.partial(role="data analysis")

# 使用时只需传入 input
python_assistant.invoke({"input": "How to use list comprehension?"})

# 2. 预填充函数动态值
prompt = ChatPromptTemplate.from_messages([
    ("system", "Current time: {current_time}"),
    ("human", "{input}")
]).partial(current_time=lambda: datetime.now().strftime("%Y-%m-%d %H:%M:%S"))

# 每次调用时自动获取当前时间
prompt.invoke({"input": "Hello"})
```

**注意事项**:
- 某些版本存在兼容性问题 (GitHub Issue #17560, #30049)
- 建议使用最新版本 (v0.2.x+)
- 函数动态值比固定值更稳定

### Q6: 如何组合多个 ChatPromptTemplate？

**难度**: ⭐⭐⭐

**答案**:

使用 `+` 操作符或 `append/extend` 方法。

**代码示例**:

```python
from langchain_core.prompts import ChatPromptTemplate

# 方法 1: + 操作符
prompt1 = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant.")
])

prompt2 = ChatPromptTemplate.from_messages([
    ("human", "{input}")
])

combined = prompt1 + prompt2

# 方法 2: append 方法
prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant.")
])

prompt.append(("human", "{input}"))

# 方法 3: extend 方法
prompt.extend([
    ("human", "Question: {question}"),
    ("ai", "Let me help you with that.")
])
```

**变量合并规则**:
- 变量会自动合并
- 重复变量会被复用
- partial_variables 可能丢失（取决于版本）

**最佳实践**:
- 避免在组合前使用 partial
- 在组合后使用 partial
- 或使用 from_messages 一次性创建

### Q7: 如何在 RAG 系统中使用 ChatPromptTemplate？

**难度**: ⭐⭐⭐

**答案**:

结合 MessagesPlaceholder 和上下文注入。

**代码示例**:

```python
from langchain_core.prompts import ChatPromptTemplate, MessagesPlaceholder
from langchain_openai import ChatOpenAI
from langchain_core.runnables import RunnablePassthrough

# 1. 创建 RAG 提示模板
prompt = ChatPromptTemplate.from_messages([
    ("system", "Answer based on the context. If unsure, say so."),
    ("system", "Context: {context}"),
    MessagesPlaceholder("chat_history", optional=True),
    ("human", "{question}")
])

# 2. 创建 RAG 链
llm = ChatOpenAI(model="gpt-4o-mini")

rag_chain = (
    {
        "context": lambda x: retrieve_context(x["question"]),
        "chat_history": lambda x: x.get("chat_history", []),
        "question": lambda x: x["question"]
    }
    | prompt
    | llm
)

# 3. 使用
response = rag_chain.invoke({
    "question": "What is LangChain?",
    "chat_history": [
        ("human", "Tell me about AI frameworks"),
        ("ai", "There are many AI frameworks...")
    ]
})
```

**关键点**:
- System 消息注入上下文
- MessagesPlaceholder 管理对话历史
- optional=True 支持首次对话

### Q8: 如何处理多模态消息（文本+图片）？

**难度**: ⭐⭐⭐

**答案**:

使用列表格式的消息内容。

**代码示例**:

```python
from langchain_core.prompts import ChatPromptTemplate

# 创建多模态模板
prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant that can analyze images."),
    ("human", [
        {"type": "text", "text": "What's in this image?"},
        {"type": "image_url", "image_url": {"url": "{image_url}"}}
    ])
])

# 使用
result = prompt.invoke({
    "image_url": "https://example.com/image.jpg"
})

# 也可以使用本地图片
prompt2 = ChatPromptTemplate.from_messages([
    ("human", [
        {"type": "text", "text": "{question}"},
        {"type": "image_url", "image_url": {"url": "data:image/jpeg;base64,{image_data}"}}
    ])
])
```

**支持的格式**:
- URL: `{"url": "https://..."}`
- Base64: `{"url": "data:image/jpeg;base64,..."}`
- detail 参数: `{"url": "...", "detail": "high"}` (控制分辨率)

## 高级题（4题）

### Q9: ChatPromptTemplate 的 partial_variables 有什么兼容性问题？

**难度**: ⭐⭐⭐⭐

**答案**:

在某些版本中存在已知问题。

**Issue #17560 (v0.1.9)**:
```python
# ❌ 在 v0.1.9 中不工作
prompt = ChatPromptTemplate.from_messages([
    ("system", "你是{role}，使用{language}回答"),
    ("human", "{input}")
])

prompt_partial = prompt.partial(role="助手", language="中文")
# 仍然会报错：KeyError: 'role'
```

**Issue #30049 (2025)**:
```python
# ❌ 模板连接时 partial variables 被忽略
prompt1 = ChatPromptTemplate.from_messages([
    ("system", "你是{role}")
]).partial(role="助手")

prompt2 = ChatPromptTemplate.from_messages([
    ("human", "{input}")
])

combined = prompt1 + prompt2
# partial variables 可能丢失
```

**解决方案**:
1. 使用最新版本 (v0.2.x+)
2. 避免使用 partial，直接传值
3. 使用函数动态值（更稳定）
4. 在组合后使用 partial

### Q10: 如何实现对话历史的自动管理？

**难度**: ⭐⭐⭐⭐

**答案**:

使用 `RunnableWithMessageHistory` 自动管理。

**代码示例**:

```python
from langchain_core.prompts import ChatPromptTemplate, MessagesPlaceholder
from langchain_core.runnables.history import RunnableWithMessageHistory
from langchain_core.chat_history import InMemoryChatMessageHistory
from langchain_openai import ChatOpenAI

# 1. 创建存储
store = {}

def get_session_history(session_id: str):
    if session_id not in store:
        store[session_id] = InMemoryChatMessageHistory()
    return store[session_id]

# 2. 创建模板
prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant."),
    MessagesPlaceholder(variable_name="history"),
    ("human", "{input}")
])

# 3. 创建链
llm = ChatOpenAI(model="gpt-4o-mini")
chain = prompt | llm

# 4. 包装为带历史的链
conversation = RunnableWithMessageHistory(
    chain,
    get_session_history,
    input_messages_key="input",
    history_messages_key="history"
)

# 5. 使用（自动管理历史）
response = conversation.invoke(
    {"input": "Hi, I'm Alice"},
    config={"configurable": {"session_id": "user_001"}}
)

response = conversation.invoke(
    {"input": "What's my name?"},
    config={"configurable": {"session_id": "user_001"}}
)
# AI 会记住名字
```

**关键点**:
- 自动加载和保存历史
- 支持多用户隔离
- 可替换为持久化存储 (Redis, 数据库)

### Q11: from_template() 和 from_messages() 有什么区别？

**难度**: ⭐⭐⭐⭐

**答案**:

`from_template()` 只能创建单个 Human 消息，是 `from_messages([("human", template)])` 的快捷方式。

**源码证明**:
```python
@classmethod
def from_template(cls, template: str, **kwargs: Any) -> ChatPromptTemplate:
    """Create a chat prompt template from a template string."""
    return cls.from_messages([("human", template)], **kwargs)
```

**代码示例**:

```python
# ✅ 适合：单个用户问题
prompt1 = ChatPromptTemplate.from_template("翻译：{text}")
# 等价于
prompt1 = ChatPromptTemplate.from_messages([("human", "翻译：{text}")])

# ❌ 不适合：需要系统消息
# prompt2 = ChatPromptTemplate.from_template("System: 你是翻译助手\nHuman: {text}")
# 这会把整个字符串当作 Human 消息

# ✅ 正确做法
prompt2 = ChatPromptTemplate.from_messages([
    ("system", "你是翻译助手"),
    ("human", "{text}")
])
```

**使用建议**:
- 简单场景用 from_template
- 需要多角色用 from_messages
- 需要系统消息用 from_messages

### Q12: 如何优化大规模对话历史的性能？

**难度**: ⭐⭐⭐⭐⭐

**答案**:

使用多种策略组合优化。

**策略 1: 限制历史长度**
```python
# 使用 n_messages 参数
MessagesPlaceholder("history", n_messages=10)

# 或手动截取
history = history[-10:]
```

**策略 2: 使用滑动窗口**
```python
from langchain.memory import ConversationBufferWindowMemory

memory = ConversationBufferWindowMemory(
    k=5,  # 只保留最近 5 轮
    memory_key="chat_history",
    return_messages=True
)
```

**策略 3: 使用摘要**
```python
from langchain.memory import ConversationSummaryMemory

memory = ConversationSummaryMemory(
    llm=ChatOpenAI(),
    memory_key="chat_history",
    return_messages=True
)
```

**策略 4: 分层存储**
```python
# 短期记忆：最近 5 轮
recent_history = history[-5:]

# 长期记忆：摘要
summary = summarize_history(history[:-5])

prompt = ChatPromptTemplate.from_messages([
    ("system", "Summary of earlier conversation: {summary}"),
    MessagesPlaceholder("recent_history"),
    ("human", "{input}")
])
```

**性能对比**:
- 无限制：Token 消耗高，响应慢
- n_messages=10：Token 消耗中等，响应快
- 滑动窗口：自动管理，性能稳定
- 摘要：Token 消耗低，但需要额外 LLM 调用
