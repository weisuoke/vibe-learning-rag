# Agent最佳实践 - 一句话总结

> Agent 最佳实践 = **用 Middleware 给不确定的 LLM 套上确定的保护壳**：限制防循环、重试防故障、降级防宕机、脱敏防泄露、审核防越权、摘要防溢出。

---

## 展开

```
一个生产级 Agent = LLM + 工具 + Middleware

                ┌─ 限制 (ModelCallLimit)   → 防无限循环
                ├─ 重试 (ModelRetry)       → 防瞬时故障
Middleware 保护 ─┼─ 降级 (ModelFallback)    → 防服务宕机
                ├─ 脱敏 (PII)             → 防数据泄露
                ├─ 审核 (HumanInTheLoop)   → 防越权操作
                └─ 摘要 (Summarization)    → 防上下文溢出

最小可用 = 限制 + 重试 + 降级（3 行 Middleware）
安全加固 = + PII + HITL（再加 2 行）
生产就绪 = + 摘要 + 工具限制（再加 2 行）
```

**记住：从单 Agent 开始，用 Middleware 按需加固，永远不要裸奔上线。**
