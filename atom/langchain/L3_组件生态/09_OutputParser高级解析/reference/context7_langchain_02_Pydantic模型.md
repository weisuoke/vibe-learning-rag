## 关键信息提取

### 1. 流式结构化输出与 Pydantic 模型

**来源**：https://docs.langchain.com/oss/python/integrations/chat/amazon_nova

**示例代码**：

```python
from pydantic import BaseModel, Field

class Person(BaseModel):
    """Information about a person."""
    name: str = Field(description="The person's name")
    age: int = Field(description="The person's age")

structured_model = model.with_structured_output(Person)

for chunk in structured_model.stream("Michael is 35 years old"):
    print(chunk)
```

**关键点**：
- 使用 `with_structured_output()` 方法绑定 Pydantic 模型
- 支持流式输出（`stream()`）
- 模型处理输入并返回解析后的对象
- 适用于实时数据提取

### 2. 定义结构化输出与 Pydantic 模型

**来源**：https://docs.langchain.com/oss/python/langchain/models

**示例代码**：

```python
from pydantic import BaseModel, Field

class Movie(BaseModel):
    """A movie with details."""
    title: str = Field(..., description="The title of the movie")
    year: int = Field(..., description="The year the movie was released")
    director: str = Field(..., description="The director of the movie")
    rating: float = Field(..., description="The movie's rating out of 10")

model_with_structure = model.with_structured_output(Movie)
response = model_with_structure.invoke("Provide details about the movie Inception")
print(response)  # Movie(title="Inception", year=2010, director="Christopher Nolan", rating=8.8)
```

**关键点**：
- 使用 Pydantic `BaseModel` 定义结构化 schema
- 使用 `Field()` 添加字段描述和验证
- `with_structured_output()` 强制模型输出符合 schema
- 输出是 Pydantic 对象，受益于 Pydantic 的丰富特性（字段验证）

### 3. 应用 Pydantic Schema 约束到 ChatOutlines 输出

**来源**：https://docs.langchain.com/oss/python/integrations/chat/outlines

**示例代码**：

```python
from pydantic import BaseModel

class Person(BaseModel):
    name: str

model.json_schema = Person
response = model.invoke("Who are the main contributors to LangChain?")
person = Person.model_validate_json(response.content)

person
```

**关键点**：
- 使用 `json_schema` 属性约束输出结构
- 模型生成符合 schema 的 JSON
- 使用 `model_validate_json()` 验证和解析为 Python 对象

### 4. 生成结构化输出与 LangChain ChatOpenAI

**来源**：https://docs.langchain.com/oss/python/integrations/chat/openai

**示例代码**：

```python
from langchain_openai import ChatOpenAI
from pydantic import BaseModel, Field

llm = ChatOpenAI(model="gpt-4.1")

class Movie(BaseModel):
    """A movie with details."""
    title: str = Field(..., description="The title of the movie")
    year: int = Field(..., description="The year the movie was released")
    director: str = Field(..., description="The director of the movie")
    rating: float = Field(..., description="The movie's rating out of 10")

structured_llm = llm.with_structured_output(Movie, method="json_schema")
response = structured_llm.invoke("Provide details about the movie Inception")
response
```

**关键点**：
- 使用 `with_structured_output()` 方法
- 设置 `method="json_schema"` 启用 OpenAI 的原生结构化输出功能
- 方法接受 schema 类并返回符合 schema 的响应

### 5. Output Parsers（来自 langchain_decorators）

**来源**：https://docs.langchain.com/oss/python/integrations/providers/langchain_decorators

**关键点**：
> The `llm_prompt` decorator is designed to automatically detect and apply the most suitable output parser based on the function's return type. If no specific output parser is configured, the decorator will simply return the raw string output from the LLM. The system natively and automatically supports common output types such as lists, dictionaries, and Pydantic models. This built-in capability streamlines the process of structuring and consuming the responses generated by the language model.

**自动检测机制**：
- `llm_prompt` 装饰器自动检测函数返回类型
- 自动应用最合适的 output parser
- 原生支持：lists、dictionaries、Pydantic models
- 无配置时返回原始字符串输出

## 与 OutputParser 的关系

### 1. 现代方法 vs 传统方法

**现代方法（2025+）**：
```python
# 使用 with_structured_output()
structured_model = model.with_structured_output(MyPydanticModel)
result = structured_model.invoke(input)
```

**传统方法（OutputParser）**：
```python
# 使用 PydanticOutputParser
from langchain_core.output_parsers import PydanticOutputParser

parser = PydanticOutputParser(pydantic_object=MyPydanticModel)
chain = model | parser
result = chain.invoke(input)
```

### 2. 两种方法的对比

| 特性 | with_structured_output() | PydanticOutputParser |
|------|--------------------------|----------------------|
| 实现方式 | LLM 原生支持（Function Calling） | 手动解析 LLM 输出 |
| 错误处理 | 自动（LLM 内置） | 手动（解析时检测） |
| 流式支持 | 部分支持 | 完全支持 |
| 适用模型 | 支持结构化输出的模型 | 所有模型 |
| 性能 | 更快（原生支持） | 较慢（需要解析） |
| 灵活性 | 受限于 LLM 能力 | 完全可控 |

### 3. 何时使用 OutputParser

根据文档：
- **模型不支持原生结构化输出**：使用 OutputParser
- **需要自定义解析逻辑**：使用 OutputParser
- **需要额外验证**：使用 OutputParser
- **需要完整流式支持**：使用 OutputParser

### 4. Pydantic 模型的作用

**在 with_structured_output() 中**：
- 定义输出 schema
- LLM 根据 schema 生成输出
- 自动验证和解析

**在 PydanticOutputParser 中**：
- 定义输出 schema
- 生成格式指令（注入到 Prompt）
- 手动验证和解析 LLM 输出

## 实战建议

### 1. 优先使用 with_structured_output()

```python
# 推荐（2025+）
from langchain_openai import ChatOpenAI
from pydantic import BaseModel, Field

class Person(BaseModel):
    name: str = Field(description="The person's name")
    age: int = Field(description="The person's age")

llm = ChatOpenAI(model="gpt-4.1")
structured_llm = llm.with_structured_output(Person, method="json_schema")
result = structured_llm.invoke("Extract info: Michael is 35 years old")
```

### 2. 回退到 PydanticOutputParser

```python
# 当模型不支持原生结构化输出时
from langchain_core.output_parsers import PydanticOutputParser

parser = PydanticOutputParser(pydantic_object=Person)
format_instructions = parser.get_format_instructions()

prompt = f"Extract info: Michael is 35 years old\n{format_instructions}"
chain = model | parser
result = chain.invoke(prompt)
```

### 3. 流式场景

```python
# with_structured_output() 流式
for chunk in structured_llm.stream("Extract info: ..."):
    print(chunk)

# PydanticOutputParser 流式
for chunk in chain.stream("Extract info: ..."):
    print(chunk)
```

## 自动检测机制（langchain_decorators）

**示例**：

```python
from langchain_decorators import llm_prompt

@llm_prompt
def extract_person(text: str) -> Person:
    """Extract person information from text."""
    return text

# 自动检测返回类型 Person，应用 PydanticOutputParser
result = extract_person("Michael is 35 years old")
# 输出: Person(name="Michael", age=35)
```

**支持的返回类型**：
- `str`：返回原始字符串
- `list`：应用 ListOutputParser
- `dict`：应用 JsonOutputParser
- `BaseModel`：应用 PydanticOutputParser
