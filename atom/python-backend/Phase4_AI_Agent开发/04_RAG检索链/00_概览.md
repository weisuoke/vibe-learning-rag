## 💡 学习建议

1. **先理解原理，再看代码**
   - 不要跳过第一性原理和核心概念
   - 理解为什么需要 RAG，而不是死记硬背

2. **动手运行所有代码**
   - 所有代码都可以直接运行
   - 修改参数观察效果变化

3. **联系实际应用**
   - 思考如何应用到自己的项目
   - 尝试用 RAG 解决实际问题

4. **避免常见误区**
   - 仔细阅读"反直觉点"章节
   - 不要陷入"检索越多越好"的误区

## 🛠️ 环境准备

### 安装依赖

```bash
# 核心依赖
uv add langchain langchain-openai chromadb openai pypdf

# 可选依赖
uv add faiss-cpu  # FAISS 向量数据库
```

### 配置 API 密钥

```bash
# .env 文件
OPENAI_API_KEY=your_key_here
OPENAI_BASE_URL=https://api.openai.com/v1  # 可选
```
