## 卡片2:三种策略

**正面**:
三种检索策略是什么?

**背面**:
1. **similarity**:相似度搜索,快速通用
2. **mmr**:多样性搜索,平衡相关性和多样性
3. **similarity_score_threshold**:阈值过滤,质量控制

**选择**:
- 通用 → similarity
- 多样性 → mmr
- 高精度 → threshold

**记忆口诀**:"相似快,MMR多样,阈值精准"

## 卡片5:MMR算法

**正面**:
MMR算法原理?

**背面**:
**公式**:
```
MMR = λ * Sim(q, d) - (1-λ) * max(Sim(d, d_i))
```

**参数**:
- λ=1:最大相关性
- λ=0.5:平衡(默认)
- λ=0:最大多样性

**流程**:
1. 获取fetch_k个候选
2. 迭代选择k个文档
3. 平衡相关性和多样性

**记忆口诀**:"相关减多样,lambda来平衡"

## 卡片8:错误处理

**正面**:
如何处理Retriever错误?

**背面**:
```python
# 1. 基础错误处理
try:
    docs = retriever.invoke(query)
except Exception as e:
    logger.error(f"检索失败: {e}")
    return []

# 2. 重试机制
for attempt in range(max_retries):
    try:
        return retriever.invoke(query)
    except Exception:
        time.sleep(2 ** attempt)

# 3. 降级策略
try:
    docs = retriever.invoke(query)
except Exception:
    docs = fallback_docs
```

**记忆口诀**:"捕获重试降级"
