## 知识点概述

### 什么是LangChain缓存机制？

LangChain提供了两种核心缓存机制：

1. **LLM缓存**：缓存LLM调用结果，避免重复API调用
   - 缓存键：`(prompt, llm_string)` 二元组
   - 支持多种后端：InMemoryCache、RedisCache、SQLiteCache等
   - 全局配置，自动生效

2. **Embedding缓存**：缓存Embedding计算结果，避免重复计算
   - 使用`CacheBackedEmbeddings`包装器
   - 缓存键：文本内容的哈希值
   - 支持namespace隔离不同模型

### 核心价值

1. **成本节省**：减少API调用次数，降低30-50%成本
2. **性能提升**：缓存命中速度提升20-100倍
3. **用户体验**：减少响应延迟，改善交互体验

### 适用场景

- 高频重复查询的应用
- 成本敏感的项目
- 需要快速响应的场景
- 大规模文档处理

## 快速参考

### 核心API

```python
# LLM缓存
from langchain_core.caches import InMemoryCache
from langchain_core.globals import set_llm_cache

set_llm_cache(InMemoryCache())

# Embedding缓存
from langchain_classic.embeddings import CacheBackedEmbeddings
from langchain_classic.storage import LocalFileStore

store = LocalFileStore("./cache/")
cached_embedder = CacheBackedEmbeddings.from_bytes_store(
    underlying_embeddings,
    store,
    namespace="model-name"
)
```
