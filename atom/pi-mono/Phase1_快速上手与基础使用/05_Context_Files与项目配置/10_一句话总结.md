# 一句话总结

**Context Files 是 Pi Agent 的"项目大脑"——AGENTS.md 定义规则，settings.json 配置行为，SYSTEM.md 塑造性格，三者协同让 AI 成为真正懂你项目的开发伙伴。**

---

## 核心要点回顾

| 维度 | 关键点 |
|------|--------|
| **本质** | 项目级配置系统，让 AI 理解项目上下文 |
| **三大文件** | AGENTS.md（规则）+ settings.json（配置）+ SYSTEM.md（性格） |
| **加载机制** | 启动时自动加载，支持全局和项目级覆盖 |
| **最佳实践** | 团队协作提交到 Git，个人配置放 .pi/ 目录 |
| **核心价值** | 减少重复沟通，提升 AI 工作效率，统一团队规范 |

---

## 记住这三句话

1. **AGENTS.md 是项目说明书** - 告诉 AI "这个项目是什么，怎么开发"
2. **settings.json 是配置中心** - 控制 AI 的模型、UI、资源加载
3. **SYSTEM.md 是 AI 人设** - 定义 AI 的工作风格和行为准则

---

## 立即行动

```bash
# 最小可用配置（30秒上手）
echo "# 项目规范\n\n- TypeScript + React\n- 使用 pnpm\n- 遵循 ESLint" > AGENTS.md
mkdir -p .pi && echo '{"defaultModel": "claude-opus-4"}' > .pi/settings.json
pi
```
