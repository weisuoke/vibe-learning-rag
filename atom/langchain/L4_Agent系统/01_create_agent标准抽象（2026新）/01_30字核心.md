# create_agent标准抽象（2026新） - 30字核心

**create_agent 是 LangChain 1.0 的统一 Agent 工厂函数，通过 Middleware 实现可组合的生产级 Agent 构建。**

---

## 关键词解析

| 关键词 | 含义 |
|--------|------|
| **统一** | 取代 `create_react_agent`、`create_openai_functions_agent`、`initialize_agent` 等十余种旧函数，一个入口覆盖所有 Agent 场景 |
| **工厂函数** | 单一入口，接收 `model` + `tools` + `middleware`，返回 `CompiledStateGraph`（LangGraph 编译图） |
| **Middleware** | 中间件系统，6 种钩子（before/after_agent、before/after_model、wrap_model_call、wrap_tool_call）实现细粒度控制 |
| **可组合** | 多个 Middleware 以列表传入，按洋葱模型嵌套执行，职责单一、自由组合，类似 Web 服务器中间件 |
| **生产级** | 内置 checkpointer（持久化）、store（共享存储）、interrupt（人机交互）等生产特性 |

## 与相关概念的关系

```
                    ┌─────────────────────┐
                    │   BaseChatModel     │  模型抽象层
                    │  (openai:gpt-4o)    │
                    └────────┬────────────┘
                             │
                             ▼
┌──────────┐    ┌────────────────────────────┐    ┌──────────┐
│  Tools   │───▶│      create_agent()        │◀───│Middleware│
│(工具列表) │    │                            │    │ (中间件)  │
└──────────┘    │  system_prompt             │    └──────────┘
                │  response_format           │
                │  checkpointer / store      │
                └────────────┬───────────────┘
                             │ 返回
                             ▼
                ┌────────────────────────────┐
                │   CompiledStateGraph       │  LangGraph 编译图
                │                            │
                │  AgentState (messages +     │
                │    structured_response)     │
                │                            │
                │  model_node ⇄ tool_node    │  核心循环
                │    (middleware 钩子穿插)      │
                └────────────────────────────┘
```

核心数据流：**model → 决策 → tool 调用 → 结果回传 → model 再决策**，Middleware 在每个环节提供钩子介入。

---

## 一句话记忆法

> **一个 `create_agent` 管所有，Middleware 洋葱层层套，model + tools 往里塞，LangGraph 底层帮你跑。**

三个关键数字速记：
- **1** 个统一入口（create_agent）
- **6** 种 Middleware 钩子（4 个生命周期 + 2 个包装器）
- **19** 个内置 Middleware（开箱即用）
