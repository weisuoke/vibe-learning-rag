## 场景 1: PyPDFLoader 基础使用

### 示例 1.1: 加载单个 PDF 文件

```python
from langchain_community.document_loaders import PyPDFLoader

# 加载 PDF 文件
loader = PyPDFLoader("document.pdf")
pages = loader.load()

# 查看结果
print(f"加载了 {len(pages)} 页")
print(f"第一页内容预览: {pages[0].page_content[:200]}...")
print(f"第一页元数据: {pages[0].metadata}")
```

**输出示例**:
```
加载了 10 页
第一页内容预览: This is the first page of the PDF document...
第一页元数据: {'source': 'document.pdf', 'page': 0}
```

**关键点**:
- PyPDFLoader 将每一页作为一个独立的 Document
- 元数据自动包含文件路径和页码
- 基于 pypdf 库实现

[来源: reference/fetch_2_pdf_a30ab0_01.md | rag101 项目]

### 示例 1.2: 懒加载模式

```python
from langchain_community.document_loaders import PyPDFLoader

# 懒加载 - 适合大 PDF 文件
loader = PyPDFLoader("large_document.pdf")

# 流式处理每一页
for page in loader.lazy_load():
    print(f"处理第 {page.metadata['page']} 页")
    print(f"内容长度: {len(page.page_content)}")
    # 逐页处理,节省内存
```

**为什么使用懒加载?**
- 大 PDF 文件不会一次性加载到内存
- 可以提前终止处理
- 适合流式处理场景

## 场景 2: PDFPlumberLoader 使用

### 示例 2.1: 基础使用

```python
from langchain_community.document_loaders import PDFPlumberLoader

# 加载 PDF 文件
loader = PDFPlumberLoader("document.pdf")
pages = loader.load()

# 查看结果
print(f"加载了 {len(pages)} 页")
print(f"第一页内容: {pages[0].page_content[:200]}...")
```

**PDFPlumberLoader 特点**:
- 基于 pdfplumber 库
- 更好的表格提取能力
- 更准确的文本定位
- 支持更复杂的 PDF 布局

### 示例 2.2: PyPDFLoader vs PDFPlumberLoader

```python
from langchain_community.document_loaders import PyPDFLoader, PDFPlumberLoader

# 使用 PyPDFLoader
pypdf_loader = PyPDFLoader("document.pdf")
pypdf_pages = pypdf_loader.load()

# 使用 PDFPlumberLoader
pdfplumber_loader = PDFPlumberLoader("document.pdf")
pdfplumber_pages = pdfplumber_loader.load()

# 对比结果
print("PyPDFLoader:")
print(f"  页数: {len(pypdf_pages)}")
print(f"  第一页长度: {len(pypdf_pages[0].page_content)}")

print("\nPDFPlumberLoader:")
print(f"  页数: {len(pdfplumber_pages)}")
print(f"  第一页长度: {len(pdfplumber_pages[0].page_content)}")
```

**对比总结**:

| 特性 | PyPDFLoader | PDFPlumberLoader |
|------|-------------|------------------|
| 速度 | 快 | 较慢 |
| 表格提取 | 一般 | 优秀 |
| 文本定位 | 一般 | 精确 |
| 复杂布局 | 一般 | 优秀 |
| 依赖库 | pypdf | pdfplumber |
| 推荐场景 | 简单文本 PDF | 复杂布局/表格 |

## 场景 3: PDF 分割与处理

### 示例 3.1: 加载并分割 PDF

```python
from langchain_community.document_loaders import PyPDFLoader
from langchain.text_splitter import RecursiveCharacterTextSplitter

# 1. 加载 PDF
loader = PyPDFLoader("document.pdf")
pages = loader.load()

# 2. 分割文档
text_splitter = RecursiveCharacterTextSplitter(
    chunk_size=1000,       # 每块最大字符数
    chunk_overlap=200,     # 块之间的重叠
    length_function=len,
    separators=["\n\n", "\n", " ", ""]
)

chunks = text_splitter.split_documents(pages)

# 3. 查看结果
print(f"原始页数: {len(pages)}")
print(f"分割后块数: {len(chunks)}")

for i, chunk in enumerate(chunks[:3]):
    print(f"\n块 {i+1}:")
    print(f"  内容: {chunk.page_content[:100]}...")
    print(f"  元数据: {chunk.metadata}")
```

**输出示例**:
```
原始页数: 10
分割后块数: 45

块 1:
  内容: Introduction to Machine Learning...
  元数据: {'source': 'document.pdf', 'page': 0}

块 2:
  内容: Machine learning is a subset of artificial intelligence...
  元数据: {'source': 'document.pdf', 'page': 0}
```

**关键参数说明**:
- `chunk_size=1000`: 适合大多数 LLM 的上下文窗口
- `chunk_overlap=200`: 保留上下文,避免语义断裂
- 元数据自动保留页码信息

## 场景 4: 完整 RAG 管道

### 示例 4.1: PDF 到 RAG 系统

```python
from langchain_community.document_loaders import PyPDFLoader
from langchain.text_splitter import RecursiveCharacterTextSplitter
from langchain_openai import OpenAIEmbeddings, ChatOpenAI
from langchain_community.vectorstores import Chroma
from langchain.chains import RetrievalQA
from dotenv import load_dotenv

# 加载环境变量
load_dotenv()

# 1. 加载 PDF
print("步骤 1: 加载 PDF...")
loader = PyPDFLoader("knowledge.pdf")
pages = loader.load()
print(f"✓ 加载了 {len(pages)} 页")

# 2. 分割文档
print("\n步骤 2: 分割文档...")
text_splitter = RecursiveCharacterTextSplitter(
    chunk_size=1000,
    chunk_overlap=200
)
chunks = text_splitter.split_documents(pages)
print(f"✓ 分割为 {len(chunks)} 个块")

# 3. 向量化
print("\n步骤 3: 向量化...")
embeddings = OpenAIEmbeddings()
vector_store = Chroma.from_documents(
    documents=chunks,
    embedding=embeddings,
    persist_directory="./chroma_db"
)
print(f"✓ 向量化完成")

# 4. 构建 QA 链
print("\n步骤 4: 构建 QA 链...")
llm = ChatOpenAI(model="gpt-3.5-turbo", temperature=0)
qa_chain = RetrievalQA.from_chain_type(
    llm=llm,
    chain_type="stuff",
    retriever=vector_store.as_retriever(
        search_type="similarity",
        search_kwargs={"k": 3}
    ),
    return_source_documents=True
)
print(f"✓ QA 链构建完成")

# 5. 查询
print("\n步骤 5: 查询...")
query = "What is the main topic of this document?"
result = qa_chain.invoke({"query": query})

print(f"\n问题: {query}")
print(f"回答: {result['result']}")
print(f"\n来源页码:")
for i, doc in enumerate(result['source_documents']):
    print(f"  {i+1}. 第 {doc.metadata['page']} 页: {doc.page_content[:100]}...")
```

**输出示例**:
```
步骤 1: 加载 PDF...
✓ 加载了 10 页

步骤 2: 分割文档...
✓ 分割为 45 个块

步骤 3: 向量化...
✓ 向量化完成

步骤 4: 构建 QA 链...
✓ QA 链构建完成

步骤 5: 查询...

问题: What is the main topic of this document?
回答: This document is about machine learning fundamentals and applications.

来源页码:
  1. 第 0 页: Introduction to Machine Learning...
  2. 第 1 页: Machine learning algorithms can be categorized...
  3. 第 2 页: Applications of machine learning include...
```

## 场景 5: 错误处理

### 示例 5.1: 处理损坏的 PDF

```python
from langchain_community.document_loaders import PyPDFLoader
import logging

logging.basicConfig(level=logging.INFO)
logger = logging.getLogger(__name__)

def safe_load_pdf(file_path: str):
    """安全加载 PDF 文件"""
    try:
        loader = PyPDFLoader(file_path)
        pages = loader.load()
        logger.info(f"✓ 成功加载: {file_path}, {len(pages)} 页")
        return pages
    except Exception as e:
        logger.error(f"✗ 加载失败: {file_path}, 错误: {e}")
        return []

# 使用示例
pages = safe_load_pdf("document.pdf")
```

### 示例 5.2: 批量加载 PDF

```python
from langchain_community.document_loaders import PyPDFLoader
from pathlib import Path
from typing import List
from langchain_core.documents import Document

def batch_load_pdfs(directory: str) -> List[Document]:
    """批量加载目录中的所有 PDF 文件"""
    all_pages = []
    failed_files = []

    # 获取所有 PDF 文件
    pdf_files = list(Path(directory).glob("*.pdf"))
    print(f"找到 {len(pdf_files)} 个 PDF 文件")

    for file_path in pdf_files:
        try:
            loader = PyPDFLoader(str(file_path))
            pages = loader.load()
            all_pages.extend(pages)
            print(f"✓ {file_path.name}: {len(pages)} 页")
        except Exception as e:
            failed_files.append((file_path.name, str(e)))
            print(f"✗ {file_path.name}: {e}")

    # 总结
    print(f"\n总结:")
    print(f"  成功: {len(all_pages)} 页")
    print(f"  失败: {len(failed_files)} 个文件")

    if failed_files:
        print(f"\n失败文件:")
        for filename, error in failed_files:
            print(f"  - {filename}: {error}")

    return all_pages

# 使用示例
pages = batch_load_pdfs("./pdfs")
```

### 示例 6.2: 并行加载多个 PDF

```python
from langchain_community.document_loaders import PyPDFLoader
from concurrent.futures import ThreadPoolExecutor
from typing import List
from langchain_core.documents import Document

def load_single_pdf(file_path: str) -> List[Document]:
    """加载单个 PDF"""
    try:
        loader = PyPDFLoader(file_path)
        return loader.load()
    except Exception as e:
        print(f"✗ {file_path}: {e}")
        return []

def parallel_load_pdfs(file_paths: List[str], max_workers: int = 4) -> List[Document]:
    """并行加载多个 PDF"""
    all_pages = []

    with ThreadPoolExecutor(max_workers=max_workers) as executor:
        results = executor.map(load_single_pdf, file_paths)

        for pages in results:
            all_pages.extend(pages)

    return all_pages

# 使用示例
file_paths = ["doc1.pdf", "doc2.pdf", "doc3.pdf"]
pages = parallel_load_pdfs(file_paths, max_workers=4)
print(f"✓ 加载了 {len(pages)} 页")
```

## 场景 7: 实战技巧

### 技巧 7.1: 提取特定页码

```python
from langchain_community.document_loaders import PyPDFLoader

def load_specific_pages(file_path: str, page_numbers: List[int]):
    """加载特定页码"""
    loader = PyPDFLoader(file_path)
    all_pages = loader.load()

    # 过滤特定页码
    selected_pages = [
        page for page in all_pages
        if page.metadata['page'] in page_numbers
    ]

    return selected_pages

# 使用示例
pages = load_specific_pages("document.pdf", [0, 1, 2])  # 加载前3页
print(f"加载了 {len(pages)} 页")
```

### 技巧 7.2: 添加自定义元数据

```python
from langchain_community.document_loaders import PyPDFLoader

def load_with_custom_metadata(file_path: str, category: str):
    """加载 PDF 并添加自定义元数据"""
    loader = PyPDFLoader(file_path)
    pages = loader.load()

    # 添加自定义元数据
    for page in pages:
        page.metadata["category"] = category
        page.metadata["file_name"] = file_path.split("/")[-1]
        page.metadata["total_pages"] = len(pages)

    return pages

# 使用示例
pages = load_with_custom_metadata("tutorial.pdf", category="tutorial")
print(f"元数据: {pages[0].metadata}")
```

## 总结

### 核心要点

1. **PyPDFLoader**: 快速、简单,适合基础文本 PDF
2. **PDFPlumberLoader**: 精确、强大,适合复杂布局和表格
3. **懒加载**: 大 PDF 文件使用 `lazy_load()` 节省内存
4. **分割策略**: 使用 RecursiveCharacterTextSplitter 保持语义完整
5. **RAG 集成**: 完整的加载 → 分割 → 向量化 → 检索 → 生成流程
6. **错误处理**: 批量加载时必须处理异常
7. **性能优化**: 懒加载、并行加载、流式处理

### 最佳实践

| 场景 | 推荐方法 | 原因 |
|------|----------|------|
| 简单文本 PDF | PyPDFLoader | 速度快 |
| 复杂布局/表格 | PDFPlumberLoader | 提取准确 |
| 大 PDF(>100MB) | lazy_load() | 节省内存 |
| 批量加载 | 并行 + 错误处理 | 提高效率 |
| RAG 应用 | load + split | 标准流程 |

### 常见问题

**Q1: PyPDFLoader 和 PDFPlumberLoader 如何选择?**
- 简单文本 PDF → PyPDFLoader (快)
- 复杂布局/表格 → PDFPlumberLoader (准确)

**Q2: 如何处理大 PDF 文件?**
- 使用 `lazy_load()` 流式加载
- 逐页处理,避免内存溢出

**Q3: 如何提升加载性能?**
- 使用并行加载处理多个文件
- 使用懒加载减少内存占用
- 批量处理时使用线程池
