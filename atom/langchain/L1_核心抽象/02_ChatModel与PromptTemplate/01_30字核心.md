# 30字核心

**ChatModel 是基于消息的模型抽象，PromptTemplate 是模板系统，两者是构建 AI Agent 对话能力的核心基础设施。**

---

## 拆解理解

### ChatModel (消息驱动的模型)
- **是什么**: 基于消息列表的模型抽象
- **核心特征**: 理解角色（System/Human/AI）
- **价值**: 专为对话场景设计，而非通用文本生成

### PromptTemplate (提示词模板)
- **是什么**: 可复用的提示词构建系统
- **核心特征**: 变量替换 + 类型验证
- **价值**: 提高提示词的可维护性和一致性

### 两者关系
```
PromptTemplate → 生成消息 → ChatModel → 返回响应
    (模板)        (格式化)      (推理)      (结果)
```

---

## 为什么这30个字很重要？

1. **ChatModel ≠ LLM**: 明确区分消息驱动 vs 字符串驱动
2. **PromptTemplate ≠ 字符串格式化**: 强调模板系统的完整性
3. **AI Agent 基础**: 点明在 AI Agent 开发中的核心地位

---

## 一句话记忆法

> **"对话用 ChatModel，模板用 PromptTemplate，两者组合构建 Agent"**
