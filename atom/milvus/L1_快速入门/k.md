# L1_快速入门 知识点列表

> 快速掌握 Milvus 的基本使用，从安装到第一次向量检索

---

## 知识点清单

1. **安装与连接** - 使用 Docker 部署 Milvus 并通过 Python SDK 建立连接
2. **Collection管理** - 创建、查看、删除 Collection，理解 Schema 定义
3. **数据插入与查询** - 插入向量数据、创建索引、执行相似度检索
4. **Milvus架构概览** - 理解 Proxy、QueryNode、DataNode、IndexNode 等核心组件
5. **数据一致性级别** - 掌握 Strong、Bounded、Eventually 三种一致性级别的选择

---

## 学习顺序建议

```
安装与连接 → Collection管理 → 数据插入与查询 → Milvus架构概览 → 数据一致性级别
    ↓              ↓              ↓                ↓                ↓
  环境准备      数据结构设计    实现检索        理解架构        性能权衡
```
