## 场景 1: 基础文本文件加载

### 示例 1.1: 最简单的文本加载

```python
from langchain_community.document_loaders import TextLoader

# 加载单个文本文件
loader = TextLoader("example.txt", encoding="utf-8")
documents = loader.load()

# 查看结果
print(f"加载了 {len(documents)} 个文档")
print(f"内容预览: {documents[0].page_content[:200]}...")
print(f"元数据: {documents[0].metadata}")
```

**输出示例**:
```
加载了 1 个文档
内容预览: This is a sample text file.
It contains multiple lines of text...
元数据: {'source': 'example.txt'}
```

**关键点**:
- TextLoader 默认将整个文件作为一个 Document
- 必须指定正确的编码(默认 utf-8)
- 元数据自动包含文件路径

[来源: reference/context7_langchain_01.md | LangChain 官方文档]

### 示例 1.2: 懒加载模式

```python
from langchain_community.document_loaders import TextLoader

# 懒加载 - 适合大文件
loader = TextLoader("large_file.txt", encoding="utf-8")

# 流式处理
for doc in loader.lazy_load():
    print(f"处理文档: {doc.metadata['source']}")
    print(f"内容长度: {len(doc.page_content)}")
    # 逐个处理,节省内存
```

**为什么使用懒加载?**
- 大文件不会一次性加载到内存
- 可以提前终止处理
- 适合流式处理场景

[来源: reference/context7_langchain_01.md | LangChain 官方文档 - 懒加载最佳实践]

## 场景 2: 文本分割与处理

### 示例 2.1: 加载并分割文本

```python
from langchain_community.document_loaders import TextLoader
from langchain.text_splitter import RecursiveCharacterTextSplitter

# 1. 加载文档
loader = TextLoader("knowledge.txt", encoding="utf-8")
documents = loader.load()

# 2. 分割文档
text_splitter = RecursiveCharacterTextSplitter(
    chunk_size=500,        # 每块最大字符数
    chunk_overlap=50,      # 块之间的重叠
    length_function=len,   # 长度计算函数
    separators=["\n\n", "\n", " ", ""]  # 分割优先级
)

chunks = text_splitter.split_documents(documents)

# 3. 查看结果
print(f"原始文档数: {len(documents)}")
print(f"分割后块数: {len(chunks)}")

for i, chunk in enumerate(chunks[:3]):
    print(f"\n块 {i+1}:")
    print(f"  内容: {chunk.page_content[:100]}...")
    print(f"  元数据: {chunk.metadata}")
```

**输出示例**:
```
原始文档数: 1
分割后块数: 15

块 1:
  内容: LangChain is a framework for developing applications powered by language models...
  元数据: {'source': 'knowledge.txt'}

块 2:
  内容: The main value props of LangChain are: Components, Off-the-shelf chains...
  元数据: {'source': 'knowledge.txt'}
```

**关键参数说明**:
- `chunk_size`: 控制每块的大小,影响检索精度
- `chunk_overlap`: 保留上下文,避免语义断裂
- `separators`: 优先按段落分割,保持语义完整

### 示例 2.2: 使用 load_and_split 便利方法

```python
from langchain_community.document_loaders import TextLoader
from langchain.text_splitter import RecursiveCharacterTextSplitter

# 创建加载器
loader = TextLoader("knowledge.txt", encoding="utf-8")

# 创建分割器
text_splitter = RecursiveCharacterTextSplitter(
    chunk_size=500,
    chunk_overlap=50
)

# 一步完成加载和分割
chunks = loader.load_and_split(text_splitter=text_splitter)

print(f"✓ 加载并分割完成,共 {len(chunks)} 个块")
```

**优点**:
- 代码更简洁
- 自动处理元数据传递
- 适合小到中等大小的文件

**注意**:
- 内部调用 `load()`,不适合超大文件
- 大文件应使用 `lazy_load()` + 手动分割

[来源: 03_核心概念_6_load_and_split集成.md]

## 场景 3: 完整 RAG 管道

### 示例 3.1: 端到端 RAG 系统

```python
from langchain_community.document_loaders import TextLoader
from langchain.text_splitter import RecursiveCharacterTextSplitter
from langchain_openai import OpenAIEmbeddings, ChatOpenAI
from langchain_community.vectorstores import Chroma
from langchain.chains import RetrievalQA
from dotenv import load_dotenv

# 加载环境变量
load_dotenv()

# 1. 加载文档
print("步骤 1: 加载文档...")
loader = TextLoader("knowledge.txt", encoding="utf-8")
documents = loader.load()
print(f"✓ 加载了 {len(documents)} 个文档")

# 2. 分割文档
print("\n步骤 2: 分割文档...")
text_splitter = RecursiveCharacterTextSplitter(
    chunk_size=500,
    chunk_overlap=50
)
chunks = text_splitter.split_documents(documents)
print(f"✓ 分割为 {len(chunks)} 个块")

# 3. 向量化
print("\n步骤 3: 向量化...")
embeddings = OpenAIEmbeddings()
vector_store = Chroma.from_documents(
    documents=chunks,
    embedding=embeddings,
    persist_directory="./chroma_db"  # 持久化存储
)
print(f"✓ 向量化完成")

# 4. 构建 QA 链
print("\n步骤 4: 构建 QA 链...")
llm = ChatOpenAI(model="gpt-3.5-turbo", temperature=0)
qa_chain = RetrievalQA.from_chain_type(
    llm=llm,
    chain_type="stuff",  # 将所有检索结果放入一个 prompt
    retriever=vector_store.as_retriever(
        search_type="similarity",
        search_kwargs={"k": 3}  # 返回前 3 个最相关的块
    ),
    return_source_documents=True  # 返回源文档
)
print(f"✓ QA 链构建完成")

# 5. 查询
print("\n步骤 5: 查询...")
query = "What is LangChain?"
result = qa_chain.invoke({"query": query})

print(f"\n问题: {query}")
print(f"回答: {result['result']}")
print(f"\n来源文档:")
for i, doc in enumerate(result['source_documents']):
    print(f"  {i+1}. {doc.metadata['source']}: {doc.page_content[:100]}...")
```

**输出示例**:
```
步骤 1: 加载文档...
✓ 加载了 1 个文档

步骤 2: 分割文档...
✓ 分割为 15 个块

步骤 3: 向量化...
✓ 向量化完成

步骤 4: 构建 QA 链...
✓ QA 链构建完成

步骤 5: 查询...

问题: What is LangChain?
回答: LangChain is a framework for developing applications powered by language models. It provides components and off-the-shelf chains to help developers build LLM applications.

来源文档:
  1. knowledge.txt: LangChain is a framework for developing applications powered by language models...
  2. knowledge.txt: The main value props of LangChain are: Components, Off-the-shelf chains...
  3. knowledge.txt: LangChain provides standard interfaces for working with LLMs...
```

**关键配置**:
- `chain_type="stuff"`: 简单但有效,适合小到中等数据量
- `search_type="similarity"`: 基于余弦相似度检索
- `k=3`: 返回前 3 个最相关的块
- `return_source_documents=True`: 返回源文档用于溯源

[来源: 04_最小可用.md | RAG 最小示例]

## 场景 4: 错误处理

### 示例 4.1: 编码错误处理

```python
import chardet
from langchain_community.document_loaders import TextLoader

def load_text_with_encoding_detection(file_path: str):
    """自动检测编码并加载文本文件"""
    try:
        # 尝试使用 UTF-8
        loader = TextLoader(file_path, encoding="utf-8")
        documents = loader.load()
        print(f"✓ 使用 UTF-8 编码成功加载")
        return documents
    except UnicodeDecodeError:
        print("✗ UTF-8 编码失败,尝试自动检测...")

        # 检测编码
        with open(file_path, "rb") as f:
            raw_data = f.read()
            result = chardet.detect(raw_data)
            encoding = result["encoding"]
            confidence = result["confidence"]

        print(f"检测到编码: {encoding} (置信度: {confidence:.2%})")

        # 使用检测到的编码
        loader = TextLoader(file_path, encoding=encoding)
        documents = loader.load()
        print(f"✓ 使用 {encoding} 编码成功加载")
        return documents

# 使用示例
documents = load_text_with_encoding_detection("chinese_file.txt")
```

**常见编码问题**:
- Windows 中文文件通常是 GBK 编码
- 旧文件可能是 Latin-1 或 ISO-8859-1
- 使用 `chardet` 库自动检测编码

[来源: 03_核心概念_8_错误处理与编码.md]

### 示例 4.2: 文件不存在处理

```python
from langchain_community.document_loaders import TextLoader
import os

def safe_load_text(file_path: str):
    """安全加载文本文件"""
    # 检查文件是否存在
    if not os.path.exists(file_path):
        print(f"✗ 文件不存在: {file_path}")
        return []

    # 检查是否是文件
    if not os.path.isfile(file_path):
        print(f"✗ 不是文件: {file_path}")
        return []

    # 检查文件是否为空
    if os.path.getsize(file_path) == 0:
        print(f"⚠ 文件为空: {file_path}")
        return []

    try:
        loader = TextLoader(file_path, encoding="utf-8")
        documents = loader.load()
        print(f"✓ 成功加载: {file_path}")
        return documents
    except Exception as e:
        print(f"✗ 加载失败: {file_path}, 错误: {e}")
        return []

# 使用示例
documents = safe_load_text("example.txt")
```

### 示例 4.3: 批量加载与错误处理

```python
from langchain_community.document_loaders import TextLoader
from pathlib import Path
from typing import List
from langchain_core.documents import Document

def batch_load_texts(directory: str) -> List[Document]:
    """批量加载目录中的所有文本文件"""
    all_documents = []
    failed_files = []

    # 获取所有 .txt 文件
    txt_files = list(Path(directory).glob("*.txt"))
    print(f"找到 {len(txt_files)} 个文本文件")

    for file_path in txt_files:
        try:
            loader = TextLoader(str(file_path), encoding="utf-8")
            documents = loader.load()
            all_documents.extend(documents)
            print(f"✓ {file_path.name}")
        except Exception as e:
            failed_files.append((file_path.name, str(e)))
            print(f"✗ {file_path.name}: {e}")

    # 总结
    print(f"\n总结:")
    print(f"  成功: {len(all_documents)} 个文档")
    print(f"  失败: {len(failed_files)} 个文件")

    if failed_files:
        print(f"\n失败文件:")
        for filename, error in failed_files:
            print(f"  - {filename}: {error}")

    return all_documents

# 使用示例
documents = batch_load_texts("./docs")
```

### 示例 5.2: 缓存加载结果

```python
from langchain_community.document_loaders import TextLoader
from langchain_core.documents import Document
import pickle
import hashlib
from pathlib import Path
from typing import List

class CachedTextLoader:
    """带缓存的文本加载器"""

    def __init__(self, file_path: str, cache_dir: str = ".cache"):
        self.file_path = file_path
        self.cache_dir = Path(cache_dir)
        self.cache_dir.mkdir(exist_ok=True)

    def _get_cache_path(self) -> Path:
        """生成缓存文件路径"""
        file_hash = hashlib.md5(self.file_path.encode()).hexdigest()
        return self.cache_dir / f"{file_hash}.pkl"

    def _is_cache_valid(self) -> bool:
        """检查缓存是否有效"""
        cache_path = self._get_cache_path()
        if not cache_path.exists():
            return False

        # 检查文件是否被修改
        file_mtime = Path(self.file_path).stat().st_mtime
        cache_mtime = cache_path.stat().st_mtime
        return cache_mtime > file_mtime

    def load(self) -> List[Document]:
        """加载文档(带缓存)"""
        cache_path = self._get_cache_path()

        # 尝试从缓存加载
        if self._is_cache_valid():
            print(f"✓ 从缓存加载: {self.file_path}")
            with open(cache_path, "rb") as f:
                return pickle.load(f)

        # 从文件加载
        print(f"✓ 从文件加载: {self.file_path}")
        loader = TextLoader(self.file_path, encoding="utf-8")
        documents = loader.load()

        # 保存到缓存
        with open(cache_path, "wb") as f:
            pickle.dump(documents, f)

        return documents

# 使用示例
import time

# 第一次加载 - 从文件
start = time.time()
loader = CachedTextLoader("large_file.txt")
documents = loader.load()
print(f"第一次加载耗时: {time.time() - start:.2f}秒")

# 第二次加载 - 从缓存
start = time.time()
loader = CachedTextLoader("large_file.txt")
documents = loader.load()
print(f"第二次加载耗时: {time.time() - start:.2f}秒")
```

**输出示例**:
```
✓ 从文件加载: large_file.txt
第一次加载耗时: 2.35秒

✓ 从缓存加载: large_file.txt
第二次加载耗时: 0.05秒
```

**缓存策略**:
- 使用文件修改时间判断缓存是否有效
- 使用 pickle 序列化 Document 对象
- 适合频繁加载相同文件的场景

## 场景 6: 实战技巧

### 技巧 6.1: 自定义元数据

```python
from langchain_community.document_loaders import TextLoader
from langchain_core.documents import Document

def load_with_custom_metadata(file_path: str, category: str):
    """加载文档并添加自定义元数据"""
    loader = TextLoader(file_path, encoding="utf-8")
    documents = loader.load()

    # 添加自定义元数据
    for doc in documents:
        doc.metadata["category"] = category
        doc.metadata["file_size"] = len(doc.page_content)
        doc.metadata["word_count"] = len(doc.page_content.split())

    return documents

# 使用示例
documents = load_with_custom_metadata("tutorial.txt", category="tutorial")
print(f"元数据: {documents[0].metadata}")
```

**输出示例**:
```
元数据: {
    'source': 'tutorial.txt',
    'category': 'tutorial',
    'file_size': 1234,
    'word_count': 256
}
```

**用途**:
- 在检索时过滤文档
- 在生成答案时提供上下文
- 用于审计和溯源

### 技巧 6.2: 过滤空文档

from langchain_community.document_loaders import TextLoader

def load_and_filter(file_path: str):
    """加载文档并过滤空内容"""
    loader = TextLoader(file_path, encoding="utf-8")
    documents = loader.load()

    # 过滤空文档
    filtered_docs = [
        doc for doc in documents
        if doc.page_content.strip()  # 过滤空白文档
    ]

    print(f"原始文档数: {len(documents)}")
    print(f"过滤后文档数: {len(filtered_docs)}")

    return filtered_docs

# 使用示例
documents = load_and_filter("example.txt")

### 技巧 6.3: 文本规范化

```python
from langchain_community.document_loaders import TextLoader
import re

def load_and_normalize(file_path: str):
    """加载文档并规范化文本"""
    loader = TextLoader(file_path, encoding="utf-8")
    documents = loader.load()

    for doc in documents:
        # 规范化换行符
        doc.page_content = doc.page_content.replace("\r\n", "\n")

        # 移除多余空白
        doc.page_content = re.sub(r"\n{3,}", "\n\n", doc.page_content)

        # 移除行尾空白
        doc.page_content = "\n".join(
            line.rstrip() for line in doc.page_content.split("\n")
        )

    return documents

# 使用示例
documents = load_and_normalize("messy_file.txt")
```

## 总结

### 核心要点

1. **基础加载**: 使用 TextLoader 加载单个文本文件
2. **懒加载**: 大文件使用 `lazy_load()` 节省内存
3. **文本分割**: 使用 RecursiveCharacterTextSplitter 分割文档
4. **RAG 集成**: 完整的加载 → 分割 → 向量化 → 检索 → 生成流程
5. **错误处理**: 编码检测、文件检查、批量加载
6. **性能优化**: 懒加载、缓存、流式处理
7. **实战技巧**: 自定义元数据、过滤、规范化

### 最佳实践

| 场景 | 推荐方法 | 原因 |
|------|----------|------|
| 小文件(<10MB) | load() | 简单方便 |
| 大文件(>100MB) | lazy_load() | 节省内存 |
| 批量加载 | 错误处理 + 日志 | 提高健壮性 |
| 频繁加载 | 缓存 | 提升性能 |
| RAG 应用 | load_and_split() | 一步到位 |

### 常见问题

**Q1: 如何处理编码错误?**
- 使用 `chardet` 库自动检测编码
- 常见编码: UTF-8, GBK, Latin-1

**Q2: 如何处理大文件?**
- 使用 `lazy_load()` 流式加载
- 逐块处理,避免内存溢出

**Q3: 如何提升加载性能?**
- 使用缓存避免重复加载
- 使用懒加载减少内存占用
- 批量处理时使用并发
