# 核心概念：从文件加载模板

## 概述

**核心价值**：将 Prompt 模板从代码中分离到独立文件，实现模板的版本管理、多语言支持和团队协作。

**适用场景**：
- 多语言 Prompt 管理（中文、英文、日文等）
- 复杂 Prompt 的版本控制
- 团队协作的 Prompt 工程
- RAG 系统的模板库管理

## 1. from_file 方法基础

### 1.1 核心 API

```python
from langchain_core.prompts import PromptTemplate

# 从文件加载模板
prompt = PromptTemplate.from_file(
    file_path="path/to/template.txt",
    encoding="utf-8"  # 可选，默认 utf-8
)
```

**关键特性**：
- 自动读取文件内容作为模板
- 自动提取模板中的变量
- 支持多种编码格式
- 返回完整的 PromptTemplate 对象

[来源: reference/source_prompttemplate_01.md:174-200]

### 1.2 自动变量提取

**源码实现**：

```python
# 源码位置: prompt.py:116-123
if values["template_format"]:
    values["input_variables"] = [
        var
        for var in get_template_variables(
            values["template"], values["template_format"]
        )
        if var not in values["partial_variables"]

```

**工作原理**：
1. 读取文件内容
2. 根据 `template_format` 解析变量
3. 排除 `partial_variables` 中的变量
4. 自动生成 `input_variables` 列表

## 2. 多编码支持

### 2.1 支持的编码格式

LangChain 支持 Python 标准库中的所有编码格式：

| 编码格式 | 说明 | 适用场景 |
|---------|------|---------|
| `utf-8` | Unicode 编码（默认） | 现代应用、多语言支持 |
| `cp1252` | Windows 西欧编码 | Windows 系统文件 |
| `gbk` | 中文编码 | 旧版中文系统 |
| `shift_jis` | 日文编码 | 日文系统文件 |
| `iso-8859-1` | Latin-1 编码 | 欧洲语言 |

### 2.2 编码处理示例

**测试用例**（来自源码）：

```python
from tempfile import NamedTemporaryFile
from langchain_core.prompts import PromptTemplate

def test_from_file_encoding():
    """测试从文件加载模板时的编码处理"""
    template = "This is a {foo} test with special character €."
    input_variables = ["foo"]

    # 使用 CP-1252 编码写入文件
    with NamedTemporaryFile(delete=True, mode="w", encoding="cp1252") as f:
        f.write(template)
        f.flush()
        file_name = f.name

        # 使用 CP-1252 编码读取
        prompt = PromptTemplate.from_file(file_name, encoding="cp1252")
        assert prompt.template == template
        assert prompt.input_variables == input_variables
```

[来源: reference/source_prompttemplate_01.md:176-194]

### 2.3 编码错误处理

```python
from langchain_core.prompts import PromptTemplate

def load_template_with_fallback(file_path: str):
    """尝试多种编码加载模板"""
    encodings = ["utf-8", "cp1252", "gbk", "shift_jis"]

    for encoding in encodings:
        try:
            prompt = PromptTemplate.from_file(file_path, encoding=encoding)
            print(f"✓ 成功使用 {encoding} 编码加载")
            return prompt
        except UnicodeDecodeError:
            print(f"✗ {encoding} 编码失败，尝试下一个...")
            continue

    raise ValueError(f"无法使用任何编码加载文件: {file_path}")

# 使用示例
prompt = load_template_with_fallback("templates/chinese_prompt.txt")
```

## 4. 实战代码示例

### 4.1 基础文件加载

```python
"""
从文件加载模板 - 基础示例
演示：基本的文件加载和使用
"""

from langchain_core.prompts import PromptTemplate
from pathlib import Path

# ===== 1. 创建模板文件 =====
template_dir = Path("prompts")
template_dir.mkdir(exist_ok=True)

# 写入模板内容
template_content = """请将以下文本翻译成 {target_language}：

原文：
{text}

翻译："""

template_file = template_dir / "translate.txt"
template_file.write_text(template_content, encoding="utf-8")

# ===== 2. 从文件加载模板 =====
prompt = PromptTemplate.from_file(
    str(template_file),
    encoding="utf-8"
)

print("=== 模板信息 ===")
print(f"模板内容:\n{prompt.template}")
print(f"\n自动提取的变量: {prompt.input_variables}")

# ===== 3. 使用模板 =====
result = prompt.format(
    target_language="英文",
    text="人工智能正在改变世界"
)

print(f"\n=== 格式化结果 ===\n{result}")
```

**运行输出**：

```
=== 模板信息 ===
模板内容:
请将以下文本翻译成 {target_language}：

原文：
{text}

翻译：

自动提取的变量: ['target_language', 'text']

=== 格式化结果 ===
请将以下文本翻译成 英文：

原文：
人工智能正在改变世界

翻译：
```

### 4.2 多语言模板管理

```python
"""
多语言模板管理系统
演示：根据语言动态加载不同的模板文件
"""

from langchain_core.prompts import PromptTemplate
from pathlib import Path
from typing import Dict

class MultiLanguagePromptManager:
    """多语言 Prompt 管理器"""

    def __init__(self, base_dir: str = "prompts"):
        self.base_dir = Path(base_dir)
        self.cache: Dict[str, PromptTemplate] = {}

    def load_prompt(
        self,
        template_name: str,
        language: str = "zh",
        encoding: str = "utf-8"
    ) -> PromptTemplate:
        """加载指定语言的模板"""
        cache_key = f"{language}:{template_name}"

        # 检查缓存
        if cache_key in self.cache:
            return self.cache[cache_key]

        # 构建文件路径
        template_path = self.base_dir / language / f"{template_name}.txt"

        if not template_path.exists():
            raise FileNotFoundError(
                f"模板文件不存在: {template_path}"
            )

        # 加载模板
        prompt = PromptTemplate.from_file(
            str(template_path),
            encoding=encoding
        )

        # 缓存
        self.cache[cache_key] = prompt
        return prompt

    def list_templates(self, language: str = "zh") -> list:
        """列出指定语言的所有模板"""
        lang_dir = self.base_dir / language
        if not lang_dir.exists():
            return []

        return [
            f.stem for f in lang_dir.glob("*.txt")
        ]

# ===== 使用示例 =====

# 1. 创建测试模板
manager = MultiLanguagePromptManager()

# 中文模板
zh_dir = Path("prompts/zh")
zh_dir.mkdir(parents=True, exist_ok=True)
(zh_dir / "summarize.txt").write_text(
    "请用中文总结以下内容：\n\n{content}\n\n摘要：",
    encoding="utf-8"
)

# 英文模板
en_dir = Path("prompts/en")
en_dir.mkdir(parents=True, exist_ok=True)
(en_dir / "summarize.txt").write_text(
    "Please summarize the following content in English:\n\n{content}\n\nSummary:",
    encoding="utf-8"
)

# 2. 加载不同语言的模板
print("=== 中文模板 ===")
zh_prompt = manager.load_prompt("summarize", language="zh")
print(zh_prompt.format(content="人工智能技术正在快速发展..."))

print("\n=== 英文模板 ===")
en_prompt = manager.load_prompt("summarize", language="en")
print(en_prompt.format(content="Artificial intelligence is rapidly evolving..."))

# 3. 列出所有模板
print("\n=== 可用模板 ===")
print(f"中文模板: {manager.list_templates('zh')}")
print(f"英文模板: {manager.list_templates('en')}")
```

**运行输出**：

```
=== 中文模板 ===
请用中文总结以下内容：

人工智能技术正在快速发展...

摘要：

=== 英文模板 ===
Please summarize the following content in English:

Artificial intelligence is rapidly evolving...

Summary:

=== 可用模板 ===
中文模板: ['summarize']
英文模板: ['summarize']
```

### 4.3 RAG 系统模板库

```python
"""
RAG 系统模板库管理
演示：在 RAG 系统中使用文件模板
"""

from langchain_core.prompts import PromptTemplate
from langchain_openai import ChatOpenAI
from pathlib import Path
import os

class RAGTemplateLibrary:
    """RAG 系统模板库"""

    def __init__(self, template_dir: str = "prompts/rag"):
        self.template_dir = Path(template_dir)
        self.template_dir.mkdir(parents=True, exist_ok=True)
        self._init_templates()

    def _init_templates(self):
        """初始化默认模板"""
        templates = {
            "query": """你是一个专业的 AI 助手。请基于以下上下文回答用户的问题。

上下文信息：
{context}

用户问题：
{query}

请提供准确、简洁的回答。如果上下文中没有相关信息，请明确说明。""",

            "query_with_source": """你是一个专业的 AI 助手。请基于以下上下文回答用户的问题，并引用来源。

上下文信息：
{context}

用户问题：
{query}

请提供准确的回答，并在回答末尾列出参考来源。""",

            "no_context": """抱歉，我在知识库中没有找到与您的问题相关的信息。

您的问题：{query}

建议：
1. 尝试使用不同的关键词重新提问
2. 确认问题是否在知识库的覆盖范围内
3. 联系管理员添加相关内容到知识库"""
        }

        for name, content in templates.items():
            file_path = self.template_dir / f"{name}.txt"
            if not file_path.exists():
                file_path.write_text(content, encoding="utf-8")

    def get_prompt(self, template_name: str) -> PromptTemplate:
        """获取指定的模板"""
        file_path = self.template_dir / f"{template_name}.txt"

        if not file_path.exists():
            raise ValueError(f"模板不存在: {template_name}")

        return PromptTemplate.from_file(str(file_path))

    def update_template(self, template_name: str, content: str):
        """更新模板内容"""
        file_path = self.template_dir / f"{template_name}.txt"
        file_path.write_text(content, encoding="utf-8")
        print(f"✓ 模板已更新: {template_name}")

# ===== 使用示例 =====

# 1. 初始化模板库
library = RAGTemplateLibrary()

# 2. 使用不同的模板
print("=== 基础查询模板 ===")
query_prompt = library.get_prompt("query")
print(query_prompt.format(
    context="LangChain 是一个用于构建 LLM 应用的框架。",
    query="什么是 LangChain？"
))

print("\n=== 带来源的查询模板 ===")
source_prompt = library.get_prompt("query_with_source")
print(source_prompt.format(
    context="LangChain 是一个用于构建 LLM 应用的框架。[来源: 官方文档]",
    query="什么是 LangChain？"
))

print("\n=== 无上下文模板 ===")
no_context_prompt = library.get_prompt("no_context")
print(no_context_prompt.format(
    query="量子计算的原理是什么？"
))

# 3. 更新模板
new_template = """你是一个专业的 AI 助手。请基于以下上下文回答用户的问题。

【上下文】
{context}

【问题】
{query}

【回答要求】
- 基于上下文提供准确回答
- 使用专业但易懂的语言
- 如果信息不足，明确说明"""

library.update_template("query", new_template)
```

## 6. 反直觉点

### 误区1：文件路径必须是绝对路径 ❌

**为什么错？**
- `from_file` 支持相对路径和绝对路径
- 相对路径相对于当前工作目录
- 使用 `Path` 对象可以更灵活地处理路径

**正确理解**：

```python
from pathlib import Path

# ✓ 相对路径（推荐）
prompt = PromptTemplate.from_file("prompts/template.txt")

# ✓ 绝对路径
prompt = PromptTemplate.from_file("/absolute/path/template.txt")

# ✓ 使用 Path 对象（最推荐）
template_path = Path("prompts") / "template.txt"
prompt = PromptTemplate.from_file(str(template_path))
```

### 误区2：必须手动指定 input_variables ❌

**为什么错？**
- `from_file` 会自动提取模板中的变量
- 无需手动指定 `input_variables`
- 自动提取支持所有模板格式（f-string、mustache、jinja2）

**正确理解**：

```python
# 模板文件内容: "Hello {name}, you are {age} years old."

# ✓ 自动提取变量
prompt = PromptTemplate.from_file("template.txt")
print(prompt.input_variables)  # ['name', 'age']

# ✗ 不需要手动指定
prompt = PromptTemplate.from_file(
    "template.txt",
    input_variables=["name", "age"]  # 多余的参数
)
```

### 误区3：文件编码错误会导致程序崩溃 ❌

**为什么错？**
- Python 的编码错误是可以捕获和处理的
- 可以实现编码自动检测和回退机制
- 使用 `chardet` 库可以自动检测编码

**正确理解**：

```python
import chardet

def load_with_auto_encoding(file_path: str) -> PromptTemplate:
    """自动检测编码并加载模板"""
    # 读取文件字节
    with open(file_path, 'rb') as f:
        raw_data = f.read()

    # 检测编码
    result = chardet.detect(raw_data)
    encoding = result['encoding']
    confidence = result['confidence']

    print(f"检测到编码: {encoding} (置信度: {confidence:.2%})")

    # 使用检测到的编码加载
    return PromptTemplate.from_file(file_path, encoding=encoding)

# 使用
prompt = load_with_auto_encoding("unknown_encoding.txt")
```

## 7. 最佳实践总结

### 7.1 文件组织

1. **按功能分类**：将模板按功能（RAG、Agent、翻译等）分类存放
2. **多语言支持**：使用语言代码（zh、en、ja）作为子目录
3. **版本控制**：将模板文件纳入 Git 版本控制
4. **文档注释**：在模板文件头部添加说明注释

### 7.2 编码处理

1. **统一使用 UTF-8**：现代应用统一使用 UTF-8 编码
2. **显式指定编码**：即使是 UTF-8 也建议显式指定
3. **错误处理**：实现编码错误的回退机制
4. **自动检测**：对于未知编码，使用 `chardet` 自动检测

### 7.3 性能优化

1. **模板缓存**：避免重复加载同一模板文件
2. **延迟加载**：只在需要时加载模板
3. **预加载**：对于高频使用的模板，可以在启动时预加载
4. **文件监控**：在开发环境中监控文件变化，自动重新加载

### 7.4 安全考虑

1. **路径验证**：验证文件路径，防止路径遍历攻击
2. **权限控制**：确保模板文件的读取权限正确设置
3. **内容审查**：对于用户上传的模板，进行内容审查
4. **沙箱隔离**：使用 jinja2 模板时，确保使用 `SandboxedEnvironment`
