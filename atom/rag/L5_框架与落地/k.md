# L5_框架与落地 知识点列表

> 掌握主流 RAG 框架的核心用法，完成从原型到生产的全流程落地

---

## 知识点清单

1. **LangChain核心概念** - 掌握 Chain/Agent/Memory 架构与 LCEL 表达式语言，快速构建 RAG 应用
2. **LlamaIndex核心概念** - 掌握数据连接器、索引类型与查询引擎，使用专注 RAG 的框架高效开发
3. **端到端项目实战** - 从需求分析到完整 RAG 系统的全流程实现，串联 L1-L4 所有知识点
4. **RAG API服务化** - 用 FastAPI 封装 RAG 系统为 REST API，实现流式响应与异步处理
5. **部署与监控** - Docker 容器化、云部署方案、日志管理与监控告警体系搭建
6. **生产环境优化** - 向量库缓存、并发处理、延迟优化与成本控制的实战策略

---

## 学习顺序建议

```
LangChain核心概念 → LlamaIndex核心概念 → 端到端项目实战 → RAG API服务化 → 部署与监控 → 生产环境优化
       ↓                  ↓                  ↓                ↓              ↓              ↓
    掌握通用框架        掌握RAG专用框架      串联全流程实战      封装为API服务    容器化与上线     性能与成本优化
```

**为什么是这个顺序？**

1. **框架基础（1-2）**：先掌握两大主流框架的核心概念，为后续实战提供工具支撑
2. **全流程实战（3）**：用框架串联 L1-L4 所有知识点，完成一个完整的 RAG 系统
3. **服务化（4）**：将实战项目封装为可调用的 API 服务，迈向生产化的第一步
4. **部署运维（5-6）**：先解决部署与监控问题，再进行性能与成本的深度优化
