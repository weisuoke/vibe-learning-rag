## 核心要点预览

### 8种索引类型速览

| 索引类型 | 核心特点 | 适用规模 | 准确率 | 内存 |
|---------|---------|---------|--------|------|
| **FLAT** | 暴力搜索 | < 1M | 100% | 1x |
| **IVF_FLAT** | k-means聚类 | 1M-10M | 95-99% | 1x |
| **IVF_SQ8** | 标量量化 | 1M-100M | 93-98% | 0.25x |
| **IVF_PQ** | 乘积量化 | 100M+ | 85-95% | 0.03x |
| **HNSW** | 多层图 | 10M-100M | 95-99% | 1.5x |
| **SCANN** | Google算法 | 10M-100M | 90-98% | 0.5x |
| **GPU_CAGRA** | GPU加速 | 100M-1B | 95-99% | 1.5x |
| **RaBitQ** | 1-bit量化 | 1B+ | 94-98% | 0.03x |

### 选型决策树

```
数据规模？
  ├─ < 1M → FLAT
  ├─ 1M-10M → IVF_FLAT
  ├─ 10M-100M → HNSW
  └─ > 100M → 内存充足？
       ├─ 是 → GPU_CAGRA
       └─ 否 → RaBitQ
```

### 关键参数

```python
# FLAT: 无参数
{"index_type": "FLAT", "params": {}}

# IVF_FLAT
{"index_type": "IVF_FLAT", "params": {"nlist": 1024}}
search_params = {"nprobe": 16}

# HNSW
{"index_type": "HNSW", "params": {"M": 16, "efConstruction": 200}}
search_params = {"ef": 100}

# GPU_CAGRA
{"index_type": "GPU_CAGRA", "params": {"intermediate_graph_degree": 128}}

# RaBitQ
{"index_type": "IVF_RABITQ", "params": {"nlist": 1024}}
```
