# 30字核心

**RunnableBranch 是根据条件动态选择执行分支的 Runnable，实现 AI Agent 中的条件路由与决策逻辑。**

---

## 核心价值

在 AI Agent 开发中，RunnableBranch 解决了**动态决策**问题：

- **条件路由**：根据输入内容选择不同的处理流程
- **可组合性**：与 LCEL 无缝集成，保持链式调用
- **可观测性**：支持 LangSmith 追踪每个分支的执行路径

---

## 快速示例

```python
from langchain_core.runnables import RunnableBranch, RunnableLambda

# 根据情感选择不同的回复策略
branch = RunnableBranch(
    (lambda x: "positive" in x, positive_handler),
    (lambda x: "negative" in x, negative_handler),
    default_handler  # 默认分支
)

result = branch.invoke("This is positive feedback!")
```

**一句话理解**：RunnableBranch 是 LangChain 中的 if-elif-else，让 AI 工作流能根据条件动态选择执行路径。
