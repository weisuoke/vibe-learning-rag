## Key RAG Patterns

### Basic RAG Pipeline
1. **Document Loading**: 加载文档（PDF, Markdown, HTML等）
2. **Text Chunking**: 文本分块处理
3. **Embedding Generation**: 生成向量嵌入
4. **Vector Storage**: 存储到Milvus
5. **Semantic Retrieval**: 语义检索
6. **LLM Generation**: LLM生成答案

### Advanced RAG Techniques
- **Hybrid Search**: Dense + Sparse vectors
- **Reranking**: 结果重排序
- **Query Rewriting**: 查询改写
- **Context Compression**: 上下文压缩
- **Multi-Query**: 多查询策略

### Integration Frameworks
- **LangChain**: 最流行的RAG框架
- **LlamaIndex**: 专注于索引和检索
- **Haystack**: 端到端NLP框架
- **Unstructured**: 文档解析工具
