# ExampleSelector示例选择 - 30字核心

**ExampleSelector 是动态选择 Few-shot 示例的机制,根据输入自动选择最相关示例,提升 Prompt 效果。**

---

## 核心要素拆解

### 1. 是什么？
**动态选择 Few-shot 示例的机制**
- 不是固定示例列表
- 不是手动选择
- 是根据输入自动选择

### 2. 做什么？
**根据输入自动选择最相关示例**
- 分析输入特征
- 匹配示例库
- 返回最相关的 k 个示例

### 3. 为什么？
**提升 Prompt 效果**
- 提高 Few-shot learning 准确性
- 控制 Prompt 长度（避免 token 超限）
- 减少无关示例干扰

---

## 核心价值

### 1. 提高准确性
选择最相关的示例 → 模型理解更准确 → 输出质量更高

### 2. 控制成本
动态控制示例数量 → 减少 token 使用 → 降低 API 成本

### 3. 提升体验
自动化示例选择 → 无需手动维护 → 开发效率更高
