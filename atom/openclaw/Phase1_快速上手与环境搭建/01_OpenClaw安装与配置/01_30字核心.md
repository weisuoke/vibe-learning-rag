# OpenClaw 安装与配置 - 30字核心

**OpenClaw 是通过 npm/pnpm 全局安装的多通道 AI 网关，需要 Node.js 22+ 环境，配置存储在 ~/.openclaw/ 目录，通过 Onboarding Wizard 交互式配置。**

---

## 核心要素拆解

### 1. 安装方式
**npm/pnpm 全局安装**
- 命令：`npm install -g openclaw@latest` 或 `pnpm add -g openclaw@latest`
- 全局 bin 目录：自动添加到系统 PATH
- 可执行文件：`openclaw` 命令全局可用

### 2. 环境要求
**Node.js 22+**
- 最低版本：Node.js 22.0.0
- 推荐版本：Node.js 22.18.0+（原生 TypeScript 支持）
- 关键特性：ESM 支持、内置 WebSocket、Maglev 编译器

### 3. 配置位置
**~/.openclaw/ 目录**
- 配置文件：`settings.json`
- 日志文件：`gateway.log`
- 跨平台标准：遵循 XDG Base Directory 规范

### 4. 配置方式
**Onboarding Wizard 交互式配置**
- 命令：`openclaw onboard`
- 交互式提示：API keys、Gateway 端口、通道选择
- 自动生成：`~/.openclaw/settings.json`

### 5. 核心价值
**统一多通道 AI 交互入口**
- 多模型支持：Claude、GPT、Gemini 等
- 统一接口：标准化 API 调用
- 本地网关：隐私保护、成本控制

## 快速验证

安装完成后，运行以下命令验证：

```bash
# 1. 检查版本
openclaw --version

# 2. 查看帮助
openclaw --help

# 3. 检查配置目录
ls -la ~/.openclaw/

# 4. 运行 Onboarding Wizard
openclaw onboard
```
