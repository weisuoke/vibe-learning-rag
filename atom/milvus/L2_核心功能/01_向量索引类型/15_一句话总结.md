# 一句话总结

**向量索引类型是 Milvus 通过不同数据结构（FLAT 暴力、IVF 聚类、HNSW 图）实现的检索加速方案，根据数据规模、召回率要求、内存预算选择合适索引，在 RAG 系统中实现从毫秒到微秒级的语义检索。**

---

## 核心记忆点

### 三大索引
- **FLAT**：暴力检索，精确但慢，小规模首选
- **IVF_FLAT**：聚类检索，平衡精度速度，中规模首选
- **HNSW**：图检索，高性能，大规模首选

### 选择决策树
```
数据规模 < 10万？
  ├─ 是 → FLAT
  └─ 否 → 数据规模 < 100万？
           ├─ 是 → IVF_FLAT
           └─ 否 → HNSW
```

### 关键参数
- **IVF_FLAT**: `nlist` (聚类数) + `nprobe` (搜索桶数)
- **HNSW**: `M` (连接数) + `efConstruction` (构建范围) + `ef` (搜索范围)

### RAG 应用
- 个人知识库 → FLAT
- 企业知识库 → IVF_FLAT
- 大型知识库 → HNSW
