# 核心概念 05：模板创建方法

> **核心要点**：ChatPromptTemplate 提供三种创建方法：`from_messages()` 灵活构建多消息模板、`from_template()` 快速创建单消息模板、直接构造函数精确控制，选择合适的方法可以提高开发效率和代码可读性。

---

## 一、三种创建方法概览

### 1.1 方法对比表

| 方法 | 适用场景 | 灵活性 | 代码简洁度 | 推荐度 |
|------|----------|--------|------------|--------|
| `from_messages()` | 多消息对话、复杂模板 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ |
| `from_template()` | 单个用户消息、快速原型 | ⭐⭐ | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ |
| 直接构造 | 需要精确控制参数 | ⭐⭐⭐⭐⭐ | ⭐⭐ | ⭐⭐ |

### 1.2 快速示例

```python
from langchain_core.prompts import ChatPromptTemplate

# 方法 1: from_messages() - 推荐
template1 = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant."),
    ("human", "{input}")
])

# 方法 2: from_template() - 快速原型
template2 = ChatPromptTemplate.from_template("{input}")

# 方法 3: 直接构造 - 精确控制
template3 = ChatPromptTemplate(
    messages=[
        ("system", "You are a helpful assistant."),
        ("human", "{input}")
    ],
    template_format="f-string"
)
```

## 二、方法一：from_messages() 详解

### 2.1 基本用法

**函数签名**（源码 lines 1101-1167）：
```python
@classmethod
def from_messages(
    cls,
    messages: Sequence[MessageLikeRepresentation],
    template_format: PromptTemplateFormat = "f-string",
    **kwargs
) -> ChatPromptTemplate:
    """从消息列表创建 ChatPromptTemplate"""
```

**核心特性**：
- 支持多种消息格式混合使用
- 自动推断 input_variables
- 灵活的消息类型定义

### 2.2 支持的消息格式

#### 格式 1：元组格式（最常用）

```python
from langchain_core.prompts import ChatPromptTemplate

# 基础元组格式: (role, content)
template = ChatPromptTemplate.from_messages([
    ("system", "You are a {role}."),
    ("human", "Tell me about {topic}.")
])

# 使用
result = template.invoke({
    "role": "helpful assistant",
    "topic": "Python"
})

print(result.to_messages())
# Output:
# [
#   SystemMessage(content="You are a helpful assistant."),
#   HumanMessage(content="Tell me about Python.")
# ]
```

**支持的角色名称**：
- `"system"` → SystemMessage
- `"human"` / `"user"` → HumanMessage
- `"ai"` / `"assistant"` → AIMessage
- 自定义角色名 → ChatMessage

#### 格式 2：消息对象

```python
from langchain_core.messages import SystemMessage, HumanMessage

# 使用消息对象
template = ChatPromptTemplate.from_messages([
    SystemMessage(content="You are a helpful assistant."),
    HumanMessage(content="What is {topic}?")
])
```

**优点**：
- 可以设置额外参数（如 `additional_kwargs`）
- 类型安全
- 适合复杂场景

#### 格式 3：模板对象

```python
from langchain_core.prompts import (
    SystemMessagePromptTemplate,
    HumanMessagePromptTemplate
)

# 使用模板对象
template = ChatPromptTemplate.from_messages([
    SystemMessagePromptTemplate.from_template("You are a {role}."),
    HumanMessagePromptTemplate.from_template("Tell me about {topic}.")
])
```

**优点**：
- 可以使用不同的 template_format（mustache, jinja2）
- 可以设置 partial_variables
- 最大灵活性

#### 格式 4：MessagesPlaceholder

```python
from langchain_core.prompts import ChatPromptTemplate, MessagesPlaceholder

# 使用 MessagesPlaceholder 插入对话历史
template = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant."),
    MessagesPlaceholder("history"),
    ("human", "{input}")
])

# 使用
result = template.invoke({
    "history": [
        ("human", "Hi!"),
        ("ai", "Hello! How can I help?")
    ],
    "input": "What's the weather?"
})
```

#### 格式 5：混合格式（推荐）

```python
from langchain_core.prompts import ChatPromptTemplate, MessagesPlaceholder
from langchain_core.messages import SystemMessage

# 混合使用多种格式
template = ChatPromptTemplate.from_messages([
    SystemMessage(content="You are a helpful assistant."),  # 消息对象
    MessagesPlaceholder("history"),                         # 占位符
    ("human", "{input}")                                    # 元组格式
])
```

**优点**：
- 结合各种格式的优势
- 代码清晰易读
- 实际项目中最常用

### 2.3 template_format 参数

```python
# f-string 格式（默认）
template1 = ChatPromptTemplate.from_messages(
    [("human", "Hello {name}")],
    template_format="f-string"
)

# mustache 格式
template2 = ChatPromptTemplate.from_messages(
    [("human", "Hello {{name}}")],
    template_format="mustache"
)

# jinja2 格式
template3 = ChatPromptTemplate.from_messages(
    [("human", "Hello {{ name }}")],
    template_format="jinja2"
)
```

**选择建议**：
- **f-string**：默认选择，Python 原生支持
- **mustache**：需要与其他语言兼容时
- **jinja2**：需要复杂模板逻辑（循环、条件）时

### 2.4 实战示例

#### 示例 1：基础对话模板

```python
"""
场景：简单的问答系统
"""
from langchain_core.prompts import ChatPromptTemplate
from langchain_openai import ChatOpenAI

# ===== 1. 创建模板 =====
template = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant that answers questions concisely."),
    ("human", "{question}")
])

# ===== 2. 配置模型 =====
model = ChatOpenAI(model="gpt-4o-mini")

# ===== 3. 创建链 =====
chain = template | model

# ===== 4. 使用 =====
response = chain.invoke({"question": "What is Python?"})
print(response.content)
```

#### 示例 2：带历史记录的对话

```python
"""
场景：上下文感知的聊天机器人
"""
from langchain_core.prompts import ChatPromptTemplate, MessagesPlaceholder
from langchain_openai import ChatOpenAI

# ===== 1. 创建带历史的模板 =====
template = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant. Use the conversation history to provide context-aware responses."),
    MessagesPlaceholder("chat_history"),
    ("human", "{input}")
])

# ===== 2. 配置模型 =====
model = ChatOpenAI(model="gpt-4o-mini")
chain = template | model

# ===== 3. 模拟对话 =====
# 第一轮对话
history = []
response1 = chain.invoke({
    "chat_history": history,
    "input": "My name is Alice."
})
print(f"AI: {response1.content}")

# 更新历史
history.extend([
    ("human", "My name is Alice."),
    ("ai", response1.content)
])

# 第二轮对话（AI 应该记住名字）
response2 = chain.invoke({
    "chat_history": history,
    "input": "What's my name?"
})
print(f"AI: {response2.content}")
# Output: "Your name is Alice."
```

#### 示例 3：Few-shot 学习模板

```python
"""
场景：通过示例教 AI 特定任务
"""
from langchain_core.prompts import ChatPromptTemplate
from langchain_openai import ChatOpenAI

# ===== 1. 创建 Few-shot 模板 =====
template = ChatPromptTemplate.from_messages([
    ("system", "You are a sentiment classifier. Classify the sentiment as positive, negative, or neutral."),
    # Few-shot 示例
    ("human", "I love this product!"),
    ("ai", "positive"),
    ("human", "This is terrible."),
    ("ai", "negative"),
    ("human", "It's okay."),
    ("ai", "neutral"),
    # 实际输入
    ("human", "{text}")
])

# ===== 2. 使用 =====
model = ChatOpenAI(model="gpt-4o-mini", temperature=0)
chain = template | model

result = chain.invoke({"text": "This is amazing!"})
print(result.content)  # Output: "positive"
```

## 三、方法二：from_template() 详解

### 3.1 基本用法

**函数签名**：
```python
@classmethod
def from_template(
    cls,
    template: str,
    template_format: PromptTemplateFormat = "f-string",
    **kwargs
) -> ChatPromptTemplate:
    """从单个模板字符串创建 ChatPromptTemplate（默认为 human 消息）"""
```

**核心特性**：
- 快速创建单消息模板
- 默认创建 HumanMessage
- 适合简单场景

### 3.2 基础示例

```python
from langchain_core.prompts import ChatPromptTemplate

# 最简单的方式
template = ChatPromptTemplate.from_template("Tell me about {topic}")

# 等价于
template = ChatPromptTemplate.from_messages([
    ("human", "Tell me about {topic}")
])
```

### 3.3 使用场景

#### 场景 1：快速原型开发

```python
"""
快速测试 LLM 响应
"""
from langchain_core.prompts import ChatPromptTemplate
from langchain_openai import ChatOpenAI

# 快速创建
template = ChatPromptTemplate.from_template("Translate '{text}' to French")
model = ChatOpenAI(model="gpt-4o-mini")
chain = template | model

result = chain.invoke({"text": "Hello"})
print(result.content)  # Output: "Bonjour"
```

#### 场景 2：单轮问答

```python
"""
不需要系统消息的简单问答
"""
from langchain_core.prompts import ChatPromptTemplate
from langchain_openai import ChatOpenAI

template = ChatPromptTemplate.from_template(
    "Answer this question in one sentence: {question}"
)

model = ChatOpenAI(model="gpt-4o-mini")
chain = template | model

result = chain.invoke({"question": "What is the capital of France?"})
print(result.content)
```

### 3.4 局限性

```python
# ❌ 不能添加系统消息
template = ChatPromptTemplate.from_template("Tell me about {topic}")
# 只有一个 HumanMessage，没有 SystemMessage

# ✅ 如果需要系统消息，使用 from_messages()
template = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant."),
    ("human", "Tell me about {topic}")
])
```

**何时不使用 from_template()**：
- 需要多个消息
- 需要系统消息定义 AI 角色
- 需要对话历史
- 需要 Few-shot 示例

## 四、方法三：直接构造详解

### 4.1 基本用法

```python
from langchain_core.prompts import ChatPromptTemplate

# 直接使用构造函数
template = ChatPromptTemplate(
    messages=[
        ("system", "You are a {role}."),
        ("human", "{input}")
    ],
    template_format="f-string",
    input_variables=["role", "input"],  # 可选，会自动推断
    partial_variables={"role": "helpful assistant"}  # 预填充变量
)
```

### 4.2 高级参数控制

#### 参数 1：validate_template

```python
# 启用模板验证
template = ChatPromptTemplate(
    messages=[("human", "{input}")],
    validate_template=True  # 验证变量是否匹配
)

# 如果传入的变量不匹配，会抛出错误
try:
    template.invoke({"wrong_var": "value"})
except Exception as e:
    print(f"Error: {e}")
```

#### 参数 2：input_variables

```python
# 手动指定输入变量（通常不需要，会自动推断）
template = ChatPromptTemplate(
    messages=[("human", "{input}")],
    input_variables=["input"]
)
```

#### 参数 3：partial_variables

```python
# 预填充部分变量
from datetime import datetime

template = ChatPromptTemplate(
    messages=[
        ("system", "Current date: {date}. You are a {role}."),
        ("human", "{input}")
    ],
    partial_variables={
        "date": lambda: datetime.now().strftime("%Y-%m-%d"),
        "role": "helpful assistant"
    }
)

# 使用时只需要提供 input
result = template.invoke({"input": "Hello"})
```

### 4.3 使用场景

#### 场景 1：需要动态默认值

```python
"""
场景：自动注入当前时间
"""
from langchain_core.prompts import ChatPromptTemplate
from datetime import datetime

def get_current_time():
    return datetime.now().strftime("%Y-%m-%d %H:%M:%S")

template = ChatPromptTemplate(
    messages=[
        ("system", "Current time: {time}. You are a helpful assistant."),
        ("human", "{question}")
    ],
    partial_variables={
        "time": get_current_time  # 函数会在每次调用时执行
    }
)

# 使用
result = template.invoke({"question": "What time is it?"})
# SystemMessage 会包含实际的当前时间
```

#### 场景 2：模板复用与配置

```python
"""
场景：创建可配置的模板工厂
"""
from langchain_core.prompts import ChatPromptTemplate

def create_assistant_template(
    role: str,
    temperature_hint: str = "",
    validate: bool = False
):
    """创建可配置的助手模板"""
    messages = [
        ("system", f"You are a {role}. {temperature_hint}"),
        ("human", "{input}")
    ]

    return ChatPromptTemplate(
        messages=messages,
        validate_template=validate
    )

# 创建不同配置的模板
creative_template = create_assistant_template(
    role="creative writer",
    temperature_hint="Be imaginative and expressive."
)

technical_template = create_assistant_template(
    role="technical expert",
    temperature_hint="Be precise and factual."
)
```

## 五、方法选择指南

### 5.1 决策树

```
需要创建 ChatPromptTemplate？
│
├─ 只有一个用户消息？
│  └─ 是 → 使用 from_template()
│
├─ 需要多个消息（系统/用户/AI）？
│  └─ 是 → 使用 from_messages()
│
├─ 需要精确控制参数（partial_variables, validate_template）？
│  └─ 是 → 使用直接构造
│
└─ 默认 → 使用 from_messages()（最灵活）
```

### 5.2 实际场景映射

| 场景 | 推荐方法 | 理由 |
|------|----------|------|
| 快速原型/测试 | `from_template()` | 最简洁 |
| 生产环境对话系统 | `from_messages()` | 需要系统消息和历史 |
| RAG 问答系统 | `from_messages()` | 需要上下文注入 |
| Few-shot 学习 | `from_messages()` | 需要多个示例消息 |
| 需要动态默认值 | 直接构造 | 需要 partial_variables |
| 模板工厂模式 | 直接构造 | 需要精确控制 |

### 5.3 代码风格建议

```python
# ✅ 推荐：清晰的多行格式
template = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant."),
    MessagesPlaceholder("history"),
    ("human", "{input}")
])

# ❌ 不推荐：单行格式（难以阅读）
template = ChatPromptTemplate.from_messages([("system", "You are a helpful assistant."), MessagesPlaceholder("history"), ("human", "{input}")])

# ✅ 推荐：使用变量提高可读性
system_message = ("system", "You are a helpful assistant.")
history_placeholder = MessagesPlaceholder("history")
user_message = ("human", "{input}")

template = ChatPromptTemplate.from_messages([
    system_message,
    history_placeholder,
    user_message
])
```

## 六、高级技巧

### 6.1 模板组合

```python
"""
使用 + 操作符组合模板
"""
from langchain_core.prompts import ChatPromptTemplate

# 基础模板
base = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant.")
])

# 添加用户消息
full_template = base + ("human", "{input}")

# 等价于
full_template = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant."),
    ("human", "{input}")
])
```

### 6.2 动态模板构建

```python
"""
根据条件动态构建模板
"""
from langchain_core.prompts import ChatPromptTemplate, MessagesPlaceholder

def create_template(include_history: bool = False, include_examples: bool = False):
    """动态创建模板"""
    messages = [
        ("system", "You are a helpful assistant.")
    ]

    # 可选：添加 Few-shot 示例
    if include_examples:
        messages.extend([
            ("human", "Hello"),
            ("ai", "Hi! How can I help you?")
        ])

    # 可选：添加历史记录
    if include_history:
        messages.append(MessagesPlaceholder("history"))

    # 添加用户输入
    messages.append(("human", "{input}"))

    return ChatPromptTemplate.from_messages(messages)

# 使用
simple_template = create_template()
history_template = create_template(include_history=True)
full_template = create_template(include_history=True, include_examples=True)
```

### 6.3 模板继承模式

```python
"""
创建可复用的基础模板
"""
from langchain_core.prompts import ChatPromptTemplate

class TemplateFactory:
    """模板工厂类"""

    @staticmethod
    def base_template():
        """基础模板"""
        return ChatPromptTemplate.from_messages([
            ("system", "You are a helpful assistant.")
        ])

    @staticmethod
    def qa_template():
        """问答模板"""
        base = TemplateFactory.base_template()
        return base + ("human", "Question: {question}")

    @staticmethod
    def translation_template():
        """翻译模板"""
        base = TemplateFactory.base_template()
        return base + ("human", "Translate '{text}' to {language}")

    @staticmethod
    def summarization_template():
        """摘要模板"""
        base = TemplateFactory.base_template()
        return base + ("human", "Summarize this text:\n\n{text}")

# 使用
qa = TemplateFactory.qa_template()
translation = TemplateFactory.translation_template()
```

## 七、常见问题与解决方案

### 7.1 问题：变量名冲突

```python
# ❌ 问题：partial_variables 和 input_variables 冲突
template = ChatPromptTemplate(
    messages=[("human", "{name}")],
    partial_variables={"name": "Alice"}
)

# 调用时再次传入 name 会被忽略
result = template.invoke({"name": "Bob"})  # 仍然使用 "Alice"

# ✅ 解决：使用不同的变量名
template = ChatPromptTemplate(
    messages=[("human", "Hello {name}, your role is {role}")],
    partial_variables={"role": "user"}
)

result = template.invoke({"name": "Bob"})  # name 可以动态传入
```

### 7.2 问题：模板格式不一致

```python
# ❌ 问题：混用不同的模板格式
template = ChatPromptTemplate.from_messages([
    ("human", "Hello {name}"),      # f-string
    ("ai", "Hi {{name}}")            # mustache - 不会工作
])

# ✅ 解决：统一使用一种格式
template = ChatPromptTemplate.from_messages(
    [
        ("human", "Hello {name}"),
        ("ai", "Hi {name}")
    ],
    template_format="f-string"
)
```

### 7.3 问题：忘记传递必需变量

```python
# ❌ 问题：缺少必需变量
template = ChatPromptTemplate.from_messages([
    ("human", "Hello {name}, tell me about {topic}")
])

try:
    result = template.invoke({"name": "Alice"})  # 缺少 topic
except Exception as e:
    print(f"Error: {e}")

# ✅ 解决 1：使用 partial_variables 设置默认值
template = ChatPromptTemplate(
    messages=[("human", "Hello {name}, tell me about {topic}")],
    partial_variables={"topic": "Python"}
)

result = template.invoke({"name": "Alice"})  # topic 使用默认值

# ✅ 解决 2：使用 optional MessagesPlaceholder
from langchain_core.prompts import MessagesPlaceholder

template = ChatPromptTemplate.from_messages([
    ("human", "Hello {name}"),
    MessagesPlaceholder("extra_info", optional=True)
])

result = template.invoke({"name": "Alice"})  # extra_info 可选
```

## 八、性能优化建议

### 8.1 模板缓存

```python
"""
缓存常用模板以提高性能
"""
from functools import lru_cache
from langchain_core.prompts import ChatPromptTemplate

@lru_cache(maxsize=10)
def get_template(template_type: str) -> ChatPromptTemplate:
    """获取缓存的模板"""
    if template_type == "qa":
        return ChatPromptTemplate.from_messages([
            ("system", "You are a helpful assistant."),
            ("human", "{question}")
        ])
    elif template_type == "translation":
        return ChatPromptTemplate.from_messages([
            ("system", "You are a translator."),
            ("human", "Translate '{text}' to {language}")
        ])
    else:
        raise ValueError(f"Unknown template type: {template_type}")

# 使用（第二次调用会从缓存获取）
template1 = get_template("qa")
template2 = get_template("qa")  # 从缓存获取
assert template1 is template2  # 同一个对象
```

### 8.2 延迟构建

```python
"""
只在需要时构建模板
"""
class LazyTemplateBuilder:
    """延迟模板构建器"""

    def __init__(self):
        self._template = None

    def get_template(self, include_history: bool = False):
        """按需构建模板"""
        if self._template is None or include_history:
            messages = [("system", "You are a helpful assistant.")]

            if include_history:
                from langchain_core.prompts import MessagesPlaceholder
                messages.append(MessagesPlaceholder("history"))

            messages.append(("human", "{input}"))

            self._template = ChatPromptTemplate.from_messages(messages)

        return self._template

# 使用
builder = LazyTemplateBuilder()
template = builder.get_template()  # 第一次构建
template = builder.get_template()  # 复用已构建的模板
```

## 九、总结

### 核心要点

1. **from_messages()**：最灵活，适合生产环境，支持多种消息格式
2. **from_template()**：最简洁，适合快速原型，只创建单个用户消息
3. **直接构造**：最精确，适合需要高级参数控制的场景

### 选择建议

- **默认选择**：`from_messages()` - 覆盖 90% 的场景
- **快速测试**：`from_template()` - 最快速
- **高级需求**：直接构造 - 最灵活

### 最佳实践

1. 优先使用 `from_messages()` 的元组格式
2. 使用 `MessagesPlaceholder` 管理对话历史
3. 通过 `partial_variables` 设置动态默认值
4. 保持模板格式一致（统一使用 f-string）
5. 使用变量提高代码可读性
