# L3_高级特性 知识点列表

> 掌握 Milvus 的高级特性，实现复杂的数据管理和检索场景

---

## 知识点清单

1. **分区管理** - 使用 Partition 实现数据分区，提升检索效率，支持 100K collections
2. **多向量检索** - 理解 Multi-vector Search 的应用场景和实现方式（多模态检索）
3. **稀疏向量与 BM25 深入** - 深入理解 BM25 算法、稀疏向量索引和混合检索权重调整
4. **数据管理 CRUD** - 掌握数据的增删改查操作
5. **动态 Schema 与高级字段** - 灵活管理 Collection 字段，支持 Int8 向量、空间数据类型、嵌套结构

---

## 学习顺序建议

```
分区管理 → 多向量检索 → 稀疏向量与BM25深入 → 数据管理CRUD → 动态Schema与高级字段
   ↓          ↓              ↓                  ↓                ↓
 数据隔离   多模态场景      全文搜索深入        完整操作        灵活扩展
```

## 2026 核心特性

### Milvus 2.6 新特性
- **100K Collections**: 支持大规模多租户场景
- **Int8 向量**: 内存优化
- **空间数据类型**: 地理位置检索
- **嵌套结构支持**: 复杂数据建模

### Milvus 2.5 核心特性
- **BM25 全文搜索**: 原生支持，4x 快于 Elasticsearch
- **稀疏向量索引**: SPARSE_INVERTED_INDEX、SPARSE_WAND

### Milvus 2.4 核心特性
- **多向量检索**: Multi-vector Search，支持多模态场景
