## 核心理念

**最小可用原则：** 掌握以下内容，就能开始使用 ExampleSelector

```
ExampleSelector 的本质 = 根据输入动态选择最相关的示例
```

**为什么需要？**
- 静态示例：所有查询使用相同的示例（效果差）
- 动态示例：根据输入选择最相关的示例（效果好）

## 最小可用知识点

### 4.1 核心知识点1：SemanticSimilarityExampleSelector（最常用）

**一句话：** 基于语义相似度自动选择最相关的示例

**为什么重要？**
- 80%的场景都用这个选择器
- 效果最好（基于语义理解）
- 2025-2026年成为主流

**最简单的用法：**

```python
from langchain.prompts.example_selector import SemanticSimilarityExampleSelector
from langchain.vectorstores import Chroma
from langchain.embeddings import OpenAIEmbeddings
from langchain.prompts import FewShotPromptTemplate, PromptTemplate
from langchain_openai import ChatOpenAI
import os
from dotenv import load_dotenv

# 加载环境变量
load_dotenv()

# ===== 1. 准备示例库 =====
examples = [
    {"input": "2+2", "output": "4"},
    {"input": "2+3", "output": "5"},
    {"input": "What is the moon?", "output": "A celestial body"},
    {"input": "Write a poem", "output": "Roses are red..."}
]

# ===== 2. 创建语义相似度选择器 =====
# 这是最核心的一步！
selector = SemanticSimilarityExampleSelector.from_examples(
    examples=examples,                          # 示例库
    embeddings=OpenAIEmbeddings(),              # Embedding 模型
    vectorstore_cls=Chroma,                     # 向量存储（推荐 Chroma）
    k=2                                         # 选择 2 个最相关的示例
)

# ===== 3. 测试选择器 =====
# 用户问题："3+3 等于多少？"
selected = selector.select_examples({"input": "3+3"})
print("选择的示例：")
for example in selected:
    print(f"  Input: {example['input']}, Output: {example['output']}")

# 输出：
# 选择的示例：
#   Input: 2+2, Output: 4
#   Input: 2+3, Output: 5
# （自动选择了数学相关的示例，忽略了诗歌和月亮）
```

**运行输出：**
```
选择的示例：
  Input: 2+2, Output: 4
  Input: 2+3, Output: 5
```

**关键参数说明：**
- `examples`：示例库（字典列表）
- `embeddings`：Embedding 模型（推荐 `OpenAIEmbeddings()`）
- `vectorstore_cls`：向量存储类（推荐 `Chroma`，易用）
- `k`：选择的示例数量（推荐 3-5）

[来源: reference/fetch_example_selector_03.md | LangChain Cookbook]
[来源: reference/fetch_example_selector_01.md | Medium 教程]

### 4.2 核心知识点2：与 FewShotPromptTemplate 集成

**一句话：** 将 ExampleSelector 集成到 Prompt 中，实现动态 Few-shot learning

**为什么重要？**
- ExampleSelector 的主要使用方式
- 与 LangChain 生态无缝集成
- 支持 LCEL 链式调用

**完整示例：**

```python
from langchain.prompts import FewShotPromptTemplate, PromptTemplate
from langchain_openai import ChatOpenAI

# ===== 1. 定义示例模板 =====
# 定义每个示例的格式
example_prompt = PromptTemplate(
    input_variables=["input", "output"],
    template="Input: {input}\nOutput: {output}"
)

# ===== 2. 创建 FewShotPromptTemplate =====
# 使用之前创建的 selector
few_shot_prompt = FewShotPromptTemplate(
    example_selector=selector,              # 使用动态选择器
    example_prompt=example_prompt,          # 示例格式
    prefix="Solve the following problems:", # 前缀（指令）
    suffix="Input: {input}\nOutput:",       # 后缀（用户输入）
    input_variables=["input"]               # 输入变量
)

# ===== 3. 测试 Prompt =====
print(few_shot_prompt.format(input="5+5"))

# 输出：
# Solve the following problems:
# Input: 2+2
# Output: 4
# Input: 2+3
# Output: 5
# Input: 5+5
# Output:

# ===== 4. 与 LLM 集成 =====
llm = ChatOpenAI(model="gpt-4o-mini", temperature=0)
chain = few_shot_prompt | llm

result = chain.invoke({"input": "5+5"})
print(f"\n最终结果：{result.content}")
# 输出：10
```

**运行输出：**
```
Solve the following problems:
Input: 2+2
Output: 4
Input: 2+3
Output: 5
Input: 5+5
Output:

最终结果：10
```

**关键点：**
- `example_selector`：使用动态选择器（而非固定的 `examples`）
- `example_prompt`：定义示例的格式
- `prefix`：Prompt 的前缀（指令部分）
- `suffix`：Prompt 的后缀（用户输入部分）

### 4.3 核心知识点3：k 参数调优

**一句话：** k 值决定选择多少个示例，直接影响效果和成本

**为什么重要？**
- k 值过小：示例太少，模型理解不足
- k 值过大：引入噪音，增加成本
- k=3-5 是 2025-2026 年的共识

**最佳实践：**

```python
# ❌ 错误：k=1（示例太少）
selector_bad = SemanticSimilarityExampleSelector.from_examples(
    examples=examples,
    embeddings=OpenAIEmbeddings(),
    vectorstore_cls=Chroma,
    k=1  # 只选择 1 个示例，效果差
)

# ❌ 错误：k=10（示例太多）
selector_bad = SemanticSimilarityExampleSelector.from_examples(
    examples=examples,
    embeddings=OpenAIEmbeddings(),
    vectorstore_cls=Chroma,
    k=10  # 选择 10 个示例，引入噪音，增加成本
)

# ✅ 推荐：k=3-5（平衡效果和成本）
selector_good = SemanticSimilarityExampleSelector.from_examples(
    examples=examples,
    embeddings=OpenAIEmbeddings(),
    vectorstore_cls=Chroma,
    k=3  # 选择 3 个示例，效果好，成本低
)
```

**k 值选择指南：**

| k 值 | 适用场景 | 优点 | 缺点 |
|------|----------|------|------|
| k=1-2 | 简单任务、示例库小 | 成本低 | 效果可能不佳 |
| k=3-5 | 大多数场景（推荐） | 平衡效果和成本 | - |
| k=6-10 | 复杂任务、需要更多上下文 | 效果可能更好 | 成本高、可能引入噪音 |

### 4.4 核心知识点4：LengthBasedExampleSelector（Token 控制）

**一句话：** 基于长度限制选择示例，避免超过 Token 限制

**为什么重要？**
- 控制 Prompt 长度，避免超过模型的 context window
- 在 Token 限制严格的场景下必须使用

**基础用法：**

```python
from langchain.prompts.example_selector import LengthBasedExampleSelector

# ===== 1. 准备示例 =====
examples = [
    {"input": "2+2", "output": "4"},
    {"input": "2+3", "output": "5"},
    {"input": "What is the moon?", "output": "A celestial body orbiting Earth"},
    {"input": "Write a poem", "output": "Roses are red, violets are blue..."}
]

# ===== 2. 定义示例模板 =====
example_prompt = PromptTemplate(
    input_variables=["input", "output"],
    template="Input: {input}\nOutput: {output}"
)

# ===== 3. 创建长度选择器 =====
selector = LengthBasedExampleSelector(
    examples=examples,
    example_prompt=example_prompt,
    max_length=100  # 限制总长度为 100 个字符
)

# ===== 4. 测试选择器 =====
# 短输入：可以选择更多示例
selected = selector.select_examples({"input": "1+1"})
print(f"短输入选择了 {len(selected)} 个示例")

# 长输入：只能选择少量示例
selected = selector.select_examples({"input": "This is a very long input that takes up a lot of space"})
print(f"长输入选择了 {len(selected)} 个示例")
```

**运行输出：**
```
短输入选择了 3 个示例
长输入选择了 1 个示例
```

**关键参数：**
- `max_length`：最大长度限制（默认按单词数计算）
- `get_text_length`：自定义长度计算函数（可选）

[来源: reference/fetch_example_selector_02.md | Pinecone 教程]

### 4.5 核心知识点5：常见错误与解决方案

**错误1：忘记设置 API Key**

```python
# ❌ 错误：没有设置 OPENAI_API_KEY
selector = SemanticSimilarityExampleSelector.from_examples(
    examples=examples,
    embeddings=OpenAIEmbeddings(),  # 需要 API Key
    vectorstore_cls=Chroma,
    k=3
)
# 报错：openai.error.AuthenticationError

# ✅ 解决方案：设置环境变量
import os
from dotenv import load_dotenv

load_dotenv()  # 从 .env 文件加载
# 或者直接设置
os.environ["OPENAI_API_KEY"] = "your-api-key"
```

**错误2：示例格式不一致**

```python
# ❌ 错误：示例格式不一致
examples = [
    {"input": "2+2", "output": "4"},
    {"question": "2+3", "answer": "5"},  # 键名不一致
]

# ✅ 解决方案：统一示例格式
examples = [
    {"input": "2+2", "output": "4"},
    {"input": "2+3", "output": "5"},  # 键名一致
]
```

**错误3：k 值超过示例数量**

```python
# ❌ 错误：k=10 但只有 3 个示例
examples = [
    {"input": "2+2", "output": "4"},
    {"input": "2+3", "output": "5"},
    {"input": "2+4", "output": "6"}
]

selector = SemanticSimilarityExampleSelector.from_examples(
    examples=examples,
    embeddings=OpenAIEmbeddings(),
    vectorstore_cls=Chroma,
    k=10  # k > len(examples)
)
# 结果：只会返回 3 个示例（不会报错）

# ✅ 解决方案：确保 k <= len(examples)
selector = SemanticSimilarityExampleSelector.from_examples(
    examples=examples,
    embeddings=OpenAIEmbeddings(),
    vectorstore_cls=Chroma,
    k=3  # k <= len(examples)
)
```

**错误4：没有安装依赖**

```bash
# ❌ 错误：没有安装 chromadb
# 报错：ModuleNotFoundError: No module named 'chromadb'

# ✅ 解决方案：安装依赖
pip install chromadb
# 或使用 uv
uv add chromadb
```

## 完整可运行示例

**场景：** 构建一个数学问答系统，根据用户问题动态选择相关的数学示例

```python
"""
ExampleSelector 完整示例：数学问答系统
演示：根据用户问题动态选择最相关的数学示例
"""

import os
from dotenv import load_dotenv
from langchain.prompts.example_selector import SemanticSimilarityExampleSelector
from langchain.vectorstores import Chroma
from langchain.embeddings import OpenAIEmbeddings
from langchain.prompts import FewShotPromptTemplate, PromptTemplate
from langchain_openai import ChatOpenAI

# ===== 1. 环境配置 =====
load_dotenv()

# ===== 2. 准备示例库 =====
print("=== 准备示例库 ===")
math_examples = [
    {"input": "2+2", "output": "4"},
    {"input": "5*3", "output": "15"},
    {"input": "10/2", "output": "5"},
    {"input": "What is 2 to the power of 3?", "output": "8"},
    {"input": "What is the square root of 16?", "output": "4"},
    {"input": "What is 15% of 100?", "output": "15"},
]

print(f"示例库包含 {len(math_examples)} 个数学示例")

# ===== 3. 创建语义相似度选择器 =====
print("\n=== 创建语义相似度选择器 ===")
selector = SemanticSimilarityExampleSelector.from_examples(
    examples=math_examples,
    embeddings=OpenAIEmbeddings(model="text-embedding-3-small"),
    vectorstore_cls=Chroma,
    k=3  # 选择 3 个最相关的示例
)
print("选择器创建成功")

# ===== 4. 定义示例模板 =====
example_prompt = PromptTemplate(
    input_variables=["input", "output"],
    template="Q: {input}\nA: {output}"
)

# ===== 5. 创建 FewShotPromptTemplate =====
few_shot_prompt = FewShotPromptTemplate(
    example_selector=selector,
    example_prompt=example_prompt,
    prefix="You are a math expert. Answer the following math questions:",
    suffix="Q: {input}\nA:",
    input_variables=["input"]
)

# ===== 6. 创建 LLM 链 =====
llm = ChatOpenAI(model="gpt-4o-mini", temperature=0)
chain = few_shot_prompt | llm

# ===== 7. 测试不同类型的问题 =====
print("\n=== 测试问题 1：加法 ===")
question1 = "What is 7+8?"
print(f"问题：{question1}")
result1 = chain.invoke({"input": question1})
print(f"答案：{result1.content}")

print("\n=== 测试问题 2：乘法 ===")
question2 = "What is 6*7?"
print(f"问题：{question2}")
result2 = chain.invoke({"input": question2})
print(f"答案：{result2.content}")

print("\n=== 测试问题 3：百分比 ===")
question3 = "What is 20% of 50?"
print(f"问题：{question3}")
result3 = chain.invoke({"input": question3})
print(f"答案：{result3.content}")

# ===== 8. 查看选择的示例 =====
print("\n=== 查看为问题 1 选择的示例 ===")
selected = selector.select_examples({"input": question1})
for i, example in enumerate(selected, 1):
    print(f"{i}. Q: {example['input']}, A: {example['output']}")
```

**运行输出：**
```
=== 准备示例库 ===
示例库包含 6 个数学示例

=== 创建语义相似度选择器 ===
选择器创建成功

=== 测试问题 1：加法 ===
问题：What is 7+8?
答案：15

=== 测试问题 2：乘法 ===
问题：What is 6*7?
答案：42

=== 测试问题 3：百分比 ===
问题：What is 20% of 50?
答案：10

=== 查看为问题 1 选择的示例 ===
1. Q: 2+2, A: 4
2. Q: 10/2, A: 5
3. Q: 5*3, A: 15
```
