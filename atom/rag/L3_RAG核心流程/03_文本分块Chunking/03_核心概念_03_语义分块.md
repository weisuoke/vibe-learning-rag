# 核心概念3：语义分块（Semantic Chunking）

**基于语义相似度切分，在语义变化的地方切分，保证每个块内容语义一致。**

---

## 一句话定义

**语义分块是通过计算文本片段的 embedding 相似度来确定分块边界的方法，在语义发生显著变化的地方切分，确保每个块内容主题一致，是质量最高但成本也最高的分块策略。**

---

## 核心原理

### 基本思想

```
传统分块：按固定规则切分
→ 可能在主题中间切断

语义分块：按语义相似度切分
→ 在主题转换点切分

示例：
"Python是一种编程语言。它简单易学。"  ← 相似度高（0.85）
                                      ↓ 不切分
"机器学习是AI的分支。深度学习很流行。" ← 相似度高（0.82）
                                      ↓ 切分！（相似度低 0.45）
"今天天气很好。我们去公园玩吧。"     ← 主题变化
```

### 算法流程

```
1. 按句子切分文本
   ↓
2. 计算每个句子的 embedding
   ↓
3. 计算相邻句子的余弦相似度
   ↓
4. 在相似度低于阈值的地方切分
   ↓
5. 合并相邻的高相似度句子
   ↓
6. 返回语义一致的块
```

## Python 实现

### 基础实现

```python
import numpy as np
from typing import List, Callable
import re

def semantic_chunk(
    text: str,
    embed_func: Callable[[str], List[float]],
    threshold: float = 0.5,
    min_chunk_size: int = 100
) -> List[str]:
    """
    语义分块

    原理：
    1. 先按句子切分
    2. 计算相邻句子的语义相似度
    3. 在相似度低于阈值的地方切分

    Args:
        text: 原始文本
        embed_func: 向量化函数
        threshold: 相似度阈值，低于此值则切分
        min_chunk_size: 最小块大小
    """
    # 1. 按句子切分
    sentences = re.split(r'(?<=[。！？.!?])', text)
    sentences = [s.strip() for s in sentences if s.strip()]

    if len(sentences) <= 1:
        return [text]

    # 2. 计算每个句子的向量
    embeddings = [embed_func(s) for s in sentences]

    # 3. 计算相邻句子的相似度
    def cosine_similarity(a, b):
        return np.dot(a, b) / (np.linalg.norm(a) * np.linalg.norm(b))

    similarities = []
    for i in range(len(embeddings) - 1):
        sim = cosine_similarity(embeddings[i], embeddings[i + 1])
        similarities.append(sim)

    # 4. 在相似度低的地方切分
    chunks = []
    current_chunk = sentences[0]

    for i, sim in enumerate(similarities):
        if sim < threshold and len(current_chunk) >= min_chunk_size:
            # 相似度低，切分
            chunks.append(current_chunk)
            current_chunk = sentences[i + 1]
        else:
            # 相似度高，合并
            current_chunk += " " + sentences[i + 1]

    if current_chunk:
        chunks.append(current_chunk)

    return chunks

# 使用示例
from openai import OpenAI

client = OpenAI()

def get_embedding(text: str) -> List[float]:
    """获取文本的 embedding"""
    response = client.embeddings.create(
        model="text-embedding-3-small",
        input=text
    )
    return response.data[0].embedding

text = """
Python是一种编程语言。它简单易学。Python支持多种编程范式。

机器学习是AI的分支。深度学习很流行。神经网络是核心技术。

今天天气很好。我们去公园玩吧。
"""

chunks = semantic_chunk(text, get_embedding, threshold=0.6)
for i, chunk in enumerate(chunks):
    print(f"Chunk {i+1}: {chunk}")
```

### 优化2：Max-Min Semantic Chunking

**核心思想**: 先 embed 整个文档，然后基于相似度矩阵确定最优切分点。

```python
import numpy as np
from typing import List

def max_min_semantic_chunk(
    text: str,
    embed_func: Callable,
    target_chunk_size: int = 512
) -> List[str]:
    """
    Max-Min 语义分块

    原理：
    1. 先 embed 整个文档的所有句子
    2. 构建相似度矩阵
    3. 找到局部最小相似度点作为切分边界
    """
    # 1. 按句子切分
    sentences = re.split(r'(?<=[。！？.!?])', text)
    sentences = [s.strip() for s in sentences if s.strip()]

    # 2. 计算所有句子的 embedding
    embeddings = np.array([embed_func(s) for s in sentences])

    # 3. 构建相似度矩阵
    n = len(embeddings)
    similarity_matrix = np.zeros((n, n))
    for i in range(n):
        for j in range(i+1, n):
            sim = np.dot(embeddings[i], embeddings[j]) / (
                np.linalg.norm(embeddings[i]) * np.linalg.norm(embeddings[j])
            )
            similarity_matrix[i][j] = sim
            similarity_matrix[j][i] = sim

    # 4. 找到局部最小相似度点
    breakpoints = [0]
    current_size = 0

    for i in range(1, n-1):
        current_size += len(sentences[i])

        # 检查是否是局部最小值
        if (similarity_matrix[i-1][i] < similarity_matrix[i][i+1] and
            current_size >= target_chunk_size * 0.5):
            breakpoints.append(i)
            current_size = 0

    breakpoints.append(n)

    # 5. 根据断点切分
    chunks = []
    for i in range(len(breakpoints)-1):
        start = breakpoints[i]
        end = breakpoints[i+1]
        chunk = " ".join(sentences[start:end])
        chunks.append(chunk)

    return chunks
```

### 优化3：Growing Window Strategy

**核心思想**: 使用滑动窗口计算相似度，提升边界检测准确性。

```python
def growing_window_semantic_chunk(
    text: str,
    embed_func: Callable,
    window_size: int = 3,
    threshold: float = 0.6
) -> List[str]:
    """
    Growing Window 语义分块

    原理：
    1. 使用滑动窗口（如3个句子）计算平均相似度
    2. 在窗口相似度低于阈值时切分
    3. 比单句相似度更稳定
    """
    sentences = re.split(r'(?<=[。！？.!?])', text)
    sentences = [s.strip() for s in sentences if s.strip()]

    if len(sentences) <= window_size:
        return [text]

    # 计算所有句子的 embedding
    embeddings = [embed_func(s) for s in sentences]

    # 计算窗口相似度
    window_similarities = []
    for i in range(len(embeddings) - window_size):
        window_emb = embeddings[i:i+window_size]
        # 计算窗口内的平均相似度
        sims = []
        for j in range(len(window_emb)-1):
            sim = np.dot(window_emb[j], window_emb[j+1]) / (
                np.linalg.norm(window_emb[j]) * np.linalg.norm(window_emb[j+1])
            )
            sims.append(sim)
        window_similarities.append(np.mean(sims))

    # 在相似度低的地方切分
    chunks = []
    current_chunk = " ".join(sentences[:window_size])

    for i, sim in enumerate(window_similarities):
        if sim < threshold:
            chunks.append(current_chunk)
            current_chunk = sentences[i+window_size]
        else:
            current_chunk += " " + sentences[i+window_size]

    if current_chunk:
        chunks.append(current_chunk)

    return chunks
```

**研究来源**: Growing Window Strategy 在某些数据集上比传统方法提升 4% 准确率。

## 相似度阈值的影响

### 阈值选择指南

| 阈值 | 效果 | 适用场景 |
|------|------|---------|
| **0.8-0.9（高）** | 切分点多，块更小，语义更纯粹 | 主题频繁变化的文档 |
| **0.6-0.7（中）** | 平衡切分，适中块大小 | 通用场景（推荐） |
| **0.3-0.5（低）** | 切分点少，块更大，可能混合主题 | 主题连贯的长文档 |

### 阈值对比实验

```python
def compare_thresholds(text: str, embed_func: Callable):
    """对比不同阈值的效果"""
    thresholds = [0.3, 0.5, 0.7, 0.9]

    for threshold in thresholds:
        chunks = semantic_chunk(text, embed_func, threshold=threshold)
        avg_size = sum(len(c) for c in chunks) // len(chunks)

        print(f"\n阈值 {threshold}:")
        print(f"  块数: {len(chunks)}")
        print(f"  平均大小: {avg_size} 字符")
        print(f"  第一块: {chunks[0][:100]}...")
```

## 优缺点分析

### 优点

| 优点 | 说明 | 适用场景 |
|------|------|---------|
| ✅ **语义完整性最好** | 在主题转换点切分，每个块主题一致 | 高质量要求 |
| ✅ **适合主题多变文档** | 自动识别主题边界 | 新闻、百科 |
| ✅ **检索准确率高** | 块内容语义纯粹，匹配更精准 | 问答系统 |
| ✅ **无需手动调参** | 自动确定边界，无需调整分隔符 | 多种文档类型 |

### 缺点

| 缺点 | 说明 | 影响 |
|------|------|------|
| ❌ **需要调用 Embedding API** | 每个句子都需要 embedding | 成本高 |
| ❌ **处理速度慢** | API 调用耗时 | 不适合实时 |
| ❌ **块大小不可控** | 由语义决定，无法精确控制 | 可能过大或过小 |
| ❌ **成本较高** | 大量 API 调用 | 预算有限不适用 |

### 实践1：批量处理优化

```python
from typing import List
import asyncio

async def batch_embed(texts: List[str], batch_size: int = 100) -> List[List[float]]:
    """批量获取 embedding（降低成本）"""
    embeddings = []

    for i in range(0, len(texts), batch_size):
        batch = texts[i:i+batch_size]
        response = await client.embeddings.create(
            model="text-embedding-3-small",
            input=batch
        )
        embeddings.extend([d.embedding for d in response.data])

    return embeddings

# 使用
sentences = ["句子1", "句子2", ...]
embeddings = await batch_embed(sentences)
```

### 实践2：缓存 Embedding

```python
import hashlib
import json
from pathlib import Path

class EmbeddingCache:
    """Embedding 缓存（避免重复计算）"""

    def __init__(self, cache_dir: str = ".embedding_cache"):
        self.cache_dir = Path(cache_dir)
        self.cache_dir.mkdir(exist_ok=True)

    def get_cache_key(self, text: str) -> str:
        """生成缓存键"""
        return hashlib.md5(text.encode()).hexdigest()

    def get(self, text: str) -> List[float]:
        """获取缓存的 embedding"""
        key = self.get_cache_key(text)
        cache_file = self.cache_dir / f"{key}.json"

        if cache_file.exists():
            with open(cache_file) as f:
                return json.load(f)
        return None

    def set(self, text: str, embedding: List[float]):
        """保存 embedding 到缓存"""
        key = self.get_cache_key(text)
        cache_file = self.cache_dir / f"{key}.json"

        with open(cache_file, 'w') as f:
            json.dump(embedding, f)

# 使用
cache = EmbeddingCache()

def get_embedding_cached(text: str) -> List[float]:
    """带缓存的 embedding 获取"""
    cached = cache.get(text)
    if cached:
        return cached

    embedding = get_embedding(text)
    cache.set(text, embedding)
    return embedding
```

### 实践3：混合策略（成本优化）

def hybrid_chunking(text: str, use_semantic: bool = False) -> List[str]:
    """
    混合分块策略：
    - 重要文档：使用语义分块
    - 普通文档：使用递归分块
    """
    if use_semantic:
        # 高质量：语义分块
        return semantic_chunk(text, get_embedding)
    else:
        # 成本优先：递归分块
        from langchain_text_splitters import RecursiveCharacterTextSplitter
        splitter = RecursiveCharacterTextSplitter(
            chunk_size=512,
            chunk_overlap=77
        )
        return splitter.split_text(text)
