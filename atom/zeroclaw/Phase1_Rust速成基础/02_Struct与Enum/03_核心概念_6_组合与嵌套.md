# 核心概念 6：组合与嵌套

> **一句话定位：** Rust 没有继承，用组合构建复杂类型 -- Struct 嵌套 Struct、Option\<T\> 表达可选、Vec\<T\> 表达集合、Box\<T\> 放堆上、Arc\<T\> 跨线程共享，这些"积木"拼出 ZeroClaw 整个 Agent 架构。

---

## 1. Struct 组合（Composition over Inheritance）

### 1.1 Rust 没有继承，只有组合

```typescript
// TypeScript: 继承链
class BaseAgent { name: string; }
class ChatAgent extends BaseAgent { model: string; }
class ToolAgent extends ChatAgent { tools: string[]; }
// 改 BaseAgent 可能炸掉所有子类
```

```rust
// Rust: 组合（Agent "拥有" SecurityPolicy，不是 "是" SecurityPolicy）
struct SecurityPolicy { autonomy_level: AutonomyLevel, max_actions: u32 }
struct Agent {
    name: String,
    security: SecurityPolicy,  // 组合，不是继承
}
```

### 1.2 ZeroClaw 的 Agent 组合结构

```rust
// ZeroClaw src/agent/agent.rs（简化）
pub struct Agent {
    name: String,
    security_policy: SecurityPolicy,        // 安全策略（值类型）
    identity: AieosIdentity,                // 身份标识（深度嵌套）
    provider: Box<dyn Provider>,            // LLM 提供者（多态）
    memory: Arc<dyn Memory>,                // 记忆系统（线程安全共享）
    tools: Vec<Box<dyn Tool>>,              // 工具列表
    conversation: Vec<ConversationMessage>,  // 对话历史
}
```

> **前端类比：** React 组合 `<App><Header /><Sidebar /><Content /></App>`，不用继承。

> **日常类比：** 乐高积木 -- 小零件拼成大模型，每个零件独立可替换。

## 2. Option\<T\> 可选字段

### 2.1 TypeScript 的 `?` vs Rust 的 Option\<T\>

```typescript
// TypeScript: 可选字段（可能忘记检查 undefined）
interface AgentConfig { name: string; description?: string; maxRetries?: number; }
```

```rust
// Rust: Option<T>（编译器强制你处理 None）
struct AgentConfig {
    name: String,                    // 必填
    description: Option<String>,     // Some("...") 或 None
    max_retries: Option<u32>,        // Some(3) 或 None
}
```

### 2.2 常用处理方法

```rust
let desc: Option<String> = Some("A helpful bot".into());
let retries: Option<u32> = None;

retries.unwrap_or(3);                    // None -> 3（提供默认值）
desc.as_ref().map(|d| d.to_uppercase()); // Some("A HELPFUL BOT")
desc.as_ref().and_then(|d| d.split_whitespace().next().map(String::from)); // Some("A")
desc.as_deref();                         // Option<String> -> Option<&str>（常用！）
desc.clone().unwrap_or_default();        // None -> ""（类型默认值）

// ok_or: Option -> Result（Builder 模式核心！）
let name: Option<String> = None;
name.ok_or("name is required");          // Err("name is required")
```

### 2.3 ZeroClaw Builder 模式中的 Option

```rust
// AgentBuilder: 所有字段都是 Option<T>，build() 时验证必填项
struct AgentBuilder {
    provider: Option<Box<dyn Provider>>,
    memory: Option<Arc<dyn Memory>>,
    tools: Option<Vec<Box<dyn Tool>>>,
}

impl AgentBuilder {
    fn build(self) -> Result<Agent, String> {
        Ok(Agent {
            provider: self.provider.ok_or("provider required")?,  // 必填
            memory: self.memory.ok_or("memory required")?,        // 必填
            tools: self.tools.unwrap_or_default(),                // 可选，默认空 Vec
        })
    }
}
```

## 3. Vec\<T\> 集合字段

### 3.1 Vec\<T\> = TypeScript 的 Array\<T\>

```typescript
// TypeScript
interface Agent {
  tools: Tool[];
  conversation: ConversationMessage[];
}
```

```rust
// Rust
struct Agent {
    tools: Vec<Box<dyn Tool>>,              // 工具列表（多态）
    conversation: Vec<ConversationMessage>,  // 对话历史
}
```

### 3.2 ZeroClaw 中的 Vec 使用

```rust
let mut tools: Vec<String> = vec!["search".into(), "write".into()];

// 常用操作
tools.push("deploy".into());                    // 追加
println!("count: {}", tools.len());              // 长度: 3
println!("empty: {}", tools.is_empty());         // false
tools.pop();                                     // 移除最后一个 -> Some("deploy")

// 遍历
for tool in &tools { println!("{}", tool); }

// 迭代器链（Rust 的强项）
let long_names: Vec<&String> = tools.iter()
    .filter(|t| t.len() > 4)
    .collect();
```

## 5. Box\<T\> 堆分配与大变体优化

### 5.1 基本概念

`Box<T>` 把数据放到堆上，栈上只存一个 8 字节指针。TypeScript 中所有对象天然在堆上；Rust 默认放栈上，需要堆分配时显式用 `Box`：

```rust
let level = AutonomyLevel::Full;                        // 栈上，1 字节
let boxed: Box<AutonomyLevel> = Box::new(AutonomyLevel::Full); // 堆上
println!("{:?}", *boxed);  // * 解引用访问
```

### 5.2 Box\<dyn Trait\> 多态 -- 最常见用途

```rust
// ZeroClaw: Box<dyn Provider> = "实现了 Provider 的某个类型，编译时不知道具体是哪个"
struct Agent {
    provider: Box<dyn Provider>,  // OpenAI、Anthropic、本地模型...
    tools: Vec<Box<dyn Tool>>,    // 每个工具可能是不同类型
}
```

```typescript
// TypeScript 等价（天然多态，不需要 Box）
class Agent { provider: Provider; tools: Tool[]; }
// TS 所有对象默认在堆上；Rust 需要 Box 显式堆分配
```

### 5.3 大变体优化与递归类型

```rust
// 问题：enum 大小 = 最大变体。大变体浪费所有小变体的内存
enum Bad { Click, BigPayload([u8; 1024]) }  // ~1032 bytes 每个值

// 解决：Box 把大数据放堆上
enum Good { Click, BigPayload(Box<[u8; 1024]>) }  // ~16 bytes 每个值

// 递归类型也必须用 Box（否则编译器无法计算无限大小）
enum List {
    Cons(i32, Box<List>),  // 8 字节指针，不是无限嵌套
    Nil,
}
```

> **前端类比：** Box 就像 React.lazy() -- 用一个小引用按需加载大组件。

> **日常类比：** 快递柜取件码 -- 口袋里放小纸条（指针），包裹存在快递柜（堆）里。

## 6. Arc\<T\> 与线程安全共享

`Arc<T>`（Atomic Reference Counted）让多个所有者共享同一份数据，跨线程安全：

```rust
use std::sync::Arc;

struct Agent {
    memory: Arc<dyn Memory>,  // 多个组件共享同一个 Memory
}

let mem: Arc<dyn Memory> = Arc::new(InMemoryStore::new());
let mem_clone = Arc::clone(&mem);  // 引用计数 +1，不复制数据
// mem 和 mem_clone 指向同一份数据
```

```typescript
// TypeScript: 天然共享引用，不需要 Arc
const mem = new InMemoryStore();
const agent1 = { memory: mem };  // 同一个引用，GC 自动管理
```

| 特性 | Box\<T\> | Arc\<T\> |
|------|----------|----------|
| 所有者 | 唯一一个 | 多个 |
| 线程安全 | 单线程 | 跨线程 |
| ZeroClaw | `Box<dyn Provider>` | `Arc<dyn Memory>` |
| 何时用 | 独占所有权 | 需要共享 |

> 详细的 Arc 用法将在后续 "并发与异步" 知识点中展开。这里只需记住：`Arc<T>` = "这个数据要被多个地方共享"。

## 7. 深度嵌套实战

### 7.1 ZeroClaw AieosIdentity: 三层嵌套

```rust
struct AuthorInfo { name: String, contact: Option<String> }
struct CoreIdentity { name: String, version: String, author: AuthorInfo }
struct AieosIdentity { core: CoreIdentity, capabilities: Vec<String> }
struct Agent { name: String, identity: AieosIdentity }
```

### 7.2 访问与解构

```rust
// 链式访问（和 TypeScript 一样用 .）
println!("{}", agent.identity.core.author.name);

// 嵌套 Option 访问
if let Some(contact) = &agent.identity.core.author.contact {
    println!("contact: {}", contact);
}

// 便捷访问器方法，避免长链
impl Agent {
    fn author_name(&self) -> &str { &self.identity.core.author.name }
}

// 一次性解构多层嵌套
let AieosIdentity {
    core: CoreIdentity { name, author: AuthorInfo { contact, .. }, .. }, ..
} = &identity;
```

```typescript
// TypeScript 可选链（Rust 没有 ?. 语法）
console.log(agent.identity?.core?.author?.contact);
// Rust 需要用 if let / map / and_then 来处理嵌套 Option
```

## 8. Enum 内嵌 Struct / Struct 内嵌 Enum

### 8.1 AND 包含 OR：struct 中的 enum 字段

```rust
struct SecurityPolicy {
    autonomy_level: AutonomyLevel,  // OR：选一种级别
    max_actions: u32,               // AND：同时存在
}

fn check(agent: &Agent) {
    match &agent.security.autonomy_level {
        AutonomyLevel::ReadOnly => println!("Read-only"),
        AutonomyLevel::Supervised => println!("Need approval"),
        AutonomyLevel::Full => println!("Full autonomy"),
    }
}
```

### 8.2 OR 包含 AND：enum 变体携带 struct 数据

```rust
// ZeroClaw ObserverEvent
enum ObserverEvent {
    AgentStarted,
    ToolCallCompleted { tool_name: String, result: String, duration_ms: u64 },
    Error(String),
}

fn handle(event: &ObserverEvent) {
    match event {
        ObserverEvent::ToolCallCompleted { tool_name, duration_ms, .. } => {
            println!("'{}' completed in {}ms", tool_name, duration_ms);
        }
        ObserverEvent::Error(msg) => println!("Error: {}", msg),
        ObserverEvent::AgentStarted => println!("Started"),
    }
}
```

### 8.3 设计原则

| 关系 | 用什么 | 例子 |
|------|--------|------|
| AND 中有 OR | struct + enum 字段 | `SecurityPolicy { autonomy_level: AutonomyLevel }` |
| OR 中有 AND | enum 变体 + 命名字段 | `ObserverEvent::ToolCallCompleted { ... }` |
| OR 中有复杂 AND | enum 变体包装独立 struct | `ConversationMessage::Chat(ChatMessage)` |

> **记忆口诀：** Struct = AND（所有字段都要），Enum = OR（只能选一个）。自由组合，无限嵌套。

## 9. 完整可运行代码示例

```rust
use std::collections::HashMap;

// ===== 基础类型 =====
#[derive(Debug, Clone, Copy, PartialEq, Default)]
enum AutonomyLevel { ReadOnly, #[default] Supervised, Full }

#[derive(Debug, Clone)]
struct SecurityPolicy { autonomy_level: AutonomyLevel, max_actions: u32 }
impl Default for SecurityPolicy {
    fn default() -> Self { SecurityPolicy { autonomy_level: AutonomyLevel::default(), max_actions: 10 } }
}

// ===== 深度嵌套 =====
#[derive(Debug, Clone)]
struct AuthorInfo { name: String, contact: Option<String> }
#[derive(Debug, Clone)]
struct Identity { name: String, version: String, author: AuthorInfo }

// ===== 组合：Agent =====
#[derive(Debug)]
struct Agent {
    identity: Identity,
    security: SecurityPolicy,
    tools: Vec<String>,
    metadata: HashMap<String, String>,
    description: Option<String>,
}

// ===== Enum 内嵌 struct 数据 =====
#[derive(Debug)]
enum Event {
    Started { agent_name: String },
    ToolUsed { name: String, duration_ms: u64 },
    Error(String),
}

impl Event {
    fn severity(&self) -> &str {
        match self {
            Event::Error(_) => "ERROR",
            Event::ToolUsed { duration_ms, .. } if *duration_ms > 5000 => "WARN",
            _ => "INFO",
        }
    }
}

fn main() {
    // 1. 组合构建 Agent（深度嵌套 + Vec + HashMap + Option）
    let mut agent = Agent {
        identity: Identity {
            name: "zeroclaw".into(), version: "2.0".into(),
            author: AuthorInfo { name: "dev".into(), contact: Some("dev@example.com".into()) },
        },
        security: SecurityPolicy::default(),
        tools: vec!["shell".into(), "file_reader".into()],
        metadata: HashMap::new(),
        description: Some("Security AI agent".into()),
    };

    // 2. 访问嵌套字段
    println!("Name: {} v{}", agent.identity.name, agent.identity.version);
    println!("Author: {}", agent.identity.author.name);
    if let Some(contact) = &agent.identity.author.contact {
        println!("Contact: {}", contact);
    }

    // 3. Option 处理
    let desc = agent.description.as_deref().unwrap_or("(none)");
    println!("Description: {}", desc);

    // 4. Vec + HashMap
    println!("Tools ({}): {:?}", agent.tools.len(), agent.tools);
    agent.metadata.insert("env".into(), "prod".into());
    println!("Env: {}", agent.metadata.get("env").unwrap());

    // 5. Struct 内嵌 Enum
    match agent.security.autonomy_level {
        AutonomyLevel::Supervised => println!("Mode: supervised (default)"),
        other => println!("Mode: {:?}", other),
    }

    // 6. Enum 内嵌 Struct 数据
    let events = vec![
        Event::Started { agent_name: "zeroclaw".into() },
        Event::ToolUsed { name: "shell".into(), duration_ms: 150 },
        Event::ToolUsed { name: "deploy".into(), duration_ms: 8000 },
        Event::Error("Permission denied".into()),
    ];
    for event in &events {
        println!("[{}] {:?}", event.severity(), event);
    }
}
```

**编译运行：** `rustc main.rs && ./main`

**运行输出：**

```
Name: zeroclaw v2.0
Author: dev
Contact: dev@example.com
Description: Security AI agent
Tools (2): ["shell", "file_reader"]
Env: prod
Mode: supervised (default)
[INFO] Started { agent_name: "zeroclaw" }
[INFO] ToolUsed { name: "shell", duration_ms: 150 }
[WARN] ToolUsed { name: "deploy", duration_ms: 8000 }
[ERROR] Error("Permission denied")
```

## 速查卡

```
组合与嵌套速查：

Struct 组合（替代继承）：
  struct Agent { policy: SecurityPolicy }   Agent "有" SecurityPolicy
  agent.policy.autonomy_level               链式 . 访问

Option<T>（TS 的 field?: string）：
  .unwrap_or(default)                       提供默认值
  .ok_or("error")?                          Option -> Result（Builder 核心）
  .map(|v| transform(v))                    变换内部值
  .and_then(|v| might_none(v))              链式 Option（flatMap）
  .as_deref()                               Option<String> -> Option<&str>

Vec<T>（TS 的 Array<T>）：
  .push() / .pop() / .len() / .is_empty()   增 / 删 / 长度 / 判空
  .iter().filter().collect()                迭代器链

HashMap<K,V>（TS 的 Record<string,T>）：
  .insert(k, v) / .get(k)                  写 / 读

Box<T>（堆分配，三大用途）：
  Box<dyn Trait>                            Trait object 多态
  Box<BigData>                              大变体放堆上省内存
  Box<Recursive>                            递归类型打断无限大小

Arc<T>（线程安全共享）：
  Arc::clone(&x)                            引用计数 +1，不复制数据

嵌套：AND 含 OR -> struct { level: Enum }
       OR 含 AND -> enum { Variant { f1, f2 } }
```
