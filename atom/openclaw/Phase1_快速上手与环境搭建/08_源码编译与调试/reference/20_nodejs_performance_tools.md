# Node.js 性能分析工具

> 来源: Grok Web Search
> 查询时间: 2026-02-24

## 搜索结果

### 1. Clinic.js 官网 - Node.js 性能分析套件
**URL**: https://clinicjs.org/
**描述**: NearForm 开源 Node.js 性能剖析工具集,包含 Flame 火焰图、Doctor 等模块诊断性能瓶颈

### 2. Clinic.js Flame 官方文档
**URL**: https://clinicjs.org/flame/
**描述**: CPU 采样生成交互式火焰图,精准揭示 Node.js 代码热点路径和性能瓶颈

### 3. 0x - Node.js 单命令火焰图剖析
**URL**: https://github.com/davidmarkclements/0x
**描述**: 单命令生成交互火焰图,发现 Node.js 应用 CPU 热点和性能问题

### 4. 使用 Clinic 优化 Node.js 性能
**URL**: https://pavel-romanov.com/optimizing-nodejs-identifying-and-fixing-performance-problems-with-clinic
**描述**: 实战教程: Clinic Flame 解决 Node.js 高 CPU、内存泄漏和异步瓶颈

## 使用示例

### Clinic.js
```bash
# 安装
npm install -g clinic

# 生成火焰图
clinic flame -- node app.js

# 诊断性能
clinic doctor -- node app.js
```

### 0x
```bash
# 安装
npm install -g 0x

# 生成火焰图
0x node app.js
```
