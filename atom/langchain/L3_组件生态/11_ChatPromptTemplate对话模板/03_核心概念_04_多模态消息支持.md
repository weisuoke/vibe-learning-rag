# 核心概念 04：多模态消息支持

> **核心要点**：ChatPromptTemplate 支持文本+图像的多模态消息，通过 `_StringImageMessagePromptTemplate` 实现，可以在对话中同时处理文本和视觉信息，是构建多模态 RAG 和视觉 AI 应用的基础。

---

## 一、什么是多模态消息支持？

### 1.1 基本概念

**多模态消息**是指在一条消息中同时包含多种类型的内容（如文本、图像、音频等）。在 LangChain 的 ChatPromptTemplate 中，主要支持**文本+图像**的组合。

**为什么需要多模态？**
- GPT-4o、Claude 3 等现代 LLM 都支持视觉理解
- 实际应用中经常需要处理包含图表、截图、照片的文档
- 多模态 RAG 可以同时检索文本和视觉内容

### 1.2 架构设计

```
ChatPromptTemplate
    ↓
HumanMessagePromptTemplate (继承自 _StringImageMessagePromptTemplate)
    ↓
支持两种格式:
1. 纯文本: "Hello {name}"
2. 文本+图像列表: [{"text": "..."}, {"image_url": "..."}]
```

**关键类**：`_StringImageMessagePromptTemplate` (源码 lines 396-662)

## 二、多模态消息的三种格式

### 2.1 格式一：纯文本（传统方式）

```python
from langchain_core.prompts import ChatPromptTemplate

# 纯文本消息
template = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant."),
    ("human", "What is {topic}?")
])

# 使用
result = template.invoke({"topic": "machine learning"})
```

**特点**：
- 最简单的方式
- 向后兼容
- 不支持图像

### 2.2 格式二：文本+图像 URL

```python
from langchain_core.prompts import ChatPromptTemplate

# 文本+图像 URL
template = ChatPromptTemplate.from_messages([
    ("system", "You are a vision assistant."),
    ("human", [
        {"text": "Describe this image:"},
        {"image_url": "{image_path}"}
    ])
])

# 使用
result = template.invoke({
    "image_path": "https://example.com/image.jpg"
})
```

**特点**：
- 支持远程图像 URL
- 图像会被传递给支持视觉的 LLM
- 适合处理在线图片

### 2.3 格式三：文本+本地图像路径

```python
from langchain_core.prompts import ChatPromptTemplate

# 文本+本地图像
template = ChatPromptTemplate.from_messages([
    ("system", "You are a vision assistant."),
    ("human", [
        {"text": "What's in this image: {description}"},
        {"image_url": "file:///path/to/image.jpg"}
    ])
])

# 使用
result = template.invoke({
    "description": "a product photo"
})
```

**特点**：
- 支持本地文件路径
- 使用 `file://` 协议
- 适合处理本地图片

## 三、图像参数详解

### 3.1 detail 参数

控制图像处理的精细度，影响 token 消耗和处理速度。

```python
from langchain_core.prompts import ChatPromptTemplate

template = ChatPromptTemplate.from_messages([
    ("human", [
        {"text": "Analyze this image:"},
        {
            "image_url": "{image_path}",
            "detail": "high"  # low | high | auto
        }
    ])
])
```

**detail 参数说明**：

| 值 | 说明 | 适用场景 | Token 消耗 |
|---|---|---|---|
| `low` | 低分辨率处理 | 简单图像识别、快速预览 | 低 |
| `high` | 高分辨率处理 | 详细分析、OCR、图表理解 | 高 |
| `auto` | 自动选择 | 让模型自动决定 | 中等 |

### 3.2 多图像支持

一条消息中可以包含多张图像：

```python
template = ChatPromptTemplate.from_messages([
    ("human", [
        {"text": "Compare these two images:"},
        {"image_url": "{image1}"},
        {"image_url": "{image2}"},
        {"text": "What are the differences?"}
    ])
])

result = template.invoke({
    "image1": "https://example.com/before.jpg",
    "image2": "https://example.com/after.jpg"
})
```

### 4.2 场景二：文档图表分析（RAG 应用）

```python
"""
场景：分析 PDF 文档中的图表
"""
from langchain_core.prompts import ChatPromptTemplate
from langchain_openai import ChatOpenAI
import base64

def encode_image(image_path):
    """将本地图像编码为 base64"""
    with open(image_path, "rb") as image_file:
        return base64.b64encode(image_file.read()).decode('utf-8')

# ===== 1. 创建图表分析模板 =====
template = ChatPromptTemplate.from_messages([
    ("system", """You are a data analyst. Extract key insights from charts and graphs.
    Provide:
    1. Chart type
    2. Key trends
    3. Notable data points
    4. Conclusions"""),
    ("human", [
        {"text": "Analyze this chart from the document:"},
        {"image_url": "data:image/jpeg;base64,{image_data}", "detail": "high"},
        {"text": "\nContext: {context}"}
    ])
])

# ===== 2. 配置模型 =====
model = ChatOpenAI(model="gpt-4o", temperature=0)

# ===== 3. 创建链 =====
chain = template | model

# ===== 4. 处理文档中的图表 =====
image_data = encode_image("./charts/sales_chart.png")

result = chain.invoke({
    "image_data": image_data,
    "context": "Q4 2025 sales report"
})

print(result.content)
```

**输出示例**：
```

Key Trends:
- Steady growth from Q1 to Q3
- Sharp increase in Q4 (holiday season)
- Peak sales in December

Notable Data Points:
- Q4 sales: $2.5M (40% increase from Q3)

Conclusions:
The data indicates strong seasonal patterns with significant
holiday season impact...
```

### 4.3 场景三：多模态 RAG 系统

```python
"""
场景：构建支持图像的 RAG 问答系统
"""
from langchain_core.prompts import ChatPromptTemplate
from langchain_openai import ChatOpenAI
from langchain_core.output_parsers import StrOutputParser

# ===== 1. 创建多模态 RAG 模板 =====
template = ChatPromptTemplate.from_messages([
    ("system", """You are a helpful assistant with access to both text and visual information.
    Use the provided context and images to answer questions accurately."""),
    ("human", [
        {"text": "Context: {text_context}\n\n"},
        {"text": "Related image:"},
        {"image_url": "{image_url}", "detail": "auto"},
        {"text": "\n\nQuestion: {question}"}
    ])
])

# ===== 2. 配置模型和输出解析器 =====
model = ChatOpenAI(model="gpt-4o", temperature=0)
output_parser = StrOutputParser()

# ===== 3. 创建完整链 =====
chain = template | model | output_parser

# ===== 4. 模拟检索结果 =====
# 假设从向量数据库检索到相关文本和图像
retrieved_context = {
    "text_context": "The product features a 12MP camera with optical image stabilization...",
    "image_url": "https://example.com/product_camera.jpg",
    "question": "What are the camera specifications?"
}

# ===== 5. 执行查询 =====
answer = chain.invoke(retrieved_context)
print(answer)
```

### 4.4 场景四：视频帧分析

```python
"""
场景：分析视频关键帧（多模态视频 RAG）
"""
from langchain_core.prompts import ChatPromptTemplate
from langchain_openai import ChatOpenAI
from typing import List

# ===== 1. 创建视频帧分析模板 =====
def create_video_analysis_template(num_frames: int):
    """动态创建包含多个视频帧的模板"""
    messages = [
        ("system", "You are a video content analyst. Analyze the sequence of frames to understand the video content.")
    ]

    # 添加文本说明
    human_content = [{"text": "Analyze these video frames in sequence:\n\n"}]

    # 添加多个帧
    for i in range(num_frames):
        human_content.append({
            "text": f"Frame {i+1}:",
        })
        human_content.append({
            "image_url": f"{{frame_{i}}}",
            "detail": "low"  # 视频帧使用低分辨率
        })
        human_content.append({"text": "\n"})

    # 添加问题
    human_content.append({"text": "Question: {question}"})

    messages.append(("human", human_content))

    return ChatPromptTemplate.from_messages(messages)

# ===== 2. 使用模板 =====
template = create_video_analysis_template(num_frames=3)
model = ChatOpenAI(model="gpt-4o")
chain = template | model

# ===== 3. 分析视频帧 =====
result = chain.invoke({
    "frame_0": "https://example.com/video/frame_001.jpg",
    "frame_1": "https://example.com/video/frame_030.jpg",
    "frame_2": "https://example.com/video/frame_060.jpg",
    "question": "What is happening in this video?"
})

print(result.content)
```

## 五、与 RAG 系统集成

### 5.1 多模态文档索引

```python
"""
将文档中的文本和图像分别索引
"""
from langchain_community.document_loaders import PyPDFLoader
from langchain_text_splitters import RecursiveCharacterTextSplitter
from langchain_openai import OpenAIEmbeddings
from langchain_community.vectorstores import Chroma

# ===== 1. 加载文档并提取图像 =====
loader = PyPDFLoader("document.pdf")
pages = loader.load()

# ===== 2. 分离文本和图像引用 =====
text_chunks = []
image_references = []

for page in pages:
    # 文本分块
    text_splitter = RecursiveCharacterTextSplitter(
        chunk_size=1000,
        chunk_overlap=200
    )
    chunks = text_splitter.split_text(page.page_content)
    text_chunks.extend(chunks)

    # 提取图像引用（假设图像已保存）
    if hasattr(page, 'images'):
        for img in page.images:
            image_references.append({
                "page": page.metadata['page'],
                "image_path": img['path'],
                "context": page.page_content[:200]  # 周围文本
            })

# ===== 3. 创建向量存储 =====
embeddings = OpenAIEmbeddings()
vectorstore = Chroma.from_texts(
    texts=text_chunks,
    embedding=embeddings,
    metadatas=[{"type": "text"} for _ in text_chunks]
)

# ===== 4. 存储图像元数据 =====
# 可以使用单独的数据库或向量存储
```

### 5.2 多模态检索

```python
"""
根据查询检索相关文本和图像
"""
from langchain_core.prompts import ChatPromptTemplate
from langchain_openai import ChatOpenAI

def multimodal_rag_query(question: str, vectorstore, image_db):
    """多模态 RAG 查询"""

    # ===== 1. 检索相关文本 =====
    text_results = vectorstore.similarity_search(question, k=3)
    text_context = "\n\n".join([doc.page_content for doc in text_results])

    # ===== 2. 检索相关图像 =====
    # 简化示例：基于文本结果找到相关图像
    relevant_images = []
    for doc in text_results:
        page_num = doc.metadata.get('page')
        images = image_db.get(page_num, [])
        relevant_images.extend(images[:1])  # 每页取一张

    # ===== 3. 构建多模态提示 =====
    human_content = [
        {"text": f"Text Context:\n{text_context}\n\n"}
    ]

    # 添加图像
    for i, img in enumerate(relevant_images[:2]):  # 最多2张图
        human_content.append({"text": f"Related Image {i+1}:"})
        human_content.append({"image_url": img['path'], "detail": "auto"})
        human_content.append({"text": "\n"})

    human_content.append({"text": f"Question: {question}"})

    # ===== 4. 创建模板并执行 =====
    template = ChatPromptTemplate.from_messages([
        ("system", "Answer based on the provided text and images."),
        ("human", human_content)
    ])

    model = ChatOpenAI(model="gpt-4o")
    chain = template | model

    result = chain.invoke({})
    return result.content

# 使用
answer = multimodal_rag_query(
    "What are the key findings in the sales report?",
    vectorstore,
    image_db
)
```

## 六、最佳实践

### 6.1 图像大小优化

```python
"""
优化图像大小以减少 token 消耗
"""
from PIL import Image
import io
import base64

def optimize_image(image_path: str, max_size: tuple = (1024, 1024)) -> str:
    """
    优化图像大小并转换为 base64

    Args:
        image_path: 图像路径
        max_size: 最大尺寸 (width, height)

    Returns:
        base64 编码的图像数据
    """
    # 打开图像
    img = Image.open(image_path)

    # 调整大小（保持宽高比）
    img.thumbnail(max_size, Image.Resampling.LANCZOS)

    # 转换为 JPEG（压缩）
    buffer = io.BytesIO()
    img.convert('RGB').save(buffer, format='JPEG', quality=85)

    # 编码为 base64
    img_str = base64.b64encode(buffer.getvalue()).decode()

    return f"data:image/jpeg;base64,{img_str}"

# 使用
optimized_image = optimize_image("large_image.png")

template = ChatPromptTemplate.from_messages([
    ("human", [
        {"text": "Analyze this image:"},
        {"image_url": optimized_image, "detail": "high"}
    ])
])
```

### 6.2 批量图像处理

```python
"""
批量处理多张图像
"""
from langchain_core.prompts import ChatPromptTemplate
from langchain_openai import ChatOpenAI
from typing import List
import asyncio

async def process_images_batch(image_urls: List[str], prompt_text: str):
    """异步批量处理图像"""

    model = ChatOpenAI(model="gpt-4o")

    async def process_single(image_url: str):
        template = ChatPromptTemplate.from_messages([
            ("human", [
                {"text": prompt_text},
                {"image_url": image_url, "detail": "low"}
            ])
        ])
        chain = template | model
        result = await chain.ainvoke({})
        return result.content

    # 并发处理
    tasks = [process_single(url) for url in image_urls]
    results = await asyncio.gather(*tasks)

    return results

# 使用
image_urls = [
    "https://example.com/img1.jpg",
    "https://example.com/img2.jpg",
    "https://example.com/img3.jpg"
]

results = asyncio.run(process_images_batch(
    image_urls,
    "Describe this image in one sentence:"
))
```

### 6.3 错误处理

```python
"""
处理图像加载失败等错误
"""
from langchain_core.prompts import ChatPromptTemplate
from langchain_openai import ChatOpenAI
import requests

def safe_multimodal_invoke(template, model, inputs: dict):
    """安全的多模态调用，带错误处理"""

    # 验证图像 URL
    if 'image_url' in inputs:
        try:
            response = requests.head(inputs['image_url'], timeout=5)
            if response.status_code != 200:
                print(f"Warning: Image URL not accessible: {inputs['image_url']}")
                # 降级为纯文本模式
                return fallback_text_only(template, model, inputs)
        except Exception as e:
            print(f"Error checking image: {e}")
            return fallback_text_only(template, model, inputs)

    # 正常执行
    try:
        chain = template | model
        return chain.invoke(inputs)
    except Exception as e:
        print(f"Error in multimodal invoke: {e}")
        return fallback_text_only(template, model, inputs)

def fallback_text_only(template, model, inputs: dict):
    """降级为纯文本模式"""
    # 创建纯文本模板
    text_template = ChatPromptTemplate.from_messages([
        ("human", "{question}")
    ])
    chain = text_template | model
    return chain.invoke({"question": inputs.get('question', 'Please help.')})
```

## 七、常见问题与解决方案

### 7.1 问题：图像 token 消耗过高

**原因**：高分辨率图像会消耗大量 tokens

**解决方案**：
```python
# 1. 使用 detail="low" 参数
{"image_url": url, "detail": "low"}

# 2. 预处理图像，降低分辨率
optimized_image = optimize_image(image_path, max_size=(512, 512))

# 3. 只在必要时使用高分辨率
if requires_detailed_analysis:
    detail = "high"
else:
    detail = "low"
```

### 7.2 问题：本地图像路径不工作

**原因**：需要使用正确的 URL 格式

**解决方案**：
```python
# ❌ 错误
{"image_url": "/path/to/image.jpg"}

# ✅ 正确
{"image_url": "file:///path/to/image.jpg"}

# ✅ 或者转换为 base64
import base64
with open("/path/to/image.jpg", "rb") as f:
    img_data = base64.b64encode(f.read()).decode()
{"image_url": f"data:image/jpeg;base64,{img_data}"}
```

### 7.3 问题：模型不支持视觉

**原因**：使用了不支持视觉的模型

**解决方案**：
```python
# ❌ 不支持视觉
model = ChatOpenAI(model="gpt-3.5-turbo")

# ✅ 支持视觉
model = ChatOpenAI(model="gpt-4o")  # 或 gpt-4-vision-preview
```
