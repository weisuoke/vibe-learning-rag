# 00_概览

## 知识点概览

**BST（Binary Search Tree，二叉搜索树）** 是一种通过 **左 < 根 < 右** 的性质维护有序性的二叉树结构，支持 O(log n) 的动态查找、插入、删除操作，是 AI Agent 中时间序列检索、优先级任务调度、有序索引系统的核心数据结构。

## 为什么学习BST？

**1. 时间序列检索**
```python
# 查找某个时间范围内的文档

```

**2. 向量相似度排序**
```python
# Top-K 最相关文档
```

## 核心价值

### 三个关键优势

**1. 有序性维护**
- 中序遍历天然有序
- 支持前驱/后继查询
- 范围查询高效

**2. 动态操作**
- 插入/删除不需要移动元素
- O(log n)时间复杂度
- 适合频繁更新的场景

**3. 灵活扩展**
- 平衡树保证性能
- 学习增强适应访问模式
- 支持多种变体

### 与其他结构对比

| 结构 | 查找 | 插入 | 删除 | 有序性 | 范围查询 |
|------|------|------|------|--------|----------|
| 数组 | O(n) | O(n) | O(n) | ✅ | ✅ |
| 有序数组 | O(log n) | O(n) | O(n) | ✅ | ✅ |
| 链表 | O(n) | O(1) | O(1) | ❌ | ❌ |
| 哈希表 | O(1) | O(1) | O(1) | ❌ | ❌ |
| **BST** | **O(log n)** | **O(log n)** | **O(log n)** | **✅** | **✅** |

## 学习建议

### 学习顺序
1. **理解性质** → 为什么左<根<右能保证有序？
2. **掌握操作** → 查找、插入、删除的实现
3. **平衡维护** → AVL、红黑树的旋转
4. **实战应用** → 时间检索、任务调度
5. **前沿技术** → 学习增强、AI集成

### 学习重点
- **基础**：BST性质、中序遍历
- **核心**：三种操作的实现
- **进阶**：平衡树的旋转
- **应用**：AI Agent场景

### 常见陷阱
- ❌ 忽略平衡性，导致退化为链表
- ❌ 删除双子节点时逻辑错误
- ❌ 旋转操作的指针处理不当
- ❌ 范围查询时没有剪枝优化
