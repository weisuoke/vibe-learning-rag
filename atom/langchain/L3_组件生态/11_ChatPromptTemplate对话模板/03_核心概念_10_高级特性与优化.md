## 一、变量自动推断机制

### 1.1 input_variables 自动推断

ChatPromptTemplate 会自动从模板中提取所有变量，无需手动指定。

**源码实现** (reference/source_chatprompt_01.md):
```python
# 自动推断机制
# 1. 遍历所有消息模板
# 2. 提取每个模板中的变量
# 3. 合并为 input_variables 列表
```

**基础示例**:
```python
from langchain_core.prompts import ChatPromptTemplate

# 自动推断变量
template = ChatPromptTemplate.from_messages([
    ("system", "You are a {role} assistant."),
    ("human", "Tell me about {topic}.")
])

# 查看自动推断的变量
print(template.input_variables)  # ['role', 'topic']

# 使用时传递所有变量
messages = template.format_messages(
    role="Python",
    topic="list comprehension"
)
```

**优势**:
- 减少人为错误
- 自动同步模板变化
- 简化代码维护

### 1.2 变量推断的边界情况

**情况1：重复变量**
```python
from langchain_core.prompts import ChatPromptTemplate

# 同一变量在多个消息中使用
template = ChatPromptTemplate.from_messages([
    ("system", "You are a {role}."),
    ("human", "As a {role}, tell me about {topic}.")
])

# 自动去重
print(template.input_variables)  # ['role', 'topic']

# 使用时只需传递一次
messages = template.format_messages(role="expert", topic="AI")
```

**情况2：嵌套变量**
```python
from langchain_core.prompts import ChatPromptTemplate

# 复杂变量名
template = ChatPromptTemplate.from_messages([
    ("system", "User: {user_name}, ID: {user_id}"),
    ("human", "{user_input}")
])

print(template.input_variables)  # ['user_name', 'user_id', 'user_input']
```

## 二、optional_variables 处理

### 2.1 什么是 optional_variables

**定义**: 来自 `optional=True` 的 MessagesPlaceholder 的变量。

**源码实现** (reference/source_chatprompt_01.md):
```python
# MessagesPlaceholder with optional=True
# 自动添加到 optional_variables
# 在 format_messages 时可以不传递
```

### 2.2 基础用法

```python
from langchain_core.prompts import ChatPromptTemplate, MessagesPlaceholder

# 创建带可选历史的模板
template = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant."),
    MessagesPlaceholder("history", optional=True),  # 可选
    ("human", "{input}")
])

# 查看变量分类
print(template.input_variables)  # ['input']
print(template.optional_variables)  # ['history']

# 使用时可以不传递 history
messages1 = template.format_messages(input="Hello")

# 也可以传递 history
messages2 = template.format_messages(
    history=[("human", "Hi"), ("ai", "Hello!")],
    input="How are you?"
)
```

### 2.3 实际应用场景

**场景1：对话历史管理**
```python
from langchain_core.prompts import ChatPromptTemplate, MessagesPlaceholder
from langchain_openai import ChatOpenAI

# 创建灵活的对话模板
template = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant."),
    MessagesPlaceholder("history", optional=True),
    ("human", "{input}")
])

model = ChatOpenAI()
chain = template | model

# 第一轮对话（无历史）
response1 = chain.invoke({"input": "What is Python?"})

# 第二轮对话（有历史）
response2 = chain.invoke({
    "history": [
        ("human", "What is Python?"),
        ("ai", response1.content)
    ],
    "input": "What are its features?"
})
```

**场景2：Agent 工具调用历史**
```python
from langchain_core.prompts import ChatPromptTemplate, MessagesPlaceholder

# Agent 模板
agent_template = ChatPromptTemplate.from_messages([
    ("system", "You are an AI agent with tools: {tools}"),
    MessagesPlaceholder("history", optional=True),
    ("human", "{input}"),
    MessagesPlaceholder("agent_scratchpad", optional=True)  # 工具调用历史
])

# 初始调用（无历史和工具调用）
messages = agent_template.format_messages(
    tools="search, calculator",
    input="What is 2+2?"
)

# 后续调用（有工具调用历史）
messages = agent_template.format_messages(
    tools="search, calculator",
    history=[],
    input="Now multiply by 3",
    agent_scratchpad=[
        ("ai", "Using calculator: 2+2=4"),
        ("tool", "Result: 4")
    ]
)
```

### 2.4 optional_variables 的内部机制

```python
from langchain_core.prompts import ChatPromptTemplate, MessagesPlaceholder

template = ChatPromptTemplate.from_messages([
    ("system", "System"),
    MessagesPlaceholder("history", optional=True),
    ("human", "{input}")
])

# 内部机制：
# 1. optional=True 的 MessagesPlaceholder 自动添加到 optional_variables
# 2. 在 format_messages 时，如果未传递，使用空列表 []
# 3. 不会抛出 KeyError

# 验证
print(template.optional_variables)  # ['history']

# 不传递 history 不会报错
messages = template.format_messages(input="test")
```

## 三、性能优化技巧

### 3.1 模板缓存

**问题**: 重复创建模板浪费资源

**解决方案**: 使用缓存
```python
from functools import lru_cache
from langchain_core.prompts import ChatPromptTemplate

@lru_cache(maxsize=10)
def get_template(template_type: str) -> ChatPromptTemplate:
    """缓存常用模板"""
    if template_type == "qa":
        return ChatPromptTemplate.from_messages([
            ("system", "You are a helpful assistant."),
            ("human", "{question}")
        ])
    elif template_type == "translation":
        return ChatPromptTemplate.from_messages([
            ("system", "You are a translator."),
            ("human", "Translate '{text}' to {language}")
        ])
    else:
        raise ValueError(f"Unknown template: {template_type}")

# 第一次调用：创建模板
template1 = get_template("qa")

# 第二次调用：从缓存获取
template2 = get_template("qa")

# 验证是同一个对象
assert template1 is template2
```

### 3.2 消息数量限制

**问题**: 对话历史过长导致上下文窗口溢出

**解决方案**: 使用 n_messages 参数
```python
from langchain_core.prompts import ChatPromptTemplate, MessagesPlaceholder

# 限制历史消息数量
template = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant."),
    MessagesPlaceholder("history", n_messages=5),  # 只保留最后5条
    ("human", "{input}")
])

# 传入10条历史消息
long_history = [
    ("human", f"Question {i}"),
    ("ai", f"Answer {i}")
    for i in range(10)
]

# 只会使用最后5条消息
messages = template.format_messages(
    history=long_history,
    input="Current question"
)

print(len([m for m in messages if m.type in ["human", "ai"]]))  # 6 (5条历史 + 1条当前)
```

### 3.3 批量格式化优化

**问题**: 逐个格式化消息效率低

**解决方案**: 批量处理
```python
from langchain_core.prompts import ChatPromptTemplate

template = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant."),
    ("human", "{input}")
])

# 批量输入
inputs = [
    {"input": "Question 1"},
    {"input": "Question 2"},
    {"input": "Question 3"}
]

# 批量格式化
batch_messages = [template.format_messages(**inp) for inp in inputs]

# 或使用 batch 方法（如果支持）
# batch_messages = template.batch(inputs)
```

### 3.4 避免重复的 partial

**问题**: 多次 partial 同一变量

**解决方案**: 一次性 partial
```python
from langchain_core.prompts import ChatPromptTemplate

template = ChatPromptTemplate.from_messages([
    ("system", "Role: {role}, Language: {language}, Tone: {tone}"),
    ("human", "{input}")
])

# ❌ 低效：多次 partial
step1 = template.partial(role="assistant")
step2 = step1.partial(language="English")
step3 = step2.partial(tone="friendly")

# ✅ 高效：一次性 partial
optimized = template.partial(
    role="assistant",
    language="English",
    tone="friendly"
)
```

## 四、生产环境最佳实践

### 4.1 错误处理

**实践1：验证输入变量**
```python
from langchain_core.prompts import ChatPromptTemplate

template = ChatPromptTemplate.from_messages([
    ("system", "You are a {role}."),
    ("human", "{input}")
])

def safe_format(template, **kwargs):
    """安全的格式化函数"""
    # 检查必需变量
    required = set(template.input_variables)
    provided = set(kwargs.keys())

    missing = required - provided
    if missing:
        raise ValueError(f"Missing required variables: {missing}")

    # 检查多余变量
    extra = provided - required - set(template.optional_variables or [])
    if extra:
        print(f"Warning: Extra variables will be ignored: {extra}")

    return template.format_messages(**kwargs)

# 使用
try:
    messages = safe_format(template, role="assistant", input="Hello")
except ValueError as e:
    print(f"Error: {e}")
```

**实践2：处理 partial_variables 冲突**
```python
from langchain_core.prompts import ChatPromptTemplate

def safe_partial(template, **kwargs):
    """安全的 partial 函数"""
    # 检查是否与现有 partial_variables 冲突
    existing = set(template.partial_variables.keys())
    new = set(kwargs.keys())

    conflicts = existing & new
    if conflicts:
        print(f"Warning: Overwriting existing partial variables: {conflicts}")

    return template.partial(**kwargs)

# 使用
template = ChatPromptTemplate.from_messages([
    ("system", "Role: {role}"),
    ("human", "{input}")
]).partial(role="assistant")

# 尝试覆盖
new_template = safe_partial(template, role="expert")  # 会警告
```

### 4.2 模板版本管理

**实践：使用版本标识**
```python
from langchain_core.prompts import ChatPromptTemplate
from datetime import datetime

class VersionedTemplate:
    """带版本管理的模板"""

    def __init__(self, template: ChatPromptTemplate, version: str):
        self.template = template
        self.version = version
        self.created_at = datetime.now()

    def format_messages(self, **kwargs):
        """格式化消息并添加版本信息"""
        messages = self.template.format_messages(**kwargs)
        # 可以在元数据中添加版本信息
        return messages

    def __repr__(self):
        return f"VersionedTemplate(version={self.version}, created={self.created_at})"

# 使用
template_v1 = VersionedTemplate(
    ChatPromptTemplate.from_messages([
        ("system", "You are a helpful assistant."),
        ("human", "{input}")
    ]),
    version="1.0.0"
)

template_v2 = VersionedTemplate(
    ChatPromptTemplate.from_messages([
        ("system", "You are a helpful assistant. Be concise."),
        ("human", "{input}")
    ]),
    version="2.0.0"
)

# 根据需要选择版本
current_template = template_v2
messages = current_template.format_messages(input="Hello")
```

### 4.3 监控与日志

**实践：记录模板使用情况**
```python
from langchain_core.prompts import ChatPromptTemplate
import logging

logging.basicConfig(level=logging.INFO)
logger = logging.getLogger(__name__)

class MonitoredTemplate:
    """带监控的模板"""

    def __init__(self, template: ChatPromptTemplate, name: str):
        self.template = template
        self.name = name
        self.call_count = 0

    def format_messages(self, **kwargs):
        """格式化消息并记录日志"""
        self.call_count += 1

        logger.info(f"Template '{self.name}' called (count: {self.call_count})")
        logger.debug(f"Input variables: {list(kwargs.keys())}")

        try:
            messages = self.template.format_messages(**kwargs)
            logger.info(f"Template '{self.name}' formatted successfully")
            return messages
        except Exception as e:
            logger.error(f"Template '{self.name}' failed: {e}")
            raise

# 使用
monitored = MonitoredTemplate(
    ChatPromptTemplate.from_messages([
        ("system", "You are a {role}."),
        ("human", "{input}")
    ]),
    name="qa_template"
)

messages = monitored.format_messages(role="assistant", input="Hello")
```

### 4.4 配置管理

**实践：外部化模板配置**
```python
from langchain_core.prompts import ChatPromptTemplate
import json
from pathlib import Path

class TemplateManager:
    """模板管理器"""

    def __init__(self, config_path: str):
        self.config_path = Path(config_path)
        self.templates = {}
        self.load_templates()

    def load_templates(self):
        """从配置文件加载模板"""
        if not self.config_path.exists():
            return

        with open(self.config_path) as f:
            config = json.load(f)

        for name, template_config in config.items():
            messages = template_config["messages"]
            partial_vars = template_config.get("partial_variables", {})

            template = ChatPromptTemplate.from_messages(messages)
            if partial_vars:
                template = template.partial(**partial_vars)

            self.templates[name] = template

    def get_template(self, name: str) -> ChatPromptTemplate:
        """获取模板"""
        if name not in self.templates:
            raise ValueError(f"Template '{name}' not found")
        return self.templates[name]

# 配置文件示例 (templates.json):
# {
#   "qa": {
#     "messages": [
#       ["system", "You are a helpful assistant."],
#       ["human", "{input}"]
#     ]
#   },
#   "translation": {
#     "messages": [
#       ["system", "You are a translator."],
#       ["human", "Translate '{text}' to {language}"]
#     ],
#     "partial_variables": {
#       "language": "French"
#     }
#   }
# }

# 使用
manager = TemplateManager("templates.json")
qa_template = manager.get_template("qa")
messages = qa_template.format_messages(input="Hello")
```

## 六、实战案例

### 案例1：生产级 RAG 系统

```python
from langchain_core.prompts import ChatPromptTemplate, MessagesPlaceholder
from langchain_openai import ChatOpenAI
from functools import lru_cache
import logging

logger = logging.getLogger(__name__)

class ProductionRAG:
    """生产级 RAG 系统"""

    def __init__(self):
        self.model = ChatOpenAI(model="gpt-4o-mini")
        self.template = self._create_template()
        self.call_count = 0

    @lru_cache(maxsize=1)
    def _create_template(self) -> ChatPromptTemplate:
        """创建并缓存模板"""
        return ChatPromptTemplate.from_messages([
            ("system", """You are a RAG assistant.
Rules:
1. Answer based on context only
2. Cite sources
3. Say "I don't know" if unsure

Context: {context}"""),
            MessagesPlaceholder("history", optional=True, n_messages=5),
            ("human", "{question}")
        ])

    def query(self, question: str, context: str, history: list = None):
        """查询 RAG 系统"""
        self.call_count += 1
        logger.info(f"RAG query #{self.call_count}: {question[:50]}...")

        try:
            # 格式化消息
            messages = self.template.format_messages(
                context=context,
                history=history or [],
                question=question
            )

            # 调用模型
            response = self.model.invoke(messages)

            logger.info(f"RAG query #{self.call_count} completed")
            return response.content

        except Exception as e:
            logger.error(f"RAG query #{self.call_count} failed: {e}")
            raise

# 使用
rag = ProductionRAG()

# 第一次查询
answer1 = rag.query(
    question="What is Python?",
    context="Python is a programming language."
)

# 第二次查询（带历史）
answer2 = rag.query(
    question="What are its features?",
    context="Python is easy to learn and powerful.",
    history=[
        ("human", "What is Python?"),
        ("ai", answer1)
    ]
)
```
