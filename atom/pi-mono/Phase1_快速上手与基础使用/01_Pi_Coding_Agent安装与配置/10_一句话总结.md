# 一句话总结

**Pi Coding Agent 是一个极简的终端 AI 编码助手，通过 `npm install -g @mariozechner/pi-coding-agent` 全局安装后，使用 API 密钥或 OAuth 认证连接 15+ LLM Provider，提供 read/write/edit/bash 四大工具，支持通过 Extensions、Skills、Prompt Templates 扩展功能，是构建自定义 AI Agent 应用的理想基础框架。**

---

**核心价值：**
- ✅ 极简设计，只提供核心功能，避免功能臃肿
- ✅ 高度可扩展，通过 TypeScript Extensions 定制任何功能
- ✅ 多 Provider 支持，灵活切换不同 LLM 模型
- ✅ 开放架构，可作为 SDK 嵌入自己的应用

**适用场景：**
- 日常编码助手（替代 Claude Code、Cursor）
- 构建自定义 AI Agent 应用
- 集成到 Slack Bot、Web UI 等场景
- 学习 AI Agent 架构设计
