# 一句话总结

**进阶 Prompt 技巧通过 Few-shot（示例引导）、Chain-of-Thought（分步推理）、结构化输出（格式约束）三大核心方法，配合 Query 分解、Self-RAG 等 RAG 专项优化，让 LLM 输出更准确、更可控、更稳定，是构建高质量 RAG 系统的关键技能。**
