## 必须掌握的5个核心知识点

### 4.1 基础模板创建与使用

**一句话**：使用 `from_template` 快速创建模板，使用 `format` 或 `invoke` 填充变量。

**为什么重要**：这是所有高级用法的基础，90%的场景都从这里开始。

**代码示例**：

```python
from langchain_core.prompts import PromptTemplate

# 方式1：from_template（推荐）
prompt = PromptTemplate.from_template(
    "Translate {text} to {language}"
)

# 方式2：显式指定（更灵活）
prompt = PromptTemplate(
    template="Translate {text} to {language}",
    input_variables=["text", "language"]
)

# 使用方式1：format（返回字符串）
result = prompt.format(text="Hello", language="French")
print(result)
# 输出: "Translate Hello to French"

# 使用方式2：invoke（返回 PromptValue）
result = prompt.invoke({"text": "Hello", "language": "French"})
print(result.to_string())
# 输出: "Translate Hello to French"
```

[来源: reference/context7_langchain_01.md | LangChain 官方文档]

**实际应用**：
- RAG 系统的查询模板
- Agent 的指令模板
- 多语言翻译模板

**这个知识足以**：
- 创建基础的提示词模板
- 动态填充变量
- 与 LLM 集成使用

### 4.2 Partial Variables（部分变量预填充）

**一句话**：使用 `partial_variables` 预填充常用变量，避免每次调用都传递相同的值。

**为什么重要**：减少重复代码，提升开发效率，特别适合日期、系统设置等共享变量。

**代码示例**：

```python
from langchain_core.prompts import PromptTemplate
from datetime import datetime

# ===== 场景1：字符串固定值 =====
prompt = PromptTemplate(
    template="You are a {role}. Answer the question: {question}",
    input_variables=["question"],
    partial_variables={"role": "helpful AI assistant"}
)

# 只需传递 question，role 已预填充
result = prompt.format(question="What is AI?")
print(result)
# 输出: "You are a helpful AI assistant. Answer the question: What is AI?"

# ===== 场景2：函数动态值 =====
def get_current_date():
    return datetime.now().strftime("%Y-%m-%d")

prompt = PromptTemplate(
    template="Today is {date}. {query}",
    input_variables=["query"],
    partial_variables={"date": get_current_date}  # 注意：传递函数本身，不是调用结果
)

# 每次调用时，date 都会动态计算
result = prompt.format(query="What's the weather?")
print(result)
# 输出: "Today is 2026-02-26. What's the weather?"
```

[来源: reference/search_partial_01.md | 社区资料]

**实际应用**：
- 预填充系统角色设定
- 动态注入当前日期时间
- 共享配置参数（如温度、模型名称）

**这个知识足以**：
- 减少重复传递相同变量
- 实现动态变量注入
- 优化模板复用性

### 4.3 模板组合（使用 + 操作符）

**一句话**：使用 `+` 操作符组合多个模板，自动合并变量和部分变量。

**为什么重要**：实现模板复用和模块化，避免重复定义相同的模板片段。

**代码示例**：

```python
from langchain_core.prompts import PromptTemplate

# ===== 场景1：基础组合 =====
system_prompt = PromptTemplate.from_template(
    "You are a {role}. "
)

task_prompt = PromptTemplate.from_template(
    "Please {task} the following: {content}"
)

# 使用 + 组合
combined = system_prompt + task_prompt

result = combined.format(
    role="translator",
    task="translate",
    content="Hello world"
)
print(result)
# 输出: "You are a translator. Please translate the following: Hello world"

# ===== 场景2：组合字符串 =====
prompt = PromptTemplate.from_template("Hello {name}. ")
combined = prompt + "How are you today?"

result = combined.format(name="Alice")
print(result)
# 输出: "Hello Alice. How are you today?"

# ===== 场景3：组合 partial_variables =====
prompt1 = PromptTemplate(
    template="Today is {date}. ",
    input_variables=[],
    partial_variables={"date": "2026-02-26"}
)

prompt2 = PromptTemplate.from_template("Answer: {question}")

combined = prompt1 + prompt2

result = combined.format(question="What is AI?")
print(result)
# 输出: "Today is 2026-02-26. Answer: What is AI?"
```

[来源: reference/source_prompttemplate_01.md | 源码分析 prompt.py:142-184]

**实际应用**：
- 构建多层次的提示词结构
- 复用常见的系统提示
- 动态组装不同的任务指令

**这个知识足以**：
- 实现模板模块化
- 避免重复定义
- 灵活组合不同的提示词片段

### 4.4 多种模板格式（f-string、mustache、jinja2）

**一句话**：根据需求选择合适的模板格式，f-string 简单快速，mustache 支持嵌套，jinja2 功能强大。

**为什么重要**：不同场景需要不同的模板能力，选对格式可以大幅简化开发。

**代码示例**：

```python
from langchain_core.prompts import PromptTemplate

# ===== 格式1：f-string（默认，推荐） =====
prompt = PromptTemplate.from_template(
    "Say {foo}",
    template_format="f-string"  # 可省略，默认就是 f-string
)
print(prompt.format(foo="hello"))
# 输出: "Say hello"

# ===== 格式2：mustache（支持嵌套） =====
prompt = PromptTemplate.from_template(
    "This is a {{foo}} test.",
    template_format="mustache"
)
print(prompt.format(foo="bar"))
# 输出: "This is a bar test."

# Mustache 嵌套变量
prompt = PromptTemplate.from_template(
    "Name: {{person.name}}, Age: {{person.age}}",
    template_format="mustache"
)
print(prompt.format(person={"name": "Alice", "age": 25}))
# 输出: "Name: Alice, Age: 25"

# ===== 格式3：jinja2（功能强大，但有安全风险） =====
prompt = PromptTemplate.from_template(
    "Say {{ foo }}",
    template_format="jinja2"
)
print(prompt.format(foo="hello"))
# 输出: "Say hello"

# Jinja2 支持条件和循环
prompt = PromptTemplate.from_template(
    """
    {% if name %}
    Hello {{ name }}!
    {% else %}
    Hello stranger!
    {% endif %}
    """,
    template_format="jinja2"
)
print(prompt.format(name="Alice"))
# 输出: "Hello Alice!"
```

[来源: reference/source_prompttemplate_01.md | 源码分析]

**格式选择指南**：

| 格式 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| **f-string** | 简单快速、Python 原生 | 不支持嵌套 | 90%的场景 |
| **mustache** | 支持嵌套、跨语言 | 语法稍复杂 | 需要嵌套变量 |
| **jinja2** | 功能强大、支持逻辑 | 安全风险 | 复杂模板逻辑 |

**实际应用**：
- f-string：简单的变量替换
- mustache：处理嵌套的 JSON 数据
- jinja2：需要条件判断和循环的复杂模板

**这个知识足以**：
- 根据需求选择合适的模板格式
- 处理简单和复杂的模板场景
- 避免格式选择错误导致的问题

### 4.5 与 LCEL 链式组合

**一句话**：使用 `|` 操作符将 PromptTemplate 与 LLM 和其他组件链式组合。

**为什么重要**：这是 LangChain 的核心编程范式，掌握它才能构建完整的 AI 应用。

**代码示例**：

```python
import os
from dotenv import load_dotenv
from langchain_core.prompts import PromptTemplate
from langchain_openai import ChatOpenAI
from langchain_core.output_parsers import StrOutputParser

load_dotenv()

# ===== 场景1：基础链式组合 =====
prompt = PromptTemplate.from_template(
    "Translate {text} to {language}"
)

llm = ChatOpenAI(temperature=0)

# 使用 | 操作符链式组合
chain = prompt | llm

result = chain.invoke({
    "text": "Hello, how are you?",
    "language": "French"
})

print(result.content)
# 输出: "Bonjour, comment allez-vous?"

# ===== 场景2：完整的 LCEL 链 =====
prompt = PromptTemplate.from_template(
    "List 3 {adjective} {items}"
)

llm = ChatOpenAI(temperature=0.7)
parser = StrOutputParser()

# 完整的链：Prompt → LLM → Parser
chain = prompt | llm | parser

result = chain.invoke({
    "adjective": "interesting",
    "items": "facts about AI"
})

print(result)
# 输出: "1. AI can learn from data...\n2. ..."

# ===== 场景3：结合 partial_variables =====
from datetime import datetime

def get_current_date():
    return datetime.now().strftime("%Y-%m-%d")

prompt = PromptTemplate(
    template="Today is {date}. {query}",
    input_variables=["query"],
    partial_variables={"date": get_current_date}
)

llm = ChatOpenAI(temperature=0)
chain = prompt | llm | StrOutputParser()

result = chain.invoke({"query": "What day of the week is it?"})
print(result)
# 输出: "Today is Wednesday, February 26, 2026..."
```

[来源: reference/context7_langchain_03.md | LangChain 官方文档]

**实际应用**：
- 构建 RAG 检索链
- 实现 Agent 推理链
- 创建多步骤的 AI 工作流

**这个知识足以**：
- 构建完整的 AI 应用链
- 实现 Prompt → LLM → Parser 的标准流程
- 与其他 LangChain 组件无缝集成

## 完整实战示例

### 示例：构建一个智能翻译助手

```python
"""
智能翻译助手 - 综合运用 PromptTemplate 高级特性
演示：partial_variables + 模板组合 + LCEL 链式组合
"""

import os
from dotenv import load_dotenv
from datetime import datetime
from langchain_core.prompts import PromptTemplate
from langchain_openai import ChatOpenAI
from langchain_core.output_parsers import StrOutputParser

load_dotenv()

# ===== 1. 定义动态函数 =====
def get_current_date():
    return datetime.now().strftime("%Y-%m-%d")

# ===== 2. 创建系统提示（使用 partial_variables） =====
system_prompt = PromptTemplate(
    template="You are a professional translator. Today is {date}. ",
    input_variables=[],
    partial_variables={"date": get_current_date}
)

# ===== 3. 创建任务提示 =====
task_prompt = PromptTemplate.from_template(
    "Translate the following {source_lang} text to {target_lang}:\n\n{text}\n\nTranslation:"
)

# ===== 4. 组合模板 =====
combined_prompt = system_prompt + task_prompt

# ===== 5. 构建 LCEL 链 =====
llm = ChatOpenAI(temperature=0, model="gpt-4o-mini")
parser = StrOutputParser()

translation_chain = combined_prompt | llm | parser

# ===== 6. 使用翻译链 =====
print("=== 智能翻译助手 ===\n")

# 测试1：英译中
result1 = translation_chain.invoke({
    "source_lang": "English",
    "target_lang": "Chinese",
    "text": "Artificial Intelligence is transforming the world."
})
print(f"英译中: {result1}\n")

# 测试2：中译英
result2 = translation_chain.invoke({
    "source_lang": "Chinese",
    "target_lang": "English",
    "text": "人工智能正在改变世界。"
})
print(f"中译英: {result2}\n")

# 测试3：英译法
result3 = translation_chain.invoke({
    "source_lang": "English",
    "target_lang": "French",
    "text": "Hello, how are you?"
})
print(f"英译法: {result3}\n")

# ===== 7. 查看完整的提示词（调试用） =====
debug_prompt = combined_prompt.format(
    source_lang="English",
    target_lang="Chinese",
    text="Test"
)
print(f"=== 完整提示词 ===\n{debug_prompt}")
```

**运行输出示例**：
```
=== 智能翻译助手 ===

英译中: 人工智能正在改变世界。

中译英: Artificial Intelligence is transforming the world.

英译法: Bonjour, comment allez-vous?

=== 完整提示词 ===
You are a professional translator. Today is 2026-02-26. Translate the following English text to Chinese:

Test

Translation:
```

## 快速参考卡

### 常用操作速查

```python
# 1. 创建模板
prompt = PromptTemplate.from_template("Say {foo}")

# 2. 预填充变量
prompt = PromptTemplate(
    template="...",
    partial_variables={"key": "value"}
)

# 3. 组合模板
combined = prompt1 + prompt2

# 4. 链式组合
chain = prompt | llm | parser

# 5. 使用模板
result = prompt.format(foo="bar")
result = prompt.invoke({"foo": "bar"})
```

### 常见错误与解决

| 错误 | 原因 | 解决方案 |
|------|------|----------|
| `KeyError: 'variable'` | 缺少必需变量 | 检查 `input_variables` 和传入的参数 |
| `ValueError: Cannot add templates` | 组合时格式不一致 | 确保两个模板的 `template_format` 相同 |
| `ValueError: Cannot have same variable partialed twice` | partial_variables 冲突 | 检查组合的模板是否有重复的 partial 变量 |
| 函数不执行 | 传递了函数调用结果 | 传递函数本身：`{"date": get_date}` 而非 `{"date": get_date()}` |
