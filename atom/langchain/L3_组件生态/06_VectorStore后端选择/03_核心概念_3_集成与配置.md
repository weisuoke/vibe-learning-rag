# 核心概念 3：集成与配置

## 概述

LangChain 为不同的向量存储后端提供了统一的集成接口，使得开发者可以轻松切换后端而无需修改业务代码。本文档详细介绍各主流向量存储后端的集成方式、配置参数和最佳实践。

**核心价值**：
- 统一接口：所有后端实现相同的 VectorStore 接口
- 灵活配置：支持多种初始化方式和配置参数
- 易于切换：更换后端只需修改初始化代码
- 生产就绪：提供完整的配置选项和最佳实践

## 1. LangChain 统一接口

### 1.1 VectorStore 抽象层

所有向量存储后端都继承自 `VectorStore` 抽象基类：

```python
from langchain_core.vectorstores import VectorStore
from langchain_core.embeddings import Embeddings
from langchain_core.documents import Document

class VectorStore(ABC):
    """向量存储的统一接口"""

    @property
    @abstractmethod
    def embeddings(self) -> Embeddings:
        """返回 embedding 函数"""
        pass

    @abstractmethod
    def similarity_search(
        self,
        query: str,
        k: int = 4,
        **kwargs
    ) -> list[Document]:
        """相似度检索"""
        pass

    def add_texts(
        self,
        texts: Iterable[str],
        metadatas: list[dict] | None = None,
        **kwargs
    ) -> list[str]:
        """添加文本"""
        pass
```

[来源: reference/source_vectorstore_base_01.md | LangChain 源码分析]

### 1.2 统一的初始化模式

所有后端都支持两种初始化方式：

```python
# 方式1：从文本创建
vector_store = VectorStoreClass.from_texts(
    texts=["文档1", "文档2"],
    embedding=embeddings,
    **backend_specific_params
)

# 方式2：从文档创建
from langchain_core.documents import Document

docs = [
    Document(page_content="文档1", metadata={"source": "file1"}),
    Document(page_content="文档2", metadata={"source": "file2"})
]

vector_store = VectorStoreClass.from_documents(
    documents=docs,
    embedding=embeddings,
    **backend_specific_params
)
```

## 2. InMemory 集成与配置

### 2.1 基础配置

```python
from langchain_core.vectorstores import InMemoryVectorStore
from langchain_openai import OpenAIEmbeddings

# 初始化（零配置）
vector_store = InMemoryVectorStore(
    embedding=OpenAIEmbeddings()
)

# 添加文档
vector_store.add_texts(
    texts=["文档1", "文档2"],
    metadatas=[{"source": "file1"}, {"source": "file2"}]
)

# 检索
results = vector_store.similarity_search("查询", k=5)
```

[来源: reference/source_inmemory_02.md | LangChain 源码分析]

### 2.2 特点与限制

**特点**：
- 零配置：无需安装额外依赖
- 快速启动：适合快速原型开发
- 完整功能：支持所有 VectorStore 接口

**限制**：
- 不持久化：程序重启后数据丢失
- 内存限制：只能处理小规模数据（< 1K 文档）
- 无并发优化：不适合生产环境

[来源: reference/source_inmemory_02.md | LangChain 源码分析]

## 5. Pinecone 集成与配置

### 5.1 基础配置

```python

from langchain_openai import OpenAIEmbeddings
from pinecone import Pinecone, ServerlessSpec

# 1. 初始化 Pinecone 客户端
pc = Pinecone(api_key="your-api-key")

# 2. 创建索引（如果不存在）
index_name = "my-index"

if index_name not in pc.list_indexes().names():
    pc.create_index(
        name=index_name,
        dimension=1536,  # OpenAI embedding 维度
        metric="cosine",
        spec=ServerlessSpec(
            cloud="aws",
            region="us-east-1"

# 4. 创建向量存储
vector_store = PineconeVectorStore(
    index=index,
    embedding=OpenAIEmbeddings(),
    text_key="text",  # 文本字段名
    namespace="default"  # 命名空间
)

# 5. 添加文档
```

## 7. 集成最佳实践

### 7.1 工厂模式

```python
from typing import Literal

def create_vector_store(
    backend: Literal["inmemory", "chroma", "qdrant", "pinecone"],
    embeddings: Embeddings,
    **kwargs
) -> VectorStore:
    """工厂函数：创建向量存储

    Args:
        backend: 后端类型
        embeddings: Embedding 函数
        **kwargs: 后端特定参数

    Returns:
        VectorStore 实例
    """
    if backend == "inmemory":
        from langchain_core.vectorstores import InMemoryVectorStore
        return InMemoryVectorStore(embedding=embeddings)

    elif backend == "chroma":
        from langchain_chroma import Chroma
        return Chroma(
            collection_name=kwargs.get("collection_name", "default"),
            embedding_function=embeddings,
            persist_directory=kwargs.get("persist_directory")
        )

    elif backend == "qdrant":
        from langchain_qdrant import QdrantVectorStore
        return QdrantVectorStore.from_documents(
            documents=kwargs.get("documents", []),
            embedding=embeddings,
            location=kwargs.get("location", ":memory:"),
            collection_name=kwargs.get("collection_name", "default")
        )

    elif backend == "pinecone":
        from langchain_pinecone import PineconeVectorStore
        from pinecone import Pinecone

        pc = Pinecone(api_key=kwargs["api_key"])
        index = pc.Index(kwargs["index_name"])

        return PineconeVectorStore(
            index=index,
            embedding=embeddings,
            namespace=kwargs.get("namespace", "default")
        )

    else:
        raise ValueError(f"不支持的后端: {backend}")

# 使用工厂函数
vector_store = create_vector_store(
    backend="chroma",
    embeddings=OpenAIEmbeddings(),
    collection_name="my_docs",
    persist_directory="./chroma_db"
)
```

```python
from typing import Optional

def create_vector_store_safe(
    backend: str,
    embeddings: Embeddings,
    **kwargs
) -> Optional[VectorStore]:
    """安全创建向量存储（带错误处理）"""
    try:
        return create_vector_store(backend, embeddings, **kwargs)

        print(f"缺少依赖: {e}")
```

```python
def export_vector_store(vector_store: VectorStore, output_file: str):
    """导出向量存储数据"""
    import json

    # 获取所有文档（如果支持）
    if hasattr(vector_store, "get_all_documents"):
        docs = vector_store.get_all_documents()
    else:
        # 手动导出
        docs = []
        for doc_id in vector_store.store.keys():
            doc = vector_store.get_by_ids([doc_id])[0]
            docs.append({
                "id": doc_id,
                "content": doc.page_content,
                "metadata": doc.metadata
            })

    # 保存到文件
    with open(output_file, "w") as f:
        json.dump(docs, f, ensure_ascii=False, indent=2)
```

## 9. 总结

### 9.1 集成难度排序

1. **InMemory**：零配置，立即可用
