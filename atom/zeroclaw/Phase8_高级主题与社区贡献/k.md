# Phase 8: 高级主题与社区贡献 - 知识点列表

> 目标：覆盖进阶功能，成为社区贡献者
> 学习时长：第 16 周
> 前置要求：Phase 1-7 完成

---

## 知识点列表

### 63. AIEOS 身份系统
- AIEOS v1.1 JSON 格式、OpenClaw Markdown 身份、人格迁移
- 前端类比：用户 Profile + 主题系统切换
- ZeroClaw 场景：identity.rs — 可移植的 AI 人格定义

### 64. RAG 能力集成
- 文档摄入、Embedding 生成、检索增强生成、与 Memory 的关系
- 前端类比：搜索引擎 + 知识库
- ZeroClaw 场景：rag/ — 内置 RAG 能力（与 RAG 学习路径关联）

### 65. Web Dashboard（React）
- React + TypeScript 前端、实时状态监控、配置界面、API 对接
- 前端类比：你的主场！React 开发
- ZeroClaw 场景：web/ — 前端开发者最容易贡献的模块

### 66. 硬件与固件集成
- Raspberry Pi GPIO、ESP32 固件、Arduino 集成、传感器/电机
- 前端类比：IoT 设备控制面板
- ZeroClaw 场景：peripherals/ + firmware/ — $10 硬件上的 AI Agent

### 67. 性能优化与 Benchmark
- 二进制大小优化（opt-level=z, LTO）、启动时间、内存占用、基准测试
- 前端类比：Lighthouse 性能评分 / Webpack bundle 优化
- ZeroClaw 场景：benches/ + Cargo.toml release profile

### 68. CI/CD 与发布流程
- GitHub Actions、跨平台编译、Release 构建、版本管理
- 前端类比：GitHub Actions + npm publish 自动发布
- ZeroClaw 场景：scripts/ — 构建、测试、跨平台发布流程

### 69. PR 工作流与代码审查
- Fork → Branch → PR、CONTRIBUTING.md 规范、CLA 签署、Code Review
- 前端类比：标准 GitHub PR 工作流
- ZeroClaw 场景：向 zeroclaw-labs/zeroclaw 提交第一个 PR

### 70. 社区参与与生态贡献
- Issue 分类与认领、RFC 流程、社区资源、生态项目
- 前端类比：开源项目维护与社区参与
- ZeroClaw 场景：成为 ZeroClaw 社区的活跃贡献者
