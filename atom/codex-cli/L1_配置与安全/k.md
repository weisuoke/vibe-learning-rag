# L1_配置与安全 - 知识点列表

> 掌握 Codex CLI 的高级配置和安全最佳实践

---

## 学习目标

- 深入理解 Codex CLI 的配置系统
- 掌握权限控制和安全最佳实践
- 为高级使用场景打下基础

---

## 知识点列表

### 01. 配置系统深度解析 ⭐⭐⭐

**核心内容：**
- 配置文件结构与优先级
- 提供商配置（OpenAI/自定义端点）
- multi_agent 实验功能启用
- 高级配置选项（超时、重试、并发限制）

**学习重点：**
- 理解配置文件的层级结构
- 掌握 multi_agent 配置启用方法
- 学会自定义 API 端点配置

**实战场景：**
- 配置自定义 API 端点
- 启用 Multi-Agent 实验功能
- 调整并发限制和超时设置

---

### 02. 权限与安全最佳实践 ⭐⭐

**核心内容：**
- 沙箱模式与文件访问控制
- API 密钥管理与轮换
- 敏感数据保护
- 审计日志与监控

**学习重点：**
- 理解沙箱模式的工作原理
- 掌握 API 密钥的安全管理
- 学会配置文件访问权限

**实战场景：**
- 配置沙箱模式限制文件访问
- 实现 API 密钥轮换策略
- 设置审计日志监控

---

## 学习路径建议

1. **先学配置系统**：理解配置文件结构是使用高级特性的基础
2. **再学安全实践**：在实际使用前建立安全意识
3. **结合实战**：每个知识点都要动手配置和验证
