# L1_NLP基础 知识点列表

> RAG 开发的 NLP 基础知识，为后续学习打下坚实基础

---

## 知识点清单

1. **BPE分词原理** - 理解大模型如何将文本切分为 Token
2. **Embedding原理与选型** - 理解文本向量化的核心技术
3. **语义相似度** - 理解如何计算和比较文本的语义相关性

---

## 学习顺序建议

```
BPE分词原理 → Embedding原理与选型 → 语义相似度
     ↓              ↓                  ↓
  理解Token      理解向量表示        理解检索基础
```

## 与 RAG 开发的关系

| 知识点 | 在 RAG 中的应用 |
|--------|----------------|
| BPE分词 | Token 计数、上下文窗口管理、成本估算 |
| Embedding | 文档向量化、语义检索、相似度计算 |
| 语义相似度 | 检索排序、相关性判断、结果过滤 |
