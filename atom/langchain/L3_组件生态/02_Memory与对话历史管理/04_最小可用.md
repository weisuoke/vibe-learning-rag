## 一、核心 API（必须掌握）

### 1.1 InMemoryChatMessageHistory

**最小用法**：
```python
from langchain_core.chat_history import InMemoryChatMessageHistory
from langchain_core.messages import HumanMessage, AIMessage

# 创建历史存储
history = InMemoryChatMessageHistory()

# 添加消息
history.add_message(HumanMessage(content="Hello!"))
history.add_message(AIMessage(content="Hi there!"))

# 获取消息
messages = history.messages
print(messages)  # [HumanMessage(...), AIMessage(...)]

# 清空历史
history.clear()
```

**核心方法**：
- `add_message(message)` - 添加单条消息
- `add_messages(messages)` - 批量添加消息
- `messages` - 获取所有消息（属性）
- `clear()` - 清空历史

### 1.2 RunnableWithMessageHistory

**最小用法**：
```python
from langchain_core.chat_history import InMemoryChatMessageHistory
from langchain_core.runnables.history import RunnableWithMessageHistory
from langchain_openai import ChatOpenAI

# 1. 创建存储字典
store = {}

# 2. 定义获取历史的函数
def get_session_history(session_id: str):
    if session_id not in store:
        store[session_id] = InMemoryChatMessageHistory()
    return store[session_id]

# 3. 创建聊天模型
chat_model = ChatOpenAI(model="gpt-4")

# 4. 包装模型
conversation = RunnableWithMessageHistory(
    chat_model,
    get_session_history,
)

# 5. 使用（需要传递 config）
config = {"configurable": {"session_id": "user_123"}}
response = conversation.invoke("Hi, I'm Alice.", config=config)
print(response.content)  # "Hello Alice! How can I help you today?"

# 6. 继续对话
response = conversation.invoke("What's my name?", config=config)
print(response.content)  # "Your name is Alice."
```

**核心参数**：
- `runnable` - 要包装的 Runnable（通常是聊天模型）
- `get_session_history` - 获取历史的函数
- `config` - 配置对象，必须包含 `{"configurable": {"session_id": "..."}}`

### 1.3 持久化存储（Redis）

**最小用法**：
```python
from langchain_redis import RedisChatMessageHistory
from langchain_core.runnables.history import RunnableWithMessageHistory
from langchain_openai import ChatOpenAI

# 1. 定义获取历史的函数（使用 Redis）
def get_session_history(session_id: str):
    return RedisChatMessageHistory(
        session_id=session_id,
        url="redis://localhost:6379",
        ttl=3600,  # 1小时过期
    )

# 2. 包装模型
conversation = RunnableWithMessageHistory(
    ChatOpenAI(model="gpt-4"),
    get_session_history,
)

# 3. 使用
config = {"configurable": {"session_id": "user_123"}}
response = conversation.invoke("Hello!", config=config)
```

## 二、常用模式（必须掌握）

**核心模式**：

**多用户隔离**：
```python

# 用户2
```

### 2.2 异步操作模式

**核心模式**：
```python

response = await conversation.ainvoke("Hello!", config=config)

# 异步流式调用
async for chunk in conversation.astream("Tell me a story", config=config):
    print(chunk.content, end="", flush=True)
```

### 2.3 内存限制模式

**核心模式**：
```python
def get_session_history(session_id: str):
    history = store.get(session_id)
    if history is None:
        history = InMemoryChatMessageHistory()
        store[session_id] = history

    # 限制消息数量
    messages = history.messages
    if len(messages) > 20:  # 只保留最近20条消息
        history.messages = messages[-20:]

    return history
```

## 三、快速决策树

### 3.1 选择存储方案

```
需要持久化？
├─ 否 → InMemoryChatMessageHistory
└─ 是
   ├─ 开发/测试 → InMemoryChatMessageHistory
   ├─ 生产环境（高性能） → Redis
   ├─ 生产环境（持久化） → PostgreSQL
   └─ 小型应用 → SQLite
```

### 3.2 选择管理方案

```
需要对话历史管理？
├─ 简单对话系统 → RunnableWithMessageHistory
└─ 复杂 Agent 系统 → LangGraph Checkpointer
```

### 3.3 选择内存限制策略

```
对话长度？
├─ 短对话（<10轮） → 完整历史
├─ 中等对话（10-50轮） → 滑动窗口（保留最近N条）
└─ 长对话（>50轮） → 自动总结（SummarizationNode）
```

## 五、常见错误和解决方案

### 5.1 错误：忘记传递 config

**错误代码**：
```python
# ❌ 错误：没有传递 config
response = conversation.invoke("Hello!")
```

**错误信息**：
```
KeyError: 'configurable'
```

**解决方案**：
```python
# ✅ 正确：传递 config
config = {"configurable": {"session_id": "user_123"}}
response = conversation.invoke("Hello!", config=config)
```

### 5.2 错误：Session ID 不唯一

**错误代码**：
```python
# ❌ 错误：所有用户使用相同的 session_id
config = {"configurable": {"session_id": "default"}}
```

**问题**：所有用户共享同一个对话历史

**解决方案**：
```python
# ✅ 正确：每个用户使用唯一的 session_id
config = {"configurable": {"session_id": f"user_{user_id}"}}
```

### 5.3 错误：没有限制内存

**错误代码**：
```python
# ❌ 错误：没有限制内存，长对话会超出上下文窗口
def get_session_history(session_id: str):
    if session_id not in store:
        store[session_id] = InMemoryChatMessageHistory()
    return store[session_id]
```

**问题**：长对话会超出 LLM 的上下文窗口

**解决方案**：
```python
# ✅ 正确：限制消息数量
def get_session_history(session_id: str):
    history = store.get(session_id)
    if history is None:
        history = InMemoryChatMessageHistory()
        store[session_id] = history

    # 限制消息数量
    messages = history.messages
    if len(messages) > 20:
        history.messages = messages[-20:]

    return history
```

### 5.4 错误：使用传统 Memory 类

**错误代码**：
```python
# ❌ 错误：使用已弃用的 ConversationBufferMemory
from langchain.memory import ConversationBufferMemory
memory = ConversationBufferMemory()
```

**问题**：传统 Memory 类已被弃用，不支持工具调用

**解决方案**：
```python
# ✅ 正确：使用 RunnableWithMessageHistory
from langchain_core.runnables.history import RunnableWithMessageHistory
conversation = RunnableWithMessageHistory(
    chat_model,
    get_session_history,
)
```
