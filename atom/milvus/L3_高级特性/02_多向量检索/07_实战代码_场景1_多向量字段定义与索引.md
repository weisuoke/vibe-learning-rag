# 实战代码 - 场景1:多向量字段定义与索引

> **场景描述**:在 Milvus 中定义包含多个向量字段的 Collection,并为所有向量字段创建索引。

---

## 场景概述

### 业务需求

在实际的 RAG 应用中,我们经常需要在同一个 Collection 中存储多种类型的向量:
- **稠密向量**(Dense Vector):捕获语义关系,如文本的语义 Embedding
- **稀疏向量**(Sparse Vector):实现精确关键词匹配,如 BM25 向量
- **多模态向量**:图片向量、音频向量等

本场景展示如何定义包含多个向量字段的 Schema,并为所有向量字段创建索引。

### 技术要点

1. **Schema 设计**:在一个 Collection 中定义多个向量字段
2. **向量类型支持**:FloatVector、SparseFloatVector、BinaryVector、BFloat16Vector
3. **索引创建要求**:所有向量字段都必须创建索引才能加载 Collection
4. **BM25 函数集成**:自动将文本字段转换为稀疏向量

### 场景1:基础多向量字段定义

```python
def create_basic_multi_vector_collection(
    collection_name: str = "multi_vector_basic",
    dense_dim: int = 768,
    drop_old: bool = True
) -> Collection:
    """
    创建包含稠密向量和稀疏向量的基础 Collection

    Args:
        collection_name: Collection 名称
        dense_dim: 稠密向量维度
        drop_old: 是否删除已存在的 Collection

    Returns:
        Collection 对象
    """
    # 1. 连接到 Milvus
    connections.connect(
        alias="default",
        host="localhost",
        port="19530"
    )

    # 2. 删除已存在的 Collection(如果需要)
    if drop_old and utility.has_collection(collection_name):
        utility.drop_collection(collection_name)
        print(f"已删除旧 Collection: {collection_name}")

    # 3. 定义 Schema
    fields = [
        # 主键字段
        FieldSchema(
            name="id",
            dtype=DataType.INT64,
            is_primary=True,
            auto_id=True,
            description="主键 ID"
        ),

        # 文本字段
        FieldSchema(
            name="text",
            dtype=DataType.VARCHAR,
            max_length=65535,
            description="原始文本内容"
        ),

        # 稠密向量字段
        FieldSchema(
            name="dense_vector",
            dtype=DataType.FLOAT_VECTOR,
            dim=dense_dim,
            description="稠密向量(语义 Embedding)"
        ),

        # 稀疏向量字段
        FieldSchema(
            name="sparse_vector",
            dtype=DataType.SPARSE_FLOAT_VECTOR,
            description="稀疏向量(关键词匹配)"
        )
    ]

    schema = CollectionSchema(
        fields=fields,
        description="基础多向量 Collection",
        enable_dynamic_field=True
    )

    # 4. 创建 Collection
    collection = Collection(
        name=collection_name,
        schema=schema
    )

    print(f"✓ 创建 Collection: {collection_name}")
    print(f"  - 稠密向量维度: {dense_dim}")
    print(f"  - 向量字段数量: 2 (dense_vector, sparse_vector)")

    return collection


def create_indexes_for_multi_vector(collection: Collection) -> None:
    """
    为多向量 Collection 创建索引

    关键约束:所有向量字段都必须创建索引才能加载 Collection

    Args:
        collection: Collection 对象
    """
    print(f"\n开始为 Collection '{collection.name}' 创建索引...")

    # 1. 为稠密向量创建 HNSW 索引
    dense_index_params = {
        "index_type": "HNSW",
        "metric_type": "IP",  # 内积(适合归一化向量)
        "params": {
            "M": 16,              # HNSW 参数:每个节点的最大连接数
            "efConstruction": 256  # 构建索引时的搜索深度
        }
    }

    collection.create_index(
        field_name="dense_vector",
        index_params=dense_index_params,
        index_name="dense_vector_idx"
    )
    print("✓ 稠密向量索引创建完成 (HNSW)")

    # 2. 为稀疏向量创建 SPARSE_INVERTED_INDEX 索引
    sparse_index_params = {
        "index_type": "SPARSE_INVERTED_INDEX",
        "metric_type": "IP",
        "params": {
            "drop_ratio_build": 0.2  # 构建时丢弃低权重的 20% 元素
        }
    }

    collection.create_index(
        field_name="sparse_vector",
        index_params=sparse_index_params,
        index_name="sparse_vector_idx"
    )
    print("✓ 稀疏向量索引创建完成 (SPARSE_INVERTED_INDEX)")

    # 3. 加载 Collection
    collection.load()
    print(f"✓ Collection '{collection.name}' 加载完成")


# 示例:创建基础多向量 Collection
if __name__ == "__main__":
    # 创建 Collection
    collection = create_basic_multi_vector_collection(
        collection_name="multi_vector_basic",
        dense_dim=768
    )

    # 创建索引
    create_indexes_for_multi_vector(collection)
```

```python
def create_multimodal_collection(
    collection_name: str = "multimodal_collection",
    text_dim: int = 768,
    image_dim: int = 512,
    drop_old: bool = True
) -> Collection:
    """
    创建多模态 Collection(文本 + 图片)

    应用场景:
    - 电商商品检索(商品描述 + 商品图片)
    - 内容平台搜索(文章内容 + 封面图)
    - 多模态问答系统

    Args:
        collection_name: Collection 名称
        text_dim: 文本向量维度
        image_dim: 图片向量维度
        drop_old: 是否删除已存在的 Collection

    Returns:
        Collection 对象
    """
    # 1. 连接到 Milvus
    connections.connect(alias="default", host="localhost", port="19530")

    # 2. 删除已存在的 Collection
    if drop_old and utility.has_collection(collection_name):
        utility.drop_collection(collection_name)

    # 3. 定义 Schema
    fields = [
        # 主键
        FieldSchema(
            name="id",
            dtype=DataType.INT64,
            is_primary=True,
            auto_id=True
        ),

        # 文本内容
        FieldSchema(
            name="text_content",
            dtype=DataType.VARCHAR,
            max_length=65535,
            description="文本内容"
        ),

        # 图片 URL
        FieldSchema(
            name="image_url",
            dtype=DataType.VARCHAR,
            max_length=512,
            description="图片 URL"
        ),

        # 文本稠密向量
        FieldSchema(
            name="text_dense_vector",
            dtype=DataType.FLOAT_VECTOR,
            dim=text_dim,
            description="文本语义向量"
        ),

        # 文本稀疏向量
        FieldSchema(
            name="text_sparse_vector",
            dtype=DataType.SPARSE_FLOAT_VECTOR,
            description="文本关键词向量"
        ),

        # 图片向量
        FieldSchema(
            name="image_vector",
            dtype=DataType.FLOAT_VECTOR,
            dim=image_dim,
            description="图片特征向量"
        ),

        # 元数据字段
        FieldSchema(
            name="category",
            dtype=DataType.VARCHAR,
            max_length=128,
            description="分类标签"
        )
    ]

    schema = CollectionSchema(
        fields=fields,
        description="多模态 Collection(文本 + 图片)",
        enable_dynamic_field=True
    )

    # 4. 创建 Collection
    collection = Collection(name=collection_name, schema=schema)

    print(f"✓ 创建多模态 Collection: {collection_name}")
    print(f"  - 文本向量维度: {text_dim}")
    print(f"  - 图片向量维度: {image_dim}")
    print(f"  - 向量字段数量: 3 (text_dense, text_sparse, image)")

    return collection


def create_multimodal_indexes(collection: Collection) -> None:
    """
    为多模态 Collection 创建索引

    Args:
        collection: Collection 对象
    """
    print(f"\n开始为多模态 Collection 创建索引...")

    # 1. 文本稠密向量索引
    collection.create_index(
        field_name="text_dense_vector",
        index_params={
            "index_type": "HNSW",
            "metric_type": "COSINE",
            "params": {"M": 16, "efConstruction": 256}
        },
        index_name="text_dense_idx"
    )
    print("✓ 文本稠密向量索引创建完成")

    # 2. 文本稀疏向量索引
    collection.create_index(
        field_name="text_sparse_vector",
        index_params={
            "index_type": "SPARSE_INVERTED_INDEX",
            "metric_type": "IP",
            "params": {"drop_ratio_build": 0.2}
        },
        index_name="text_sparse_idx"
    )
    print("✓ 文本稀疏向量索引创建完成")

    # 3. 图片向量索引
    collection.create_index(
        field_name="image_vector",
        index_params={
            "index_type": "IVF_FLAT",
            "metric_type": "L2",
            "params": {"nlist": 128}
        },
        index_name="image_idx"
    )
    print("✓ 图片向量索引创建完成")

    # 4. 加载 Collection
    collection.load()
    print(f"✓ 多模态 Collection 加载完成")


# 示例:创建多模态 Collection
if __name__ == "__main__":
    collection = create_multimodal_collection(
        collection_name="multimodal_collection",
        text_dim=768,
        image_dim=512
    )

    create_multimodal_indexes(collection)
```

### 场景3:BM25 函数集成

```python
def create_bm25_collection(
    collection_name: str = "bm25_collection",
    dense_dim: int = 768,
    drop_old: bool = True
) -> Collection:
    """
    创建集成 BM25 函数的 Collection

    BM25 函数会自动将文本字段转换为稀疏向量,无需手动生成

    Args:
        collection_name: Collection 名称
        dense_dim: 稠密向量维度
        drop_old: 是否删除已存在的 Collection

    Returns:
        Collection 对象
    """
    # 1. 连接到 Milvus
    connections.connect(alias="default", host="localhost", port="19530")

    # 2. 删除已存在的 Collection
    if drop_old and utility.has_collection(collection_name):
        utility.drop_collection(collection_name)

    # 3. 定义 Schema
    fields = [
        FieldSchema(
            name="id",
            dtype=DataType.INT64,
            is_primary=True,
            auto_id=True
        ),

        # 文本字段(需要启用 analyzer)
        FieldSchema(
            name="text",
            dtype=DataType.VARCHAR,
            max_length=65535,
            enable_analyzer=True,  # 启用文本分析器
            enable_match=True,     # 启用全文匹配
            analyzer_params={
                "type": "standard"  # 使用标准分析器
            }
        ),

        # 稠密向量字段
        FieldSchema(
            name="dense_vector",
            dtype=DataType.FLOAT_VECTOR,
            dim=dense_dim
        ),

        # 稀疏向量字段(由 BM25 函数自动生成)
        FieldSchema(
            name="sparse_vector",
            dtype=DataType.SPARSE_FLOAT_VECTOR
        )
    ]

    schema = CollectionSchema(
        fields=fields,
        description="BM25 集成 Collection"
    )

    # 4. 添加 BM25 函数
    bm25_function = Function(
        name="bm25_function",
        function_type=FunctionType.BM25,
        input_field_names=["text"],      # 输入:文本字段
        output_field_names=["sparse_vector"],  # 输出:稀疏向量字段
        params={}
    )

    schema.add_function(bm25_function)

    # 5. 创建 Collection
    collection = Collection(name=collection_name, schema=schema)

    print(f"✓ 创建 BM25 Collection: {collection_name}")
    print(f"  - BM25 函数已集成")
    print(f"  - 输入字段: text")
    print(f"  - 输出字段: sparse_vector")

    return collection


def create_bm25_indexes(collection: Collection) -> None:
    """
    为 BM25 Collection 创建索引

    Args:
        collection: Collection 对象
    """
    print(f"\n开始为 BM25 Collection 创建索引...")

    # 1. 稠密向量索引
    collection.create_index(
        field_name="dense_vector",
        index_params={
            "index_type": "HNSW",
            "metric_type": "IP",
            "params": {"M": 16, "efConstruction": 256}
        }
    )
    print("✓ 稠密向量索引创建完成")

    # 2. 稀疏向量索引(BM25 生成)
    collection.create_index(
        field_name="sparse_vector",
        index_params={
            "index_type": "SPARSE_INVERTED_INDEX",
            "metric_type": "BM25",  # 使用 BM25 度量
            "params": {}
        }
    )
    print("✓ BM25 稀疏向量索引创建完成")

    # 3. 加载 Collection
    collection.load()
    print(f"✓ BM25 Collection 加载完成")


# 示例:创建 BM25 Collection
if __name__ == "__main__":
    collection = create_bm25_collection(
        collection_name="bm25_collection",
        dense_dim=768
    )

    create_bm25_indexes(collection)
```

```python
def insert_multi_vector_data(
    collection: Collection,
    num_entities: int = 100
) -> None:
    """
    向多向量 Collection 插入测试数据

    Args:
        collection: Collection 对象
        num_entities: 插入的实体数量
    """
    print(f"\n开始插入 {num_entities} 条测试数据...")

    # 1. 生成测试数据
    data = []

    for i in range(num_entities):
        # 生成稠密向量(归一化)
        dense_vec = np.random.rand(768).astype(np.float32)
        dense_vec = dense_vec / np.linalg.norm(dense_vec)

        # 生成稀疏向量(CSR 格式)
        # 稀疏向量:只有少数维度有值
        indices = random.sample(range(10000), 20)  # 随机选择 20 个维度
        values = np.random.rand(20).astype(np.float32)
        sparse_vec = {idx: val for idx, val in zip(indices, values)}

        entity = {
            "text": f"这是第 {i} 条测试文本内容",
            "dense_vector": dense_vec.tolist(),
            "sparse_vector": sparse_vec
        }

        data.append(entity)

    # 2. 插入数据
    insert_result = collection.insert(data)

    print(f"✓ 成功插入 {len(insert_result.primary_keys)} 条数据")
    print(f"  - 主键范围: {insert_result.primary_keys[0]} ~ {insert_result.primary_keys[-1]}")

    # 3. 刷新数据(确保数据持久化)
    collection.flush()
    print("✓ 数据已刷新到磁盘")


# 示例:插入数据
if __name__ == "__main__":
    # 假设已经创建了 Collection
    connections.connect(alias="default", host="localhost", port="19530")
    collection = Collection("multi_vector_basic")

    insert_multi_vector_data(collection, num_entities=100)
```

## 关键技术要点

### 1. 索引创建的强制要求

```python
# ❌ 错误:只为部分向量字段创建索引
collection.create_index(field_name="dense_vector", ...)
# 缺少 sparse_vector 的索引
collection.load()  # 会失败!

# ✓ 正确:为所有向量字段创建索引
collection.create_index(field_name="dense_vector", ...)
collection.create_index(field_name="sparse_vector", ...)
collection.load()  # 成功
```

### 2. 不同向量类型的索引选择

| 向量类型 | 推荐索引类型 | 适用场景 |

| FloatVector | HNSW, IVF_FLAT | 稠密语义向量 |

| BinaryVector | BIN_IVF_FLAT | 二值化向量 |
| BFloat16Vector | HNSW | 内存优化的稠密向量 |

## 常见错误与解决方案

### 错误1:加载 Collection 失败

**错误信息**:
```
MilvusException: collection not loaded, collection_name=xxx
```

**原因**:部分向量字段没有创建索引

**解决方案**:
```python
# 检查所有向量字段是否都有索引
indexes = collection.indexes
print(f"已创建的索引: {[idx.field_name for idx in indexes]}")

# 为缺失的向量字段创建索引
collection.create_index(field_name="missing_vector_field", ...)
```

### 错误2:稀疏向量格式错误

**错误信息**:
```
TypeError: sparse vector must be dict type
```

**原因**:稀疏向量必须是字典格式 `{index: value}`

**解决方案**:
```python
# ❌ 错误:使用 CSR 矩阵
sparse_vec = csr_matrix(...)

# ✓ 正确:使用字典格式
sparse_vec = {0: 0.5, 10: 0.8, 100: 0.3}
```

### 错误3:BM25 函数无法使用

**错误信息**:
```
MilvusException: enable_analyzer must be true for BM25 function
```

**原因**:输入字段没有启用 analyzer

**解决方案**:
```python
FieldSchema(
    name="text",
    dtype=DataType.VARCHAR,
    enable_analyzer=True,  # 添加这一行
    enable_match=True
)
```

## 性能优化建议

### 1. 索引参数调优

```python
# 稠密向量:HNSW 参数
{
    "M": 16,              # 增大 M 提高召回率,但增加内存
    "efConstruction": 256  # 增大 ef 提高索引质量,但增加构建时间
}

# 稀疏向量:SPARSE_INVERTED_INDEX 参数
{
    "drop_ratio_build": 0.2  # 丢弃低权重元素,减少索引大小
}
```

### 2. 批量插入优化

```python
# 分批插入大量数据
batch_size = 1000
for i in range(0, len(data), batch_size):
    batch = data[i:i+batch_size]
    collection.insert(batch)

    # 每 10 批刷新一次
    if (i // batch_size) % 10 == 0:
        collection.flush()
```

### 3. 内存优化

```python
# 使用 BFloat16Vector 减少内存占用
FieldSchema(
```

本场景展示了如何在 Milvus 中定义和管理多向量字段:

4. **BM25 集成**:使用 Function 自动生成稀疏向量
5. **数据插入**:稀疏向量使用字典格式 `{index: value}`
