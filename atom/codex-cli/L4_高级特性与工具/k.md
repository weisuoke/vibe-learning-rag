# L4_高级特性与工具 - 知识点列表

> 掌握 Codex CLI 的高级定制化能力和社区工具生态

---

## 学习目标

- 深入理解技能系统（Skills）
- 掌握 AGENTS.md 配置
- 学会使用社区工具生态
- 能够进行性能优化和监控

## 知识点列表

### 01. 技能系统（Skills） ⭐⭐⭐

**核心内容：**
- 创建自定义技能（SKILL.md）
- 驱动-专家循环模式
- 技能市场与社区技能
- 技能与 Multi-Agent 集成

**学习重点：**
- 理解技能系统的设计理念
- 掌握自定义技能的创建方法
- 学会使用驱动-专家循环模式

**实战场景：**
- 创建自定义技能扩展 Codex 能力
- 使用社区技能加速开发
- 将技能与 Multi-Agent 结合使用

---

### 02. AGENTS.md 配置 ⭐⭐⭐

**核心内容：**
- AGENTS.md 的作用与结构
- 测试与 Lint 配置
- 项目特定指令
- 与 Ralph Loop 结合使用

**学习重点：**
- 理解 AGENTS.md 的配置结构
- 掌握测试和 Lint 配置方法
- 学会编写项目特定指令

**实战场景：**
- 配置 AGENTS.md 定制项目行为
- 设置测试和 Lint 规则
- 与 Ralph Loop 结合实现自动化

---

### 03. 社区工具生态 ⭐⭐

**核心内容：**
- TSK: 开源 agent sandbox 与并行化工具
- Emdash 2.0: 多 worktree 并行运行
- ralph CLI: 自动 PRD 和计划生成
- feiskyer/codex-settings: 配置集合与提示模板

**学习重点：**
- 了解社区工具的功能和特点
- 掌握主要工具的使用方法
- 学会选择合适的工具

**实战场景：**
- 使用 TSK 实现并行化开发
- 使用 Emdash 管理多 worktree
- 使用 ralph CLI 自动生成 PRD

---

### 04. 性能优化与监控 ⭐

**核心内容：**
- 减少 API 调用与成本控制
- 内存管道优化
- 并发子代理配额管理
- 避免 Pro 计划配额瞬间耗尽

**学习重点：**
- 理解 API 调用的成本结构
- 掌握配额管理的最佳实践
- 学会监控和优化性能

**实战场景：**
- 优化 API 调用减少成本
- 配置并发限制避免配额耗尽
- 监控任务执行性能

## 学习路径建议

4. **优化性能**：提升效率降低成本
