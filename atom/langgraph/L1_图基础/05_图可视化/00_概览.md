## 一句话概括

**图可视化是 LangGraph 提供的将计算图结构转换为可视化图表的功能,通过 `get_graph()` 方法获取图对象并使用 Mermaid 格式渲染,是理解和调试复杂工作流的核心工具。**

3. **文档阶段**: 为项目文档生成高质量图表,提升可读性
4. **团队协作**: 分享图结构给团队成员,促进讨论和代码审查

- **理解复杂工作流**: 将抽象的代码逻辑转换为直观的可视化图表

## 常见问题

### Q1: 什么时候使用 draw_mermaid() vs draw_mermaid_png()?

**A**:
- **开发阶段**: 使用 `draw_mermaid()` 快速查看文本表示
- **文档阶段**: 使用 `draw_mermaid_png()` 生成高质量图像
- **调试阶段**: 使用 `draw_mermaid()` 配合文本编辑器查看

### Q2: xray 参数什么时候使用?

**A**:
- 当图包含子图时,使用 `xray=True` 递归展开所有子图
- 当只需要查看顶层结构时,使用 `xray=False` (默认)
- 当需要控制展开深度时,使用 `xray=N` (N 为层数)

### Q3: PNG 生成失败怎么办?

**A**:
- PNG 生成依赖 Mermaid.ink API,可能在某些环境中失败
- 可以先使用 `draw_mermaid()` 获取文本,然后使用在线工具渲染
- 检查网络连接和防火墙设置

### Q4: 如何在 Jupyter 外使用可视化?

**A**:
- 使用 `draw_mermaid()` 获取文本,保存到文件
- 使用 `draw_mermaid_png()` 获取字节数据,保存为 PNG 文件
- 使用在线 Mermaid 编辑器渲染文本
