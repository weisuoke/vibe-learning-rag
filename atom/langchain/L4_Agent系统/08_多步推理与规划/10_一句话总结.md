# 多步推理与规划 - 一句话总结

> LangChain Agent 系统 · 第 8 个知识点 · 维度 10/10

---

## 一句话总结

**多步推理与规划的本质是让 Agent 像项目经理一样工作——先用大模型制定 3-7 步粗粒度计划（Planner），再用小模型逐步执行（Executor），执行完评估是否需要调整（Replanner），必要时加入反思（Reflection）确保输出质量，用 LangGraph 的 StateGraph + conditional_edges 将这套 Plan-and-Execute 架构连接成可运行的状态机。**
