# 实战代码：自定义日志回调实现

## 场景说明

在 RAG 开发中，我们经常需要追踪 LLM 的调用详情，包括：
- 输入的 prompt 内容
- 输出的响应内容
- Token 用量统计
- 调用延迟时间
- 错误信息记录

通过自定义日志回调处理器，我们可以实现完整的 LLM 调用监控，为后续的性能优化和成本控制提供数据支持。

**在 RAG 中的应用场景：**
- 监控检索增强生成的每个步骤
- 追踪 LLM 调用的成本和性能
- 记录用户查询和系统响应
- 调试和优化 RAG 管道

---

## 核心概念

### 1. BaseCallbackHandler

LangChain 提供的基础回调处理器类，包含所有回调方法的默认实现。

**关键方法：**
- `on_llm_start()` - LLM 开始调用时触发
- `on_llm_end()` - LLM 调用结束时触发
- `on_llm_error()` - LLM 调用出错时触发
- `on_chain_start()` - Chain 开始执行时触发
- `on_chain_end()` - Chain 执行结束时触发

### 2. 回调参数

所有回调方法都遵循统一的参数模式：
- `run_id: UUID` - 当前运行的唯一标识
- `parent_run_id: UUID | None` - 父运行的标识（支持嵌套）
- `tags: list[str] | None` - 标签（用于分类和过滤）
- `metadata: dict[str, Any] | None` - 元数据
- `**kwargs: Any` - 额外参数

### 3. 日志记录策略

**前端类比：** 类似于 axios 的请求拦截器，在请求前后记录日志
**日常生活类比：** 像快递追踪系统，记录包裹的每个流转节点

## 完整代码

```python
"""
自定义日志回调实现
演示：记录 LLM 调用详情，包括 prompt、响应、token 用量和延迟
"""

import os
import time
import json
from typing import Any, Dict, List
from uuid import UUID
from datetime import datetime

from dotenv import load_dotenv
from langchain_openai import ChatOpenAI
from langchain_core.callbacks import BaseCallbackHandler
from langchain_core.outputs import LLMResult
from langchain_core.messages import HumanMessage

# 加载环境变量
load_dotenv()

# ===== 1. 自定义日志回调处理器 =====

class CustomLoggingCallbackHandler(BaseCallbackHandler):
    """
    自定义日志回调处理器
    记录 LLM 调用的完整生命周期
    """

    def __init__(self, log_file: str = "llm_calls.log"):
        """
        初始化回调处理器

        Args:
            log_file: 日志文件路径
        """
        super().__init__()
        self.log_file = log_file
        self.start_times: Dict[UUID, float] = {}  # 记录每次调用的开始时间

    def on_llm_start(
        self,
        serialized: Dict[str, Any],
        prompts: List[str],
        *,
        run_id: UUID,
        parent_run_id: UUID | None = None,
        tags: List[str] | None = None,
        metadata: Dict[str, Any] | None = None,
        **kwargs: Any,
    ) -> None:
        """LLM 开始调用时触发"""
        # 记录开始时间
        self.start_times[run_id] = time.time()

        # 构建日志信息
        log_entry = {
            "event": "llm_start",
            "timestamp": datetime.now().isoformat(),
            "run_id": str(run_id),
            "parent_run_id": str(parent_run_id) if parent_run_id else None,
            "model": serialized.get("name", "unknown"),
            "prompts": prompts,
            "tags": tags,
            "metadata": metadata,
        }

        # 写入日志
        self._write_log(log_entry)

        # 控制台输出
        print(f"\n{'='*60}")
        print(f"🚀 LLM 调用开始")
        print(f"Run ID: {run_id}")
        print(f"Model: {serialized.get('name', 'unknown')}")
        print(f"Prompt: {prompts[0][:100]}..." if prompts else "")
        print(f"{'='*60}\n")

    def on_llm_end(
        self,
        response: LLMResult,
        *,
        run_id: UUID,
        parent_run_id: UUID | None = None,
        tags: List[str] | None = None,
        **kwargs: Any,
    ) -> None:
        """LLM 调用结束时触发"""
        # 计算延迟
        start_time = self.start_times.get(run_id)
        latency = time.time() - start_time if start_time else 0

        # 提取响应内容
        output_text = ""
        if response.generations:
            output_text = response.generations[0][0].text

        # 提取 token 用量
        token_usage = {}
        if response.llm_output and "token_usage" in response.llm_output:
            token_usage = response.llm_output["token_usage"]

        # 构建日志信息
        log_entry = {
            "event": "llm_end",
            "timestamp": datetime.now().isoformat(),
            "run_id": str(run_id),
            "parent_run_id": str(parent_run_id) if parent_run_id else None,
            "output": output_text,
            "token_usage": token_usage,
            "latency_seconds": round(latency, 3),
            "tags": tags,
        }

        # 写入日志
        self._write_log(log_entry)

        # 控制台输出
        print(f"\n{'='*60}")
        print(f"✅ LLM 调用完成")
        print(f"Run ID: {run_id}")
        print(f"延迟: {latency:.3f}s")
        if token_usage:
            print(f"Token 用量: {token_usage}")
        print(f"响应: {output_text[:100]}..." if output_text else "")
        print(f"{'='*60}\n")

        # 清理开始时间记录
        if run_id in self.start_times:
            del self.start_times[run_id]

    def on_llm_error(
        self,
        error: BaseException,
        *,
        run_id: UUID,
        parent_run_id: UUID | None = None,
        tags: List[str] | None = None,
        **kwargs: Any,
    ) -> None:
        """LLM 调用出错时触发"""
        # 计算延迟
        start_time = self.start_times.get(run_id)
        latency = time.time() - start_time if start_time else 0

        # 构建日志信息
        log_entry = {
            "event": "llm_error",
            "timestamp": datetime.now().isoformat(),
            "run_id": str(run_id),
            "parent_run_id": str(parent_run_id) if parent_run_id else None,
            "error_type": type(error).__name__,
            "error_message": str(error),
            "latency_seconds": round(latency, 3),
            "tags": tags,
        }

        # 写入日志
        self._write_log(log_entry)

        # 控制台输出
        print(f"\n{'='*60}")
        print(f"❌ LLM 调用出错")
        print(f"Run ID: {run_id}")
        print(f"错误类型: {type(error).__name__}")
        print(f"错误信息: {str(error)}")
        print(f"{'='*60}\n")

        # 清理开始时间记录
        if run_id in self.start_times:
            del self.start_times[run_id]

    def _write_log(self, log_entry: Dict[str, Any]) -> None:
        """写入日志到文件"""
        with open(self.log_file, "a", encoding="utf-8") as f:
            f.write(json.dumps(log_entry, ensure_ascii=False) + "\n")


# ===== 2. 基础使用示例 =====

def basic_usage_example():
    """基础使用示例"""
    print("=== 场景1：基础日志记录 ===\n")

    # 创建自定义回调处理器
    logger = CustomLoggingCallbackHandler(log_file="basic_llm_calls.log")

    # 初始化 LLM（传入回调处理器）
    llm = ChatOpenAI(
        model="gpt-3.5-turbo",
        temperature=0.7,
        callbacks=[logger]  # 传入回调处理器
    )

    # 调用 LLM
    response = llm.invoke("什么是 RAG？请用一句话解释。")

    print(f"最终响应: {response.content}\n")

def rag_scenario_example():

    print("=== 场景2：RAG 场景日志追踪 ===\n")

    # 创建自定义回调处理器
    logger = CustomLoggingCallbackHandler(log_file="rag_llm_calls.log")

    # 步骤1：用户查询
    user_query = "LangChain 的 CallbackHandler 有什么作用？"

    # 步骤2：检索相关文档（模拟）

# ===== 4. 多次调用统计示例 =====

def multiple_calls_example():
    """多次调用统计示例"""
    print("=== 场景3：多次调用统计 ===\n")

    # 创建自定义回调处理器
    logger = CustomLoggingCallbackHandler(log_file="multiple_llm_calls.log")

    # 初始化 LLM
    llm = ChatOpenAI(
        model="gpt-3.5-turbo",
        temperature=0.7,
        callbacks=[logger]
    )

    # 多次调用
    questions = [
        "什么是向量数据库？",
        "什么是 Embedding？",
        "什么是语义检索？"
    ]

    for i, question in enumerate(questions, 1):
        print(f"问题 {i}: {question}")
        response = llm.invoke(question)
        print(f"答案 {i}: {response.content[:50]}...\n")
        time.sleep(1)  # 避免请求过快


# ===== 5. 日志分析工具 =====

def analyze_logs(log_file: str = "llm_calls.log"):
    """分析日志文件，统计调用情况"""
    print(f"=== 日志分析：{log_file} ===\n")

    if not os.path.exists(log_file):
        print(f"日志文件不存在: {log_file}")
        return

    total_calls = 0
    total_tokens = 0
    total_latency = 0.0
    errors = 0

    with open(log_file, "r", encoding="utf-8") as f:
        for line in f:
            try:
                log_entry = json.loads(line)

                if log_entry["event"] == "llm_end":
                    total_calls += 1

                    # 统计 token 用量
                    token_usage = log_entry.get("token_usage", {})
                    if "total_tokens" in token_usage:
                        total_tokens += token_usage["total_tokens"]

                    # 统计延迟
                    latency = log_entry.get("latency_seconds", 0)
                    total_latency += latency

                elif log_entry["event"] == "llm_error":
                    errors += 1

            except json.JSONDecodeError:
                continue

    # 输出统计结果
    print(f"总调用次数: {total_calls}")
    print(f"总 Token 用量: {total_tokens}")
    print(f"平均延迟: {total_latency / total_calls:.3f}s" if total_calls > 0 else "N/A")
    print(f"错误次数: {errors}")
    print(f"成功率: {(total_calls / (total_calls + errors) * 100):.2f}%" if (total_calls + errors) > 0 else "N/A")


# ===== 主函数 =====

if __name__ == "__main__":
    # 运行示例
    basic_usage_example()
    rag_scenario_example()
    multiple_calls_example()

    # 分析日志
    analyze_logs("basic_llm_calls.log")
    analyze_logs("rag_llm_calls.log")
    analyze_logs("multiple_llm_calls.log")
```

## 代码解析

### 1. 回调处理器设计

```python
class CustomLoggingCallbackHandler(BaseCallbackHandler):
    def __init__(self, log_file: str = "llm_calls.log"):
        super().__init__()
        self.log_file = log_file
        self.start_times: Dict[UUID, float] = {}
```

**关键点：**
- 继承 `BaseCallbackHandler`
- 使用字典记录每次调用的开始时间（通过 `run_id` 索引）
- 支持自定义日志文件路径

### 2. 延迟计算

```python
def on_llm_start(self, ..., run_id: UUID, ...):
    self.start_times[run_id] = time.time()

def on_llm_end(self, ..., run_id: UUID, ...):
    start_time = self.start_times.get(run_id)
    latency = time.time() - start_time if start_time else 0
```

**关键点：**
- 在 `on_llm_start` 中记录开始时间
- 在 `on_llm_end` 中计算延迟
- 使用 `run_id` 关联同一次调用

### 3. Token 用量提取

```python
token_usage = {}
if response.llm_output and "token_usage" in response.llm_output:
    token_usage = response.llm_output["token_usage"]
```

**关键点：**
- Token 用量信息在 `response.llm_output` 中
- 包含 `prompt_tokens`、`completion_tokens`、`total_tokens`

## 常见问题

### Q1: 如何在多线程环境中使用？

**A:** 每个线程创建独立的回调处理器实例：

```python
import threading

def worker(query: str):
    # 每个线程独立的回调处理器
    logger = CustomLoggingCallbackHandler(
        log_file=f"llm_calls_{threading.current_thread().name}.log"
    )
    llm = ChatOpenAI(callbacks=[logger])
    response = llm.invoke(query)
```

### Q2: 如何记录更详细的信息？

**A:** 重写更多回调方法：

```python
class DetailedLoggingCallbackHandler(CustomLoggingCallbackHandler):
    def on_chain_start(self, ...):
        # 记录 Chain 开始
        pass

    def on_tool_start(self, ...):
        # 记录 Tool 调用
        pass
```

### Q3: 如何实现日志轮转？

**A:** 使用 Python 的 `logging` 模块：

```python
import logging
from logging.handlers import RotatingFileHandler

class RotatingLoggingCallbackHandler(BaseCallbackHandler):
    def __init__(self):
        self.logger = logging.getLogger("llm_calls")
        handler = RotatingFileHandler(
            "llm_calls.log",
            maxBytes=10*1024*1024,  # 10MB
            backupCount=5
        )
        self.logger.addHandler(handler)
```
