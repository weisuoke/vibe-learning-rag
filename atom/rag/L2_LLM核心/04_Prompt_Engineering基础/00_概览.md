# Prompt Engineering 基础（RAG 专用）

> 设计 RAG 的系统提示词、用户问题模板

---

## 为什么要学这个？

Prompt 是你与 LLM 沟通的唯一方式。在 RAG 系统中，Prompt 设计直接决定了：

- **回答质量**：LLM 如何理解和使用你注入的检索内容
- **回答可控性**：LLM 是否按照你期望的格式和风格输出
- **幻觉控制**：LLM 是否会编造检索内容中没有的信息
- **边界处理**：当检索不到相关内容时，LLM 如何回应
