## 知识点列表

### 01. Ralph Loop 机制详解 ⭐⭐⭐

**核心内容：**
- Ralph Wiggum Loop 原理
- 循环运行多个短会话避免上下文积累
- PRD 清单验证完成
- "don't stop until..." 提示模式
- 社区工具：ralph CLI (Ian Nuttall)

**学习重点：**
- 理解 Ralph Loop 的设计原理
- 掌握循环运行短会话的技巧
- 学会使用 PRD 清单验证任务完成

**实战场景：**
- 使用 Ralph Loop 运行长时间任务
- 配置 PRD 清单自动验证
- 使用 ralph CLI 工具自动化流程

---
