class StrOutputParser(BaseTransformOutputParser[str]):
    """Extract text content from model outputs as a string.

    Converts model outputs (such as `AIMessage` or `AIMessageChunk` objects) into plain
    text strings. It's the simplest output parser and is useful when you need string
    responses for downstream processing, display, or storage.

    Supports streaming, yielding text chunks as they're generated by the model.

    Example:
        ```python
        from langchain_core.output_parsers import StrOutputParser
        from langchain_openai import ChatOpenAI

        model = ChatOpenAI(model="gpt-4o")
        parser = StrOutputParser()

        # Get string output from a model
        message = model.invoke("Tell me a joke")
        result = parser.invoke(message)
        print(result)  # plain string

        # With streaming - use transform() to process a stream
        stream = model.stream("Tell me a story")
        for chunk in parser.transform(stream):
            print(chunk, end="", flush=True)
        ```
    """
```

**关键特性**：
- 继承自 `BaseTransformOutputParser`（支持流式解析）
- 返回类型固定为 `str`
- 最简单的 OutputParser，几乎不做任何处理

### 6. _StreamingParser - 流式 XML 解析器

**类定义**（xml.py:42-100）：

```python
class _StreamingParser:
    """Streaming parser for XML.

    This implementation is pulled into a class to avoid implementation drift between
    `transform` and `atransform` of the `XMLOutputParser`.
    """

    def __init__(self, parser: Literal["defusedxml", "xml"]) -> None:
        """Initialize the streaming parser.

        Args:
            parser: Parser to use for XML parsing.

                Can be either `'defusedxml'` or `'xml'`. See documentation in
                `XMLOutputParser` for more information.

        Raises:
            ImportError: If `defusedxml` is not installed and the `defusedxml` parser is
                requested.
        """
        if parser == "defusedxml":
            if not _HAS_DEFUSEDXML:
                msg = (
                    "defusedxml is not installed. "
                    "Please install it to use the defusedxml parser."
                    "You can install it with `pip install defusedxml` "
                )
                raise ImportError(msg)
            parser_ = XMLParser(target=TreeBuilder())
        else:
            parser_ = None
        self.pull_parser = ET.XMLPullParser(["start", "end"], _parser=parser_)
        self.xml_start_re = re.compile(r"<[a-zA-Z:_]")
        self.current_path: list[str] = []
        self.current_path_has_children = False
        self.buffer = ""
        self.xml_started = False
```

**关键特性**：
- 使用 `XMLPullParser` 实现流式解析
- 支持 `defusedxml`（防止 XXE 攻击）
- 维护当前路径和缓冲区状态

### 7. defusedxml vs xml

**defusedxml（推荐）**：
- 防止 XML 外部实体（XXE）攻击
- 防止 XML 炸弹攻击
- 更安全，适合生产环境

**xml（标准库）**：
- Python 标准库，无需额外安装
- 性能更好
- 不安全，仅适合可信输入

**选择建议**：
```python
# 生产环境（推荐）
parser = XMLOutputParser(parser="defusedxml")

# 开发环境（可选）
parser = XMLOutputParser(parser="xml")
```

## 使用示例

### StrOutputParser 基础用法

```python
from langchain_core.output_parsers import StrOutputParser
from langchain_openai import ChatOpenAI

model = ChatOpenAI()
parser = StrOutputParser()

chain = model | parser

result = chain.invoke("Tell me a joke")
# 输出: "Why did the chicken cross the road? ..."
```

### StrOutputParser 流式用法

```python
# 流式输出
for chunk in chain.stream("Tell me a story"):
    print(chunk, end="", flush=True)
    # 输出: "Once", " ", "upon", " ", "a", " ", "time", ...
```

### XMLOutputParser 基础用法

```python
from langchain_core.output_parsers import XMLOutputParser
from langchain_openai import ChatOpenAI

model = ChatOpenAI()
parser = XMLOutputParser()

# 获取格式指令
format_instructions = parser.get_format_instructions(tags=["person", "name", "age"])

# 注入到 Prompt
prompt = f"Return a person's info in XML format.\n{format_instructions}"

chain = model | parser
result = chain.invoke(prompt)

```

```python
# 使用 defusedxml（推荐）

## 架构设计亮点

### StrOutputParser

1. **极简设计**：不做任何转换，直接返回输入

3. **序列化支持**：可以保存和加载配置

### XMLOutputParser

3. **格式指令**：提供 Prompt 模板
4. **路径跟踪**：维护当前 XML 路径
5. **错误处理**：捕获 XML 解析错误

## 性能考虑

### StrOutputParser

- **零开销**：不做任何转换
- **流式友好**：逐块输出，无延迟
- **内存友好**：不需要缓冲区

### XMLOutputParser

- **解析开销**：XML 解析比 JSON 慢
- **缓冲区开销**：需要维护缓冲区
- **安全开销**：`defusedxml` 比标准 `xml` 慢

## 安全考虑

### XMLOutputParser 安全问题

**XXE 攻击（XML 外部实体攻击）**：

```xml
<!DOCTYPE foo [
  <!ENTITY xxe SYSTEM "file:///etc/passwd">
]>
<foo>&xxe;</foo>
```

**防御措施**：
- 使用 `defusedxml` 解析器
- 禁用外部实体引用
- 验证 XML 输入

**XML 炸弹攻击**：

```xml
<!DOCTYPE lolz [
  <!ENTITY lol "lol">
  <!ENTITY lol2 "&lol;&lol;&lol;&lol;&lol;&lol;&lol;&lol;&lol;&lol;">
  <!ENTITY lol3 "&lol2;&lol2;&lol2;&lol2;&lol2;&lol2;&lol2;&lol2;&lol2;&lol2;">
]>
<lolz>&lol3;</lolz>
```

**防御措施**：
- 使用 `defusedxml` 解析器
- 限制 XML 大小
- 设置解析超时

## 总结

### StrOutputParser

- **最简单的解析器**：不做任何转换
- **流式友好**：逐块输出，无延迟
- **LCEL 兼容**：可以在 LCEL 中使用
- **适用场景**：需要纯文本输出时

### XMLOutputParser
