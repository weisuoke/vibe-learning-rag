# 30字核心

**交互式向导，一键配置 Gateway、Workspace、Channel、Daemon，让 OpenClaw 从零到可用**

---

## 核心定义

Onboarding Wizard（入职向导）是 OpenClaw 的**首次配置工具**，通过交互式命令行界面（CLI）引导用户完成从安装到首次对话的全部设置流程。

它的核心价值在于：**降低配置门槛，避免手动编辑 JSON 配置文件的复杂性和错误风险**。

> Onboarding Wizard 是 OpenClaw 的"安装向导"，就像 macOS 首次开机时的设置助手，通过问答式交互帮你完成所有必需配置，让你无需手动编辑配置文件就能启动 AI 助手。

---

## 核心命令

```bash
# 启动向导（推荐方式）
openclaw onboard

# 快速模式（使用默认值）
openclaw onboard --flow quickstart

# 高级模式（完全控制）
openclaw onboard --flow advanced

# 非交互模式（自动化脚本）
openclaw onboard --non-interactive \
  --auth-choice anthropic-api-key \
  --anthropic-api-key "$ANTHROPIC_API_KEY"
```

---

## 配置范围

向导会配置以下 7 个核心组件：

1. **Model/Auth** - 模型提供商和 API 密钥
2. **Workspace** - Agent 工作目录
3. **Gateway** - 网关服务器（端口、绑定、认证）
4. **Channels** - 聊天渠道（WhatsApp、Telegram、Discord 等）
5. **Daemon** - 守护进程（macOS LaunchAgent / Linux systemd）
6. **Health Check** - 健康检查（验证 Gateway 启动）
7. **Skills** - 技能安装（推荐技能和依赖）

---

## 两种模式对比

| 维度 | QuickStart（快速模式） | Advanced（高级模式） |
|------|----------------------|-------------------|
| **交互次数** | 最少（3-5 个问题） | 完整（10+ 个问题） |
| **Gateway 绑定** | Loopback (127.0.0.1) | 可选 LAN/Tailscale/自定义 |
| **认证方式** | Token（自动生成） | Token 或 Password |
| **Tailscale** | 关闭 | 可选 Serve/Funnel |
| **适用场景** | 本地开发、快速体验 | 生产部署、多机访问 |

---

## 核心价值

### 1. 降低门槛
- 无需理解 JSON5 配置语法
- 无需手动生成安全 Token
- 无需了解 systemd/launchd 配置

### 2. 避免错误
- 自动验证输入（端口号、IP 地址、API 密钥）
- 检测配置冲突（如 Tailscale Funnel 必须使用 Password 认证）
- 提供修复建议（如 `openclaw doctor --fix`）

### 3. 智能检测
- 检测已有配置（保留或更新）
- 检测 Gateway 运行状态（本地/远程）
- 检测 systemd 可用性（Linux 环境）

### 4. 一键安装
- 自动安装 LaunchAgent（macOS）
- 自动安装 systemd user service（Linux）
- 自动启动 Gateway 并验证健康状态

## 与其他配置方式的关系

```
openclaw onboard        # 首次配置（推荐）
    ↓
openclaw configure      # 后续调整（部分配置）
    ↓
openclaw config set     # 单个字段修改
    ↓
直接编辑 ~/.openclaw/openclaw.json  # 高级用户（支持热重载）
```

---

## 关键设计理念

1. **渐进式披露**：QuickStart 隐藏复杂选项，Advanced 暴露全部控制
2. **安全默认值**：Loopback 绑定 + Token 认证 + Pairing 策略
3. **幂等性**：重复运行不会破坏已有配置（除非选择 Reset）
4. **可恢复性**：配置错误时提示运行 `openclaw doctor`

## 日常生活类比

> 就像买新手机时的"设置向导"：
> - 选择语言（Model/Auth）
> - 连接 Wi-Fi（Gateway）
> - 登录账号（Channels）
> - 设置 Face ID（认证方式）
> - 安装推荐 App（Skills）
>
> 完成后手机就能正常使用，无需手动编辑系统配置文件。

## 核心文件位置

```bash
~/.openclaw/openclaw.json          # 配置文件（向导生成）
~/.openclaw/workspace/             # 默认工作目录
~/Library/LaunchAgents/            # macOS 守护进程（LaunchAgent）
~/.config/systemd/user/            # Linux 守护进程（systemd）
```
