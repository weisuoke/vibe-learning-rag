### Q2: 依赖安装失败怎么办？

**答案**：参考 07_实战代码_场景5_问题排查.md 的"依赖安装问题"章节。

常见原因：
- Node.js 版本过低（需要 22+）
- 缺少编译工具链
- 网络问题

---

### Q3: 构建速度太慢怎么办？

**答案**：参考 07_实战代码_场景6_性能优化.md 的"构建性能优化"章节。

优化策略：
- 启用 tsdown 缓存
- 优化 TypeScript 配置
- 减少入口点
- 使用 esbuild（开发环境）

---

### Q4: 如何选择 Vitest pool 模式？

**答案**：根据测试类型选择：
- 单元测试：threads（快速）
- E2E 测试：vmForks（隔离）
- Live 测试：单线程（避免并发）

详见 03_核心概念_3_Vitest测试系统.md

---

### Q5: 如何调试 OpenClaw？

**答案**：参考 07_实战代码_场景2_开发调试.md。

推荐方式：
- 配置 VS Code launch.json
- 使用断点调试
- 分析调用栈
- 使用 profiler
