# 07_实战代码_场景1_基础命令使用

## 场景概述

本文档展示如何使用 OpenClaw CLI 基础命令完成一个完整的工作流程：从安装配置到发送消息、Agent 交互、通道管理和配置调整。

**目标**：
- 掌握 CLI 基础命令的实际使用
- 理解命令之间的协作关系
- 学会构建自动化脚本
- 掌握错误处理和最佳实践

**涉及命令**：
- `openclaw gateway`
- `openclaw message send`
- `openclaw agent`
- `openclaw channels`
- `openclaw config`

## 场景 1: 快速启动工作流

### 需求描述

作为开发者，需要快速启动 OpenClaw 并发送第一条消息。

### 实现步骤

#### 1. 安装和初始化

```bash
# 1. 全局安装 OpenClaw
npm install -g openclaw@latest

# 2. 运行 Onboarding Wizard
openclaw onboard --install-daemon

# 3. 验证安装
openclaw --version
```

#### 2. 启动 Gateway

```bash
# 前台启动（开发环境）
openclaw gateway --verbose

# 或后台启动（生产环境）
openclaw gateway start

# 检查状态
openclaw gateway status
```

#### 3. 配置通道

```bash
# 添加 WhatsApp 通道
openclaw channels add whatsapp

# 扫码配对（会显示二维码）
# 等待连接成功

# 检查通道状态
openclaw channels status
```

#### 4. 发送第一条消息

```bash
# 发送消息到 WhatsApp
openclaw message send --to +1234567890 --message "Hello from OpenClaw!"

# 检查发送状态
openclaw channels logs whatsapp-1 --tail 10
```

#### 5. Agent 交互

```bash
# 简单问答
openclaw agent --message "What's the weather today?" --thinking low

# 复杂任务
openclaw agent --message "Create a deployment checklist" --thinking high
```

## 场景 2: TypeScript 自动化脚本

### 需求描述

使用 TypeScript 编写自动化脚本，封装 CLI 命令，实现完整的工作流程。

### 完整代码实现

```typescript
// src/openclaw-cli-wrapper.ts
import { exec } from 'child_process';
import { promisify } from 'util';
import * as readline from 'readline';

const execAsync = promisify(exec);

/**
 * OpenClaw CLI 封装类
 */
export class OpenClawCLI {
  private verbose: boolean;

  constructor(verbose = false) {
    this.verbose = verbose;
  }

  /**
   * 执行命令并返回结果
   */
  private async execute(command: string): Promise<string> {
    if (this.verbose) {
      console.log(`[CMD] ${command}`);
    }

    try {
      const { stdout, stderr } = await execAsync(command);

      if (stderr && this.verbose) {
        console.warn(`[WARN] ${stderr}`);
      }

      return stdout.trim();
    } catch (error: any) {
      throw new Error(`Command failed: ${command}\n${error.message}`);
    }
  }

  /**
   * 检查 OpenClaw 是否已安装
   */
  async isInstalled(): Promise<boolean> {
    try {
      await this.execute('openclaw --version');
      return true;
    } catch {
      return false;
    }
  }

  /**
   * 启动 Gateway
   */
  async startGateway(options: {
    port?: number;
    verbose?: boolean;
    daemon?: boolean;
  } = {}): Promise<void> {
    const { port = 18789, verbose = false, daemon = false } = options;

    if (daemon) {
      await this.execute('openclaw gateway start');
      console.log('✅ Gateway started as daemon');
    } else {
      const args = [
        `--port ${port}`,
        verbose ? '--verbose' : '',
      ].filter(Boolean).join(' ');

      console.log(`🚀 Starting Gateway on port ${port}...`);
      // 前台运行需要在单独的进程中
      exec(`openclaw gateway ${args}`);

      // 等待 Gateway 就绪
      await this.waitForGateway(port);
      console.log('✅ Gateway is ready');
    }
  }

  /**
   * 停止 Gateway
   */
  async stopGateway(): Promise<void> {
    await this.execute('openclaw gateway stop');
    console.log('✅ Gateway stopped');
  }

  /**
   * 重启 Gateway
   */
  async restartGateway(): Promise<void> {
    await this.execute('openclaw gateway restart');
    console.log('✅ Gateway restarted');
  }

  /**
   * 获取 Gateway 状态
   */
  async getGatewayStatus(): Promise<{
    status: string;
    port: number;
    uptime: number;
  }> {
    const output = await this.execute('openclaw gateway status --json');
    return JSON.parse(output);
  }

  /**
   * 等待 Gateway 就绪
   */
  private async waitForGateway(port: number, timeout = 30000): Promise<void> {
    const startTime = Date.now();

    while (Date.now() - startTime < timeout) {
      try {
        const response = await fetch(`http://localhost:${port}/health`);
        if (response.ok) {
          return;
        }
      } catch {
        // Gateway 还未就绪
      }

      await new Promise(resolve => setTimeout(resolve, 1000));
    }

    throw new Error('Gateway failed to start within timeout');
  }

  /**
   * 发送消息
   */
  async sendMessage(to: string, message: string): Promise<void> {
    await this.execute(`openclaw message send --to "${to}" --message "${message}"`);
    console.log(`✅ Message sent to ${to}`);
  }

  /**
   * Agent 交互
   */
  async askAgent(
    message: string,
    options: {
      thinking?: 'low' | 'medium' | 'high';
      deliver?: boolean;
    } = {}
  ): Promise<string> {
    const { thinking = 'medium', deliver = false } = options;

    const args = [
      `--message "${message}"`,
      `--thinking ${thinking}`,
      deliver ? '' : '--deliver=false',
    ].filter(Boolean).join(' ');

    const response = await this.execute(`openclaw agent ${args}`);
    return response;
  }

  /**
   * 列出所有通道
   */
  async listChannels(): Promise<Array<{
    id: string;
    type: string;
    status: string;
  }>> {
    const output = await this.execute('openclaw channels list --json');
    const result = JSON.parse(output);
    return result.channels;
  }

  /**
   * 获取通道状态
   */
  async getChannelStatus(): Promise<Record<string, string>> {
    const output = await this.execute('openclaw channels status --json');
    return JSON.parse(output);
  }

  /**
   * 添加通道
   */
  async addChannel(type: string, options?: Record<string, string>): Promise<void> {
    let command = `openclaw channels add ${type}`;

    if (options) {
      for (const [key, value] of Object.entries(options)) {
        command += ` --${key} ${value}`;
      }
    }

    await this.execute(command);
    console.log(`✅ Channel ${type} added`);
  }

  /**
   * 获取配置
   */
  async getConfig(key?: string): Promise<any> {
    if (key) {
      return await this.execute(`openclaw config get ${key}`);
    }

    const output = await this.execute('openclaw config get');
    return JSON.parse(output);
  }

  /**
   * 设置配置
   */
  async setConfig(key: string, value: string): Promise<void> {
    await this.execute(`openclaw config set ${key} ${value}`);
    console.log(`✅ Config set: ${key} = ${value}`);
  }

  /**
   * 健康检查
   */
  async healthCheck(): Promise<{
    gateway: boolean;
    channels: boolean;
    overall: boolean;
  }> {
    try {
      // 检查 Gateway
      const gatewayStatus = await this.getGatewayStatus();
      const gatewayHealthy = gatewayStatus.status === 'running';

      // 检查通道
      const channels = await this.listChannels();
      const connectedChannels = channels.filter(c => c.status === 'connected');
      const channelsHealthy = connectedChannels.length > 0;

      return {
        gateway: gatewayHealthy,
        channels: channelsHealthy,
        overall: gatewayHealthy && channelsHealthy,
      };
    } catch (error) {
      return {
        gateway: false,
        channels: false,
        overall: false,
      };
    }
  }
}

/**
 * 交互式 CLI 工具
 */
export class InteractiveCLI {
  private cli: OpenClawCLI;
  private rl: readline.Interface;

  constructor() {
    this.cli = new OpenClawCLI(true);
    this.rl = readline.createInterface({
      input: process.stdin,
      output: process.stdout,
    });
  }

  /**
   * 启动交互式会话
   */
  async start(): Promise<void> {
    console.log('🤖 OpenClaw Interactive CLI');
    console.log('Type "help" for available commands, "exit" to quit\n');

    await this.loop();
  }

  /**
   * 主循环
   */
  private async loop(): Promise<void> {
    const command = await this.prompt('> ');

    if (command === 'exit') {
      this.rl.close();
      return;
    }

    await this.handleCommand(command);
    await this.loop();
  }

  /**
   * 处理命令
   */
  private async handleCommand(command: string): Promise<void> {
    const [cmd, ...args] = command.split(' ');

    try {
      switch (cmd) {
        case 'help':
          this.showHelp();
          break;

        case 'status':
          await this.showStatus();
          break;

        case 'start':
          await this.cli.startGateway({ daemon: true });
          break;

        case 'stop':
          await this.cli.stopGateway();
          break;

        case 'restart':
          await this.cli.restartGateway();
          break;

        case 'send':
          const to = args[0];
          const message = args.slice(1).join(' ');
          await this.cli.sendMessage(to, message);
          break;

        case 'ask':
          const question = args.join(' ');
          const response = await this.cli.askAgent(question);
          console.log(`\n🤖 ${response}\n`);
          break;

        case 'channels':
          await this.showChannels();
          break;

        case 'config':
          const config = await this.cli.getConfig();
          console.log(JSON.stringify(config, null, 2));
          break;

        case 'health':
          await this.showHealth();
          break;

        default:
          console.log(`Unknown command: ${cmd}`);
          this.showHelp();
      }
    } catch (error: any) {
      console.error(`❌ Error: ${error.message}`);
    }
  }

  /**
   * 显示帮助
   */
  private showHelp(): void {
    console.log(`
Available commands:
  help                    - Show this help message
  status                  - Show Gateway status
  start                   - Start Gateway
  stop                    - Stop Gateway
  restart                 - Restart Gateway
  send <to> <message>     - Send message
  ask <question>          - Ask Agent
  channels                - List channels
  config                  - Show configuration
  health                  - Health check
  exit                    - Exit
    `);
  }

  /**
   * 显示状态
   */
  private async showStatus(): Promise<void> {
    const status = await this.cli.getGatewayStatus();
    console.log(`
Gateway Status:
  Status: ${status.status}
  Port: ${status.port}
  Uptime: ${Math.floor(status.uptime / 60)}m ${status.uptime % 60}s
    `);
  }

  /**
   * 显示通道
   */
  private async showChannels(): Promise<void> {
    const channels = await this.cli.listChannels();
    console.log('\nChannels:');
    channels.forEach(channel => {
      const icon = channel.status === 'connected' ? '✅' : '❌';
      console.log(`  ${icon} ${channel.type} (${channel.id}): ${channel.status}`);
    });
    console.log();
  }

  /**
   * 显示健康状态
   */
  private async showHealth(): Promise<void> {
    const health = await this.cli.healthCheck();
    console.log(`
Health Check:
  Gateway: ${health.gateway ? '✅' : '❌'}
  Channels: ${health.channels ? '✅' : '❌'}
  Overall: ${health.overall ? '✅ Healthy' : '❌ Unhealthy'}
    `);
  }

  /**
   * 提示输入
   */
  private prompt(question: string): Promise<string> {
    return new Promise(resolve => {
      this.rl.question(question, resolve);
    });
  }
}
```

```typescript
import { OpenClawCLI } from './openclaw-cli-wrapper';

async function main() {
  const cli = new OpenClawCLI(true);

    deliver: false,

  // 7. 查看通道
  const channels = await cli.listChannels();
  console.log('Channels:', channels);

  // 8. 查看配置
  const model = await cli.getConfig('model');
  console.log('Current model:', model);
}

main().catch(console.error);
```

## 场景 3: 批量消息发送

### 需求描述

批量发送消息到多个联系人，支持错误重试和进度跟踪。

### 实现代码

```typescript
// src/batch-sender.ts
import { OpenClawCLI } from './openclaw-cli-wrapper';

interface Message {
  to: string;
  content: string;
}

interface SendResult {
  to: string;
  success: boolean;
  error?: string;
}

export class BatchSender {
  private cli: OpenClawCLI;
  private maxRetries: number;
  private retryDelay: number;

  constructor(maxRetries = 3, retryDelay = 1000) {
    this.cli = new OpenClawCLI();
    this.maxRetries = maxRetries;
    this.retryDelay = retryDelay;
  }

  /**
   * 批量发送消息
   */
  async sendBatch(messages: Message[]): Promise<SendResult[]> {
    const results: SendResult[] = [];

    console.log(`📤 Sending ${messages.length} messages...`);

    for (let i = 0; i < messages.length; i++) {
      const message = messages[i];
      console.log(`[${i + 1}/${messages.length}] Sending to ${message.to}...`);

      const result = await this.sendWithRetry(message);
      results.push(result);

      if (result.success) {
        console.log(`✅ Sent to ${message.to}`);
      } else {
        console.error(`❌ Failed to send to ${message.to}: ${result.error}`);
      }

      // 避免速率限制
      await this.delay(500);
    }

    this.printSummary(results);
    return results;
  }

  /**
   * 发送消息（带重试）
   */
  private async sendWithRetry(message: Message): Promise<SendResult> {
    for (let attempt = 1; attempt <= this.maxRetries; attempt++) {
      try {
        await this.cli.sendMessage(message.to, message.content);
        return { to: message.to, success: true };
      } catch (error: any) {
        if (attempt === this.maxRetries) {
          return {
            to: message.to,
            success: false,
            error: error.message,
          };
        }

        console.warn(`⚠️  Retry ${attempt}/${this.maxRetries} for ${message.to}`);
        await this.delay(this.retryDelay * attempt);
      }
    }

    return {
      to: message.to,
      success: false,
      error: 'Max retries reached',
    };
  }

  /**
   * 打印摘要
   */
  private printSummary(results: SendResult[]): void {
    const successful = results.filter(r => r.success).length;
    const failed = results.filter(r => !r.success).length;

    console.log(`
📊 Batch Send Summary:
  Total: ${results.length}
  Successful: ${successful}
  Failed: ${failed}
  Success Rate: ${((successful / results.length) * 100).toFixed(1)}%
    `);

    if (failed > 0) {
      console.log('Failed messages:');
      results
        .filter(r => !r.success)
        .forEach(r => {
          console.log(`  - ${r.to}: ${r.error}`);
        });
    }
  }

  /**
   * 延迟
   */
  private delay(ms: number): Promise<void> {
    return new Promise(resolve => setTimeout(resolve, ms));
  }
}

// 使用示例
async function main() {
  const sender = new BatchSender();

  const messages: Message[] = [
    { to: '+1234567890', content: 'Hello from batch sender!' },
    { to: '+0987654321', content: 'This is a test message' },
    { to: '@telegram_user', content: 'Telegram message' },
  ];

  await sender.sendBatch(messages);
}

main().catch(console.error);
```

## 错误处理

### 常见错误和解决方案

```typescript
// src/error-handler.ts
export class OpenClawError extends Error {
  constructor(
    message: string,
    public code: string,
    public details?: any
  ) {
    super(message);
    this.name = 'OpenClawError';
  }
}

export class ErrorHandler {
  /**
   * 处理命令执行错误
   */
  static handleCommandError(error: any): never {
    if (error.message.includes('Port') && error.message.includes('in use')) {
      throw new OpenClawError(
        'Gateway port is already in use',
        'PORT_IN_USE',
        { suggestion: 'Use a different port or stop the existing Gateway' }
      );
    }

    if (error.message.includes('not found')) {
      throw new OpenClawError(
        'OpenClaw CLI not found',
        'CLI_NOT_FOUND',
        { suggestion: 'Install OpenClaw: npm install -g openclaw' }
      );
    }

    if (error.message.includes('timeout')) {
      throw new OpenClawError(
        'Gateway failed to start within timeout',
        'GATEWAY_TIMEOUT',
        { suggestion: 'Check Gateway logs for errors' }
      );
    }

    if (error.message.includes('Channel') && error.message.includes('disconnected')) {
      throw new OpenClawError(
        'Channel is disconnected',
        'CHANNEL_DISCONNECTED',
        { suggestion: 'Reconnect the channel: openclaw channels add <type>' }
      );
    }

    throw new OpenClawError(
      error.message,
      'UNKNOWN_ERROR',
      { originalError: error }
    );
  }

  /**
   * 安全执行命令
   */
  static async safeExecute<T>(
    fn: () => Promise<T>,
    fallback?: T
  ): Promise<T | undefined> {
    try {
      return await fn();
    } catch (error) {
      console.error('Error:', error);
      return fallback;
    }
  }
}
```
