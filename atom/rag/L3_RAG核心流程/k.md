# L3_RAG核心流程 知识点列表

> 掌握 RAG 系统的完整构建流程，从文档到答案的端到端实现

---

## 知识点清单

1. **RAG架构全景** - 理解 RAG 的整体架构、核心组件与数据流
2. **文档加载与解析** - 掌握多格式文档的加载、解析与预处理
3. **文本分块Chunking** - 理解分块策略对检索质量的关键影响
4. **向量存储（RAG视角）** - 掌握面向 RAG 的向量库配置与索引策略
5. **检索器设计** - 实现高效的语义检索与相关性匹配
6. **上下文注入与生成** - 将检索结果组装成 Prompt 并生成高质量答案

---

## 学习顺序建议

```
RAG架构全景 → 文档加载与解析 → 文本分块Chunking → 向量存储 → 检索器设计 → 上下文注入与生成
     ↓              ↓                ↓              ↓            ↓              ↓
  建立全局认知    掌握数据入口      理解分块策略    配置存储层    实现检索逻辑    完成生成闭环
```

**为什么是这个顺序？**

1. **先全景后细节**：理解 RAG 整体架构，才能明白每个组件的作用和位置
2. **按数据流顺序**：文档 → 分块 → 向量化 → 存储 → 检索 → 生成，符合实际处理流程
3. **离线与在线分离**：前4个是离线索引阶段，后2个是在线查询阶段
