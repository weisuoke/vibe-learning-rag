## 常见问题

### Q1: 如何选择实战项目？
**A:** 根据你的需求和技能选择：Coding Agent（个人工具）、Slack Bot（团队协作）、Web 应用（用户界面）、多 Agent（复杂任务）。

### Q2: 生产环境部署需要注意什么？
**A:** 安全性（API key 保护）、可靠性（错误处理）、性能（资源优化）、监控（日志和指标）。

### Q3: 如何优化 Agent 的成本？
**A:** 使用更便宜的模型（如 Haiku）、减少 API 调用、缓存结果、优化 Prompt。

### Q4: 多 Agent 协作的最佳实践？
**A:** 清晰的职责划分、高效的通信协议、错误处理和重试、性能监控。

### Q5: 如何调试生产环境的问题？
**A:** 使用日志（Winston/Pino）、监控（Prometheus/Grafana）、错误追踪（Sentry）、性能分析（Node.js Profiler）。

### Q6: 如何处理 Agent 的幻觉问题？
**A:** 使用 ReRank、验证输出、提供清晰的上下文、使用更强的模型。
