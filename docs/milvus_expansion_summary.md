## 🆕 新增知识点详情

### L1_快速入门（+2个知识点）

**04. Milvus架构概览**
- 理解 Proxy、QueryNode、DataNode、IndexNode 等核心组件
- 掌握 Milvus 的分布式架构设计
- 理解各组件的职责和交互方式

**05. 数据一致性级别**
- 掌握 Strong、Bounded、Eventually 三种一致性级别
- 理解一致性与性能的权衡
- 根据场景选择合适的一致性级别

---

### L2_核心功能（+2个知识点）

**04. 高级索引类型**
- GPU 索引（GPU_IVF_FLAT, GPU_IVF_PQ）
- 量化索引（PQ, SQ）的原理和应用
- 稀疏向量索引
- 索引类型的性能对比

**05. 多向量检索**
- Multi-vector Search 的应用场景
- 多模态检索（文本+图片）
- 实现方式和性能优化

---

### L3_高级特性（+2个知识点）

**04. 动态Schema**
- 灵活管理 Collection 字段
- 支持动态字段添加
- Schema 演化的最佳实践

**05. 数据一致性与持久化**
- WAL（Write-Ahead Log）机制
- Compaction 压缩策略
- Segment 管理
- 数据持久化保障

---

### L4_性能优化（+2个知识点）

**04. 性能基准测试**
- 性能测试方法论
- 测试工具（VectorDBBench, Milvus Benchmark）
- 关键指标分析（QPS, P99延迟, 召回率）
- 性能瓶颈诊断

**05. 分布式优化**
- 负载均衡策略
- 网络优化（gRPC 参数调优）
- 存储引擎优化（RocksDB 配置）
- 内存映射（mmap）配置

---

### L5_生产实践（+3个知识点）

**04. Kubernetes部署**
- 使用 Milvus Operator 部署
- Helm Charts 配置
- K8s 资源管理
- 滚动更新和版本管理

**05. 安全与权限管理**
- 用户认证配置
- 角色和权限管理
- 数据加密（传输加密、存储加密）
- 审计日志

**06. 高可用集群**
- 多副本配置
- 故障转移机制
- 灾难恢复方案
- 跨数据中心部署

---

### L6_RAG集成实战（+5个知识点，新增层级）

**01. 文档问答系统实现**
- 端到端 RAG 系统构建
- 文档加载、分块、向量化
- Milvus 存储与检索
- LLM 生成答案

**02. 多租户知识库**
- 租户隔离策略（Partition vs 标量过滤）
- 权限控制实现
- 数据安全保障
- 性能优化

**03. 大规模向量检索优化**
- 百万级、千万级向量处理
- 索引选择和参数调优
- 查询优化策略
- 成本控制

**04. Milvus与LangChain集成**
- LangChain VectorStore 接口
- 快速构建 RAG 应用
- Chain 和 Agent 集成
- 实战案例

**05. Milvus与LlamaIndex集成**
- LlamaIndex 数据连接器
- 索引类型选择
- 查询引擎配置
- 高级 RAG 模式
