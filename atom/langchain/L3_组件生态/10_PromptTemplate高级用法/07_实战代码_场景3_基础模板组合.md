# 实战代码 - 场景3：基础模板组合

> 使用 `+` 操作符实现 PromptTemplate 的基础组合模式

---

## 场景概述

**核心目标：** 掌握 LangChain PromptTemplate 的基础组合技术，使用 `+` 操作符实现模板的灵活拼接和复用。

**适用场景：**
- 构建多段式提示词（系统提示 + 用户输入）
- 复用通用模板片段（如角色设定、输出格式）
- 动态组装提示词结构
- 模块化提示词工程

**技术要点：**
- PromptTemplate + PromptTemplate 组合
- PromptTemplate + str 组合
- 变量自动合并机制
- 常见组合模式

---

## 双重类比

### 前端类比：字符串模板拼接

```javascript
// 前端：模板字符串拼接
const header = `<header>${title}</header>`;
const body = `<body>${content}</body>`;
const page = header + body;  // 直接拼接

// LangChain：PromptTemplate 组合
const header_prompt = PromptTemplate.from_template("Role: {role}");
const body_prompt = PromptTemplate.from_template("Task: {task}");
const full_prompt = header_prompt + body_prompt;  // 智能合并
```

**相似点：**
- 都是将多个模板片段组合成完整模板
- 都支持变量占位符
- 都可以动态填充内容

**区别：**
- LangChain 会自动合并变量列表
- LangChain 会检查 partial_variables 冲突
- LangChain 要求模板格式一致

### 日常生活类比：乐高积木拼装

**场景：** 用乐高积木搭建房子

```
基础模块：地基（foundation）
中间模块：墙壁（walls）
顶部模块：屋顶（roof）

完整房子 = 地基 + 墙壁 + 屋顶
```

**对应关系：**
- 每个乐高模块 = 一个 PromptTemplate
- 模块的连接点 = 变量（input_variables）
- 拼装后的房子 = 组合后的完整 Prompt
- 模块可以重复使用 = 模板复用

## 源码原理解析

### `__add__` 方法实现

**来源：** [sourcecode/langchain/libs/core/langchain_core/prompts/prompt.py:142-184]

```python
def __add__(self, other: Any) -> PromptTemplate:
    """Override the `+` operator to allow for combining prompt templates."""
    if isinstance(other, PromptTemplate):
        # 1. 检查模板格式是否一致
        if self.template_format != other.template_format:
            msg = "Cannot add templates of different formats"
            raise ValueError(msg)

        # 2. 合并 input_variables（取并集）
        input_variables = list(
            set(self.input_variables) | set(other.input_variables)
        )

        # 3. 拼接模板字符串
        template = self.template + other.template

        # 4. 合并 validate_template（都为 True 才为 True）
        validate_template = self.validate_template and other.validate_template

        # 5. 合并 partial_variables（检查冲突）
        partial_variables = dict(self.partial_variables.items())
        for k, v in other.partial_variables.items():
            if k in partial_variables:
                msg = "Cannot have same variable partialed twice."
                raise ValueError(msg)
            partial_variables[k] = v

        # 6. 返回新的 PromptTemplate
        return PromptTemplate(
            template=template,
            input_variables=input_variables,
            partial_variables=partial_variables,
            template_format=self.template_format,
            validate_template=validate_template,
        )

    if isinstance(other, str):
        # 支持 PromptTemplate + str
        prompt = PromptTemplate.from_template(
            other,
            template_format=self.template_format,
        )
        return self + prompt
```

**关键特性：**
1. **自动合并变量**：input_variables 取并集，无需手动指定
2. **冲突检查**：partial_variables 不能有重复的键
3. **格式一致性**：两个模板的 template_format 必须相同
4. **字符串支持**：可以直接 + 字符串，自动转换为 PromptTemplate

## 完整代码示例

```python
"""
场景3：基础模板组合实战
演示：使用 + 操作符组合 PromptTemplate
"""

import os
from dotenv import load_dotenv
from langchain_core.prompts import PromptTemplate
from langchain_openai import ChatOpenAI

# 加载环境变量
load_dotenv()

print("=" * 80)
print("场景3：基础模板组合实战")
print("=" * 80)

# ===== 1. PromptTemplate + PromptTemplate 组合 =====
print("\n【示例1】PromptTemplate + PromptTemplate 组合")
print("-" * 80)

# 创建两个独立的模板
system_prompt = PromptTemplate.from_template(
    "You are a {role} assistant.\n"
)

user_prompt = PromptTemplate.from_template(
    "Please help me with: {task}\n"
)

# 使用 + 操作符组合
combined_prompt = system_prompt + user_prompt

print("✓ 组合后的模板：")
print(combined_prompt.template)
print(f"\n✓ 自动合并的变量：{combined_prompt.input_variables}")

# 格式化输出
result = combined_prompt.format(
    role="Python programming",
    task="write a function to calculate fibonacci numbers"
)
print(f"\n✓ 格式化结果：\n{result}")

# ===== 2. PromptTemplate + str 组合 =====
print("\n【示例2】PromptTemplate + str 组合")
print("-" * 80)

# 创建基础模板
base_prompt = PromptTemplate.from_template(
    "Translate the following text to {language}:\n"
)

# 直接 + 字符串（自动转换为 PromptTemplate）
full_prompt = base_prompt + "Text: {text}\n"

print("✓ 组合后的模板：")
print(full_prompt.template)
print(f"\n✓ 自动合并的变量：{full_prompt.input_variables}")

# 格式化输出
result = full_prompt.format(
    language="French",
    text="Hello, how are you?"
)
print(f"\n✓ 格式化结果：\n{result}")

# ===== 3. 变量自动合并机制 =====
print("\n【示例3】变量自动合并机制")
print("-" * 80)

# 创建三个模板，包含不同的变量
prompt1 = PromptTemplate.from_template("Role: {role}\n")
prompt2 = PromptTemplate.from_template("Task: {task}\n")
prompt3 = PromptTemplate.from_template("Context: {context}\n")

# 链式组合
final_prompt = prompt1 + prompt2 + prompt3

print("✓ 组合后的模板：")
print(final_prompt.template)
print(f"\n✓ 自动合并的变量（并集）：{final_prompt.input_variables}")

# 格式化输出
result = final_prompt.format(
    role="Data Analyst",
    task="Analyze sales data",
    context="Q4 2025 sales report"
)
print(f"\n✓ 格式化结果：\n{result}")

# ===== 4. 常见模式：系统提示 + 用户输入 =====
print("\n【示例4】常见模式：系统提示 + 用户输入")
print("-" * 80)

# 系统提示（固定部分）
system_template = PromptTemplate.from_template(
    """You are an AI assistant specialized in {domain}.
Your responses should be:
- Clear and concise
- Technically accurate
- Beginner-friendly

"""
)

# 用户输入（动态部分）
user_template = PromptTemplate.from_template(
    "User Question: {question}\n"
)

# 输出格式（固定部分）
format_template = PromptTemplate.from_template(
    "Please provide your answer in {format} format.\n"
)

# 组合完整提示词
complete_prompt = system_template + user_template + format_template

print("✓ 完整提示词模板：")
print(complete_prompt.template)
print(f"\n✓ 需要的变量：{complete_prompt.input_variables}")

# 格式化输出
result = complete_prompt.format(
    domain="Python programming",
    question="What is a decorator?",
    format="markdown"
)
print(f"\n✓ 格式化结果：\n{result}")

# ===== 5. 模板复用模式 =====
print("\n【示例5】模板复用模式")

# 场景1：代码生成

print("✓ 代码生成提示词：")

# 场景2：文档撰写

doc_prompt = role_prefix + doc_task + output_format

print("\n✓ 文档撰写提示词：")

    role="Technical writer",
    topic="REST API endpoints",
    format="Markdown with examples"
)
print(result2)

# ===== 6. 与 LLM 集成使用 =====
print("\n【示例6】与 LLM 集成使用")
print("-" * 80)

# 创建组合提示词

    "You are a helpful assistant.\n"
)

    "Task: {task}\n"

print("\n【示例8】常见组合模式总结")
print("-" * 80)

# 模式1：前缀 + 主体 + 后缀

print(pattern2.format(
    role="Teacher",
    task="Explain quantum physics",
    constraint="Use simple language"

# 模式3：上下文 + 问题 + 格式
context = PromptTemplate.from_template("Context: {context}\n")
question = PromptTemplate.from_template("Question: {question}\n")
format_spec = PromptTemplate.from_template("Format: {format}\n")
pattern3 = context + question + format_spec

print("\n✓ 模式3（上下文+问题+格式）：")
print(pattern3.format(
    context="Machine learning basics",
    question="What is overfitting?",
    format="Bullet points"
))

print("\n" + "=" * 80)
print("✓ 基础模板组合实战完成！")
```

## 核心要点总结

### 1. 组合方式

| 组合类型 | 语法 | 说明 |
|---------|------|------|
| 模板 + 模板 | `prompt1 + prompt2` | 自动合并变量和模板字符串 |
| 模板 + 字符串 | `prompt + "text"` | 字符串自动转换为模板 |
| 链式组合 | `p1 + p2 + p3` | 支持多个模板连续组合 |

### 2. 变量合并规则

**input_variables：** 取并集（union）
```python
prompt1.input_variables = ['a', 'b']
prompt2.input_variables = ['b', 'c']
combined.input_variables = ['a', 'b', 'c']  # 自动去重
```

**partial_variables：** 检查冲突
```python
prompt1.partial_variables = {'x': 'value1'}
prompt2.partial_variables = {'x': 'value2'}  # ✗ 冲突！
# 会抛出 ValueError: Cannot have same variable partialed twice.
```

### 3. 注意事项

**必须满足的条件：**
- ✅ 两个模板的 `template_format` 必须一致
- ✅ `partial_variables` 不能有重复的键
- ✅ 组合后的模板仍然是 PromptTemplate 对象

**推荐做法：**
- ✅ 使用 `+` 操作符而非 PipelinePromptTemplate（更简洁）
- ✅ 将通用片段提取为可复用模板
- ✅ 按照逻辑顺序组合（系统提示 → 用户输入 → 输出格式）

## 实际应用场景

### 场景1：多语言提示词管理

```python
# 定义语言无关的结构

```

### 场景2：RAG 系统提示词构建

```python
# RAG 系统的标准结构
system_role = PromptTemplate.from_template(
    "You are a helpful assistant that answers questions based on provided context.\n"
)

    "Context:\n{context}\n\n"
)

    "Question: {question}\n"

    "Please provide a concise answer based only on the context above.\n"

```
