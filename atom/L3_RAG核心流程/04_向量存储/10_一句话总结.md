# 一句话总结

**向量存储是将 Embedding 向量持久化并通过索引（HNSW/IVF）实现毫秒级相似度搜索的专用数据库，是 RAG 系统实现高效语义检索的核心基础设施。**

---

## 核心要点回顾

| 维度 | 核心内容 |
|------|----------|
| 本质 | 持久化 + 索引 + 相似度搜索 |
| 价值 | 避免重复计算，加速检索 |
| 索引 | HNSW（推荐）、IVF、Flat |
| 度量 | 余弦相似度（RAG 首选） |
| 选型 | ChromaDB（入门）→ Milvus（生产） |

## 快速参考

```python
# ChromaDB 最小示例
import chromadb
client = chromadb.Client()
collection = client.create_collection("docs")
collection.add(ids=["1"], documents=["内容"])
results = collection.query(query_texts=["查询"], n_results=5)

# Milvus 最小示例
from pymilvus import MilvusClient
client = MilvusClient("./db.db")
client.create_collection("docs", dimension=384)
client.insert("docs", [{"id": 1, "vector": [...]}])
results = client.search("docs", data=[query_vec], limit=5)
```
