# 状态验证 - 一句话总结

**状态验证是通过 Pydantic BaseModel 为 LangGraph 状态添加运行时类型检查、自动类型转换和自定义业务约束的机制，在图的入口处拦截非法数据，确保状态在多节点工作流中安全流转。**

## 核心公式

```
状态验证 = Pydantic BaseModel（运行时验证）
         + field_validator（字段级约束）
         + model_validator（跨字段约束）
         + ValidationError（结构化错误处理）
```

## 记忆锚点

> 状态验证就像机场安检：在入口处（图的首个节点）进行严格检查，
> 一旦通过安检进入候机区（图的内部节点），就不再重复检查。
> TypedDict 是登机牌上的信息（静态提示），Pydantic 是安检设备（实际检查）。
