## 一句话总结

**LangGraph 的循环与迭代是通过边指向已有节点形成有向环，由 PregelLoop 引擎驱动 tick-execute-apply 迭代，以条件边控制终止、递归限制保障安全，支撑 ReAct 推理、自我修正、迭代检索等核心 Agent 模式的基础能力。**

---

## 核心要点速查

| 维度 | 核心内容 |
|------|----------|
| 本质 | 边指向已有节点 → 有向环 → `branch:to:{node}` channel 触发 |
| 构建方式 | 静态边回连、条件边、Command(goto)、Send |
| 终止策略 | 条件终止、TTL计数器、RemainingSteps、recursion_limit |
| 执行引擎 | PregelLoop: tick() → runner.tick() → after_tick() |
| 安全机制 | 递归限制（默认25）、GraphRecursionError、优雅降级 |
| 典型模式 | ReAct循环、Generator-Critic、迭代检索、Map-Reduce |
