## 核心知识点

掌握以下5个核心知识点，就能开始使用LangChain缓存机制：

### 1. 启用LLM缓存（3分钟）

**核心概念**：
- 使用`set_llm_cache()`全局启用缓存
- 所有LLM调用自动使用缓存
- 开发环境用InMemoryCache

**代码示例**：
```python
from langchain_core.caches import InMemoryCache
from langchain_core.globals import set_llm_cache
from langchain_openai import ChatOpenAI

# 1. 启用缓存
set_llm_cache(InMemoryCache())

# 2. 正常使用LLM
llm = ChatOpenAI(model="gpt-4o-mini")
response = llm.invoke("What is LangChain?")

# 3. 再次调用（缓存命中）
response2 = llm.invoke("What is LangChain?")  # 快速返回
```

**关键点**：
- 一行代码启用缓存
- 无需修改业务代码
- 立即生效

---

### 2. 使用Embedding缓存（5分钟）

**核心概念**：
- 使用`CacheBackedEmbeddings`包装器
- 避免重复计算相同文本的embedding
- 必须设置namespace

**代码示例**：
```python
from langchain_classic.embeddings import CacheBackedEmbeddings
from langchain_classic.storage import LocalFileStore
from langchain_openai import OpenAIEmbeddings

# 1. 创建存储
store = LocalFileStore("./cache/")

# 2. 创建缓存embedder
underlying_embeddings = OpenAIEmbeddings()
cached_embedder = CacheBackedEmbeddings.from_bytes_store(
    underlying_embeddings,
    store,
    namespace=underlying_embeddings.model  # 必须设置
)

# 3. 使用缓存embedder
embeddings = cached_embedder.embed_documents(["hello", "world"])
```

**关键点**：
- 包装器模式，透明缓存
- namespace避免冲突
- 首次计算，后续缓存

---

### 3. 选择合适的缓存后端（2分钟）

**决策树**：
```
开发环境 → InMemoryCache
单机应用 + 需要持久化 → SQLiteCache
生产环境 + 分布式 → RedisCache
```

**代码示例**：
```python
import os
from langchain_core.caches import InMemoryCache
from langchain_community.cache import SQLiteCache, RedisCache
from langchain_core.globals import set_llm_cache
import redis

# 根据环境选择
env = os.getenv("ENV", "development")

if env == "development":
    # 开发环境：内存缓存
    set_llm_cache(InMemoryCache())

elif env == "testing":
    # 测试环境：SQLite缓存
    set_llm_cache(SQLiteCache(database_path=".cache.db"))

elif env == "production":
    # 生产环境：Redis缓存
    redis_client = redis.Redis(host='localhost', port=6379)
    set_llm_cache(RedisCache(redis_client))
```

**关键点**：
- 开发用InMemory（快速）
- 生产用Redis（可靠）
- 根据需求选择

### 4. 监控缓存效果（5分钟）

**核心概念**：
- 监控命中率
- 计算成本节省
- 分析性能提升

**代码示例**：
```python
from langchain_core.caches import InMemoryCache
from langchain_core.globals import set_llm_cache, get_llm_cache
from langchain_openai import ChatOpenAI
import time

# 自定义监控缓存
class MonitoredCache(InMemoryCache):
    def __init__(self, **kwargs):
        super().__init__(**kwargs)
        self.hits = 0
        self.misses = 0
        self.total_time_saved = 0

    def lookup(self, prompt, llm_string):
        result = super().lookup(prompt, llm_string)
        if result:
            self.hits += 1
            self.total_time_saved += 2.0  # 假设每次节省2秒
        else:
            self.misses += 1
        return result

    def get_stats(self):
        total = self.hits + self.misses
        hit_rate = self.hits / total if total > 0 else 0
        return {
            "命中率": f"{hit_rate:.2%}",
            "命中次数": self.hits,
            "未命中次数": self.misses,
            "节省时间": f"{self.total_time_saved:.2f}秒"
        }

# 使用监控缓存
cache = MonitoredCache()
set_llm_cache(cache)

llm = ChatOpenAI(model="gpt-4o-mini")

# 模拟请求
for i in range(10):
    llm.invoke("What is AI?")

# 查看统计
print(cache.get_stats())
```

**关键点**：
- 继承基础缓存类
- 记录命中/未命中
- 定期查看统计

### 5. 常见问题排查（3分钟）

**问题1：缓存没有命中**

**检查清单**：
```python
# 1. 检查prompt是否完全相同
prompt1 = "What is AI?"
prompt2 = "What is AI? "  # 注意末尾空格

# 2. 检查配置是否相同
llm1 = ChatOpenAI(model="gpt-4o-mini", temperature=0.7)
llm2 = ChatOpenAI(model="gpt-4o-mini", temperature=0.9)  # 配置不同

# 3. 规范化prompt
def normalize_prompt(prompt: str) -> str:
    return prompt.strip()

prompt = normalize_prompt(user_input)
response = llm.invoke(prompt)
```

**问题2：内存占用过高**

**解决方案**：
```python
# 设置maxsize限制
set_llm_cache(InMemoryCache(maxsize=1000))

# 或使用持久化缓存
from langchain_community.cache import SQLiteCache
set_llm_cache(SQLiteCache(database_path=".cache.db"))
```

**问题3：生产环境缓存丢失**

**解决方案**：
```python
# 不要使用InMemoryCache
# ❌ set_llm_cache(InMemoryCache())

# 使用持久化缓存
# ✅ Redis或SQLite
import redis
from langchain_redis import RedisCache

redis_client = redis.Redis(host='localhost', port=6379)
set_llm_cache(RedisCache(redis_client))
```

### 场景2：RAG系统Embedding缓存

```python
from langchain_classic.embeddings import CacheBackedEmbeddings
from langchain_classic.storage import LocalFileStore
from langchain_openai import OpenAIEmbeddings
from langchain_community.vectorstores import Chroma
from langchain_community.document_loaders import DirectoryLoader
from langchain_text_splitters import RecursiveCharacterTextSplitter

# 1. 创建缓存embedder
store = LocalFileStore("./embedding_cache/")
underlying_embeddings = OpenAIEmbeddings()
cached_embedder = CacheBackedEmbeddings.from_bytes_store(
    underlying_embeddings,
    store,
    namespace=underlying_embeddings.model
)

# 2. 加载文档
loader = DirectoryLoader("./docs/", glob="**/*.md")
documents = loader.load()

# 3. 分块
text_splitter = RecursiveCharacterTextSplitter(chunk_size=1000)
chunks = text_splitter.split_documents(documents)

# 4. 构建向量库（使用缓存）
vectorstore = Chroma.from_documents(
    documents=chunks,
    embedding=cached_embedder  # 使用缓存embedder
)

print("向量库构建完成，重复文本已缓存！")
```

### 场景3：生产环境Redis缓存

```python
import os
import redis
from langchain_redis import RedisCache
from langchain_core.globals import set_llm_cache
from langchain_openai import ChatOpenAI

# 1. 连接Redis
redis_client = redis.Redis(
    host=os.getenv("REDIS_HOST", "localhost"),
    port=int(os.getenv("REDIS_PORT", 6379)),
    db=0,
    decode_responses=True
)

# 2. 启用Redis缓存
set_llm_cache(RedisCache(redis_client))

# 3. 使用LLM
llm = ChatOpenAI(model="gpt-4o-mini")
response = llm.invoke("What is LangChain?")

print("生产环境缓存已启用！")
```
