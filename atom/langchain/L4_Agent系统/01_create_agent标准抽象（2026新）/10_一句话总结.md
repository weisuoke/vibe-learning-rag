## 一句话总结

**create_agent 是 LangChain 1.0 的统一 Agent 工厂函数，通过 Middleware 中间件模式解决了上下文工程的可组合性问题，底层基于 LangGraph StateGraph 提供生产级 Agent 构建能力。**

---

## 关键记忆点

- `create_agent` = 模型 + 工具 + 中间件 → `CompiledStateGraph`
- Middleware = 6 种钩子 × 可组合 = 细粒度控制
- 一个函数取代 6+ 种旧版 Agent 创建方式
