# OpenClaw 基础故障排查 - 核心概念 2：日志系统与查看

## 一、概念定义

**OpenClaw 日志系统是基于 tslog 的结构化日志记录机制，通过实时日志跟踪和特征识别，快速定位系统运行时的问题根源。**

**来源**: `reference/source_debugging_tools.md`

---

## 二、核心价值

### 2.1 为什么需要日志系统？

**问题场景**：
- 错误发生后无法复现
- 不知道系统在做什么
- 多个组件交互时难以追踪
- 瞬时错误难以捕获
- 生产环境无法调试

**传统解决方式的问题**：
- ❌ console.log 调试 → 生产环境不可用
- ❌ 断点调试 → 无法追踪异步流程
- ❌ 手动记录 → 容易遗漏关键信息
- ❌ 事后分析 → 缺少上下文

**日志系统的价值**：
- ✅ 实时跟踪系统行为
- ✅ 记录完整的上下文信息
- ✅ 支持多级别日志过滤
- ✅ 结构化数据便于分析
- ✅ 持久化存储便于回溯

**来源**: `reference/source_debugging_tools.md`

### 4.3 日志特征匹配

```typescript
// 日志特征匹配引擎
interface LogMatcher {
  signatures: LogSignature[];
  match(logLine: string): LogSignature | null;
}

class LogAnalyzer implements LogMatcher {
  signatures: LogSignature[] = [
    {
      pattern: /EADDRINUSE.*18789/,
      severity: 'fatal',
      category: 'port_conflict',
      solution: 'Stop other gateway instances'
    },
    {
      pattern: /pairing request/,
      severity: 'warn',
      category: 'pairing_required',
      solution: 'Approve pairing request'
    },
    // ... 更多特征
  ];

  match(logLine: string): LogSignature | null {
    for (const sig of this.signatures) {
      if (sig.pattern.test(logLine)) {
        return sig;
      }
    }
    return null;
  }

  analyze(logs: string[]): DiagnosticReport {
    const issues: Issue[] = [];

    for (const log of logs) {
      const signature = this.match(log);
      if (signature) {
        issues.push({
          type: signature.category,
          severity: signature.severity,
          solution: signature.solution,
          logLine: log
        });
      }
    }

    return { issues, summary: this.generateSummary(issues) };
  }
}
```

## 八、进阶使用

### 8.1 自定义日志分析脚本

```typescript
// analyze-logs.ts
import * as fs from 'fs';

interface LogEntry {
  timestamp: string;
  level: string;
  component: string;
  message: string;
}

function parseLog(line: string): LogEntry | null {
  const regex = /(\d{4}-\d{2}-\d{2} \d{2}:\d{2}:\d{2}) (\w+)\s+\[(\w+)\] (.+)/;
  const match = line.match(regex);

  if (!match) return null;

  return {
    timestamp: match[1],
    level: match[2],
    component: match[3],
    message: match[4]
  };
}

function analyzeLogs(logFile: string): void {
  const content = fs.readFileSync(logFile, 'utf-8');
  const lines = content.split('\n');

  const stats = {
    total: 0,
    byLevel: {} as Record<string, number>,
    byComponent: {} as Record<string, number>,
    errors: [] as LogEntry[]
  };

  for (const line of lines) {
    const entry = parseLog(line);
    if (!entry) continue;

    stats.total++;
    stats.byLevel[entry.level] = (stats.byLevel[entry.level] || 0) + 1;
    stats.byComponent[entry.component] = (stats.byComponent[entry.component] || 0) + 1;

    if (entry.level === 'ERROR' || entry.level === 'FATAL') {
      stats.errors.push(entry);
    }
  }

  console.log('Log Analysis Report:');
  console.log(`Total entries: ${stats.total}`);
  console.log('\nBy Level:');
  console.log(stats.byLevel);
  console.log('\nBy Component:');
  console.log(stats.byComponent);
  console.log(`\nErrors found: ${stats.errors.length}`);

  if (stats.errors.length > 0) {
    console.log('\nTop 10 Errors:');
    stats.errors.slice(0, 10).forEach(err => {
      console.log(`${err.timestamp} [${err.component}] ${err.message}`);
    });
  }
}

// 使用
analyzeLogs('openclaw.log');
```

### 8.2 实时日志告警

```typescript
// log-monitor.ts
import { spawn } from 'child_process';

interface AlertRule {
  pattern: RegExp;
  severity: 'low' | 'medium' | 'high';
  action: (line: string) => void;
}

const alertRules: AlertRule[] = [
  {
    pattern: /FATAL/,
    severity: 'high',
    action: (line) => {
      console.error('🚨 FATAL ERROR DETECTED:');
      console.error(line);
      // 发送告警通知
      sendAlert('FATAL error in OpenClaw', line);
    }
  },
  {
    pattern: /EADDRINUSE/,
    severity: 'high',
    action: (line) => {
      console.error('🚨 PORT CONFLICT:');
      console.error(line);
      // 自动尝试修复
      spawn('openclaw', ['gateway', 'stop']);
    }
  },
  {
    pattern: /unauthorized/,
    severity: 'medium',
    action: (line) => {
      console.warn('⚠️  AUTH FAILURE:');
      console.warn(line);
    }
  }
];

function monitorLogs(): void {
  const logs = spawn('openclaw', ['logs', '--follow']);

  logs.stdout.on('data', (data) => {
    const lines = data.toString().split('\n');

    for (const line of lines) {
      // 检查所有告警规则
      for (const rule of alertRules) {
        if (rule.pattern.test(line)) {
          rule.action(line);
        }
      }
    }
  });
}

function sendAlert(title: string, message: string): void {
  // 实现告警通知（邮件、Slack、钉钉等）
  console.log(`Alert: ${title}`);
  console.log(message);
}

// 启动监控
monitorLogs();
```
