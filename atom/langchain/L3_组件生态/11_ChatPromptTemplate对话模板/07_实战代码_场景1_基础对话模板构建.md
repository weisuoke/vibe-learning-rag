## 实战1：最简单的对话模板

### 代码实现

```python
"""
实战1：最简单的对话模板
演示如何创建一个基础的 System + Human 对话模板
"""

from langchain_core.prompts import ChatPromptTemplate
from langchain_openai import ChatOpenAI
from dotenv import load_dotenv

# 加载环境变量
load_dotenv()

def demo_basic_template():
    """演示最简单的对话模板"""
    print("=== 实战1：最简单的对话模板 ===\n")

    # 1. 创建最简单的模板（使用元组格式）
    template = ChatPromptTemplate.from_messages([
        ("system", "You are a helpful assistant."),
        ("human", "{input}")
    ])

    print("1. 模板创建成功")
    print(f"   输入变量: {template.input_variables}")

    # 2. 格式化模板
    messages = template.invoke({"input": "What is the capital of France?"})

    print("\n2. 格式化后的消息:")
    for msg in messages.messages:
        print(f"   [{msg.__class__.__name__}] {msg.content}")

    # 3. 与 LLM 集成
    llm = ChatOpenAI(model="gpt-4o-mini", temperature=0)
    chain = template | llm

    print("\n3. 调用 LLM:")
    response = chain.invoke({"input": "What is the capital of France?"})
    print(f"   回答: {response.content}")

if __name__ == "__main__":
    demo_basic_template()
```

### 运行结果

```
=== 实战1：最简单的对话模板 ===

1. 模板创建成功
   输入变量: ['input']

2. 格式化后的消息:
   [SystemMessage] You are a helpful assistant.
   [HumanMessage] What is the capital of France?

3. 调用 LLM:
   回答: The capital of France is Paris.
```

### 代码解析

**关键点1：元组格式**
```python
("system", "You are a helpful assistant.")
```
- 第一个元素是角色类型：`"system"`, `"human"`, `"ai"`, `"placeholder"`
- 第二个元素是消息内容
- 这是最简洁的写法，推荐使用

**关键点2：变量替换**
```python
("human", "{input}")
```
- 使用 `{变量名}` 进行占位
- 调用 `invoke()` 时传入字典进行替换
- 自动推断 `input_variables`

**关键点3：LCEL 链式组合**
```python
chain = template | llm
```
- 使用 `|` 操作符连接模板和模型
- 自动处理消息格式转换
- 支持流式处理

## 实战2：多变量对话模板

### 代码实现

```python
"""
实战2：多变量对话模板
演示如何在模板中使用多个变量
"""

from langchain_core.prompts import ChatPromptTemplate
from langchain_openai import ChatOpenAI
from dotenv import load_dotenv

load_dotenv()

def demo_multi_variable_template():
    """演示多变量对话模板"""
    print("=== 实战2：多变量对话模板 ===\n")

    # 1. 创建包含多个变量的模板
    template = ChatPromptTemplate.from_messages([
        ("system", "You are a {role}. You specialize in {specialty}."),
        ("human", "{task}")
    ])

    print("1. 模板创建成功")
    print(f"   输入变量: {template.input_variables}")

    # 2. 场景1：翻译助手
    print("\n2. 场景1：翻译助手")
    messages = template.invoke({
        "role": "professional translator",

        "task": "Translate 'Hello, how are you?' to Chinese"
    })

    for msg in messages.messages:
        print(f"   [{msg.__class__.__name__}] {msg.content}")

    # 3. 场景2：代码助手
    print("\n3. 场景2：代码助手")
    messages = template.invoke({
        "role": "senior Python developer",
        "specialty": "data processing and algorithms",
        "task": "Write a function to calculate Fibonacci numbers"
    })

    for msg in messages.messages:
        print(f"   [{msg.__class__.__name__}] {msg.content}")

    # 4. 与 LLM 集成
    llm = ChatOpenAI(model="gpt-4o-mini", temperature=0)
    chain = template | llm

    print("\n4. 调用 LLM (翻译场景):")
    response = chain.invoke({
        "role": "professional translator",
        "specialty": "English to Chinese translation",
        "task": "Translate 'Hello, how are you?' to Chinese"

    print(f"   回答: {response.content}")

if __name__ == "__main__":
    demo_multi_variable_template()
```

## 实战3：使用消息对象创建模板

### 代码实现

```python
"""
实战3：使用消息对象创建模板
演示如何使用 Message 对象而非元组格式
"""

from langchain_core.prompts import ChatPromptTemplate
from langchain_core.messages import SystemMessage, HumanMessage
from langchain_openai import ChatOpenAI
from dotenv import load_dotenv

load_dotenv()

def demo_message_objects():
    """演示使用消息对象创建模板"""
    print("=== 实战3：使用消息对象创建模板 ===\n")

    # 1. 使用消息对象创建模板
    template = ChatPromptTemplate.from_messages([
        SystemMessage(content="You are a helpful assistant that provides concise answers."),
        HumanMessage(content="{question}")
    ])

    print("1. 使用消息对象创建模板")
    print(f"   输入变量: {template.input_variables}")

    # 2. 格式化模板
    messages = template.invoke({"question": "What is Python?"})

    print("\n2. 格式化后的消息:")
    for msg in messages.messages:
        print(f"   [{msg.__class__.__name__}] {msg.content}")

    # 3. 混合使用元组和消息对象
    mixed_template = ChatPromptTemplate.from_messages([
        SystemMessage(content="You are a {role}."),
        ("human", "Tell me about {topic}"),
        SystemMessage(content="Remember to be {style}.")
    ])

    print("\n3. 混合格式模板:")
    print(f"   输入变量: {mixed_template.input_variables}")

    messages = mixed_template.invoke({
        "role": "science teacher",
        "topic": "photosynthesis",
        "style": "simple and clear"
    })

    print("\n4. 混合格式的消息:")
    for msg in messages.messages:
        print(f"   [{msg.__class__.__name__}] {msg.content}")

if __name__ == "__main__":
    demo_message_objects()
```

### 运行结果

```

   输入变量: ['question']

   [SystemMessage] You are a helpful assistant that provides concise answers.
```

### 代码解析

**关键点1：消息对象 vs 元组格式**

| 格式 | 优点 | 缺点 | 适用场景 |
|------|------|------|----------|
| 元组格式 | 简洁、易读 | 功能有限 | 简单场景 |
| 消息对象 | 功能完整、可扩展 | 代码较长 | 需要额外参数 |

**关键点2：混合使用**
```python
ChatPromptTemplate.from_messages([
    SystemMessage(content="..."),  # 消息对象
    ("human", "..."),              # 元组格式
])
```
- 可以在同一个模板中混合使用
- 根据需要选择合适的格式
- 推荐：简单消息用元组，复杂消息用对象

## 实战4：RAG 场景的对话模板

### 代码实现

```python
"""
实战4：RAG 场景的对话模板
演示如何为 RAG 系统构建对话模板
"""

from langchain_core.prompts import ChatPromptTemplate
from langchain_openai import ChatOpenAI
from dotenv import load_dotenv

load_dotenv()

def demo_rag_template():
    """演示 RAG 场景的对话模板"""
    print("=== 实战4：RAG 场景的对话模板 ===\n")

    # 1. 创建 RAG 对话模板
    rag_template = ChatPromptTemplate.from_messages([
        ("system", """You are a helpful assistant that answers questions based on the provided context.

Rules:
1. Only use information from the context to answer
2. If the context doesn't contain the answer, say "I don't have enough information"
3. Be concise and accurate
4. Cite the source when possible"""),
        ("human", """Context:
{context}

Question: {question}""")
    ])

    print("1. RAG 模板创建成功")
    print(f"   输入变量: {rag_template.input_variables}")

    # 2. 模拟检索到的上下文
    context = """
Document 1: Python is a high-level programming language created by Guido van Rossum in 1991.
Document 2: Python is known for its simple syntax and readability.
Document 3: Python is widely used in web development, data science, and AI.
"""

    question = "When was Python created and by whom?"

    # 3. 格式化模板
    messages = rag_template.invoke({
        "context": context,
        "question": question
    })

    print("\n2. 格式化后的消息:")
    for i, msg in enumerate(messages.messages, 1):
        print(f"\n   消息 {i} [{msg.__class__.__name__}]:")
        print(f"   {msg.content[:200]}...")

    # 4. 与 LLM 集成
    llm = ChatOpenAI(model="gpt-4o-mini", temperature=0)
    chain = rag_template | llm

    print("\n3. 调用 LLM:")
    response = chain.invoke({
        "context": context,
        "question": question
    })
    print(f"   回答: {response.content}")

    # 5. 测试无答案场景
    print("\n4. 测试无答案场景:")
    response = chain.invoke({
        "context": context,
        "question": "What is the latest version of Python?"
    })
    print(f"   回答: {response.content}")

if __name__ == "__main__":
    demo_rag_template()
```

## 完整示例：对话模板工具类

### 代码实现

```python
"""
完整示例：对话模板工具类
封装常用的对话模板，便于复用
"""

from langchain_core.prompts import ChatPromptTemplate
from langchain_openai import ChatOpenAI
from typing import Dict, Any
from dotenv import load_dotenv

load_dotenv()

class ChatTemplateFactory:
    """对话模板工厂类"""

    @staticmethod
    def create_simple_qa() -> ChatPromptTemplate:
        """创建简单问答模板"""
        return ChatPromptTemplate.from_messages([
            ("system", "You are a helpful assistant."),
            ("human", "{question}")
        ])

    @staticmethod
    def create_role_based(role: str, specialty: str) -> ChatPromptTemplate:
        """创建基于角色的模板"""
        return ChatPromptTemplate.from_messages([
            ("system", f"You are a {role}. You specialize in {specialty}."),
            ("human", "{task}")
        ])

    @staticmethod
    def create_rag_template() -> ChatPromptTemplate:
        """创建 RAG 模板"""
        return ChatPromptTemplate.from_messages([
            ("system", """You are a helpful assistant that answers questions based on context.
Rules:
1. Only use information from the context
2. If no answer, say "I don't have enough information"
3. Be concise and accurate"""),
            ("human", "Context:\n{context}\n\nQuestion: {question}")
        ])

    @staticmethod
    def create_translation_template() -> ChatPromptTemplate:
        """创建翻译模板"""
        return ChatPromptTemplate.from_messages([
            ("system", "You are a professional translator."),
            ("human", "Translate the following {source_lang} text to {target_lang}:\n\n{text}")
        ])

def demo_template_factory():
    """演示模板工厂的使用"""
    print("=== 完整示例：对话模板工具类 ===\n")

    llm = ChatOpenAI(model="gpt-4o-mini", temperature=0)

    # 1. 简单问答
    print("1. 简单问答模板:")
    qa_template = ChatTemplateFactory.create_simple_qa()
    qa_chain = qa_template | llm
    response = qa_chain.invoke({"question": "What is 2+2?"})
    print(f"   问题: What is 2+2?")
    print(f"   回答: {response.content}\n")

    # 2. 角色模板
    print("2. 角色模板:")
    role_template = ChatTemplateFactory.create_role_based(
        role="Python expert",
        specialty="code optimization"
    )
    role_chain = role_template | llm
    response = role_chain.invoke({"task": "How to optimize a for loop?"})
    print(f"   任务: How to optimize a for loop?")
    print(f"   回答: {response.content[:100]}...\n")

    # 3. RAG 模板
    print("3. RAG 模板:")
    rag_template = ChatTemplateFactory.create_rag_template()
    rag_chain = rag_template | llm
    response = rag_chain.invoke({
        "context": "LangChain is a framework for building LLM applications.",
        "question": "What is LangChain?"
    })
    print(f"   问题: What is LangChain?")
    print(f"   回答: {response.content}\n")

    # 4. 翻译模板
    print("4. 翻译模板:")
    trans_template = ChatTemplateFactory.create_translation_template()
    trans_chain = trans_template | llm
    response = trans_chain.invoke({
        "source_lang": "English",
        "target_lang": "Chinese",
        "text": "Hello, how are you?"
    })
    print(f"   原文: Hello, how are you?")
    print(f"   译文: {response.content}")

if __name__ == "__main__":
    demo_template_factory()
```

## 最佳实践

### 1. 模板设计原则

**清晰的角色定义**
```python
# ✅ 好的做法
("system", "You are a professional translator specializing in technical documents.")

# ❌ 不好的做法
("system", "You are helpful.")
```

**合理的变量命名**
```python
# ✅ 好的做法
("human", "Translate {source_text} from {source_lang} to {target_lang}")

# ❌ 不好的做法
("human", "Translate {x} from {y} to {z}")
```

### 2. 性能优化

**模板复用**
```python
# ✅ 创建一次，多次使用
template = ChatTemplateFactory.create_simple_qa()
chain = template | llm

for question in questions:
    response = chain.invoke({"question": question})
```

**批量处理**
```python
# ✅ 使用 batch 方法
responses = chain.batch([
    {"question": q1},
    {"question": q2},
    {"question": q3}
])
```

### 3. 错误处理

**验证输入变量**
```python
def safe_invoke(template, llm, inputs: Dict[str, Any]):
    """安全调用模板"""
    required_vars = template.input_variables
    missing_vars = set(required_vars) - set(inputs.keys())

    if missing_vars:
        raise ValueError(f"Missing required variables: {missing_vars}")

    chain = template | llm
    return chain.invoke(inputs)
```

## 常见问题

### Q1: 元组格式和消息对象有什么区别？

**A:**
- **元组格式**：`("role", "content")` - 简洁，适合简单场景
- **消息对象**：`SystemMessage(content="...")` - 功能完整，支持额外参数

推荐：简单场景用元组，需要额外参数时用对象。

### Q2: 如何在模板中使用多行文本？

**A:** 使用三引号字符串：
```python
("system", """You are a helpful assistant.
You should:
1. Be concise
2. Be accurate
3. Be helpful""")
```

### Q3: 变量名可以包含特殊字符吗？

**A:** 变量名遵循 Python 标识符规则：
- ✅ 可以：`{user_name}`, `{question_1}`
- ❌ 不可以：`{user-name}`, `{1_question}`

### Q4: 如何处理可选变量？

**A:** 使用默认值或条件判断：
```python
# 方法1：使用 partial
template = template.partial(optional_var="default_value")

# 方法2：在调用时提供默认值
inputs = {"required_var": "value"}
inputs.setdefault("optional_var", "default")
```
