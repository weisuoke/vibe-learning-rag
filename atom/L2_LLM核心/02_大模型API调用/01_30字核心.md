# 大模型API调用 - 30字核心

**大模型 API 是通过 HTTP 请求调用云端 LLM 的标准接口，是 RAG 系统生成模块的核心基础设施。**

---

## 关键词解析

| 关键词 | 含义 |
|--------|------|
| 大模型 API | OpenAI、Anthropic 等厂商提供的 LLM 调用接口 |
| HTTP 请求 | 通过网络发送请求，获取模型响应 |
| 云端 LLM | 模型部署在云服务器，无需本地运行 |
| RAG 生成模块 | RAG 系统的最后一环：基于检索结果生成答案 |

---

## 为什么这是核心？

```
用户问题 → 检索相关文档 → 【调用 LLM API 生成答案】→ 返回给用户
                              ↑
                         这就是 API 调用的位置
```

没有 API 调用能力，RAG 系统就无法生成最终答案。
