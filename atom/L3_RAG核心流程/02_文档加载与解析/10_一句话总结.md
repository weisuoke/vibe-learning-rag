# 一句话总结

**文档加载与解析是将 PDF、Word、HTML 等多格式文档转换为统一 Document 对象的过程，通过格式解析和预处理提取纯文本并保留元数据，是 RAG 系统数据入口的关键环节，直接决定后续检索和生成的质量。**

---

## 关键词拆解

| 关键词 | 含义 |
|-------|------|
| 多格式文档 | PDF、Word、HTML、Markdown、TXT |
| 统一 Document 对象 | page_content + metadata |
| 格式解析 | 从二进制/标签中提取文本 |
| 预处理 | 清理噪音、标准化格式 |
| 元数据 | 来源、页码等追踪信息 |
| 数据入口 | RAG 离线索引的第一步 |

---

## 核心公式

```
文档加载与解析 = 读取文件 + 格式解析 + 预处理 + 元数据附加
                    ↓
              Document(page_content, metadata)
                    ↓
              传给分块器继续处理
```

loader = PyPDFLoader("document.pdf")

    print(doc.metadata["source"]) # 来源文件
