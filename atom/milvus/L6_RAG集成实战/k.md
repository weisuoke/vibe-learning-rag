# L6_RAG集成实战 知识点列表

> 将 Milvus 应用于实际 RAG 系统，掌握端到端的集成实践

---

## 知识点清单

1. **文档问答系统实现** - 构建完整的文档问答 RAG 系统（文档加载、向量化、检索、生成）
2. **多租户知识库** - 实现多租户隔离的企业级知识库系统
3. **大规模向量检索优化** - 处理百万级、千万级向量的检索优化策略
4. **Milvus与LangChain集成** - 使用 LangChain 框架快速构建 RAG 应用
5. **Milvus与LlamaIndex集成** - 使用 LlamaIndex 实现高级 RAG 模式

---

## 学习顺序建议

```
文档问答系统实现 → 多租户知识库 → 大规模向量检索优化 → LangChain集成 → LlamaIndex集成
       ↓                 ↓                ↓                    ↓              ↓
   基础RAG          企业应用          性能优化            快速开发        高级模式
```

## 核心应用场景

### 场景1：企业文档问答
- 文档加载与解析（PDF、Word、Markdown）
- 文本分块与向量化
- Milvus 存储与检索
- LLM 生成答案

### 场景2：多租户SaaS知识库
- 租户隔离（Partition 或标量过滤）
- 权限控制
- 数据安全

### 场景3：电商商品推荐
- 商品向量化（图片+文本）
- 多向量检索
- 个性化推荐

### 场景4：代码搜索引擎
- 代码向量化
- 语义代码搜索
- 代码片段推荐
