## 场景概述

### 业务场景

**电商图文搜索系统**：用户可以通过文本描述或上传图片来搜索商品。

**核心需求**：
- 支持文本搜索商品（"红色连衣裙"）
- 支持图片搜索商品（上传图片找相似商品）
- 支持图文混合搜索（图片 + 文本描述）
- 融合多模态检索结果

### 技术方案

**多模态向量检索**：
1. 使用 CLIP 模型生成图片和文本的统一向量表示
2. 在 Milvus 中定义多个向量字段（图片向量、文本向量）
3. 使用 `hybrid_search` 融合多模态检索结果
4. 使用 `RRFRanker` 或 `WeightedRanker` 排序

### 2. CLIP 模型封装

```python
class CLIPEmbedding:
    """CLIP 模型封装 - 生成图片和文本的统一向量表示"""

    def __init__(self, model_name: str = CLIP_MODEL_NAME):
        """
        初始化 CLIP 模型

        Args:
            model_name: CLIP 模型名称
        """
        print(f"加载 CLIP 模型: {model_name}")
        self.device = "cuda" if torch.cuda.is_available() else "cpu"
        self.model = CLIPModel.from_pretrained(model_name).to(self.device)
        self.processor = CLIPProcessor.from_pretrained(model_name)
        self.embedding_dim = self.model.config.projection_dim  # 512
        print(f"模型加载完成，向量维度: {self.embedding_dim}")

    def encode_text(self, texts: Union[str, List[str]]) -> np.ndarray:
        """
        文本编码

        Args:
            texts: 单个文本或文本列表

        Returns:
            文本向量 (batch_size, embedding_dim)
        """
        if isinstance(texts, str):
            texts = [texts]

        # 处理文本
        inputs = self.processor(
            text=texts,
            return_tensors="pt",
            padding=True,
            truncation=True
        ).to(self.device)

        # 生成向量
        with torch.no_grad():
            text_features = self.model.get_text_features(**inputs)
            # L2 归一化
            text_features = text_features / text_features.norm(dim=-1, keepdim=True)

        return text_features.cpu().numpy()

    def encode_image(self, images: Union[str, Image.Image, List]) -> np.ndarray:
        """
        图片编码

        Args:
            images: 图片路径、PIL Image 对象或列表

        Returns:
            图片向量 (batch_size, embedding_dim)
        """
        # 统一处理为列表
        if isinstance(images, (str, Image.Image)):
            images = [images]

        # 加载图片
        pil_images = []
        for img in images:
            if isinstance(img, str):
                pil_images.append(Image.open(img).convert("RGB"))
            elif isinstance(img, Image.Image):
                pil_images.append(img.convert("RGB"))
            else:
                raise ValueError(f"不支持的图片类型: {type(img)}")

        # 处理图片
        inputs = self.processor(
            images=pil_images,
            return_tensors="pt"
        ).to(self.device)

        # 生成向量
        with torch.no_grad():
            image_features = self.model.get_image_features(**inputs)
            # L2 归一化
            image_features = image_features / image_features.norm(dim=-1, keepdim=True)

        return image_features.cpu().numpy()
```

### 3. Milvus Collection 创建

```python
class MultimodalProductSearch:
    """多模态商品搜索系统"""

    def __init__(self, uri: str = MILVUS_URI):
        """初始化"""
        self.client = MilvusClient(uri=uri)
        self.clip = CLIPEmbedding()
        self.collection_name = COLLECTION_NAME

    def create_collection(self):
        """
        创建多向量 Collection

        Schema 设计：
        - id: 主键
        - product_name: 商品名称
        - description: 商品描述
        - image_url: 图片 URL
        - price: 价格
        - text_vector: 文本向量（CLIP 编码）
        - image_vector: 图片向量（CLIP 编码）
        """
        # 删除已存在的 Collection
        if self.client.has_collection(self.collection_name):
            self.client.drop_collection(self.collection_name)
            print(f"已删除旧 Collection: {self.collection_name}")

        # 定义 Schema
        schema = self.client.create_schema(
            auto_id=False,
            enable_dynamic_field=True
        )

        # 添加字段
        schema.add_field(field_name="id", datatype=DataType.INT64, is_primary=True)
        schema.add_field(field_name="product_name", datatype=DataType.VARCHAR, max_length=256)
        schema.add_field(field_name="description", datatype=DataType.VARCHAR, max_length=1024)
        schema.add_field(field_name="image_url", datatype=DataType.VARCHAR, max_length=512)
        schema.add_field(field_name="price", datatype=DataType.FLOAT)

        # 添加多个向量字段
        schema.add_field(
            field_name="text_vector",
            datatype=DataType.FLOAT_VECTOR,
            dim=self.clip.embedding_dim
        )
        schema.add_field(
            field_name="image_vector",
            datatype=DataType.FLOAT_VECTOR,
            dim=self.clip.embedding_dim
        )

        # 创建索引参数
        index_params = self.client.prepare_index_params()

        # 为文本向量创建索引
        index_params.add_index(
            field_name="text_vector",
            index_type="IVF_FLAT",
            metric_type="IP",  # 内积（CLIP 向量已归一化，等价于余弦相似度）
            params={"nlist": 128}
        )

        # 为图片向量创建索引
        index_params.add_index(
            field_name="image_vector",
            index_type="IVF_FLAT",
            metric_type="IP",
            params={"nlist": 128}
        )

        # 创建 Collection
        self.client.create_collection(
            collection_name=self.collection_name,
            schema=schema,
            index_params=index_params
        )

        print(f"✓ Collection 创建成功: {self.collection_name}")
        print(f"  - 文本向量维度: {self.clip.embedding_dim}")
        print(f"  - 图片向量维度: {self.clip.embedding_dim}")
        print(f"  - 索引类型: IVF_FLAT")
        print(f"  - 相似度度量: IP (内积)")

    def insert_products(self, products: List[Dict]):
        """
        插入商品数据

        Args:
            products: 商品列表，每个商品包含：
                - id: 商品 ID
                - product_name: 商品名称
                - description: 商品描述
                - image_path: 图片路径（本地）
                - image_url: 图片 URL（存储）
                - price: 价格
        """
        data = []

        for product in products:
            # 生成文本向量（商品名称 + 描述）
            text = f"{product['product_name']}. {product['description']}"
            text_vector = self.clip.encode_text(text)[0].tolist()

            # 生成图片向量
            image_vector = self.clip.encode_image(product['image_path'])[0].tolist()

            # 构造数据
            data.append({
                "id": product["id"],
                "product_name": product["product_name"],
                "description": product["description"],
                "image_url": product["image_url"],
                "price": product["price"],
                "text_vector": text_vector,
                "image_vector": image_vector
            })

        # 批量插入
        self.client.insert(
            collection_name=self.collection_name,
            data=data
        )

        print(f"✓ 插入 {len(data)} 条商品数据")

    def search_by_text(
        self,
        query_text: str,
        limit: int = 10,
        output_fields: List[str] = None
    ) -> List[Dict]:
        """
        纯文本搜索

        Args:
            query_text: 查询文本
            limit: 返回结果数量
            output_fields: 返回字段

        Returns:
            搜索结果列表
        """
        if output_fields is None:
            output_fields = ["product_name", "description", "price", "image_url"]

        # 生成查询向量
        query_vector = self.clip.encode_text(query_text)[0].tolist()

        # 搜索
        results = self.client.search(
            collection_name=self.collection_name,
            data=[query_vector],
            anns_field="text_vector",
            limit=limit,
            output_fields=output_fields
        )

        return results[0]

    def search_by_image(
        self,
        image_path: str,
        limit: int = 10,
        output_fields: List[str] = None
    ) -> List[Dict]:
        """
        纯图片搜索

        Args:
            image_path: 图片路径
            limit: 返回结果数量
            output_fields: 返回字段

        Returns:
            搜索结果列表
        """
        if output_fields is None:
            output_fields = ["product_name", "description", "price", "image_url"]

        # 生成查询向量
        query_vector = self.clip.encode_image(image_path)[0].tolist()

        # 搜索
        results = self.client.search(
            collection_name=self.collection_name,
            data=[query_vector],
            anns_field="image_vector",
            limit=limit,
            output_fields=output_fields
        )

        return results[0]

    def hybrid_search(
        self,
        query_text: str = None,
        image_path: str = None,
        text_weight: float = 1.0,
        image_weight: float = 1.0,
        limit: int = 10,
        use_rrf: bool = False,
        output_fields: List[str] = None
    ) -> List[Dict]:
        """
        多模态混合搜索

        Args:
            query_text: 查询文本（可选）
            image_path: 查询图片路径（可选）
            text_weight: 文本权重
            image_weight: 图片权重
            limit: 返回结果数量
            use_rrf: 是否使用 RRF 排序（否则使用加权排序）
            output_fields: 返回字段

        Returns:
            搜索结果列表
        """
        if output_fields is None:
            output_fields = ["product_name", "description", "price", "image_url"]

        # 构建检索请求列表
        requests = []

        # 文本检索请求
        if query_text:
            text_vector = self.clip.encode_text(query_text)[0].tolist()
            text_req = AnnSearchRequest(
                data=[text_vector],
                anns_field="text_vector",
                param={"metric_type": "IP", "params": {"nprobe": 10}},
                limit=limit
            )
            requests.append(text_req)

        # 图片检索请求
        if image_path:
            image_vector = self.clip.encode_image(image_path)[0].tolist()
            image_req = AnnSearchRequest(
                data=[image_vector],
                anns_field="image_vector",
                param={"metric_type": "IP", "params": {"nprobe": 10}},
                limit=limit
            )
            requests.append(image_req)

        if not requests:
            raise ValueError("至少需要提供 query_text 或 image_path")

        # 选择 Ranker
        if use_rrf:
            ranker = RRFRanker()
            ranker_name = "RRFRanker"
        else:
            # 根据请求数量设置权重
            if len(requests) == 1:
                weights = [1.0]
            else:
                weights = [text_weight, image_weight]
            ranker = WeightedRanker(*weights)
            ranker_name = f"WeightedRanker(weights={weights})"

        # 执行混合搜索
        results = self.client.hybrid_search(
            collection_name=self.collection_name,
            reqs=requests,
            ranker=ranker,
            limit=limit,
            output_fields=output_fields
        )

        print(f"✓ 混合搜索完成 (Ranker: {ranker_name})")
        return results[0]
```

## 核心技术点

### 1. CLIP 模型原理

**CLIP (Contrastive Language-Image Pre-training)**：
- OpenAI 开发的多模态模型
- 将图片和文本映射到同一向量空间
- 向量维度：512（base 模型）
- 使用对比学习训练

**关键特性**：
```python
# 图片和文本的向量在同一空间
text_vec = clip.encode_text("红色连衣裙")    # shape: (512,)
image_vec = clip.encode_image("dress.jpg")   # shape: (512,)

# 可以直接计算相似度
similarity = np.dot(text_vec, image_vec)  # 余弦相似度（已归一化）
```

### 2. 多向量字段设计

**Schema 设计原则**：
```python
# 为不同模态定义独立的向量字段
schema.add_field("text_vector", DataType.FLOAT_VECTOR, dim=512)
schema.add_field("image_vector", DataType.FLOAT_VECTOR, dim=512)

# 所有向量字段都必须创建索引
index_params.add_index("text_vector", ...)
index_params.add_index("image_vector", ...)
```

### 3. 混合检索策略

**RRFRanker vs WeightedRanker**：

| 特性 | RRFRanker | WeightedRanker |
|------|-----------|----------------|
| 算法 | Reciprocal Rank Fusion | 加权求和 |
| 权重 | 自动平衡 | 手动指定 |
| 适用场景 | 不确定权重 | 已知权重 |
| 参数 | k=60（默认） | weights=[w1, w2] |

**选择建议**：
- 初期探索：使用 RRFRanker
- 生产环境：根据 A/B 测试结果使用 WeightedRanker

## 性能优化

### 1. 批量编码

```python

    text_vec = clip.encode_text(product['name'])

texts = [p['name'] for p in products]

image_vecs = clip.encode_image(images)   # 批量处理
```

## 总结

**核心要点**：
1. CLIP 模型将图片和文本映射到统一向量空间
2. Milvus 支持多向量字段，每个字段独立索引
3. `hybrid_search` + `RRFRanker`/`WeightedRanker` 融合多模态结果
4. 批量编码和 GPU 加速提升性能

**最佳实践**：
- 使用 CLIP 等预训练模型，无需自己训练
