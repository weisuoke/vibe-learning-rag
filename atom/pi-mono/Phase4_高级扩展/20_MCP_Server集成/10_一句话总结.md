# 一句话总结

> **用一句话总结整个 MCP Server 集成知识点**

---

## 核心总结

**MCP Server 集成通过标准化的 JSON-RPC 协议让 AI Agent 能够动态发现和调用外部工具，实现了三种能力扩展模式：构建自定义服务器提供专有工具、创建 pi-mono 扩展集成 MCP 客户端、以及配置使用社区现有服务器，从而在不修改核心代码的前提下无限扩展 Agent 的能力边界。**

---

## 关键要素拆解

这句话包含了 MCP Server 集成的所有核心要素：

### 1. 技术基础
- **标准化的 JSON-RPC 协议**：统一的通信规范

### 2. 核心价值
- **动态发现和调用外部工具**：运行时能力扩展

### 3. 三种模式
- **构建自定义服务器**：创建专有工具提供者
- **创建 pi-mono 扩展**：集成 MCP 客户端
- **使用现有服务器**：配置社区资源

### 4. 最终目标
- **无限扩展 Agent 的能力边界**：不修改核心代码
