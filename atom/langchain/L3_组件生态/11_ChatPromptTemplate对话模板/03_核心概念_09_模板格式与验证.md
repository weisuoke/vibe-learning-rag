# 核心概念 09: 模板格式与验证

## 概述

ChatPromptTemplate 支持多种模板格式（f-string、mustache、jinja2）和可选的模板验证机制。正确选择模板格式和启用验证可以提升模板的灵活性、可读性和健壮性。

**核心问题**: 为什么需要不同的模板格式？

**答案**: 因为不同的模板格式适用于不同的场景：f-string 简洁高效、mustache 支持嵌套变量、jinja2 支持复杂逻辑。模板验证则可以在构造时发现错误，避免运行时失败。

**来源**: [源码分析](reference/source_chatprompt_01.md) | lines 899-901, 1046-1098

## 1. 模板格式类型

### 1.1 三种模板格式概览

ChatPromptTemplate 支持三种模板格式，通过 `template_format` 参数指定。

**支持的格式**:
- **f-string** (默认): Python 格式化字符串，使用 `{variable}`
- **mustache**: Mustache 模板语法，使用 `{{variable}}`
- **jinja2**: Jinja2 模板引擎，使用 `{{ variable }}`，支持复杂逻辑

**来源**: [源码分析](reference/source_chatprompt_01.md) | ChatPromptTemplate 构造函数

### 1.2 格式对比表

| 特性 | f-string | mustache | jinja2 |
|------|----------|----------|--------|
| 语法 | `{var}` | `{{var}}` | `{{ var }}` |
| 简洁性 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐ |
| 嵌套变量 | ❌ | ✅ | ✅ |
| 条件逻辑 | ❌ | ❌ | ✅ |
| 循环 | ❌ | ✅ | ✅ |
| 过滤器 | ❌ | ❌ | ✅ |
| 性能 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐⭐ |
| 适用场景 | 简单变量替换 | 嵌套数据结构 | 复杂模板逻辑 |

## 2. f-string 格式（默认）

### 2.1 基础用法

**f-string** 是 Python 的原生格式化字符串语法，也是 ChatPromptTemplate 的默认格式。

```python
from langchain_core.prompts import ChatPromptTemplate

# 默认使用 f-string 格式
prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a {role}."),
    ("human", "{input}")
])

# 格式化
messages = prompt.format_messages(role="assistant", input="Hello")
```

**语法规则**:
- 使用单层花括号 `{variable}`
- 支持基本的格式化选项（如 `{value:.2f}`）
- 不支持嵌套变量或复杂逻辑

### 2.2 格式化选项

f-string 支持 Python 的格式化选项：

```python
from langchain_core.prompts import ChatPromptTemplate

# 数字格式化
prompt = ChatPromptTemplate.from_messages([
    ("system", "Temperature: {temp:.2f}°C"),
    ("human", "{input}")
])

messages = prompt.format_messages(temp=25.678, input="What's the weather?")
# Temperature: 25.68°C
```

**支持的格式化选项**:
- `{value:.2f}` - 保留2位小数
- `{value:>10}` - 右对齐，宽度10
- `{value:0>5}` - 左侧填充0，宽度5

### 2.3 适用场景

**场景 1: 简单变量替换**

```python
from langchain_core.prompts import ChatPromptTemplate

# 基础对话模板
prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a {role} assistant."),
    ("human", "{question}")
])
```

**场景 2: RAG 系统**

```python
# RAG 提示词模板
rag_prompt = ChatPromptTemplate.from_messages([
    ("system", "Answer based on the context."),
    ("human", "Context: {context}\nQuestion: {question}")
])
```

**场景 3: 多变量模板**

```python
# 多变量模板
prompt = ChatPromptTemplate.from_messages([
    ("system", "You are {role}. Language: {language}. Style: {style}."),
    ("human", "{input}")
])
```

## 3. mustache 格式

### 3.1 基础用法

**mustache** 使用双层花括号 `{{variable}}`，支持嵌套变量和简单的逻辑。

```python
from langchain_core.prompts import ChatPromptTemplate

# 使用 mustache 格式
prompt = ChatPromptTemplate.from_messages(
    [
        ("system", "You are {{role}}."),
        ("human", "{{input}}")
    ],
    template_format="mustache"
)

messages = prompt.format_messages(role="assistant", input="Hello")
```

**语法规则**:
- 使用双层花括号 `{{variable}}`
- 支持嵌套对象访问 `{{user.name}}`
- 支持条件渲染 `{{#condition}}...{{/condition}}`
- 支持列表循环 `{{#list}}...{{/list}}`

### 3.2 嵌套变量访问

mustache 支持访问嵌套对象的属性：

```python
from langchain_core.prompts import ChatPromptTemplate

# 嵌套变量模板
prompt = ChatPromptTemplate.from_messages(
    [
        ("system", "You are {{user.role}}."),
        ("human", "Name: {{user.name}}, Question: {{input}}")
    ],
    template_format="mustache"
)

# 传递嵌套对象
messages = prompt.format_messages(
    user={"role": "assistant", "name": "Alice"},
    input="Hello"
)
```

### 3.3 条件渲染

mustache 支持基于条件的内容渲染：

```python
from langchain_core.prompts import ChatPromptTemplate

# 条件渲染模板
prompt = ChatPromptTemplate.from_messages(
    [
        ("system", "You are an assistant.{{#expert}} You are an expert.{{/expert}}"),
        ("human", "{{input}}")
    ],
    template_format="mustache"
)

# expert=True 时会渲染额外内容
messages = prompt.format_messages(expert=True, input="Hello")
# "You are an assistant. You are an expert."

# expert=False 时不渲染
messages = prompt.format_messages(expert=False, input="Hello")
# "You are an assistant."
```

### 3.4 列表循环

mustache 支持遍历列表：

```python
from langchain_core.prompts import ChatPromptTemplate

# 列表循环模板
prompt = ChatPromptTemplate.from_messages(
    [
        ("system", "Available tools: {{#tools}}{{name}}, {{/tools}}"),
        ("human", "{{input}}")
    ],
    template_format="mustache"
)

messages = prompt.format_messages(
    tools=[
        {"name": "search"},
        {"name": "calculator"},
        {"name": "translator"}
    ],
    input="Use a tool"
)
# "Available tools: search, calculator, translator,"
```

### 3.5 适用场景

**场景 1: 嵌套数据结构**

```python
# 用户配置模板
prompt = ChatPromptTemplate.from_messages(
    [
        ("system", "User: {{user.name}}, Role: {{user.role}}, Language: {{user.settings.language}}"),
        ("human", "{{input}}")
    ],
    template_format="mustache"
)
```

**场景 2: 动态工具列表**

```python
# Agent 工具列表
prompt = ChatPromptTemplate.from_messages(
    [
        ("system", "Tools: {{#tools}}{{name}} - {{description}}; {{/tools}}"),
        ("human", "{{input}}")
    ],
    template_format="mustache"
)
```

## 4. jinja2 格式

### 4.1 基础用法

**jinja2** 是功能最强大的模板引擎，支持复杂的逻辑和控制流。

```python
from langchain_core.prompts import ChatPromptTemplate

# 使用 jinja2 格式
prompt = ChatPromptTemplate.from_messages(
    [
        ("system", "You are {{ role }}."),
        ("human", "{{ input }}")
    ],
    template_format="jinja2"
)

messages = prompt.format_messages(role="assistant", input="Hello")
```

**语法规则**:
- 使用 `{{ variable }}` 输出变量
- 使用 `{% if %}...{% endif %}` 条件判断
- 使用 `{% for %}...{% endfor %}` 循环
- 支持过滤器 `{{ variable|filter }}`
- 支持表达式 `{{ 1 + 1 }}`

### 4.2 条件判断

jinja2 支持完整的条件逻辑：

```python
from langchain_core.prompts import ChatPromptTemplate

# 条件判断模板
prompt = ChatPromptTemplate.from_messages(
    [
        ("system", """You are an assistant.
{% if expert %}
You are an expert in {{ domain }}.
{% else %}
You are a general assistant.
{% endif %}"""),
        ("human", "{{ input }}")
    ],
    template_format="jinja2"
)

# expert=True
messages = prompt.format_messages(expert=True, domain="Python", input="Hello")

# expert=False
messages = prompt.format_messages(expert=False, input="Hello")
```

### 4.3 循环遍历

jinja2 支持强大的循环功能：

```python
from langchain_core.prompts import ChatPromptTemplate

# 循环遍历模板
prompt = ChatPromptTemplate.from_messages(
    [
        ("system", """Available tools:
{% for tool in tools %}
{{ loop.index }}. {{ tool.name }}: {{ tool.description }}
{% endfor %}"""),
        ("human", "{{ input }}")
    ],
    template_format="jinja2"
)

messages = prompt.format_messages(
    tools=[
        {"name": "search", "description": "Search the web"},
        {"name": "calculator", "description": "Calculate math"}
    ],
    input="Use a tool"
)
```

**循环变量**:
- `loop.index` - 当前索引（从1开始）
- `loop.index0` - 当前索引（从0开始）
- `loop.first` - 是否第一个元素
- `loop.last` - 是否最后一个元素

### 4.4 过滤器

jinja2 支持丰富的过滤器：

```python
from langchain_core.prompts import ChatPromptTemplate

# 过滤器模板
prompt = ChatPromptTemplate.from_messages(
    [
        ("system", "User: {{ name|upper }}, Role: {{ role|capitalize }}"),
        ("human", "{{ input|trim }}")
    ],
    template_format="jinja2"
)

messages = prompt.format_messages(
    name="alice",
    role="assistant",
    input="  Hello  "
)
# User: ALICE, Role: Assistant
# Hello
```

**常用过滤器**:
- `upper` - 转大写
- `lower` - 转小写
- `capitalize` - 首字母大写
- `trim` - 去除首尾空格
- `length` - 获取长度
- `default(value)` - 默认值

### 4.5 适用场景

**场景 1: 复杂的 Agent 提示词**

```python
# Agent 系统提示词
prompt = ChatPromptTemplate.from_messages(
    [
        ("system", """You are an AI agent.

Available tools:
{% for tool in tools %}
- {{ tool.name }}: {{ tool.description }}
{% endfor %}

{% if constraints %}
Constraints:
{% for constraint in constraints %}
- {{ constraint }}
{% endfor %}
{% endif %}

{% if examples %}
Examples:
{% for example in examples %}
Input: {{ example.input }}
Output: {{ example.output }}
{% endfor %}
{% endif %}"""),
        ("human", "{{ input }}")
    ],
    template_format="jinja2"
)
```

**场景 2: 多语言模板**

```python
# 多语言支持
prompt = ChatPromptTemplate.from_messages(
    [
        ("system", """{% if language == 'zh' %}
你是一个助手。
{% elif language == 'en' %}
You are an assistant.
{% elif language == 'fr' %}
Vous êtes un assistant.
{% endif %}"""),
        ("human", "{{ input }}")
    ],
    template_format="jinja2"
)
```

## 5. 模板验证机制

### 5.1 validate_template 参数

ChatPromptTemplate 支持通过 `validate_template=True` 启用模板验证。

```python
from langchain_core.prompts import ChatPromptTemplate

# 启用模板验证
prompt = ChatPromptTemplate.from_messages(
    [
        ("system", "You are a {role}."),
        ("human", "{input}")
    ],
    validate_template=True
)
```

**验证内容**:
- 检查模板变量是否与 input_variables 匹配
- 检查模板语法是否正确
- 在构造时验证，而非运行时

**来源**: [源码分析](reference/source_chatprompt_01.md) | lines 1046-1098

### 5.2 验证示例

**示例 1: 变量不匹配**

```python
from langchain_core.prompts import ChatPromptTemplate

# ❌ 这会在构造时抛出错误
try:
    prompt = ChatPromptTemplate.from_messages(
        [
            ("system", "You are a {role}."),
            ("human", "{input}")
        ],
        input_variables=["role"],  # 缺少 input 变量
        validate_template=True
    )
except ValueError as e:
    print(f"Validation error: {e}")
```

**示例 2: 语法错误**

```python
# ❌ 模板语法错误
try:
    prompt = ChatPromptTemplate.from_messages(
        [
            ("system", "You are a {role."),  # 缺少右括号
            ("human", "{input}")
        ],
        validate_template=True
    )
except Exception as e:
    print(f"Syntax error: {e}")
```

### 5.3 自动变量推断

默认情况下，ChatPromptTemplate 会自动推断 input_variables，无需手动指定。

```python
from langchain_core.prompts import ChatPromptTemplate

# 自动推断变量
prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a {role}."),
    ("human", "{input}")
])

# 查看推断的变量
print(prompt.input_variables)  # ['role', 'input']
```

**推断规则**:
- 从所有消息模板中提取变量
- 自动去重
- 排除 partial_variables
- 排除 optional MessagesPlaceholder

## 6. 错误处理

### 6.1 常见错误类型

**错误 1: 变量未提供**

```python
from langchain_core.prompts import ChatPromptTemplate

prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a {role}."),
    ("human", "{input}")
])

# ❌ 缺少 role 变量
try:
    messages = prompt.format_messages(input="Hello")
except KeyError as e:
    print(f"Missing variable: {e}")
```

**错误 2: 模板格式不匹配**

```python
# ❌ 使用 f-string 格式但传递了 mustache 语法
prompt = ChatPromptTemplate.from_messages(
    [
        ("system", "You are {{role}}."),  # mustache 语法
        ("human", "{input}")
    ],
    template_format="f-string"  # 但指定了 f-string
)

# 不会报错，但 {{role}} 会被当作普通文本
messages = prompt.format_messages(role="assistant", input="Hello")
# "You are {{role}}."
```

**错误 3: jinja2 语法错误**

```python
# ❌ jinja2 语法错误
try:
    prompt = ChatPromptTemplate.from_messages(
        [
            ("system", "{% if role %}You are {{ role }}."),  # 缺少 {% endif %}
            ("human", "{{ input }}")
        ],
        template_format="jinja2"
    )
except Exception as e:
    print(f"Jinja2 syntax error: {e}")
```

### 6.2 错误处理最佳实践

**实践 1: 使用 try-except 捕获错误**

```python
from langchain_core.prompts import ChatPromptTemplate

prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a {role}."),
    ("human", "{input}")
])

try:
    messages = prompt.format_messages(role="assistant", input="Hello")
except KeyError as e:
    print(f"Missing variable: {e}")
    # 提供默认值或提示用户
except Exception as e:
    print(f"Unexpected error: {e}")
```

**实践 2: 验证输入参数**

```python
def safe_format(prompt, **kwargs):
    """安全的格式化函数"""
    # 检查必需的变量
    required_vars = set(prompt.input_variables)
    provided_vars = set(kwargs.keys())

    missing_vars = required_vars - provided_vars
    if missing_vars:
        raise ValueError(f"Missing variables: {missing_vars}")

    return prompt.format_messages(**kwargs)

# 使用
try:
    messages = safe_format(prompt, role="assistant", input="Hello")
except ValueError as e:
    print(e)
```

**实践 3: 提供默认值**

```python
from langchain_core.prompts import ChatPromptTemplate

# 使用 partial 提供默认值
prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a {role}."),
    ("human", "{input}")
]).partial(role="assistant")

# 现在只需提供 input
messages = prompt.format_messages(input="Hello")
```

### 7.3 性能优化

**优化 1: 避免复杂的 jinja2 逻辑**

```python
# ❌ 不好的做法 - 模板中包含复杂逻辑
prompt = ChatPromptTemplate.from_messages(
    [
        ("system", """{% for i in range(100) %}
{% if i % 2 == 0 %}
{{ i }}
{% endif %}
{% endfor %}"""),
        ("human", "{{ input }}")
    ],
    template_format="jinja2"
)

# ✅ 好的做法 - 在 Python 中处理逻辑
even_numbers = [i for i in range(100) if i % 2 == 0]
prompt = ChatPromptTemplate.from_messages(
    [
        ("system", "Numbers: {numbers}"),
        ("human", "{input}")
    ]
)
messages = prompt.format_messages(numbers=str(even_numbers), input="Hello")
```

**优化 2: 缓存模板实例**

```python
# ✅ 缓存模板实例，避免重复创建
_template_cache = {}

def get_template(template_key):
    if template_key not in _template_cache:
        _template_cache[template_key] = ChatPromptTemplate.from_messages([
            ("system", "You are a {role}."),
            ("human", "{input}")
        ])
    return _template_cache[template_key]

# 使用缓存的模板
prompt = get_template("basic")
messages = prompt.format_messages(role="assistant", input="Hello")
```

## 9. 总结

### 核心要点

1. **三种模板格式**: f-string（简单）、mustache（嵌套）、jinja2（复杂）
2. **格式选择**: 根据场景选择合适的格式
3. **模板验证**: 使用 validate_template 在构造时发现错误
4. **自动推断**: input_variables 自动推断，无需手动指定
5. **错误处理**: 使用 try-except 捕获格式化错误
6. **性能优化**: 避免复杂逻辑，缓存模板实例

### 最佳实践

- ✅ 简单场景使用 f-string（默认）
- ✅ 嵌套数据使用 mustache
- ✅ 复杂逻辑使用 jinja2
- ✅ 开发时启用 validate_template
- ✅ 生产环境关闭验证（性能）
