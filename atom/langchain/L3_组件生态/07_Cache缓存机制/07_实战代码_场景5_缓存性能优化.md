# 实战代码 - 场景5：缓存性能优化

> **场景目标**：掌握缓存性能监控、分析和优化策略

---

## 场景说明

### 优化目标
- 提升缓存命中率
- 降低 API 调用成本
- 改善响应速度
- 优化资源使用

### 核心指标
- **命中率**：缓存命中次数 / 总请求次数
- **成本节省**：避免的 API 调用成本
- **响应时间**：平均响应时间改善
- **内存使用**：缓存占用的内存大小

"""
LangChain 缓存性能优化实战
演示：缓存监控、分析和优化策略

环境要求：
- Python 3.13+
- langchain-core

- python-dotenv

import os
import time
from typing import List, Dict, Any
from collections import defaultdict
from dotenv import load_dotenv

from langchain_core.caches import InMemoryCache
from langchain_core.globals import set_llm_cache, get_llm_cache
from langchain_openai import ChatOpenAI

# 加载环境变量
load_dotenv()

class MonitoredCache(InMemoryCache):
    """带监控功能的缓存包装器"""

    def __init__(self, *args, **kwargs):
        super().__init__(*args, **kwargs)
        self.stats = {
            'hits': 0,
            'misses': 0,
            'total_requests': 0,
            'hit_times': [],
            'miss_times': [],
        }

    def lookup(self, prompt: str, llm_string: str):
        """查询缓存并记录统计"""
        start_time = time.time()
        result = super().lookup(prompt, llm_string)
        elapsed = time.time() - start_time

        self.stats['total_requests'] += 1
        if result is not None:
            self.stats['hits'] += 1
            self.stats['hit_times'].append(elapsed)
        else:
            self.stats['misses'] += 1
            self.stats['miss_times'].append(elapsed)

        return result

    def get_stats(self) -> Dict[str, Any]:
        """获取统计信息"""
        total = self.stats['total_requests']
        hits = self.stats['hits']
        misses = self.stats['misses']

        hit_rate = (hits / total * 100) if total > 0 else 0

        avg_hit_time = (
            sum(self.stats['hit_times']) / len(self.stats['hit_times'])
            if self.stats['hit_times'] else 0
        )
        avg_miss_time = (
            sum(self.stats['miss_times']) / len(self.stats['miss_times'])
            if self.stats['miss_times'] else 0
        )

        return {
            'total_requests': total,
            'cache_hits': hits,
            'cache_misses': misses,
            'hit_rate': hit_rate,
            'avg_hit_time': avg_hit_time,
            'avg_miss_time': avg_miss_time,
        }

    def reset_stats(self):
        """重置统计信息"""
        self.stats = {
            'hits': 0,
            'misses': 0,
            'total_requests': 0,
            'hit_times': [],
            'miss_times': [],
        }

# 创建监控缓存
monitored_cache = MonitoredCache()
set_llm_cache(monitored_cache)

print("✓ 监控缓存已启用")

stats = monitored_cache.get_stats()
print(f"\n📊 性能统计:")

print(f"   缓存命中: {stats['cache_hits']}")
print(f"   缓存未命中: {stats['cache_misses']}")
print(f"   命中率: {stats['hit_rate']:.1f}%")
print(f"   平均命中时间: {stats['avg_hit_time']:.4f}s")
print(f"   平均未命中时间: {stats['avg_miss_time']:.3f}s")

# ===== 3. 成本分析 =====
print("\n【步骤3】成本节省分析")
print("-" * 80)

# 假设成本

total_cost = stats['cache_misses'] * COST_PER_CALL
saved_cost = stats['cache_hits'] * COST_PER_CALL
potential_cost = stats['total_requests'] * COST_PER_CALL

print(f"💰 成本分析（假设 ${COST_PER_CALL}/次）:")
print(f"   实际成本: ${total_cost:.2f}")
print(f"   节省成本: ${saved_cost:.2f}")
print(f"   潜在成本: ${potential_cost:.2f}")
print(f"   节省比例: {(saved_cost/potential_cost*100):.1f}%")

# ===== 4. 缓存键优化 =====
print("\n【步骤4】缓存键优化策略")
print("-" * 80)

# 测试不同配置的影响

    {"temperature": 0.7, "max_tokens": 100},

## 关键点解释

### 1. 监控包装器模式

通过继承 InMemoryCache 实现监控功能，不侵入原有代码。

### 2. 核心性能指标

- **命中率**：最重要的指标，直接影响成本和性能
- **响应时间**：缓存命中 vs 未命中的时间差
- **成本节省**：避免的 API 调用成本

### 3. 查询规范化

统一查询格式提高命中率：
- 大小写统一
- 去除多余空格
- 标准化标点符号

### 4. 配置固定化

保持 LLM 配置参数一致：
- 固定 temperature
- 固定 max_tokens
- 避免动态参数

### 5. 批量处理优化

批量执行可以显著提升性能，因为缓存命中率会随着批量大小增加而提高。
