## 最小可用API

### 1. as_retriever() - 转换方法

**核心签名**:
```python
def as_retriever(
    search_type: str = "similarity",
    search_kwargs: dict = {},
    tags: list[str] | None = None
) -> VectorStoreRetriever
```

**最常用配置**:
```python
# 配置1:默认相似度搜索
retriever = vectorstore.as_retriever()

# 配置2:指定返回数量
retriever = vectorstore.as_retriever(
    search_kwargs={"k": 5}
)

# 配置3:MMR多样性搜索
retriever = vectorstore.as_retriever(
    search_type="mmr",
    search_kwargs={"k": 10, "lambda_mult": 0.5}
)

# 配置4:阈值过滤
retriever = vectorstore.as_retriever(
    search_type="similarity_score_threshold",
    search_kwargs={"score_threshold": 0.8}
)
```

---

### 2. invoke() - 同步调用

**核心签名**:
```python
def invoke(self, query: str, **kwargs) -> list[Document]
```

**基础用法**:
```python
# 基础调用
docs = retriever.invoke("What is LangChain?")

# 运行时覆盖参数
docs = retriever.invoke("query", k=10)

# 带元数据过滤
docs = retriever.invoke("query", filter={"source": "docs"})
```

---

### 3. ainvoke() - 异步调用

**核心签名**:
```python
async def ainvoke(self, query: str, **kwargs) -> list[Document]
```

**基础用法**:
```python
# 异步调用
docs = await retriever.ainvoke("What is LangChain?")

# 批量异步
import asyncio
tasks = [retriever.ainvoke(q) for q in queries]
results = await asyncio.gather(*tasks)
```

---

### 4. batch() - 批量调用

**核心签名**:
```python
def batch(self, queries: list[str], **kwargs) -> list[list[Document]]
```

**基础用法**:
```python
# 批量检索
queries = ["query1", "query2", "query3"]
batch_results = retriever.batch(queries)

# 配置并发数
batch_results = retriever.batch(
    queries,
    config={"max_concurrency": 5}
)
```

## 最小可用参数

### search_type - 检索策略

**3种策略**:
```python
# 策略1:相似度搜索(默认)
search_type="similarity"

# 策略2:MMR多样性搜索
search_type="mmr"

# 策略3:阈值过滤
search_type="similarity_score_threshold"
```

**选择建议**:
- 通用场景 → similarity
- 需要多样性 → mmr
- 需要质量控制 → similarity_score_threshold

---

### search_kwargs - 检索参数

**核心参数**:
```python
search_kwargs = {
    "k": 4,                    # 返回文档数量
    "score_threshold": 0.8,    # 相似度阈值(threshold策略)
    "fetch_k": 20,             # 候选文档数量(mmr策略)
    "lambda_mult": 0.5,        # 多样性参数(mmr策略)
    "filter": {"key": "value"} # 元数据过滤
}
```

**常用配置**:
```python
# 简单问答
search_kwargs={"k": 4}

# 复杂分析
search_kwargs={"k": 15}

# MMR多样性
search_kwargs={"k": 10, "fetch_k": 20, "lambda_mult": 0.5}

# 高精度过滤
search_kwargs={"k": 10, "score_threshold": 0.85}

# 元数据过滤
search_kwargs={"k": 5, "filter": {"source": "docs"}}
```

## 最小可用模式

### 模式1:基础检索

```python
from langchain_core.vectorstores import InMemoryVectorStore
from langchain_openai import OpenAIEmbeddings

# 1. 创建VectorStore
vectorstore = InMemoryVectorStore(embedding=OpenAIEmbeddings())

# 2. 添加文档
texts = [
    "LangChain is a framework for LLM applications",
    "LCEL is LangChain Expression Language",
    "RAG is Retrieval-Augmented Generation"
]
vectorstore.add_texts(texts)

# 3. 转换为Retriever
retriever = vectorstore.as_retriever()

# 4. 检索
docs = retriever.invoke("What is LangChain?")
for doc in docs:
    print(doc.page_content)
```

---

### 模式2:LCEL集成

```python
from langchain_core.prompts import ChatPromptTemplate
from langchain_core.runnables import RunnablePassthrough
from langchain_openai import ChatOpenAI

# 1. 创建Retriever
retriever = vectorstore.as_retriever()

# 2. 定义Prompt
prompt = ChatPromptTemplate.from_template("""
根据以下上下文回答问题:

上下文:
{context}

问题:{question}

回答:
""")

# 3. 格式化函数
def format_docs(docs):
    return "\n\n".join([doc.page_content for doc in docs])

# 4. 构建RAG管道
rag_chain = (
    {
        "context": retriever | format_docs,
        "question": RunnablePassthrough()
    }
    | prompt
    | ChatOpenAI(model="gpt-4")
)

# 5. 调用
answer = rag_chain.invoke("What is LangChain?")
print(answer.content)
```

### 模式4:批量处理

```python
# 批量检索
queries = [
    "What is LangChain?",
    "What is LCEL?",
    "What is RAG?"
]

# 同步批量
batch_results = retriever.batch(queries)
for query, docs in zip(queries, batch_results):
    print(f"{query}: {len(docs)} docs")

# 异步批量
import asyncio

async def batch_retrieve():
    tasks = [retriever.ainvoke(q) for q in queries]
    return await asyncio.gather(*tasks)

results = asyncio.run(batch_retrieve())
```

### 模板3:生产级配置

```python
from langchain_core.vectorstores import InMemoryVectorStore
from langchain_openai import OpenAIEmbeddings
from langchain_core.callbacks import StdOutCallbackHandler
from dotenv import load_dotenv
import logging

load_dotenv()

# 配置日志
logging.basicConfig(level=logging.INFO)
logger = logging.getLogger(__name__)

# 创建VectorStore
vectorstore = InMemoryVectorStore(embedding=OpenAIEmbeddings())

# 添加文档
texts = ["doc1", "doc2", "doc3"]
vectorstore.add_texts(texts)

# 创建Retriever(带标签和回调)
retriever = vectorstore.as_retriever(
    search_type="similarity",
    search_kwargs={"k": 5},
    tags=["production", "v1.0"]
)

# 检索(带错误处理)
def safe_retrieve(query: str):
    try:
        docs = retriever.invoke(
            query,
            config={"callbacks": [StdOutCallbackHandler()]}
        )
        logger.info(f"Retrieved {len(docs)} documents")
        return docs
    except Exception as e:
        logger.error(f"Retrieval failed: {e}")
        return []

# 使用
docs = safe_retrieve("query")
```

## 参数速查表

### search_type参数

| 值 | 说明 | 适用场景 |
|----|------|----------|
| similarity | 相似度搜索 | 通用场景 |
| mmr | 多样性搜索 | 推荐系统 |
| similarity_score_threshold | 阈值过滤 | 高精度问答 |

### search_kwargs参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| k | int | 4 | 返回文档数量 |
| score_threshold | float | - | 相似度阈值(0-1) |
| fetch_k | int | 20 | MMR候选数量 |
| lambda_mult | float | 0.5 | MMR多样性(0-1) |
| filter | dict | {} | 元数据过滤 |

### k值选择建议

| 场景 | 推荐k值 |
|------|---------|
| 简单问答 | 3-5 |
| 复杂分析 | 10-20 |
| 推荐系统 | 20-50 |

### lambda_mult选择建议

| 需求 | 推荐值 |
|------|--------|
| 高相关性 | 0.8-1.0 |
| 平衡 | 0.5 |
| 高多样性 | 0.0-0.2 |

### score_threshold选择建议

| 需求 | 推荐值 |
|------|--------|
| 高精度 | 0.8-0.9 |
| 平衡 | 0.7-0.8 |
| 高召回 | 0.6-0.7 |

## 常见错误及解决

### 错误1:无效的search_type

```python
# 错误
retriever = vectorstore.as_retriever(search_type="invalid")
# ValueError: search_type of invalid not allowed

# 解决
retriever = vectorstore.as_retriever(search_type="similarity")
```

### 错误2:缺少score_threshold

```python
# 错误
retriever = vectorstore.as_retriever(
    search_type="similarity_score_threshold"
)
# ValueError: score_threshold is not specified

# 解决
retriever = vectorstore.as_retriever(
    search_type="similarity_score_threshold",
    search_kwargs={"score_threshold": 0.8}
)
```

### 错误3:返回文档为空

```python
# 检查
docs = retriever.invoke("query")
if len(docs) == 0:
    # 可能原因:
    # 1. VectorStore中没有文档
    # 2. score_threshold设置过高
    # 3. 查询与文档不相关

    # 解决:降低阈值或检查数据
    retriever = vectorstore.as_retriever(
        search_type="similarity_score_threshold",
        search_kwargs={"score_threshold": 0.6}  # 降低阈值
    )
```
