## 核心命令速查

| 命令 | 用途 | 常用选项 |
|------|------|---------|
| `openclaw gateway` | 启动 Gateway | `--port`, `--verbose` |
| `openclaw message send` | 发送消息 | `--to`, `--message` |
| `openclaw agent` | AI 交互 | `--message`, `--thinking` |
| `openclaw channels` | 通道管理 | `list`, `status`, `add` |
| `openclaw config` | 配置管理 | `get`, `set`, `unset` |
