# 核心概念 08: 与 LCEL 和其他组件集成

## 概述

ChatPromptTemplate 的核心价值之一是与 LangChain 生态系统的无缝集成。通过 LCEL (LangChain Expression Language) 的管道操作符 `|`，ChatPromptTemplate 可以与 ChatModel、Memory、Agent 等组件链式组合，构建完整的对话式 AI 应用。

**核心问题**: 为什么需要组件集成？

**答案**: 因为实际的对话式 AI 应用不是孤立的模板，而是由多个组件协作完成的完整流程：提示词构建 → LLM 调用 → 输出解析 → 历史管理 → 工具调用。

---

## 1. LCEL 管道操作符集成

### 1.1 基础概念

**LCEL (LangChain Expression Language)** 是 LangChain 的核心表达式语言，使用管道操作符 `|` 连接不同的组件。

**核心特性**:
- 使用 `|` 操作符链式组合
- 自动处理输入输出格式转换
- 支持同步和异步调用
- 支持流式处理

**来源**: [Context7 文档](reference/context7_langchain_02.md) | [源码分析](reference/source_chatprompt_01.md)

### 1.2 与 ChatModel 的基础集成

**场景 1: 最简单的链式组合**

```python
from langchain_core.prompts import ChatPromptTemplate
from langchain_openai import ChatOpenAI

# 创建提示词模板
prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant."),
    ("human", "{input}")
])

# 创建 LLM
llm = ChatOpenAI(model="gpt-4")

# 使用 | 操作符组合
chain = prompt | llm

# 调用
response = chain.invoke({"input": "What is Python?"})
print(response.content)
```

**工作流程**:
1. `prompt.invoke({"input": "..."})` → 生成消息列表
2. `llm.invoke(messages)` → 调用 LLM
3. 返回 AIMessage 对象

**场景 2: 添加输出解析器**

```python
from langchain_core.prompts import ChatPromptTemplate
from langchain_openai import ChatOpenAI
from langchain_core.output_parsers import StrOutputParser

# 创建完整处理链
prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant."),
    ("human", "{input}")
])

llm = ChatOpenAI()
output_parser = StrOutputParser()

# 三段式链式组合
chain = prompt | llm | output_parser

# 调用
result = chain.invoke({"input": "Hello"})
print(result)  # 直接输出字符串，而非 AIMessage 对象
```

**优势**:
- 自动提取 AIMessage 的 content
- 返回纯字符串，便于后续处理
- 代码更简洁

**场景 3: 流式处理**

```python
from langchain_core.prompts import ChatPromptTemplate
from langchain_openai import ChatOpenAI

prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant."),
    ("human", "{input}")
])

llm = ChatOpenAI()
chain = prompt | llm

# 流式调用
for chunk in chain.stream({"input": "Tell me a story"}):
    print(chunk.content, end="", flush=True)
```

**特点**:
- 实时输出 LLM 生成的内容
- 提升用户体验
- 适合长文本生成

### 1.3 LCEL 的高级特性

**特性 1: 批量处理**

```python
from langchain_core.prompts import ChatPromptTemplate
from langchain_openai import ChatOpenAI

prompt = ChatPromptTemplate.from_messages([
    ("system", "Translate to French."),
    ("human", "{text}")
])

llm = ChatOpenAI()
chain = prompt | llm

# 批量调用
inputs = [
    {"text": "Hello"},
    {"text": "Goodbye"},
    {"text": "Thank you"}
]

results = chain.batch(inputs)
for result in results:
    print(result.content)
```

**特性 2: 异步调用**

```python
import asyncio
from langchain_core.prompts import ChatPromptTemplate
from langchain_openai import ChatOpenAI

prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant."),
    ("human", "{input}")
])

llm = ChatOpenAI()
chain = prompt | llm

# 异步调用
async def main():
    result = await chain.ainvoke({"input": "Hello"})
    print(result.content)

asyncio.run(main())
```

**特性 3: 并行处理**

```python
from langchain_core.prompts import ChatPromptTemplate
from langchain_openai import ChatOpenAI
from langchain_core.runnables import RunnableParallel

prompt1 = ChatPromptTemplate.from_template("Translate to French: {text}")
prompt2 = ChatPromptTemplate.from_template("Translate to Spanish: {text}")

llm = ChatOpenAI()

# 并行执行两个翻译任务
parallel_chain = RunnableParallel(
    french=prompt1 | llm,
    spanish=prompt2 | llm
)

result = parallel_chain.invoke({"text": "Hello"})
print(result["french"].content)
print(result["spanish"].content)
```

## 2. 与 Memory 组件集成

### 2.1 对话历史管理

**场景 1: 使用 MessagesPlaceholder 管理历史**

```python
from langchain_core.prompts import ChatPromptTemplate, MessagesPlaceholder
from langchain_openai import ChatOpenAI
from langchain_core.messages import HumanMessage, AIMessage

# 创建带历史记录的模板
prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant."),
    MessagesPlaceholder("chat_history"),
    ("human", "{input}")
])

llm = ChatOpenAI()
chain = prompt | llm

# 第一轮对话
response1 = chain.invoke({
    "chat_history": [],
    "input": "My name is Alice"
})
print(response1.content)

# 第二轮对话（带历史）
response2 = chain.invoke({
    "chat_history": [
        HumanMessage(content="My name is Alice"),
        AIMessage(content="Nice to meet you, Alice!")
    ],
    "input": "What's my name?"
})
print(response2.content)  # 应该回答 "Alice"
```

**场景 2: 使用 ConversationBufferMemory**

```python
from langchain_core.prompts import ChatPromptTemplate, MessagesPlaceholder
from langchain_openai import ChatOpenAI
from langchain.memory import ConversationBufferMemory
from langchain.chains import LLMChain

# 创建 Memory
memory = ConversationBufferMemory(
    memory_key="chat_history",
    return_messages=True
)

# 创建模板
prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant."),
    MessagesPlaceholder(variable_name="chat_history"),
    ("human", "{input}")
])

llm = ChatOpenAI()

# 创建 LLMChain（自动管理 Memory）
chain = LLMChain(
    llm=llm,
    prompt=prompt,
    memory=memory
)

# 对话
response1 = chain.invoke({"input": "My name is Bob"})
print(response1["text"])

response2 = chain.invoke({"input": "What's my name?"})
print(response2["text"])  # 应该回答 "Bob"
```

**场景 3: 使用 ConversationBufferWindowMemory（限制历史长度）**

```python
from langchain_core.prompts import ChatPromptTemplate, MessagesPlaceholder
from langchain_openai import ChatOpenAI
from langchain.memory import ConversationBufferWindowMemory

# 只保留最近 2 轮对话
memory = ConversationBufferWindowMemory(
    k=2,  # 保留最近 2 轮
    memory_key="chat_history",
    return_messages=True
)

prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant."),
    MessagesPlaceholder(variable_name="chat_history"),
    ("human", "{input}")
])

llm = ChatOpenAI()
chain = LLMChain(llm=llm, prompt=prompt, memory=memory)

# 多轮对话
chain.invoke({"input": "My name is Alice"})
chain.invoke({"input": "I like Python"})
chain.invoke({"input": "I live in Paris"})
chain.invoke({"input": "What do you know about me?"})
# 只会记得最近 2 轮对话
```

### 2.2 使用 MessagesPlaceholder 的 n_messages 参数

```python
from langchain_core.prompts import ChatPromptTemplate, MessagesPlaceholder
from langchain_openai import ChatOpenAI

# 限制历史消息数量
prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant."),
    MessagesPlaceholder("chat_history", n_messages=4),  # 只保留最后 4 条消息
    ("human", "{input}")
])

llm = ChatOpenAI()
chain = prompt | llm

# 传递大量历史消息
long_history = [
    ("human", "Message 1"),
    ("ai", "Response 1"),
    ("human", "Message 2"),
    ("ai", "Response 2"),
    ("human", "Message 3"),
    ("ai", "Response 3"),
    ("human", "Message 4"),
    ("ai", "Response 4"),
]

response = chain.invoke({
    "chat_history": long_history,
    "input": "Current question"
})
# 只会使用最后 4 条消息
```

## 3. 与 Agent 系统集成

### 3.1 Agent 提示词模板

**场景 1: 基础 Agent 模板**

```python
from langchain_core.prompts import ChatPromptTemplate, MessagesPlaceholder
from langchain_openai import ChatOpenAI
from langchain.agents import AgentExecutor, create_openai_functions_agent
from langchain.tools import Tool

# 定义工具
def search_tool(query: str) -> str:
    return f"Search results for: {query}"

def calculator_tool(expression: str) -> str:
    return str(eval(expression))

tools = [
    Tool(name="Search", func=search_tool, description="Search the web"),
    Tool(name="Calculator", func=calculator_tool, description="Calculate math expressions")
]

# 创建 Agent 提示词模板
prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant with access to tools."),
    MessagesPlaceholder("chat_history", optional=True),
    ("human", "{input}"),
    MessagesPlaceholder("agent_scratchpad")
])

llm = ChatOpenAI(model="gpt-4")

# 创建 Agent
agent = create_openai_functions_agent(llm, tools, prompt)
agent_executor = AgentExecutor(agent=agent, tools=tools)

# 使用 Agent
response = agent_executor.invoke({
    "input": "What is 25 * 4?",
    "chat_history": []
})
print(response["output"])
```

**场景 2: 自定义 Agent 提示词**

```python
from langchain_core.prompts import ChatPromptTemplate, MessagesPlaceholder

# 更详细的 Agent 提示词
agent_prompt = ChatPromptTemplate.from_messages([
    ("system", """You are a helpful AI assistant with access to the following tools:

{tools}

To use a tool, respond with a JSON object in this format:
{{
    "action": "tool_name",
    "action_input": "input_for_tool"
}}

To give a final answer, respond with:
{{
    "action": "Final Answer",
    "action_input": "your final answer"
}}

Always think step by step."""),
    MessagesPlaceholder("chat_history", optional=True),
    ("human", "{input}"),
    MessagesPlaceholder("agent_scratchpad")
])
```

### 3.2 Agent Scratchpad 的使用

**Agent Scratchpad** 是 Agent 的"草稿纸"，记录工具调用的历史。

## 4. RAG 系统集成

### 4.1 基础 RAG 链

```python
from langchain_core.prompts import ChatPromptTemplate
from langchain_openai import ChatOpenAI, OpenAIEmbeddings
from langchain_community.vectorstores import Chroma
from langchain_core.runnables import RunnablePassthrough
from langchain_core.output_parsers import StrOutputParser

# 创建向量存储
embeddings = OpenAIEmbeddings()
vectorstore = Chroma.from_texts(
    texts=[
        "Python is a high-level programming language.",
        "Python was created by Guido van Rossum in 1991.",
        "Python is known for its simplicity and readability."
    ],
    embedding=embeddings
)

# 创建检索器
retriever = vectorstore.as_retriever()

# 创建 RAG 提示词模板
prompt = ChatPromptTemplate.from_messages([
    ("system", "Answer based on the following context:\n\n{context}"),
    ("human", "{question}")
])

llm = ChatOpenAI()

# 构建 RAG 链
rag_chain = (
    {"context": retriever, "question": RunnablePassthrough()}
    | prompt
    | llm
    | StrOutputParser()
)

# 使用
response = rag_chain.invoke("When was Python created?")
print(response)
```

### 4.2 对话式 RAG

```python
from langchain_core.prompts import ChatPromptTemplate, MessagesPlaceholder
from langchain_openai import ChatOpenAI
from langchain_core.runnables import RunnablePassthrough

# 对话式 RAG 模板
prompt = ChatPromptTemplate.from_messages([
    ("system", "Answer based on the context. Context:\n{context}"),
    MessagesPlaceholder("chat_history"),
    ("human", "{question}")
])

llm = ChatOpenAI()

# 构建对话式 RAG 链
conversational_rag_chain = (
    {
        "context": retriever,
        "question": RunnablePassthrough(),
        "chat_history": lambda x: x.get("chat_history", [])
    }
    | prompt
    | llm
)

# 使用
response = conversational_rag_chain.invoke({
    "question": "What is Python?",
    "chat_history": []
})
```

## 5. 完整对话系统示例

### 5.1 端到端对话系统

```python
from langchain_core.prompts import ChatPromptTemplate, MessagesPlaceholder
from langchain_openai import ChatOpenAI
from langchain.memory import ConversationBufferMemory
from langchain_core.output_parsers import StrOutputParser

# 创建完整的对话系统
class ConversationalSystem:
    def __init__(self):
        # 初始化 Memory
        self.memory = ConversationBufferMemory(
            memory_key="chat_history",
            return_messages=True
        )

        # 创建提示词模板
        self.prompt = ChatPromptTemplate.from_messages([
            ("system", "You are a helpful assistant. Current date: {date}"),
            MessagesPlaceholder("chat_history"),
            ("human", "{input}")
        ])

        # 创建 LLM
        self.llm = ChatOpenAI(model="gpt-4")

        # 创建处理链
        self.chain = self.prompt | self.llm | StrOutputParser()

    def chat(self, user_input: str) -> str:
        # 获取历史记录
        chat_history = self.memory.load_memory_variables({})["chat_history"]

        # 调用链
        response = self.chain.invoke({
            "date": "2026-02-26",
            "chat_history": chat_history,
            "input": user_input
        })

        # 保存到 Memory
        self.memory.save_context(
            {"input": user_input},
            {"output": response}
        )

        return response

# 使用
system = ConversationalSystem()
print(system.chat("My name is Alice"))
print(system.chat("What's my name?"))
```

## 6. 最佳实践

### 6.1 链式组合的原则

**原则 1: 保持链的简洁性**

```python
# ✅ 好的做法 - 清晰的三段式
chain = prompt | llm | output_parser

# ❌ 不好的做法 - 过于复杂
chain = prompt | preprocessor | llm | postprocessor | validator | formatter
```

**原则 2: 使用 RunnablePassthrough 传递数据**

```python
from langchain_core.runnables import RunnablePassthrough

# ✅ 使用 RunnablePassthrough
chain = (
    {"context": retriever, "question": RunnablePassthrough()}
    | prompt
    | llm
)

# 调用时直接传递问题
chain.invoke("What is Python?")
```

**原则 3: 合理使用 Memory**

```python
# ✅ 对话系统使用 Memory
# ❌ 单次查询不需要 Memory
```

### 6.2 性能优化

**优化 1: 使用批量处理**

```python
# 批量处理多个请求
inputs = [{"input": q} for q in questions]
results = chain.batch(inputs)
```

**优化 2: 使用异步调用**

```python
# 异步处理提升并发性能
async def process_multiple():
    tasks = [chain.ainvoke({"input": q}) for q in questions]
    results = await asyncio.gather(*tasks)
    return results
```

## 7. 常见问题

### Q1: 如何在链中传递额外的参数？

**A**: 使用字典格式传递多个参数

```python
chain = prompt | llm

response = chain.invoke({
    "input": "Hello",
    "language": "French",
    "style": "formal"
})
```

### Q2: 如何处理链中的错误？

**A**: 使用 try-except 捕获异常

```python
try:
    response = chain.invoke({"input": "..."})
except Exception as e:
    print(f"Error: {e}")
```

### Q3: 如何调试 LCEL 链？

**A**: 使用 `chain.get_graph()` 查看链的结构

```python
graph = chain.get_graph()
print(graph.print_ascii())
```

## 8. 总结

### 核心要点

1. **LCEL 管道操作符**: 使用 `|` 连接组件，构建完整处理链
2. **与 ChatModel 集成**: prompt | llm | output_parser
3. **与 Memory 集成**: 使用 MessagesPlaceholder 管理对话历史
4. **与 Agent 集成**: 使用 agent_scratchpad 记录工具调用
5. **RAG 系统集成**: 结合 retriever 构建知识库问答
6. **流式处理**: 使用 stream() 实现实时输出
7. **批量处理**: 使用 batch() 提升性能
8. **异步调用**: 使用 ainvoke() 提升并发性能

### 最佳实践

- ✅ 保持链的简洁性（3-5 个组件）
- ✅ 使用 RunnablePassthrough 传递数据
- ✅ 合理使用 Memory 管理历史
- ✅ 限制历史消息数量（n_messages）
- ✅ 使用批量和异步提升性能
- ❌ 避免过于复杂的链
- ❌ 避免在单次查询中使用 Memory
- ❌ 避免无限制的历史消息
