## 4.1 三大主流Agent类型

### OpenAI Functions Agent

**一句话**: 使用函数调用能力的Agent，适合生产环境。

```python
from langchain_openai import ChatOpenAI
from langchain.agents import create_openai_functions_agent, AgentExecutor
from langchain.tools import tool
from langchain.prompts import ChatPromptTemplate, MessagesPlaceholder

# 定义工具
@tool
def search_web(query: str) -> str:
    """搜索网页内容"""
    return f"搜索结果: {query}"

# 创建模型
llm = ChatOpenAI(model="gpt-3.5-turbo")

# 创建Prompt
prompt = ChatPromptTemplate.from_messages([
    ("system", "你是一个有用的助手"),
    ("user", "{input}"),
    MessagesPlaceholder(variable_name="agent_scratchpad"),
])

# 创建Agent
agent = create_openai_functions_agent(llm, [search_web], prompt)
executor = AgentExecutor(agent=agent, tools=[search_web], verbose=True)

# 运行
result = executor.invoke({"input": "搜索Python教程"})
print(result["output"])
```

**适用场景**:
- ✅ 使用OpenAI或Anthropic模型
- ✅ 工具参数简单（1-3个参数）
- ✅ 生产环境，需要高可靠性

---

### ReAct Agent

**一句话**: 基于推理-行动循环的Agent，适合开源模型和调试。

```python
from langchain.agents import create_react_agent, AgentExecutor
from langchain_openai import ChatOpenAI
from langchain.prompts import PromptTemplate

# 创建模型
llm = ChatOpenAI(model="gpt-3.5-turbo")

# ReAct Prompt模板
template = """Answer the following questions as best you can. You have access to the following tools:

{tools}

Use the following format:

Question: the input question you must answer
Thought: you should always think about what to do
Action: the action to take, should be one of [{tool_names}]
Action Input: the input to the action
Observation: the result of the action
... (this Thought/Action/Action Input/Observation can repeat N times)
Thought: I now know the final answer
Final Answer: the final answer to the original input question

Begin!

Question: {input}
Thought:{agent_scratchpad}"""

prompt = PromptTemplate.from_template(template)

# 创建Agent
agent = create_react_agent(llm, [search_web], prompt)
executor = AgentExecutor(agent=agent, tools=[search_web], verbose=True)

# 运行
result = executor.invoke({"input": "搜索Python教程"})
```

**适用场景**:
- ✅ 使用开源模型（Llama, Mistral等）
- ✅ 需要查看推理过程
- ✅ 调试和开发阶段

---

### Structured Chat Agent

**一句话**: 支持复杂参数的Agent，适合多输入工具。

```python
from langchain.agents import create_structured_chat_agent, AgentExecutor
from langchain.tools import tool
from typing import List, Dict

# 定义复杂工具
@tool
def advanced_search(
    query: str,
    filters: List[str],
    max_results: int = 10
) -> str:
    """高级搜索工具

    Args:
        query: 搜索关键词
        filters: 过滤条件列表
        max_results: 最大结果数
    """
    return f"搜索 {query}，过滤 {filters}，返回 {max_results} 条结果"

# 创建Agent
agent = create_structured_chat_agent(llm, [advanced_search], prompt)
executor = AgentExecutor(agent=agent, tools=[advanced_search], verbose=True)

# 运行
result = executor.invoke({
    "input": "搜索Python教程，只要视频类型，最多10条"
})
```

**适用场景**:
- ✅ 工具参数复杂（多个参数、嵌套结构）
- ✅ 需要精确控制工具调用
- ✅ 数据分析、复杂查询场景

## 4.2 快速选择规则（3步决策）

### 步骤1：检查模型能力

```python
# 检查模型是否支持函数调用
def supports_function_calling(llm) -> bool:
    return hasattr(llm, 'bind_tools')

# 示例
from langchain_openai import ChatOpenAI
from langchain_community.llms import Ollama

openai_llm = ChatOpenAI(model="gpt-4")
print(supports_function_calling(openai_llm))  # True

ollama_llm = Ollama(model="llama2")
print(supports_function_calling(ollama_llm))  # False
```

**决策**:
- 不支持函数调用 → **使用 ReAct Agent**（唯一选择）
- 支持函数调用 → 进入步骤2

---

### 步骤2：检查工具复杂度

```python
# 简单工具：参数 ≤ 2个
@tool
def simple_tool(query: str) -> str:
    """简单工具"""
    return f"结果: {query}"

# 复杂工具：参数 > 2个 或 包含List/Dict
@tool
def complex_tool(
    query: str,
    filters: List[str],
    options: Dict[str, any]
) -> str:
    """复杂工具"""
    return f"结果: {query}"
```

**决策**:
- 工具参数复杂 → **使用 Structured Chat Agent**
- 工具参数简单 → 进入步骤3

---

### 步骤3：选择优先级

```python
# 生产环境：优先可靠性
agent = create_openai_functions_agent(llm, tools, prompt)

# 最新API：优先简洁性
agent = create_agent(llm, tools)

# 调试需求：优先可解释性
agent = create_react_agent(llm, tools, prompt)
```

**决策**:
- 生产环境 → **OpenAI Functions Agent**
- 最新API → **Tool Calling Agent (create_agent)**
- 调试需求 → **ReAct Agent**

## 4.3 2026推荐API：create_agent()

### 统一的Agent创建接口

```python
from langchain.agents import create_agent
from langchain_openai import ChatOpenAI
from langchain.tools import tool

# 定义工具
@tool
def calculator(expression: str) -> str:
    """计算数学表达式"""
    return str(eval(expression))

@tool
def search(query: str) -> str:
    """搜索网页"""
    return f"搜索结果: {query}"

# 创建Agent（自动选择最佳类型）
llm = ChatOpenAI(model="gpt-4")
agent = create_agent(
    llm,
    tools=[calculator, search],
    system_prompt="你是一个有用的助手，可以进行计算和搜索。"
)

# 运行（新的调用方式）
result = agent.invoke({
    "messages": [("user", "计算 123 + 456，然后搜索结果的含义")]
})

print(result["messages"][-1].content)
```

**优势**:
- ✅ 自动选择最佳Agent类型
- ✅ 统一的API接口
- ✅ 更简洁的代码
- ✅ 2026年官方推荐

### 模板2：生产环境（OpenAI Functions）

```python
"""
生产环境Agent - 高可靠性
"""

from langchain_openai import ChatOpenAI
from langchain.agents import create_openai_functions_agent, AgentExecutor
from langchain.prompts import ChatPromptTemplate, MessagesPlaceholder
from langchain.tools import tool

# 1. 定义工具
@tool
def my_tool(input: str) -> str:
    """工具描述"""
    return f"处理结果: {input}"

# 2. 创建Prompt
prompt = ChatPromptTemplate.from_messages([
    ("system", "你是一个专业的助手"),
    ("user", "{input}"),
    MessagesPlaceholder(variable_name="agent_scratchpad"),
])

# 3. 创建Agent
llm = ChatOpenAI(model="gpt-3.5-turbo", temperature=0)
agent = create_openai_functions_agent(llm, [my_tool], prompt)

# 4. 创建Executor
executor = AgentExecutor(
    agent=agent,
    tools=[my_tool],
    verbose=True,
    max_iterations=5,  # 最大迭代次数
    handle_parsing_errors=True  # 处理解析错误
)

# 5. 运行
result = executor.invoke({"input": "你的问题"})
print(result["output"])
```

## 4.5 常见问题排查

### 问题1：Agent不调用工具

**症状**:
```
Agent直接返回答案，没有调用工具
```

**排查步骤**:

```python
# 1. 检查工具描述是否清晰
@tool
def search(query: str) -> str:
    """搜索网页内容。当用户需要查找信息时使用此工具。"""  # ✅ 清晰的描述
    return f"结果: {query}"

# 2. 检查Prompt是否包含工具使用指令
prompt = ChatPromptTemplate.from_messages([
    ("system", "你是一个助手。当需要查找信息时，使用search工具。"),  # ✅ 明确指令
    ("user", "{input}"),
    MessagesPlaceholder(variable_name="agent_scratchpad"),
])

# 3. 启用verbose查看推理过程
executor = AgentExecutor(agent=agent, tools=tools, verbose=True)  # ✅ 调试模式
```

---

### 问题2：工具调用失败

**症状**:
```
Error: Tool execution failed
```

**排查步骤**:

```python
# 1. 检查工具参数类型
@tool
def calculator(expression: str) -> str:  # ✅ 明确类型注解
    """计算数学表达式"""
    try:
        return str(eval(expression))
    except Exception as e:
        return f"计算错误: {e}"  # ✅ 错误处理

# 2. 启用错误处理
executor = AgentExecutor(
    agent=agent,
    tools=tools,
    handle_parsing_errors=True,  # ✅ 自动处理解析错误
    verbose=True
)

# 3. 测试工具独立运行
result = calculator.invoke("123 + 456")
print(result)  # 确保工具本身可以运行
```

---

### 问题3：模型不支持函数调用

**症状**:
```
Error: Model does not support function calling
```

**解决方案**:

```python
# 检查模型能力
from langchain_openai import ChatOpenAI

llm = ChatOpenAI(model="gpt-3.5-turbo")

# 方案1: 使用支持函数调用的模型
if hasattr(llm, 'bind_tools'):
    agent = create_openai_functions_agent(llm, tools, prompt)  # ✅
else:
    # 方案2: 切换到ReAct Agent
    agent = create_react_agent(llm, tools, prompt)  # ✅ 兼容所有模型
```

---

### 问题4：Agent陷入无限循环

**症状**:
```
Agent一直重复调用工具，不返回最终答案
```

**解决方案**:

```python
# 设置最大迭代次数
executor = AgentExecutor(
    agent=agent,
    tools=tools,
    max_iterations=5,  # ✅ 限制最大迭代
    early_stopping_method="generate"  # ✅ 提前停止策略
)

# 改进工具描述，明确返回格式
@tool
def search(query: str) -> str:
    """搜索网页内容。返回搜索结果的摘要。"""  # ✅ 明确返回内容
    return f"搜索结果: {query}"
```

## 快速参考卡

### Agent类型选择速查

```
模型支持函数调用？
├─ 否 → ReAct Agent
└─ 是 → 工具参数复杂？
    ├─ 是 → Structured Chat Agent
    └─ 否 → 优先级？
        ├─ 生产环境 → OpenAI Functions Agent
        ├─ 最新API → Tool Calling Agent (create_agent)
        └─ 调试需求 → ReAct Agent
```

### API对比速查

| API | 适用场景 | 代码复杂度 | 2026状态 |
|-----|---------|-----------|---------|
| `create_agent()` | 通用 | ⭐ | 推荐 |
| `create_openai_functions_agent()` | 生产环境 | ⭐⭐ | 稳定 |
| `create_react_agent()` | 开源模型 | ⭐⭐⭐ | 稳定 |
| `create_structured_chat_agent()` | 复杂工具 | ⭐⭐⭐ | 稳定 |

### 常见错误速查

| 错误 | 原因 | 解决方案 |
|------|------|---------|
| 不调用工具 | 工具描述不清晰 | 改进描述，添加使用场景 |
| 调用失败 | 参数类型错误 | 添加类型注解和错误处理 |
| 不支持函数调用 | 模型不兼容 | 切换到ReAct Agent |
| 无限循环 | 没有停止条件 | 设置max_iterations |
