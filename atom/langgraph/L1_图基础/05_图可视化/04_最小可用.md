# 图可视化 - 最小可用

## 核心目标

掌握以下内容,就能开始使用 LangGraph 图可视化:

### 4.1 获取图对象

**核心方法**: `get_graph()`

```python
from langgraph.graph import StateGraph, START, END
from typing import TypedDict

# 定义状态
class State(TypedDict):
    messages: list[str]

# 创建图
workflow = StateGraph(State)
workflow.add_node("process", lambda x: {"messages": x["messages"] + ["processed"]})
workflow.add_edge(START, "process")
workflow.add_edge("process", END)

# 编译图
app = workflow.compile()

# 获取图对象
graph = app.get_graph()
```

**关键点**:
- `get_graph()` 返回 Graph 对象
- Graph 对象包含节点和边的完整信息
- 支持 `xray` 参数控制子图展开

[来源: reference/source_图可视化_01.md - get_graph() 方法]

### 4.2 生成 Mermaid 文本

**核心方法**: `draw_mermaid()`

```python
# 生成 Mermaid 文本
mermaid_text = graph.draw_mermaid()
print(mermaid_text)

# 保存到文件
with open("workflow.mmd", "w") as f:
    f.write(mermaid_text)
```

**关键点**:
- 返回 Mermaid 格式的文本字符串
- 可以保存到文件或嵌入 Markdown
- 支持样式定制参数

[来源: reference/context7_langchain_01.md - Python API]

### 4.3 生成 PNG 图像

**核心方法**: `draw_mermaid_png()`

```python
# 生成 PNG 图像
png_data = graph.draw_mermaid_png()

# 保存到文件
with open("workflow.png", "wb") as f:
    f.write(png_data)
```

**关键点**:
- 返回 PNG 图像的字节数据
- 需要网络连接(使用 Mermaid.ink API)
- 可以保存到文件或在 Jupyter 中显示

[来源: reference/context7_langchain_01.md - Python API]

### 4.4 Jupyter 自动显示

**核心方法**: 直接显示对象

```python
# 在 Jupyter 中直接显示
app  # 自动调用 _repr_mimebundle_() 并显示图表
```

**关键点**:
- Jupyter 自动调用 `_repr_mimebundle_()` 方法
- 自动生成并显示 PNG 图像
- 无需手动调用任何方法

[来源: reference/source_图可视化_01.md - Jupyter 集成]

### 4.5 使用 xray 参数

**核心方法**: `get_graph(xray=True)`

```python
# 展开所有子图
graph_detailed = app.get_graph(xray=True)

# 在 Jupyter 中显示
from IPython.display import Image, display
display(Image(graph_detailed.draw_mermaid_png()))
```

**关键点**:
- `xray=False`: 只显示顶层结构(默认)
- `xray=True`: 递归展开所有子图
- `xray=N`: 只展开 N 层子图

[来源: reference/source_图可视化_01.md - get_graph() 方法实现]

## 完整示例

```python
from langgraph.graph import StateGraph, START, END
from typing import TypedDict
from IPython.display import Image, display

# 1. 定义状态
class State(TypedDict):
    messages: list[str]

# 2. 创建图
workflow = StateGraph(State)
workflow.add_node("retrieve", lambda x: {"messages": x["messages"] + ["retrieved"]})
workflow.add_node("process", lambda x: {"messages": x["messages"] + ["processed"]})
workflow.add_node("generate", lambda x: {"messages": x["messages"] + ["generated"]})

# 3. 添加边
workflow.add_edge(START, "retrieve")
workflow.add_edge("retrieve", "process")
workflow.add_edge("process", "generate")
workflow.add_edge("generate", END)

# 4. 编译图
app = workflow.compile()

# 5. 可视化 - 方法1: Jupyter 自动显示
app

# 6. 可视化 - 方法2: 生成 Mermaid 文本
mermaid_text = app.get_graph().draw_mermaid()
print(mermaid_text)

# 7. 可视化 - 方法3: 生成 PNG 图像
png_data = app.get_graph().draw_mermaid_png()
with open("workflow.png", "wb") as f:
    f.write(png_data)

# 8. 可视化 - 方法4: 手动显示
display(Image(app.get_graph().draw_mermaid_png()))

# 9. 可视化 - 方法5: 展开子图
display(Image(app.get_graph(xray=True).draw_mermaid_png()))
```
