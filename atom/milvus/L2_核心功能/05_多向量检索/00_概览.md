## 💡 为什么要学习多向量检索？

### 实际应用场景

**场景1：多模态检索**
- 电商搜索：同时检索商品标题向量 + 商品图片向量
- 文档问答：同时检索文本向量 + 图表向量

**场景2：多粒度检索**
- 同时检索文档级向量 + 段落级向量
- 同时检索标题向量 + 正文向量

**场景3：多语言检索**
- 同时检索中文向量 + 英文向量
- 跨语言文档检索

**场景4：多模型检索**
- 同时使用不同的 Embedding 模型（如 OpenAI + BGE）
- 融合多个模型的检索结果

## ⚡ 核心价值

多向量检索是 Milvus 的高级特性，能够：

1. **提升检索准确率** - 融合多个向量的检索结果
2. **支持多模态应用** - 同时处理文本、图片、音频等
3. **增强系统灵活性** - 根据场景动态调整检索策略
4. **优化用户体验** - 提供更精准的搜索结果
