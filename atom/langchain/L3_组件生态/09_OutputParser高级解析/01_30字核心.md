# 30字核心

**OutputParser 是 LLM 输出的结构化转换器，将文本转为 JSON、Pydantic 对象等格式，支持流式解析与错误修复。**

---

## 核心要素拆解

### 1. 本质定义
- **结构化转换器**：将非结构化的 LLM 文本输出转换为结构化数据

### 2. 输入输出
- **输入**：LLM 生成的文本（字符串或 BaseMessage）
- **输出**：结构化数据（JSON、Pydantic 对象、列表、XML 等）

### 3. 核心能力
- **多格式支持**：JSON、Pydantic、列表、XML、字符串等
- **流式解析**：支持逐块解析和累积解析
- **错误修复**：自动检测和修复格式错误

### 4. 技术特性
- **Runnable 协议**：无缝集成到 LCEL 管道
- **类型安全**：泛型设计确保类型正确
- **异步优先**：完整的异步流式支持

---

## 一句话记忆

> OutputParser = LLM 输出 → 结构化数据的桥梁

---

## 类比理解

**前端类比**：
- OutputParser 就像前端的 JSON.parse()，但更智能
- 可以处理不完整的数据（流式）
- 可以自动修复格式错误（容错）

**日常生活类比**：
- 就像翻译官，把 LLM 的"自然语言"翻译成程序能理解的"结构化语言"
- 支持"同声传译"（流式解析）
- 可以"纠正口误"（错误修复）

## 为什么需要 OutputParser？

### 问题场景
```python
# LLM 输出（不可预测的格式）
llm_output = """
Here's the person info:
```json
{"name": "Alice", "age": 30}
```
"""

# 程序需要的格式
expected = {"name": "Alice", "age": 30}
```

### OutputParser 的价值
1. **自动提取**：从 Markdown 代码块中提取 JSON
2. **类型验证**：确保数据符合 Pydantic 模型
3. **错误处理**：自动修复格式错误
4. **流式支持**：逐块解析，减少延迟

---

## 核心使用场景

### 1. 结构化数据提取
```python
# 从 LLM 输出中提取结构化信息
parser = PydanticOutputParser(pydantic_object=Person)
person = parser.parse(llm_output)
```

### 2. 流式解析
```python
# 实时解析 LLM 流式输出
for chunk in chain.stream("Extract person info"):
    print(chunk)  # 逐步输出解析结果
```

### 3. 错误修复
```python
# 自动修复格式错误的输出
fixing_parser = OutputFixingParser.from_llm(parser=parser, llm=llm)
result = fixing_parser.parse(malformed_output)
```

## 与现代方法的对比

### 现代方法（2025+）
```python
# 使用 LLM 原生结构化输出
structured_llm = llm.with_structured_output(Person)
result = structured_llm.invoke("Extract person info")
```

### 传统方法（OutputParser）
```python
# 手动解析 LLM 输出
parser = PydanticOutputParser(pydantic_object=Person)
chain = llm | parser
result = chain.invoke("Extract person info")
```

### 何时使用 OutputParser
- ✅ 模型不支持原生结构化输出
- ✅ 需要自定义解析逻辑
- ✅ 需要额外验证
- ✅ 需要完整流式支持

## 关键数字

- **17种**：LangChain 提供的 OutputParser 类型
- **3层**：OutputParser 的抽象层次（BaseLLMOutputParser → BaseGenerationOutputParser → BaseOutputParser）
- **2种**：流式解析模式（逐块解析 vs 累积解析）
- **100%**：Runnable 协议覆盖率（所有 OutputParser 都是 Runnable）
