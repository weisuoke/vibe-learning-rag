# 一句话总结

**RAG（检索增强生成）是通过"离线索引 + 在线查询"双阶段架构，让 LLM 能够检索外部知识库并基于真实信息生成答案的技术，解决了 LLM 知识过时、知识缺失和幻觉问题，是构建企业级 AI 应用的核心范式。**

---

## 核心要点回顾

| 维度 | 要点 |
|-----|------|
| **本质** | 让 LLM 从"闭卷考试"变成"开卷考试" |
| **架构** | 离线索引 + 在线查询 双阶段 |
| **组件** | Loader → Chunker → Embedder → VectorStore → Retriever → Generator |
| **价值** | 知识可更新、答案可追溯、成本可控 |
| **适用** | 知识密集型问答、需要引用来源的场景 |
