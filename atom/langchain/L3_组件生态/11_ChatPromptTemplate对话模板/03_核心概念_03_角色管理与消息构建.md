# 核心概念 03：角色管理与消息构建

## 一句话定义

**角色管理与消息构建是 ChatPromptTemplate 的核心机制，通过定义不同角色（System/Human/AI/Chat）和消息顺序来设计对话流程和实现 Few-shot 学习。**

---

## 核心概念详解

### 1. 消息角色类型

ChatPromptTemplate 支持四种基本消息角色，每种角色在对话中扮演不同的功能。

#### 1.1 System 角色

**定义**：系统消息，用于定义 AI 的角色、行为规则和任务说明。

**源码定义**（来自 langchain_core/prompts/chat.py:681-688）：

```python
class SystemMessagePromptTemplate(_StringImageMessagePromptTemplate):
    """System message prompt template.

    This is a message that is not sent to the user.
    """

    _msg_class: type[BaseMessage] = SystemMessage
```

**典型用法**：

```python
from langchain_core.prompts import ChatPromptTemplate

# 方式 1：使用元组格式
prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant."),
    ("human", "{input}")
])

# 方式 2：使用模板类
from langchain_core.prompts import SystemMessagePromptTemplate

system_template = SystemMessagePromptTemplate.from_template(
    "You are a {role}. Your task is to {task}."
)

prompt = ChatPromptTemplate.from_messages([
    system_template,
    ("human", "{input}")
])

# 使用
result = prompt.invoke({
    "role": "translator",
    "task": "translate English to French",
    "input": "Hello, world!"
})
```

**最佳实践**：
- 始终放在消息列表的最前面
- 清晰定义 AI 的身份和行为
- 避免过于复杂的指令
- 使用变量实现动态角色配置

#### 1.2 Human 角色

**定义**：用户消息，表示来自用户的输入、问题或请求。

**源码定义**（来自 langchain_core/prompts/chat.py:663-670）：

```python
class HumanMessagePromptTemplate(_StringImageMessagePromptTemplate):
    """Human message prompt template.

    This is a message sent from the user.
    """

    _msg_class: type[BaseMessage] = HumanMessage
```

**典型用法**：

```python
# 方式 1：简单文本
prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant."),
    ("human", "{question}")
])

# 方式 2：包含上下文
prompt = ChatPromptTemplate.from_messages([
    ("system", "Answer based on the context."),
    ("human", "Context: {context}\nQuestion: {question}")
])

# 方式 3：多模态（文本+图片）
prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a vision assistant."),
    ("human", [
        {"type": "text", "text": "What's in this image?"},
        {"type": "image_url", "image_url": "{image_url}"}
    ])
])
```

**最佳实践**：
- 支持变量替换（如 `{question}`）
- 可以包含上下文信息
- 保持简洁明了
- 使用清晰的变量命名

#### 1.3 AI 角色

**定义**：AI 消息，表示 AI 的回复，主要用于 Few-shot 学习和对话历史。

**源码定义**（来自 langchain_core/prompts/chat.py:672-679）：

```python
class AIMessagePromptTemplate(_StringImageMessagePromptTemplate):
    """AI message prompt template.

    This is a message sent from the AI.
    """

    _msg_class: type[BaseMessage] = AIMessage
```

**典型用法**：

```python
# Few-shot 学习示例
prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a translator."),
    ("human", "Hello"),
    ("ai", "Bonjour"),
    ("human", "Goodbye"),
    ("ai", "Au revoir"),
    ("human", "{input}")

# 使用

# AI 会学习前面的翻译模式，输出类似 "Bon matin"
```

**最佳实践**：
- 用于 Few-shot 示例
- 展示期望的回答格式
- 提供对话上下文
- 保持示例的一致性

#### 1.4 Chat 角色（自定义角色）

**定义**：自定义角色消息，可以指定任意角色名称。

**源码定义**（来自 langchain_core/prompts/chat.py:353-386）：

```python
class ChatMessagePromptTemplate(BaseStringMessagePromptTemplate):
    """Chat message prompt template."""

    role: str
    """Role of the message."""

    def format(self, **kwargs: Any) -> BaseMessage:
        text = self.prompt.format(**kwargs)
        return ChatMessage(
            content=text, role=self.role, additional_kwargs=self.additional_kwargs
        )
```

**典型用法**：

```python
from langchain_core.prompts import ChatMessagePromptTemplate

# 创建自定义角色
custom_role = ChatMessagePromptTemplate.from_template(
    template="{content}",
    role="expert"
)

prompt = ChatPromptTemplate.from_messages([
    ("system", "You are facilitating a discussion."),
    custom_role,
    ("human", "{question}")
])

# 使用
result = prompt.invoke({
    "content": "As an expert, I recommend...",
    "question": "What do you think?"
})
```

**最佳实践**：
- 用于特殊场景（如多角色对话）
- 明确角色名称
- 保持角色一致性

### 2. 消息顺序管理

消息的顺序在对话中至关重要，直接影响 AI 的理解和回复质量。

#### 2.1 标准顺序模式

```python
# 模式 1：最简单的问答
prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant."),
    ("human", "{question}")
])

# 模式 2：带上下文的问答
prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant."),
    ("human", "Context: {context}"),
    ("human", "Question: {question}")
])

# 模式 3：带历史的对话
from langchain_core.prompts import MessagesPlaceholder

prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant."),
    MessagesPlaceholder("history"),
    ("human", "{input}")
])
```

#### 2.2 消息顺序的重要性

```python
# 错误示例：System 消息不在最前面

    ("human", "{question}")

# 正确示例：System 消息在最前面

    ("system", "You are a helpful assistant."),  # ✅ 正确位置
```

### 3. 对话流程设计

#### 3.1 单轮对话流程

```python
from langchain_openai import ChatOpenAI

# 创建简单的单轮对话
llm = ChatOpenAI(model="gpt-4")

prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant."),
    ("human", "{question}")
])

chain = prompt | llm

# 使用
response = chain.invoke({"question": "What is Python?"})
print(response.content)
```

#### 3.2 多轮对话流程

```python
from langchain_core.prompts import MessagesPlaceholder
from langchain.memory import ConversationBufferMemory

# 创建多轮对话系统
memory = ConversationBufferMemory(
    memory_key="chat_history",
    return_messages=True
)

prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant."),
    MessagesPlaceholder("chat_history"),
    ("human", "{input}")
])

chain = prompt | llm

# 对话管理器
class ConversationManager:
    def __init__(self, chain, memory):
        self.chain = chain
        self.memory = memory

    def chat(self, user_input: str) -> str:
        # 获取历史
        history = self.memory.load_memory_variables({})["chat_history"]

        # 调用链
        response = self.chain.invoke({
            "chat_history": history,
            "input": user_input
        })

        # 保存到内存
        self.memory.save_context(
            {"input": user_input},
            {"output": response.content}
        )

        return response.content

# 使用
manager = ConversationManager(chain, memory)
print(manager.chat("What is Python?"))
print(manager.chat("Can you give me an example?"))  # 带上下文
```

```python
def create_conditional_prompt(task_type: str):
    """根据任务类型创建不同的对话流程"""
    if task_type == "translation":
        return ChatPromptTemplate.from_messages([

            ("system", "You are a helpful assistant."),
            ("human", "{input}")
        ])

# 使用
```

#### 3.4 复杂对话流程（多阶段）

```python
# 阶段 1：信息收集
collection_prompt = ChatPromptTemplate.from_messages([
    ("system", "You are collecting user requirements."),
    ("human", "{user_input}")
])

# 阶段 2：信息处理
processing_prompt = ChatPromptTemplate.from_messages([
    ("system", "You are processing the collected information."),
    ("human", "Requirements: {requirements}\nTask: {task}")
])

# 阶段 3：结果生成
generation_prompt = ChatPromptTemplate.from_messages([
    ("system", "You are generating the final result."),
    ("human", "Processed data: {processed_data}\nGenerate: {output_format}")
])

# 完整流程
def multi_stage_conversation(user_input: str):
    # 阶段 1
    requirements = (collection_prompt | llm).invoke({"user_input": user_input})

    # 阶段 2
    processed = (processing_prompt | llm).invoke({
        "requirements": requirements.content,
        "task": "analyze and structure"
    })

    # 阶段 3
    result = (generation_prompt | llm).invoke({
        "processed_data": processed.content,
        "output_format": "markdown"
    })

    return result.content
```

### 4. Few-shot 学习示例构建

Few-shot 学习是通过提供示例来引导 AI 学习期望的行为模式。

#### 4.1 基础 Few-shot 模式

```python
# 翻译任务的 Few-shot
translation_prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a translator. Translate English to French."),
    ("human", "Hello"),
    ("ai", "Bonjour"),
    ("human", "Goodbye"),
    ("ai", "Au revoir"),
    ("human", "Thank you"),
    ("ai", "Merci"),
    ("human", "{input}")
])

# 使用
result = translation_prompt.invoke({"input": "Good morning"})
# AI 会学习模式，输出类似 "Bon matin"
```

#### 4.2 格式化输出 Few-shot


#### 4.4 动态 Few-shot 示例

```python
def create_fewshot_prompt(examples: list[tuple[str, str]], task: str):
    """动态创建 Few-shot 提示"""
    messages = [("system", f"You are performing the task: {task}")]

    # 添加示例
    for human_msg, ai_msg in examples:
        messages.append(("human", human_msg))
        messages.append(("ai", ai_msg))

    # 添加用户输入
    messages.append(("human", "{input}"))

    return ChatPromptTemplate.from_messages(messages)

# 使用
examples = [
    ("Classify: This movie is amazing!", "Sentiment: Positive"),
    ("Classify: This movie is terrible.", "Sentiment: Negative"),
    ("Classify: This movie is okay.", "Sentiment: Neutral"),
]

sentiment_prompt = create_fewshot_prompt(
    examples=examples,
    task="sentiment classification"
)

result = sentiment_prompt.invoke({"input": "Classify: I love this movie!"})
```

#### 4.5 Few-shot 与 RAG 结合

```python
from langchain_community.vectorstores import Chroma
from langchain_openai import OpenAIEmbeddings

# 创建示例存储
embeddings = OpenAIEmbeddings()
example_store = Chroma.from_texts(
    texts=[
        "Q: What is Python? A: Python is a programming language.",
        "Q: What is JavaScript? A: JavaScript is a web programming language.",
        "Q: What is SQL? A: SQL is a database query language.",
    ],
    embedding=embeddings
)

# 动态检索相关示例
def create_dynamic_fewshot_prompt(query: str, k: int = 2):
    """根据查询动态检索相关示例"""
    # 检索相关示例
    relevant_examples = example_store.similarity_search(query, k=k)

    messages = [("system", "You are a helpful assistant.")]

    # 添加检索到的示例
    for example in relevant_examples:
        parts = example.page_content.split(" A: ")
        if len(parts) == 2:
            messages.append(("human", parts[0].replace("Q: ", "")))
            messages.append(("ai", parts[1]))

    messages.append(("human", "{question}"))

    return ChatPromptTemplate.from_messages(messages)

# 使用
prompt = create_dynamic_fewshot_prompt("What is Java?")
result = (prompt | llm).invoke({"question": "What is Java?"})
```

### 6. 在 RAG 系统中的应用

#### 6.1 RAG 对话流程设计

```python
from langchain_openai import OpenAIEmbeddings
from langchain_community.vectorstores import Chroma

# 创建 RAG 系统
embeddings = OpenAIEmbeddings()
vectorstore = Chroma.from_texts(
    texts=[
        "LangChain is a framework for LLM applications.",
        "RAG combines retrieval and generation.",
        "Vector stores enable semantic search."
    ],
    embedding=embeddings
)
retriever = vectorstore.as_retriever()

# 设计 RAG 对话流程
rag_prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a helpful assistant. Use the context to answer:\n\n{context}"),
    MessagesPlaceholder("chat_history", optional=True),
    ("human", "{question}")
])

# 创建 RAG 链
from langchain_core.runnables import RunnablePassthrough

rag_chain = (
    {
        "context": lambda x: "\n".join([doc.page_content for doc in retriever.get_relevant_documents(x["question"])]),
        "chat_history": lambda x: x.get("chat_history", []),
        "question": lambda x: x["question"]
    }
    | rag_prompt
    | llm
)

# 使用
response = rag_chain.invoke({
    "question": "What is LangChain?",
    "chat_history": []
})
```

```python
    ("human", "{student_question}")
])
```

## 总结

角色管理与消息构建是 ChatPromptTemplate 的核心能力：

1. **四种角色**：System（定义行为）、Human（用户输入）、AI（示例/历史）、Chat（自定义）
2. **消息顺序**：System 在前，历史在中，当前输入在后
3. **对话流程**：单轮、多轮、条件分支、多阶段
4. **Few-shot 学习**：通过示例引导 AI 行为
5. **RAG 集成**：结合检索和对话管理

**核心价值**：通过精心设计的角色和消息顺序，实现高质量的对话系统和智能应用。
