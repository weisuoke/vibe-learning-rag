# Prompt Engineering 进阶

> 优化 RAG 输出质量、处理复杂推理任务

---

## 为什么要学这个？

基础篇教你"怎么写 Prompt"，进阶篇教你"怎么让 Prompt 更强大"。

在实际 RAG 系统中，你会遇到这些挑战：

- **输出格式不稳定**：LLM 有时返回 JSON，有时返回纯文本
- **复杂问题推理错误**：需要多步推理时，LLM 容易跳步或出错
- **回答风格不一致**：同样的问题，每次回答的格式和详细程度不同
- **多文档综合困难**：检索到多个文档时，LLM 不知道如何整合

进阶技巧就是解决这些问题的武器。

## 核心要点预览

学完本知识点，你将掌握：

1. **Few-shot**：通过示例教 LLM 你想要的输出格式和风格
2. **Chain-of-Thought**：让 LLM 分步推理，提高复杂问题的准确率
3. **结构化输出**：强制 LLM 输出 JSON 等固定格式
4. **RAG 专项优化**：Query 分解、Self-RAG、Prompt Chaining 等高级技巧
