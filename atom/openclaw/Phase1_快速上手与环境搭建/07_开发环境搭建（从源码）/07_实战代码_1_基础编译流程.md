## 一、环境准备

### 1.1 检查系统要求

```bash
#!/bin/bash

echo "=== 检查系统环境 ==="

# 检查 Node.js
if command -v node &> /dev/null; then
    NODE_VERSION=$(node --version | cut -d'v' -f2)
    echo "✅ Node.js: $NODE_VERSION"

    # 检查版本是否符合要求
    REQUIRED="22.12.0"
    if [ "$(printf '%s\n' "$REQUIRED" "$NODE_VERSION" | sort -V | head -n1)" != "$REQUIRED" ]; then
        echo "❌ Node.js 版本必须 >= $REQUIRED"
        exit 1
    fi
else
    echo "❌ Node.js 未安装"
    exit 1
fi

# 检查 pnpm
if command -v pnpm &> /dev/null; then
    PNPM_VERSION=$(pnpm --version)
    echo "✅ pnpm: $PNPM_VERSION"
else
    echo "⚙️  启用 pnpm..."
    corepack enable pnpm
fi

# 检查 Git
if command -v git &> /dev/null; then
    GIT_VERSION=$(git --version | cut -d' ' -f3)
    echo "✅ Git: $GIT_VERSION"
else
    echo "❌ Git 未安装"
    exit 1
fi

echo "=== 环境检查完成 ==="
```

**数据来源**: `sourcecode/openclaw/package.json` - engines

## 四、完整脚本

### 4.1 一键安装脚本

```bash
#!/bin/bash

set -e  # 遇到错误立即退出

echo "=== OpenClaw 开发环境搭建 ==="

# 1. 检查 Node.js
if ! command -v node &> /dev/null; then
    echo "❌ Node.js 未安装"
    exit 1
fi

NODE_VERSION=$(node --version | cut -d'v' -f2)
REQUIRED="22.12.0"

if [ "$(printf '%s\n' "$REQUIRED" "$NODE_VERSION" | sort -V | head -n1)" != "$REQUIRED" ]; then
    echo "❌ Node.js 版本必须 >= $REQUIRED，当前版本: $NODE_VERSION"
    exit 1
fi

echo "✅ Node.js: $NODE_VERSION"

# 2. 启用 pnpm
if ! command -v pnpm &> /dev/null; then
    echo "⚙️  启用 pnpm..."
    corepack enable pnpm
fi

echo "✅ pnpm: $(pnpm --version)"

# 3. 克隆仓库
if [ ! -d "openclaw" ]; then
    echo "⚙️  克隆仓库..."
    git clone https://github.com/openclaw/openclaw.git
fi

cd openclaw

# 4. 安装依赖
echo "⚙️  安装依赖..."
pnpm install

# 5. 构建 UI
echo "⚙️  构建 UI..."
pnpm ui:build

# 6. 编译 TypeScript
echo "⚙️  编译 TypeScript..."
pnpm build

# 7. 验证
echo "⚙️  运行检查..."
pnpm check

echo "✅ OpenClaw 开发环境搭建完成！"
echo "运行 'pnpm gateway:watch' 启动开发模式"
```

**使用方法**:

```bash
# 保存为 setup.sh
chmod +x setup.sh
./setup.sh
```

### 4.2 Makefile 配置

```makefile
# Makefile

.PHONY: install build dev clean check test

# 安装依赖
install:
	pnpm install

# 构建项目
build:
	pnpm ui:build
	pnpm build

# 开发模式
dev:
	pnpm gateway:watch

# 清理构建产物
clean:
	rm -rf dist ui/dist node_modules

# 代码检查
check:
	pnpm check

# 运行测试
test:
	pnpm test

# 完整流程
all: install build check test
	@echo "✅ 所有步骤完成"
```

**使用方法**:

```bash
# 完整流程
make all

# 开发模式
make dev

# 清理
make clean
```

## 五、常见问题排查

### 5.1 pnpm install 失败

**错误**:

```
ERR_PNPM_UNSUPPORTED_ENGINE  Unsupported environment
```

**解决方案**:

```bash
# 检查 Node.js 版本
node --version

# 升级 Node.js
asdf install nodejs 22.12.0
asdf global nodejs 22.12.0

# 重新安装
pnpm install
```

---

### 5.2 pnpm build 失败

**错误**:

```
Error: Cannot find module 'tsdown'
```

**解决方案**:

```bash
# 清理并重新安装
rm -rf node_modules pnpm-lock.yaml
pnpm install
pnpm build
```

---

### 5.3 端口被占用

**错误**:

```
Error: Port 3000 is already in use
```

**解决方案**:

```bash
# 查找占用端口的进程
lsof -i :3000

# 杀死进程
kill -9 <PID>

# 或使用其他端口
PORT=3001 pnpm gateway:watch
```
