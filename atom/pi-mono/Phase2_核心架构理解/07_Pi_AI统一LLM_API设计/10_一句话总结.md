# 一句话总结

**Pi AI 统一 LLM API 通过 Provider Adapter 模式和统一消息格式，让开发者用一套代码调用 25+ LLM 模型，支持流式响应、工具调用、成本追踪，是构建生产级 AI Agent 的极简基础设施。**

---

## 记住这三点

1. **Adapter Pattern**：每个 Provider 一个适配器，统一接口
2. **Wire Protocol**：自动识别 4 种主流协议（OpenAI、Anthropic、Google、Ollama）
3. **Event-based Streaming**：start/delta/end 三段式流式架构
