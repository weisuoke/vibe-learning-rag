# 最小可用：16 分钟上手 Struct 与 Enum

> **一句话定位：** 用 6 个最小代码片段，从零构建一个可运行的 Agent 配置系统——覆盖 struct 定义、enum 变体、match 分发、derive 宏、Option 处理、组合嵌套。

| 步骤 | 时间 | 内容 | 你将学会 |

## Step 1：定义 Struct（2 min）

```rust
struct AgentConfig {
    name: String,
    model: String,
    temperature: f64,
}

fn main() {
    let config = AgentConfig {
        name: "assistant".into(),
        model: "gpt-4".into(),
        temperature: 0.7,
    };
    println!("Agent: {}, model: {}", config.name, config.model);
}
```

**要点**：字段用 `,` 分隔，类型必须标注，创建时所有字段必须赋值。

---

## Step 2：impl 方法（2 min）

```rust
impl AgentConfig {
    // 关联函数（构造器）
    fn new(name: &str, model: &str) -> Self {
        AgentConfig {
            name: name.into(),
            model: model.into(),
            temperature: 0.7,
        }
    }

    // 方法（&self 只读借用）
    fn summary(&self) -> String {
        format!("{} ({})", self.name, self.model)
    }
}

fn main() {
    let config = AgentConfig::new("bot", "gpt-4");
    println!("{}", config.summary());
}
```

**要点**：`Self` = 当前类型，`&self` = 只读借用，`::` 调关联函数，`.` 调方法。

## Step 3：定义 Enum（3 min）

```rust
enum AutonomyLevel {
    ReadOnly,                        // 单元变体
    Supervised,
    Full,
    Custom(String),                  // 元组变体：携带数据
}

enum ConversationMessage {
    Chat { role: String, content: String },  // 结构体变体
    ToolCall { name: String },
    Error(String),                           // 元组变体
}
```

**要点**：enum 变体可以不带数据、带元组数据、带命名字段——远比 TypeScript union type 强大。

---

## Step 4：match 分发（3 min）

```rust
fn describe(level: &AutonomyLevel) -> &str {
    match level {
        AutonomyLevel::ReadOnly => "read only",
        AutonomyLevel::Supervised => "needs approval",
        AutonomyLevel::Full => "autonomous",
        AutonomyLevel::Custom(s) => "custom",  // 绑定内部数据
    }
}

fn process(msg: &ConversationMessage) {
    match msg {
        ConversationMessage::Chat { role, content } => {
            println!("[{}] {}", role, content);
        }
        ConversationMessage::ToolCall { name } => {
            println!("Calling tool: {}", name);
        }
        ConversationMessage::Error(e) => {
            println!("Error: {}", e);
        }
    }
}
```

**要点**：漏掉任何变体 → 编译错误。这就是"穷尽性检查"。

## Step 5：derive + Option（3 min）

```rust
#[derive(Debug, Clone, PartialEq)]
struct AgentConfig {
    name: String,
    model: String,
    temperature: f64,
    description: Option<String>,  // 可选字段
}

fn main() {
    let config = AgentConfig {
        name: "bot".into(),
        model: "gpt-4".into(),
        temperature: 0.7,
        description: None,
    };

    // Debug: {:?} 打印
    println!("{:?}", config);

    // Clone: 深拷贝
    let config2 = config.clone();

    // PartialEq: == 比较
    println!("same? {}", config == config2);

    // Option 处理
    let desc = config.description.as_deref().unwrap_or("(none)");
    println!("desc: {}", desc);
}
```

## Step 6：组合嵌套（3 min）

```rust
#[derive(Debug, Clone, Copy, PartialEq)]
enum AutonomyLevel { ReadOnly, Supervised, Full }

#[derive(Debug, Clone)]
struct SecurityPolicy {
    autonomy_level: AutonomyLevel,
    max_actions: u32,
}

#[derive(Debug)]
struct Agent {
    name: String,
    security: SecurityPolicy,       // struct 嵌入 struct
    tools: Vec<String>,             // 集合
    description: Option<String>,    // 可选
}

fn main() {
    let agent = Agent {
        name: "assistant".into(),
        security: SecurityPolicy {
            autonomy_level: AutonomyLevel::Supervised,
            max_actions: 10,
        },
        tools: vec!["search".into(), "write".into()],
        description: Some("A helpful agent".into()),
    };

    println!("{:#?}", agent);

    // 访问嵌套字段
    if agent.security.autonomy_level == AutonomyLevel::Full {
        println!("Warning: full autonomy!");
    }
}
```

## 完整可运行模板

把上面 6 步合并成一个可直接 `rustc main.rs && ./main` 运行的文件：

```rust
#[derive(Debug, Clone, Copy, PartialEq)]
enum AutonomyLevel { ReadOnly, Supervised, Full }

#[derive(Debug, Clone)]
struct SecurityPolicy {
    autonomy_level: AutonomyLevel,
    max_actions: u32,
}

#[derive(Debug, Clone, PartialEq)]
struct AgentConfig {
    name: String,
    model: String,
    temperature: f64,
    description: Option<String>,
}

impl AgentConfig {
    fn new(name: &str, model: &str) -> Self {
        AgentConfig {
            name: name.into(), model: model.into(),
            temperature: 0.7, description: None,
        }
    }
    fn summary(&self) -> String {
        format!("{} ({})", self.name, self.model)
    }
}

#[derive(Debug)]
enum ConversationMessage {
    Chat { role: String, content: String },
    ToolCall { name: String },
    Error(String),
}

impl ConversationMessage {
    fn label(&self) -> &str {
        match self {
            ConversationMessage::Chat { .. } => "chat",
            ConversationMessage::ToolCall { .. } => "tool",
            ConversationMessage::Error(_) => "error",
        }
    }
}

fn main() {
    // Struct + impl
    let config = AgentConfig::new("bot", "gpt-4");
    println!("{}", config.summary());
    println!("{:?}", config);

    // Enum + match
    let msgs = vec![
        ConversationMessage::Chat { role: "user".into(), content: "Hello".into() },
        ConversationMessage::ToolCall { name: "search".into() },
        ConversationMessage::Error("timeout".into()),
    ];
    for msg in &msgs {
        println!("[{}] {:?}", msg.label(), msg);
    }

    // 组合嵌套
    let policy = SecurityPolicy {
        autonomy_level: AutonomyLevel::Supervised,
        max_actions: 10,
    };
    println!("level: {:?}, max: {}", policy.autonomy_level, policy.max_actions);

    // Option
    let desc = config.description.as_deref().unwrap_or("(none)");
    println!("description: {}", desc);
}
```
