# 文档加载与解析

> 掌握多格式文档的加载、解析与预处理

---

## 知识点概述

**文档加载与解析**是 RAG 系统离线索引阶段的第一步，负责将 PDF、Word、HTML 等多种格式的文档转换为统一的纯文本格式，为后续的文本分块和向量化做准备。

### 核心价值

| 价值 | 说明 |
|-----|------|
| 格式统一化 | 多种格式 → 统一 Document 对象 |
| 信息提取 | 从复杂格式中提取纯文本 |
| 元数据保留 | 保留来源信息用于答案溯源 |

### 在 RAG 系统中的位置

```

│  │ PDF │       │chunk│       │ vec │         │     │       │
```

```python
from langchain_community.document_loaders import PyPDFLoader

loader = PyPDFLoader("document.pdf")
docs = loader.load()

    print(doc.metadata["source"]) # 来源文件
```
